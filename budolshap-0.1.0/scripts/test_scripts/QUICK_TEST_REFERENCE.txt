┌──────────────────────────────────────────────────────────────────────────────┐
│ 🔧 COMMON ISSUES & QUICK FIXES                                               │
└──────────────────────────────────────────────────────────────────────────────┘

❌ Order not in Lalamove Portal
   → Check: Browser console for API errors
   → Check: Lalamove credentials in .env.local
   → Check: Wallet balance in Partner Portal
   → Check: Address within Metro Manila

❌ Payment not updating order
   → Check: Webhook configured in PayMongo dashboard
   → Check: Webhook endpoint accessible (use ngrok for local)
   → Check: Webhook secret in .env.local

❌ Driver not assigned
   → Sandbox: May be simulated/delayed
   → Production: Wait 5-15 minutes
   → Check: Service hours and area coverage

❌ Quote expired
   → Solution: Request new quote
   → Tip: Complete checkout faster

┌──────────────────────────────────────────────────────────────────────────────┐
│ 📊 EXPECTED API SEQUENCE                                                     │
└──────────────────────────────────────────────────────────────────────────────┘

1. POST /api/orders
   → Order created (status: pending)

2. POST /api/payment/gcash/create
   → GCash checkout URL returned

3. [User completes payment in GCash]

4. POST /api/webhooks/paymongo (webhook)
   → Order updated (isPaid: true)

5. POST /api/shipping/lalamove/book
   → Lalamove order created

6. POST /api/webhooks/lalamove (webhook)
   → Order status updated based on delivery
