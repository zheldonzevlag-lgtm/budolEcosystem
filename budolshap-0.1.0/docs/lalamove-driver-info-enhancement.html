Sync Mechanism

⚠️ Important: Dual Data Source

Driver information is obtained from TWO sources:

  1. Webhooks (Primary): Real-time updates when events occur
  2. API Polling (Fallback): Manual sync every 10 seconds for active orders

This ensures data is always up-to-date even if webhooks fail.
