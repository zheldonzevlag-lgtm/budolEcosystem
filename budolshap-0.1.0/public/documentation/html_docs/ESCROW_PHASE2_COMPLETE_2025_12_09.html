🚀 Key Features Implemented

1. creditPendingBalance()

Credits seller's pending balance when payment is received 

await creditPendingBalance({
  orderId: 'order_123',
  amount: 1000,
  gateway: 'PAYMONGO'
})

// Calculates:
// - Platform fee: ₱50 (5%)
// - Net earnings: ₱950
// - Credits to pendingBalance

2. releaseFunds()

Releases funds from pending to available balance

await releaseFunds({
  orderId: 'order_123',
  amount: 950
})

// Transfers:
// - From: pendingBalance
// - To: balance (available for withdrawal)

3. Gateway Agnostic Design

Works with any payment gateway

// PayMongo
await creditPendingBalance({ ..., gateway: 'PAYMONGO' })

// Xendit (future)
await creditPendingBalance({ ..., gateway: 'XENDIT' })

// Stripe (future)
await creditPendingBalance({ ..., gateway: 'STRIPE' })

💰 New Payment Flow

1
Customer Pays

Payment processed via PayMongo (GCash)

2
Webhook Triggered

PayMongo sends payment.paid event

3
Escrow Service Called

creditPendingBalance() calculates fees and credits seller

4
Funds Held in Escrow

Money in pendingBalance (not yet available)

7d
Order Completed

After 7 days or manual completion

5
Funds Released

releaseFunds() moves to available balance
