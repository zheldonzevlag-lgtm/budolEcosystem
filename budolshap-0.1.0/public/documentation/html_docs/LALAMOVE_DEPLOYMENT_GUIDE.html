🔄 Redeployment Process

Whenever you make changes to the code or environment variables, you must redeploy the application to Vercel.

1

Push Changes to GitHub

Ensure your latest code is committed and pushed to the main branch.

git add .
git commit -m "Update Lalamove integration"
git push origin main
2

Update Environment Variables (If Needed)

If you've added new features requiring keys, add them via Vercel CLI.

vercel env add VARIABLE_NAME production
3

Trigger Deployment

Deploy the application to production.

vercel --prod

Follow the prompts if asked, but usually, it will auto-detect settings.

🔧 Common Issues & Fixes

Below are issues encountered during the development and deployment of the Lalamove integration and their solutions.

🔴 Issue: 500 Internal Server Error on Quote API

Symptom: Calling POST /api/shipping/lalamove/quote returns a 500 error, and logs show "shippingFactory is not a function" or module resolution errors.

Cause: The shippingFactory.js service was using CommonJS (require/module.exports) while the Next.js API route was using ES Modules (import/export). This caused import failures.

✅ Fix: Refactored services/shippingFactory.js to use ES Module syntax (export const, export default) to match the rest of the Next.js application structure.
🔴 Issue: "Return URL format is invalid" (PayMongo)

Symptom: PayMongo checkout fails with an error about the return URL.

Cause: Vercel preview URLs or local hostnames were being sent to PayMongo, which requires a valid, publicly accessible HTTPS URL.

✅ Fix: Ensured NEXT_PUBLIC_SITE_URL is set to the production URL (https://budolshap.vercel.app) and is correctly prepended to the return path in the payment intent creation.
🔴 Issue: Prisma Response Size Limit (P6009)

Symptom: Fetching orders fails with PrismaClientKnownRequestError: Response size limit exceeded.

Cause: The /api/orders endpoint was fetching all orders with full product details and base64 images, exceeding Vercel's Serverless Function payload limits (approx 4-5MB).

✅ Fix: Implemented pagination in the API and optimized the Prisma query to select only necessary fields, significantly reducing payload size.
🟡 Issue: Lalamove Logo Not Showing

Symptom: The Lalamove logo appears as a broken image or generic placeholder.

Cause: Incorrect asset path or missing image file in the public directory.

✅ Fix: Added lalamove_logo.png to the assets folder and ensured the LalamoveWidget component references the correct path.
