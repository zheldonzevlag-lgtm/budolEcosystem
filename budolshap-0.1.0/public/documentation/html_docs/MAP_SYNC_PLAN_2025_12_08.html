Plan: Sync Map Status with Order Status

Problem

The app shows the order as DELIVERED (based on webhook), but the embedded Lalamove map (iframe) still shows the driver at the "Drop-off Point". This visual discrepancy confuses users.

Proposed Solution

We will override the iframe display when the local order status is DELIVERED. Instead of showing the live map (which might be lagging or showing 'Drop-off'), we will display a clear "Package Delivered" success state.

Implementation Logic

In components/LalamoveTracking.jsx:

if (order.status === 'DELIVERED') { return ( <div className="delivered-banner"> <h3>✅ Package Delivered</h3> <p>Your driver has completed the delivery.</p> </div> ); } // Otherwise, show the live map iframe
