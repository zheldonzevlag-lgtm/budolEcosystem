📋 Checkout Flow Integration

User Journey

  1. Cart Page: User selects "Same Day Delivery via Lalamove"
  2. Quote Fetching: LalamoveWidget automatically fetches quote based on:
    • Delivery address (from selected address)
    • Store location (pickup point)
    • Package details (weight, dimensions from cart items)
  3. Quote Display: Shows price breakdown, ETA, and service type
  4. Order Placement: Quote data (including stops with stopIds) is passed to order creation
  5. Booking: After payment confirmation, delivery is booked with Lalamove
  6. Tracking: Real-time tracking available in order details

⚙️ Configuration Requirements

  • LALAMOVE_API_URL - Base API URL (sandbox or production)
