🔐 Authentication Implementation

HMAC SHA256 Signature Process

Lalamove requires each API request to be signed with an HMAC SHA256 signature to ensure authenticity and prevent tampering.

// Generate signature const timestamp = new Date().getTime().toString(); const rawSignature = `${timestamp}\r\n${method}\r\n${path}\r\n\r\n${body}`; const signature = crypto .createHmac('sha256', apiSecret) .update(rawSignature) .digest('hex'); // Add to request headers headers: { 'Authorization': `hmac ${apiKey}:${timestamp}:${signature}`, 'Content-Type': 'application/json', 'Market': 'PH' }
