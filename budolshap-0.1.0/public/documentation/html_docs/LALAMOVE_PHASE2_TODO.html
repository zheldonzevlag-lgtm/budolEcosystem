📋 API Endpoints
5 Endpoints
  • 1. Quote Endpoint Critical
    Create API endpoint to get shipping quotes from Lalamove based on pickup/delivery locations and package details.
    File:
    app/api/shipping/lalamove/quote/route.js
    Method:
    POST
    Input:
    pickup address, delivery address, package details
    Output:
    price, ETA, distance, quoteId
    💡 Implementation Notes:
    • Use the getQuote() method from services/lalamove.js
    • Validate input addresses and package dimensions
    • Return formatted quote with pricing breakdown
    • Handle "no drivers available" scenarios
  • 2. Booking Endpoint Critical
    Create API endpoint to book a Lalamove delivery and store the booking details in the database.
    File:
    app/api/shipping/lalamove/book/route.js
    Method:
    POST
    Input:
    orderId, quoteId, pickup/delivery details, contact info
    Output:
    bookingId, status, trackingUrl
    💡 Implementation Notes:
    • Use the createOrder() method from services/lalamove.js
    • Update Order.shipping field in database with booking details
    • Generate unique client_reference_id for idempotency
    • Return tracking URL for customer
  • 3. Tracking Endpoint High
    Create API endpoint to retrieve real-time delivery status and driver information.
    File:
    app/api/shipping/lalamove/track/[orderId]/route.js
    Method:
    GET
    Input:
    orderId (from URL params)
    Output:
    status, driver info, location, ETA
    💡 Implementation Notes:
    • Use the trackOrder() method from services/lalamove.js
    • Fetch bookingId from Order.shipping field
    • Return formatted status timeline
    • Include driver name, phone, and vehicle details if assigned
  • 4. Cancel Endpoint High
    Create API endpoint to cancel a pending Lalamove delivery order.
    File:
    app/api/shipping/lalamove/cancel/[orderId]/route.js
    Method:
    POST
    Input:
    orderId (from URL params)
    Output:
    success status, cancellation confirmation
    💡 Implementation Notes:
    • Use the cancelOrder() method from services/lalamove.js
    • Check if order is within cancellation window
    • Update Order.shipping.status to "cancelled"
    • Handle "cannot cancel" errors gracefully
  • 5. Webhook Receiver Critical
    Create webhook endpoint to receive real-time delivery status updates from Lalamove.
    File:
    app/api/webhooks/lalamove/route.js
    Method:
    POST
    Input:
    webhook payload with signature header
    Events:
    ASSIGNING, PICKED_UP, IN_TRANSIT, COMPLETED, CANCELLED
    💡 Implementation Notes:
    • Use the verifyWebhookSignature() method from services/lalamove.js
    • Verify HMAC signature before processing
    • Update Order.shipping field with new status and driver info
    • Return 200 OK immediately to acknowledge receipt
    • Log all webhook events for debugging
