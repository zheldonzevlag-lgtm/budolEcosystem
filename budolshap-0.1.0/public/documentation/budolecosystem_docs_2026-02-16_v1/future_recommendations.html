Future Recommendations

Technical roadmap and enhancement proposals.

← Back to Index

1. Migrate to PostgreSQL

Infrastructure High Priority

SQLite is suitable for development but not for concurrent production traffic. Migrating to PostgreSQL will resolve database locking issues and improve scalability.

2. Implement Redis for Session Management

Performance Medium Priority

Instead of querying the database for session validation on every request, use Redis for faster session lookups and caching.

3. Enhanced Cart Validation

Feature Medium Priority

Implement stock checking during cart sync to alert users if items in their cart have gone out of stock since they were added.

4. Automated CI/CD Pipeline

DevOps Low Priority

Set up GitHub Actions to run tests automatically on every pull request to prevent regression bugs like the cart persistence issue.

5. Content Delivery Network (CDN) for Images

Performance Medium Priority

Offload image serving to a CDN (e.g., Cloudinary, AWS S3 + CloudFront) to reduce server load and improve load times.
