Known Issues

1. Post-Logout Empty Cart

Status: RESOLVED

Description: Cart data was being wiped on the server side during logout due to a race condition where the client cleared local state and then synced the empty state to the server.

Impact: High - Users lost their shopping cart contents upon logging out.

2. Image Rendering Failures

Status: RESOLVED

Description: Console errors "Empty string passed to src attribute" when product images were missing or empty strings.

Impact: Medium - UI glitches and console noise.

3. Prisma Client Lock (SQLite)

Status: MONITORING

Description: Occasional "database is locked" errors when multiple operations (test scripts + running server) access the SQLite database simultaneously.

Workaround: Ensure server is restarted after schema changes and avoid running heavy test scripts concurrently with user traffic in dev mode.
