Technical Achievements

Security & Trust

The implementation of atomic escrow operations ensures that funds are always accounted for. By locking funds during a return request, we eliminate the risk of a seller withdrawing funds while a refund is pending. The integration of Order Completion triggers ensures funds are only released from pending to available balance after the buyer confirms receipt or the guarantee period expires.

Fair Logistics & Mediation

The RSF (Return Shipping Fee) logic balances fairness between buyers and sellers. By identifying the party at fault, the system automatically adjusts refund amounts. Admin Mediation logic allows for professional resolution of disputes with manual overrides for fund release or refund.

Fraud Prevention & Automation

COD failure tracking provides the data necessary to identify and restrict high-risk buyers. Auto-completion ensures that the marketplace remains liquid by automatically finalizing orders that have passed their Shopee Guarantee period without buyer action.

Marketplace Performance (Phase 5)

The Performance Service now tracks Non-Fulfilment Rate (NFR) for stores and Buyer Return Rate (BRR) for users. By analyzing a rolling 30-day window, the system automatically calculates penalty points for stores exceeding a 10% NFR threshold, providing a scalable way to maintain marketplace quality.
