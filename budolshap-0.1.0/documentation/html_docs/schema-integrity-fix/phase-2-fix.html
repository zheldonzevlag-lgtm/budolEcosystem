Schema Updates Applied

The following changes were applied to prisma/schema.prisma to resolve the identified discrepancies.

  ...
  adminId          String?
  adminNotes       String?
  ...
}

2. ReturnStatus Enum Synchronization

Added TO_PICKUP status to align with Lalamove webhook integrations and universal status mapping.

enum ReturnStatus {
  ...
  DISPUTED
  TO_PICKUP
}

Next Steps: Database Migration

To apply these changes to your database and regenerate the Prisma client, run the following commands in your terminal:

npx prisma generate
npx prisma db push

Note: Using db push is recommended for development environments to quickly sync the schema. For production, use prisma migrate dev.
