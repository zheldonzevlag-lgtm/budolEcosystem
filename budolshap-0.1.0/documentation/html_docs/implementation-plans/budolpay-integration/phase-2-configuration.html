Phase 2: Configuration Completed

Overview

Phase 2 involved configuring the application to connect with the external budolPay services using secure environment variables.

Environment Variables

The following variables were added to .env and .env.example:

BUDOLPAY_API_KEY="bs_key_2025"
BUDOLPAY_GATEWAY_URL="http://localhost:8004"

Key Descriptions

  • BUDOLPAY_API_KEY: The unique identifier for budolShap within the budolPay ecosystem.
  • BUDOLPAY_GATEWAY_URL: The base URL for the centralized API Gateway (Port 8004).

Internal Service Mapping

The BudolPayAdapter was designed to automatically use these variables for all outgoing requests, ensuring that switching between local development and production environments only requires changing the .env file.

Security

The API key is passed via the x-api-key header in all requests to the Gateway, as per the security requirements of the budolPay ecosystem.
