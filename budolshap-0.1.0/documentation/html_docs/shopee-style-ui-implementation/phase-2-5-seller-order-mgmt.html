Phase 2.5: Seller Order Management UI

Objective

Enhance the seller's order management interface to align with the Shopee-style milestones, providing clear status badges and streamlined actions for shipping and fulfillment.

Technical Changes

SellerDashboard & Order Management Components

  • Status Badges: Updated seller-side order status badges to use "To Ship", "To Receive", and "Completed" terminology for consistency with the buyer's view.
  • Action Buttons: Streamlined fulfillment actions. For "To Ship" orders, the "Arrange Shipment" button is prioritized.
  • Lalamove Integration: For sellers using Lalamove, the "Track Driver" button is prominently displayed once a driver is assigned.
  • Status Filtering: Updated the seller dashboard filters to match the new Shopee-style milestones (All, To Ship, To Receive, Completed, Cancelled, Return/Refund).

Code Snippets

Updated status mapping for Seller views:

const getStatusLabel = (status) => {
    switch (status) {
        case 'ORDER_PLACED':
        case 'PAID':
        case 'PROCESSING':
            return 'TO SHIP';
        case 'SHIPPED':
        case 'IN_TRANSIT':
            return 'TO RECEIVE';
        case 'DELIVERED':
        case 'COMPLETED':
            return 'COMPLETED';
        default:
            return status;
    }
};
