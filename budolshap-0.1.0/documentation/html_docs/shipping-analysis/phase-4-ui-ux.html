1. Shopee-Style Shipping Display

Transparency is key to reducing disputes over shipping fee variances.

Key UI Elements

  • Estimated Label: Display "Estimated Shipping" instead of just "Shipping Fee".
  • Info Tooltip: ⓘ "Final fee may vary based on actual traffic, tolls, and waiting time."
  • Dynamic Updates: Real-time price updates if the user changes address pins or vehicle type.
  • Cost Breakdown: Show subsidies clearly (e.g., ₱150 ₱80).

2. Buffer Logic Implementation

To prevent negative margins, we can implement a "Price Buffer" in the UI.

// Example Buffer Logic
const displayShipping = (quotedAmount) => {
    const buffer = 1.05; // 5% safety margin
    return Math.ceil(quotedAmount * buffer);
}

3. Error & Edge Case Handling

Scenario UI/UX Response
Lalamove API Down Fall back to "Flat Rate Shipping" with a "Service Unavailable" notice.
Invalid Address Pin Highlight the map and prompt user to "Refine Pickup/Drop-off Location".
Extreme Surge Display a warning: "High demand in your area. Shipping rates are currently higher than usual."

4. Post-Order Analysis UI (Admin)

For internal monitoring, an admin view should compare estimated vs. actual costs.

  • Variance Alert: Flag orders where actual > quoted by more than 20%.
  • Heatmap: Identify regions with frequent shipping fee fluctuations.

Phase 4 Success Criteria

  • "Estimated" labels and tooltips integrated into Checkout.
  • Address Map Picker accurately captures coordinates.
  • Admin dashboard shows shipping variance analytics.
