Executive Summary

This implementation plan outlines the technical strategy for integrating real-world shipping calculations via Lalamove and implementing a flexible cost-allocation model inspired by Shopee's platform dynamics. The goal is to provide accurate shipping estimates while managing the variance between quoted and final fees.

Project Objectives

  • Integrate Lalamove API for real-time, road-distance based shipping quotations.
  • Implement a dynamic pricing engine that accounts for base fares, per-km rates, and surge factors.
  • Develop a Shopee-style cost allocation system (Buyer-pays, Seller-absorbs, Platform-subsidized).
  • Design a UI/UX that communicates "Estimated Fees" clearly to reduce customer friction.
  • Establish a robust logging system for analytical comparison between quoted and actual shipping costs.
