Phase 2: Branding Implementation

Executing the transition from orange to green branding across the address component suite.

1. Global Class Replacements

Systematic replacement of Tailwind utility classes was performed across all target files:

Legacy (Orange)

  • text-orange-500
  • bg-orange-500
  • border-orange-500
  • focus:border-orange-500
  • hover:bg-orange-50

New (Green)

  • text-green-500
  • bg-green-600
  • border-green-500
  • focus:border-green-500
  • hover:bg-green-50

2. Map Integration (MapPicker.jsx)

The custom Leaflet marker was updated from orange to green by modifying the inline SVG and the divIcon definition.

// Before
const OrangeIcon = L.divIcon({
  html: '<div style="color: #f97316;">...</div>',
  ...
});

// After
const GreenIcon = L.divIcon({
  html: '<div style="color: #16a34a;">...</div>',
  ...
});

3. Component Specific Updates

AddressAutocomplete.jsx

  • Updated search icon color to green.
  • Updated dropdown suggestion hover state to bg-green-50.
  • Updated loading spinner to text-green-500.

AddressFormManager.jsx

  • Updated "Home/Work/Other" label buttons to use green background when active.
  • Updated form field icons to text-green-500.
  • Updated "Set as Default" toggle switch to peer-checked:bg-green-500.
  • Implemented phone number auto-formatting to nnn-nnn-nnnn format.
  • Added validation to enforce exactly 10 digits for phone numbers.
  • Added auto-prefilling of "Full Name" and "Phone Number" from the seller's user profile.
  • Fixed "Address not found" bug in StoreAddressesManager.jsx by correctly distinguishing between new and existing addresses using ID presence.
  • Standardized "Default" badge and "Set as Default" button colors to branded green.

AddressModal.jsx

  • Updated loading spinner border color to border-t-green-500.

MapPicker.jsx

  • Updated custom marker to branded green icon.
  • Implemented custom zoom controls (Plus/Minus buttons) to replace default Leaflet controls.
  • Styled controls with white background and green hover states to match branding.
