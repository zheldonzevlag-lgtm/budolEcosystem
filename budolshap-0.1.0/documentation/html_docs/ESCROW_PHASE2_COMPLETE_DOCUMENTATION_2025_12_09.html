3. Dashboard Elements Explained

💭 Potential Earnings

What: Total value of ALL orders

Includes: Paid + Unpaid orders

Purpose: Shows maximum possible revenue

Formula: Sum of all order totals (except refunded)

🔒 In Escrow

What: Funds held for 7 days

Includes: Only PAID orders

Purpose: Buyer protection period

Formula: (Order Total - 5% Fee) in pendingBalance

💰 Available Balance

What: Funds ready for withdrawal

Includes: Released from escrow

Purpose: Withdrawable earnings

Formula: Funds in wallet.balance
