📋 API Endpoints Created

  • POST /api/shipping/lalamove/quote
    Get shipping quotes from Lalamove based on pickup/delivery locations and package details.
    File:
    app/api/shipping/lalamove/quote/route.js
    Auth:
    Required (Session)
    Input:
    pickup, delivery, package details, serviceType
    Output:
    quoteId, price breakdown, distance, ETA
  • POST /api/shipping/lalamove/book
    Book a Lalamove delivery and store booking details in the database with idempotency support.
    File:
    app/api/shipping/lalamove/book/route.js
    Auth:
    Required (Session)
    Input:
    orderId, quoteId, pickup/delivery details, contact info
    Output:
    bookingId, status, trackingUrl, shareLink
    Features:
    UUID-based client reference ID, database update
  • GET /api/shipping/lalamove/track/[orderId]
    Retrieve real-time delivery status, driver information, and location updates.
    File:
    app/api/shipping/lalamove/track/[orderId]/route.js
    Auth:
    Required (Owner or Admin)
    Input:
    orderId (URL parameter)
    Output:
    status, driver info, location, timeline, ETA
  • POST /api/shipping/lalamove/cancel/[orderId]
    Cancel a pending Lalamove delivery order with proper validation and error handling.
    File:
    app/api/shipping/lalamove/cancel/[orderId]/route.js
    Auth:
    Required (Owner or Admin)
    Input:
    orderId (URL parameter)
    Output:
    cancellation confirmation, updated status
    Validation:
    Checks cancellation window and order status
  • POST /api/webhooks/lalamove
    Receive real-time delivery status updates from Lalamove with HMAC signature verification.
    File:
    app/api/webhooks/lalamove/route.js
    Auth:
    HMAC Signature Verification
    Events:
    ASSIGNING_DRIVER, ON_GOING, PICKED_UP, IN_TRANSIT, COMPLETED, CANCELLED
    Features:
    Timeline tracking, driver updates, location updates

🧪 Testing Guide

📝 How to Test the Endpoints
1. Start the development server:
npm run dev
2. Test Quote Endpoint:

3. Test Webhook (Health Check):
GET http://localhost:3000/api/webhooks/lalamove
4. Check Database:
npm run prisma:studio
⚠️ Important Notes
• All endpoints require authentication (except webhook)
• Use valid coordinates for Manila/Philippines area
• Webhook signature verification requires LALAMOVE_WEBHOOK_SECRET
• Test with sandbox environment first (LALAMOVE_ENV=sandbox)
• Monitor console logs for debugging information
