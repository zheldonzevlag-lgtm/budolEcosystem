Phase 5: Seller UI Standardization

Overview

The Seller Order Management interface was updated to align perfectly with the Buyer's view and the Shopee-style 5-stage terminology.

Key Improvements

  • Integrated Tracker: The OrderProgressTracker component was added to the Seller's Order Detail Modal, providing sellers with the same visual context as the buyer.
  • Terminology Alignment: Standardized labels across filters, action columns, and dashboard statistics.

Label Mapping Changes

Old Label New Standardized Label Description
To Ship (Pending) TO SHIP Order is paid and needs courier booking.
To Ship (Booked) PROCESSING Courier is booked, awaiting pickup.
Shipped / In Transit TO RECEIVE Package is with the courier.
Delivered COMPLETED Order finalized and escrow released.

Visual Consistency

Dashboard cards were updated to use the new "Processing" terminology for booked orders, reducing confusion between "needs booking" and "waiting for pickup" states.
