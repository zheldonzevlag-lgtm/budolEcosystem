📜 Privacy Policy Content

1. Collection of Personal Data

The modal discloses collection of:

  • Name
  • Email address
  • Mailing address
  • Phone number
  • Payment details

2. Use of Collected Information

  • To process payments: Information used only for order fulfillment
  • To send periodic emails: Order updates, marketing purposes, and customer support

3. Data Protection Measures

Commitment to appropriate security measures to protect against unauthorized access, alteration, disclosure, or destruction of personal information.

4. User Rights under RA 10173

Right to be Informed

Right to Access

Right to Object

Right to Erasure or Blocking

Right to Damages

Right to Data Portability

💻 Technical Implementation

State Management

// Footer.jsx const [isPrivacyOpen, setIsPrivacyOpen] = useState(false); // Trigger in linkSections array { text: "Privacy Policy", action: () => setIsPrivacyOpen(true), icon: null } // Render modal <PrivacyPolicyModal isOpen={isPrivacyOpen} onClose={() => setIsPrivacyOpen(false)} />

Styling & Animations

  • Backdrop: bg-black/50 backdrop-blur-sm
  • Animations: animate-fadeIn, animate-scaleIn
  • Max Height: max-h-[85vh] with scrollable content
  • Color Scheme: Green (#10b981) for brand consistency

Icons Used (Lucide React)

  • Shield - Header icon
  • FileText - Data Collection section
  • UserCheck - Use of Information section
  • Lock - Data Protection section
  • Eye - User Rights section
  • X - Close button

🎯 User Experience

Interaction Flow

  1. User clicks "Privacy Policy" link in footer
  2. Modal fades in with backdrop blur
  3. Content is scrollable with hidden scrollbar for clean UI
  4. User can close via:
    • Close button (X) in header
    • "I Understand" button in footer
    • Clicking backdrop (optional)

Responsive Design

  • Desktop: 2-column grid for user rights, full-width content
  • Mobile: Single column layout, optimized padding
  • Max Width: 3xl (768px) for optimal readability
