1. Absorption Logic Engine

The system will calculate the final shipping cost based on three potential contributors:

  • Platform (Shopee-like): Via Free Shipping Vouchers.
  • Seller: Via "Free Shipping over ₱X" store settings.
  • Buyer: Pays the remaining balance.
// backend/lib/shippingEngine.js
function calculateShippingDistribution(quote, subsidies) {
    let remaining = quote.total;
    const distribution = { platform: 0, seller: 0, buyer: 0 };

    // 1. Apply Platform Voucher
    if (subsidies.voucher) {
        distribution.platform = Math.min(remaining, subsidies.voucher.maxCap);
        remaining -= distribution.platform;
    }

    // 2. Apply Seller Subsidy
    if (subsidies.sellerDiscount && remaining > 0) {
        distribution.seller = Math.min(remaining, subsidies.sellerDiscount);
        remaining -= distribution.seller;
    }

    // 3. Buyer Pays the rest
    distribution.buyer = Math.max(0, remaining);
    
    return distribution;
}

2. Lalamove Integration Adjustments

Since Lalamove is real-time, the quote must be captured and "locked" during checkout to prevent price shifts from affecting the voucher calculations.

  • Locking: Store the `quotedAmount` in the `Order` model during placement.
  • Buffer: Add a configurable platform buffer (e.g., 5%) to the quote to cover potential driver wait fees or minor route changes.

Success Criteria

  • Checkout page correctly displays "Shipping Discount" line items.
  • Order confirmation shows the breakdown of who paid for shipping.
  • Lalamove booking matches the quoted route but allows for platform-absorbed surcharges.
