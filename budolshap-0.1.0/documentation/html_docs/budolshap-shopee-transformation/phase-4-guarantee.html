The "budolShap Guarantee" Mechanism

Similar to "Shopee Guarantee," this phase focuses on building buyer trust by making the escrow process visible and interactive.

1. Protection Countdown Timer

On the Order Details page, buyers will see a countdown: "budolShap Guarantee ends in 2 days, 4 hours."

  • Trigger: Starts once the Lalamove driver marks the order as `DELIVERED`.
  • Action: Buyer can click "Order Received" to release funds early, or "Extend Guarantee" (+3 days) if the item hasn't arrived.

2. Integrated Return/Refund Flow

If a buyer is unhappy, they must "Raise a Dispute" before the guarantee expires.

Dispute Workflow:

  1. Buyer clicks "Return/Refund" and uploads evidence.
  2. Funds are frozen in Escrow (cannot be moved to Seller's balance).
  3. Seller has 48 hours to "Accept" or "Counter" the request.
  4. If no agreement, the case is escalated to BudolShap Admin for mediation.

3. Dispute Mediation Dashboard (Admin)

A new view for Admins to review evidence from both parties and choose to:

  • Refund Buyer: Return funds to Buyer's original payment source.
  • Release to Seller: Reject the dispute and credit the Seller's wallet.
  • Partial Refund: Split the funds between both parties.

Success Criteria

  • "budolShap Guarantee" badge visible on all checkout and order pages.
  • Escrow funds are successfully frozen when a return is initiated.
  • Dispute resolution logs are maintained for all mediated orders.
