Phase 6: Refund & Return UI Integration

Escrow Protection Workflow

The Shopee-style guarantee system follows a strict state machine to ensure funds are protected during disputes.

1. Order Delivered

Funds remain in Escrow. autoCompleteAt timer starts (default 7 days).

2. Guarantee Extension

Buyer can extend the window by 3 days. isGuaranteeExtended becomes true.

3. Return Requested

Funds are locked. Status changes to RETURN_REQUESTED. Timer pauses.

4. Dispute Resolution

Seller accepts (Refund) or Rejects (Admin Dispute). RSF is calculated based on reason.

RSF (Return Shipping Fee) Logic

Return Shipping Fees are automatically assigned based on the reason for return:

Reason Payer Logic
Wrong Product / Damaged Seller Full refund to buyer. Seller loses shipping fee.
Change of Mind Buyer RSF deducted from buyer refund.
Faulty Product Seller Full refund to buyer.
