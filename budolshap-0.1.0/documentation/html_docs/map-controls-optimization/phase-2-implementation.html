Implementation Details

Feature File Changes
Center Pin Size MapPicker.jsx Changed pin classes to h-8 w-8 md:h-10 md:w-10. Adjusted inner circle positioning to top-[9px] left-[9px] for mobile.
Address Info Sync AddressFormManager.jsx Added mapSearchValue state. Synchronized map geocoding results directly back to the AddressAutocomplete input.
UI De-clutter MapPicker.jsx Commented out the floating address/error overlay. Replaced with minimalist status indicators for loading/errors to maximize map visibility.

Design Decisions

  • Border Radius: Adjusted from rounded-2xl to rounded-xl on mobile for a tighter, more cohesive look on smaller buttons.
  • Shadows: Maintained shadow-xl to ensure controls remain distinct from the map background.
  • Z-Index: Ensured all controls remain at z-[2000] to stay above Leaflet's default layers.
