Phase 2: MapPicker Component

The core of the new experience is the MapPicker component. This component will handle map rendering, pin placement, and reverse geocoding.

Key Features

  • Draggable Marker: Users can drag the pin to their exact location for high precision.
  • Auto-Locate: Button to center the map on the user's current GPS coordinates.
  • Search Autocomplete: Integrate with Geoapify or Google Places to search for addresses.
  • Reverse Geocoding: Moving the pin updates the address details (City, Barangay, etc.) automatically.

Technical Specifications

The component will use a "Factory" pattern to support multiple providers:

function MapPicker({ provider, onLocationSelect }) {
    switch(provider) {
        case 'GOOGLE': return <GoogleMapPicker />;
        case 'OSM':
        case 'GEOAPIFY': return <LeafletMapPicker />;
        default: return <DefaultMapPicker />;
    }
}

Testing Protocol

  • Verify marker placement accuracy on different zoom levels.
  • Test reverse geocoding response time and accuracy.
  • Ensure the component is responsive and works well on mobile devices.
