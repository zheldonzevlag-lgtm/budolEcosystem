Phase 3: UI Redesign

Integrating the MapPicker into the existing address management workflow while improving the overall UX.

UX Improvements

  • Split Layout: On desktop, show the map and the form side-by-side.
  • Progressive Disclosure: Start with the map search, then show detailed form fields.
  • Visual Validation: Show a "Map Pin Verified" badge when coordinates are set.
  • Improved Feedback: Use loading states and toast notifications for geocoding actions.

Implementation Details

We will update the StoreAddressModal to use the new MapPicker. The modal will expand to accommodate the map, providing a larger canvas for location selection.
