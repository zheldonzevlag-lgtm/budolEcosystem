Implementation Summary

The Lalamove Shipping Fee Fix has been fully implemented, addressing critical data propagation issues and ensuring financial accuracy for shipping charges.

Technical Improvements

  • Centralized shipping quote data management.
  • End-to-end propagation of stopIds.
  • Robust phone number formatting for API compatibility.
  • Validated price calculation in both frontend and backend.

Business Benefits

  • Accurate shipping fee collection from customers.
  • Reduced delivery booking failures.
  • Improved transparency in order history.
  • Better support for same-day delivery options.

Final Technical Decisions

Stop ID Preservation: We decided to store the full stops array from Lalamove in the order's shipping metadata. This avoids redundant API calls and ensures that the exact locations quoted are the ones used for booking.

Fee Override: The backend now prioritizes Lalamove quote fees over store-specific flat rates when Lalamove is selected as the provider, preventing under-charging for premium shipping services.
