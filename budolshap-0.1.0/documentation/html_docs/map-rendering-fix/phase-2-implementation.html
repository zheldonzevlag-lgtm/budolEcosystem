1. Structural Fixes for Height

The primary fix involved ensuring the height prop is correctly applied to the outermost container of the MapPicker component.

<div style={{ height: height === '100%' ? '100%' : height }} className="relative w-full rounded-xl overflow-hidden border-2 border-slate-100 shadow-sm h-full" > <div className="z-0 h-full w-full"> <MapContainer ... style={{ height: '100%', width: '100%' }}> ... </MapContainer> </div> </div>

By applying h-full and style={{ height: '100%' }} down the tree, we ensured that the map expands to the 200px container defined in AddressFormManager.jsx.
