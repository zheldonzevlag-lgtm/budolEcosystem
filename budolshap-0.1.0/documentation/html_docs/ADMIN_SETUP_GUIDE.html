🔧 Troubleshooting

Cannot Login?

  1. Verify you're using the correct email: admin@budolshap.com
  2. Check that ADMIN_EMAILS is set in Vercel environment variables
  3. Try redeploying the application: vercel --prod
  4. Clear your browser cache and cookies

"Access Denied" Error?

  • Verify the user has isAdmin: true in the database
  • Check accountType is set to ADMIN
  • Ensure emailVerified is true
  • Confirm email is in the ADMIN_EMAILS environment variable

Need to Reset Password?

Run the create-admin script again to reset the password to default:

npx dotenv -e .env.production -- node scripts/create-admin.js

This will update the existing admin account with the password admin123.

🔒 Security Best Practices

  • Change Default Password: Never use admin123 in production
  • Use Strong Passwords: Minimum 12 characters with mixed case, numbers, and symbols
  • Limit Admin Access: Only create admin accounts for trusted users
  • Regular Audits: Review admin actions and access logs periodically
  • Secure Email: Use a secure, dedicated email account for admin access
  • Enable 2FA: If available, enable two-factor authentication
