Phase 2: Core Service Implementation

Shipping Service Fixes

Modified lib/services/shippingService.js to include null fallbacks for all Lalamove-related metadata.

Return Booking Fix:

// Before
customerOrderId: booking.customerOrderId,
shareLink: booking.shareLink

// After
customerOrderId: booking.customerOrderId || null,
shareLink: booking.shareLink || null

Standard Booking Fix:

// Before
trackingUrl: booking.trackingUrl,
shareLink: booking.shareLink

// After
trackingUrl: booking.trackingUrl || null,
shareLink: booking.shareLink || null

Tracking Info Fix:

// Before
driver: trackingInfo.driver,
location: trackingInfo.location

// After
driver: trackingInfo.driverInfo || null,
location: trackingInfo.location || null
