UI Verification Guide

Step-by-step instructions to verify the Phase 1 Shipping Absorption UI changes.

1. Checkout Transparency (Buyer View)

1
Navigate to Cart: Add any product to your cart and go to /cart.
2
Proceed to Checkout: Click "Proceed to Checkout". In the Order Summary panel, look for the shipping section.
3
Verify Breakdown: You should see a hierarchical list:
  • Shipping Fee: ₱XX.XX
  • ↳ Seller Subsidy: -₱XX.XX
  • ↳ Shipping Voucher: -₱XX.XX

2. Order History Tooltips (Buyer View)

1
Go to Orders: Navigate to /orders.
2
Hover Over Shipping: Find a recent order and hover your mouse over the "Shipping Fee" text.
3
Check Tooltip: A white popover should appear showing the full math: Total Cost - Subsidy - Voucher = Net Fee.

3. Seller Dashboard (Store Owner View)

1
Access Store Orders: Log in as a seller and go to /store/orders.
2
Select Order: Click on an order to view its details.
3
Verify Subsidy: Check the summary section to ensure the "Seller Subsidy" you provided is clearly listed as a deduction.

4. Post-Payment Summary

1
Complete Payment: Complete a test payment.
2
Check Return Page: On the /payment/return page, verify that the shipping breakdown is preserved and displayed for the final confirmation.
