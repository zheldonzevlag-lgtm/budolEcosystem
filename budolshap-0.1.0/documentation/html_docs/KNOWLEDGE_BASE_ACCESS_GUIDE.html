📚 Knowledge Base Access Guide

Welcome to the BudolShap Knowledge Base Access System!

This guide explains how to securely access and use the knowledge base documentation system, including keyboard shortcuts, password authentication, session management, and security features.

🚀 Quick Start

Accessing the Knowledge Base

1 Press the keyboard shortcut: Ctrl + Shift + K Windows/Linux Cmd + Shift + K Mac
2 Request Access Code: Click "Send Access Code to Email" button
3 Check Your Email: A 6-digit access code will be sent to the registered email address
4 Enter Code: Type the 6-digit code in the prompt
5 Access Granted: Knowledge base opens in a new browser tab

⌨️ Keyboard Shortcuts

Available Shortcuts

Platform Shortcut Action
Windows / Linux Ctrl + Shift + K Opens the knowledge base access modal
Mac Cmd + Shift + K Opens the knowledge base access modal

💡 Tip

The keyboard shortcut works from anywhere within the BudolShap application. Press it at any time to quickly access the knowledge base!

🔐 Password Authentication System

How It Works

The knowledge base uses a secure password-based authentication system:

  • 6-Digit Access Code: A random 6-digit code is generated for each access request
  • Email Delivery: The code is sent to the registered email address
  • Time-Limited: Each code expires after 10 minutes
  • One-Time Use: Codes can only be used once
  • Attempt Limiting: Maximum 5 attempts per code to prevent brute force attacks

⚠️ Important Security Notes

  • Access codes expire after 10 minutes for security purposes
  • If you exceed 5 failed attempts, you must request a new code
  • Codes are single-use only - each successful authentication invalidates the code
  • Keep your email secure - codes are sent only to the registered address

⏱️ Session Management

Session Duration

Your knowledge base session is valid for 10 minutes from the time the access code was generated. The session timer runs even when the knowledge base tab is closed.

Session Expiration Warning

When your session is about to expire (2 minutes remaining), you will see a warning modal with the following options:

🔄 Extend Session

What it does:

  • Opens the main application in a new tab
  • Automatically shows the password request modal
  • Allows you to request a new access code
  • Upon validation, your session is extended for another 10 minutes

How to use: Click the "Extend Session" button in the warning modal, then follow the steps to request and enter a new access code.

❌ Close & End Session

What it does:

  • Immediately closes the knowledge base window
  • Clears all session data
  • Requires re-authentication to access again

When to use: Click this if you're finished using the knowledge base and want to end your session.

⚠️ Automatic Session Termination

If your session expires without extending, the knowledge base will automatically close and you will need to authenticate again using a new access code.

🔒 Security Features

Implemented Security Measures

  • Email Address Obfuscation: The recipient email address is base64-encoded in the source code to prevent plain text visibility
  • Password Expiry Obfuscation: Configuration values are encoded to protect sensitive settings
  • Environment Variable Support: Critical values can be set via environment variables for enhanced security in production
  • Rate Limiting: Maximum 5 attempts per access code prevents brute force attacks
  • Time-Limited Sessions: 10-minute session expiry reduces risk of unauthorized access
  • One-Time Use Codes: Codes are invalidated after successful use

🛡️ Production Security Recommendations

For production deployments, it is recommended to use environment variables instead of hardcoded values:

# Set in your .env file or deployment environment
KB_TARGET_EMAIL=your-email@example.com
KB_PASSWORD_EXPIRY=600000

This provides an additional layer of security by keeping sensitive values out of the source code entirely.

📧 Email Configuration

Email Delivery System

The system uses the configured SMTP settings to send access codes. The email includes:

  • Your 6-digit access code (prominently displayed)
  • Expiration time information (10 minutes)
  • Instructions for using the code
  • Professional HTML email template

📬 Checking Your Email

After requesting an access code:

  1. Check your inbox for an email with subject "BudolShap Knowledge Base Access Code"
  2. If not in inbox, check your spam/junk folder
  3. Look for the 6-digit code displayed in a highlighted box
  4. Enter the code within 10 minutes before it expires

🐛 Troubleshooting

Common Issues and Solutions

❌ "Invalid password" or "Access Denied"

  • Check the code: Make sure you entered all 6 digits correctly
  • Check expiration: Codes expire after 10 minutes - request a new one if expired
  • Check attempts: Maximum 5 attempts allowed - request a new code if exceeded

📧 Email not received

  • Check your spam/junk folder
  • Verify your email address is correct
  • Wait a few moments - email delivery may take 30-60 seconds
  • Check SMTP configuration if emails consistently fail

⌨️ Keyboard shortcut not working

  • Windows/Linux: Make sure you're pressing Ctrl + Shift + K (all three keys together)
  • Mac: Make sure you're pressing Cmd + Shift + K (all three keys together)
  • Ensure the BudolShap application window is focused/active
  • Some browsers or extensions may interfere - try in a different browser

⏱️ Session expired unexpectedly

  • Sessions expire 10 minutes after code generation (not after opening)
  • Use "Extend Session" before expiration to continue working
  • If expired, simply request a new access code

🔄 Extend session not working

  • Make sure pop-ups are allowed in your browser
  • The main application should open in a new tab
  • Press Ctrl+Shift+K or Cmd+Shift+K in the main app if modal doesn't appear automatically
  • Request a new access code and validate it to extend your session

💡 Best Practices

Tips for Optimal Usage

  • Keep the knowledge base tab open: Once authenticated, keep the tab open to avoid re-authentication
  • Monitor the warning: When the 2-minute warning appears, decide whether to extend or close
  • Request code promptly: Request access codes only when ready to use them (they expire in 10 minutes)
  • Save important docs: If you find important documentation, bookmark or save it for future reference
  • Use search: The knowledge base has a powerful search feature - use it to quickly find relevant documents
