User Manual: Using the Map

Selecting Your Location

When placing an order or updating your profile, you can select your delivery address using the interactive map.

Features

  • Drag to Move: Drag the map to position the pin on your exact location.
  • Zoom Controls: Use the + and - buttons to zoom in/out for better precision.
  • Current Location: Click the navigation arrow icon to automatically center the map on your current GPS location (requires permission).
  • Search: Type your address in the search bar above the map to quickly jump to a location.

Google Maps Experience

If enabled by the administrator, you will see the familiar Google Maps interface. You can switch between map view and satellite view if configured.
