Future Recommendations

Enhancements

  • Google Places Autocomplete: Integrate the Autocomplete widget directly into the search bar for better address prediction.
  • Route Visualization: Add capability to show delivery route from store to user location.
  • Map Caching: Implement caching strategies for map tiles if possible (limited by Google Terms of Service).

Dependencies & Software Choice

  • React-Leaflet: Chosen for the base implementation due to its open-source nature and flexibility with OSM.
  • Google Maps JS API: Chosen for the premium tier to provide "Native Google Tiles" as requested. It offers superior POI data and satellite imagery compared to OSM.
