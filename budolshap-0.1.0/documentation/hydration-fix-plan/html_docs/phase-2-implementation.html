Phase 2: Implementation

Code Changes

Modified: app/(public)/orders/page.jsx

Added mounted state and updated loading logic.

// Before
const loading = isLoading && page === 1;

if (loading) {
    return <Loading />;
}

// After
const [mounted, setMounted] = useState(false);
useEffect(() => {
    setMounted(true);
}, []);

const loading = (isLoading && page === 1) || !mounted;

if (loading) {
    return <Loading />;
}

Implementation Steps

  1. Initialize mounted state to false.
  2. Use useEffect to set mounted to true after the component mounts on the client.
  3. Modify the loading condition to include !mounted.
  4. This ensures that on the server (where useEffect doesn't run) and on the initial client render, the component returns the <Loading /> component.
