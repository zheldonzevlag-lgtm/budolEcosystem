Marketplace Guide for Sellers

Managing Variation Prices

When you enable variations for your product (e.g., Color, Size), you no longer need to set a single top-level price. You can now set a unique price for each combination in the Variation Matrix.

The system will automatically calculate and display the starting price for your product based on the lowest price in your matrix. If different variations have different prices, the marketplace will show a price range to potential buyers.

Removing Variations

If a specific variation is no longer available (e.g., "Red / Low" is out of stock indefinitely), you can use the Trash icon in the Variation Matrix to hide it from buyers. You can restore it later if needed using the Restore button.
