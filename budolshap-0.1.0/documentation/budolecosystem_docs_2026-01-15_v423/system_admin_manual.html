System Admin Manual - v423

System Overview

Administrators can now manage complete user profiles including phone numbers and physical addresses. This data is stored in the User and Address tables respectively.

Managing User Profiles

Adding a New User

  1. Navigate to Admin Dashboard > Manage Users.
  2. Click Add User.
  3. Enter basic details (Name, Email, Password).
  4. Enter the Phone Number (required).
  5. Click the Address field to open the Map/Address picker.
  6. Select the address on the map or use the autocomplete search.
  7. Click Save Address.
  8. Complete the form and click Add User.

Editing a User

  1. Find the user in the list and click Edit.
  2. Modify phone number or address as needed.
  3. Click Update User.

Data Integrity & Security

Phone numbers are unique across the system. Addresses are geocoded to ensure accurate location data for future delivery integrations.
