Mitigation Strategies

Mapping Gaps (RSK-CAT-001)

Strategy: Implemented a "Partial Match" logic in getCategoryLucideIcon. If an exact slug match isn't found, the system searches for keyword overlaps. Furthermore, the Package icon serves as a safe visual fallback.

Performance Overhead (RSK-CAT-002)

Strategy: Grouped icon imports from lucide-react. Modern bundlers (Webpack/Vite) perform "tree-shaking" to ensure only the used icons are included in the production build, preventing bloated bundle sizes.

Dependency Locking (RSK-CAT-003)

Strategy: package.json uses caret ^ for compatible updates but production builds are locked via package-lock.json to prevent breaking version jumps.
