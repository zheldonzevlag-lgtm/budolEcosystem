System Admin Manual

This release involves frontend UI changes and does not require server-side configuration or database migrations.

Deployment Verification

  • Ensure Next.js build passes without lint errors.
  • Verify that responsive breakpoints (max-width: 640px for mobile) correctly trigger the flex-nowrap/overflow-x behaviors.
  • Check that browsers with "touch-simulation" enabled in DevTools correctly handle the horizontal swipe on thumbnails and categories.
