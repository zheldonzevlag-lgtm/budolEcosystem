Version 3.3.19 - User Profile Audit & Forensic Logging

Purpose & Rationale

This release extends the forensic audit logging system to include User Profile Modifications. Ensuring that all changes to user profiles (Names, Emails, Addresses) are logged with granular details (previous vs. new values) is critical for identity theft protection and regulatory compliance (BSP Circular 808, PCI DSS).

Technical Summary

  • Profile Update Auditing: Implemented createAuditLog in budolPay Auth Service and budolShap User API.
  • Granular Change Tracking: Logs now capture previous and updated values for critical fields (Name, Email) to enable detailed forensic reconstruction.
  • Compliance Controls: Enforced restrictions on name changes for FULLY_VERIFIED users in BudolPay.
  • Integration: Seamlessly integrated with the existing centralized audit logging infrastructure.
