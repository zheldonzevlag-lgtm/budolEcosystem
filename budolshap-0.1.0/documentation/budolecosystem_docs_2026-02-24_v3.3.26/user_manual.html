User Manual: Address Management Updates (v3.3.25)

Welcome to the updated Address Management system (v3.3.25). We have made it easier for you to manage your addresses.

What's New?

1. Store Address Labels & Defaults (New in v3.3.25)

Store owners can now label their addresses (e.g., "Home", "Work", "Other") to easily identify them. You can also set a default address that will be used automatically for shipping calculations.

2. Simplified Forms

District and Zip Code are no longer mandatory fields for store addresses. You can leave them blank if not applicable.

3. Optional Barangay (v3.3.24)

You are no longer required to enter a "Barangay" when adding or editing an address. This field is now optional.

4. Automatic Name Filling (v3.3.24)

When you add a new address, your First Name and Last Name will be automatically filled in based on your profile information. You can still edit them if needed.

3. Improved Error Messages

We have improved the error messages you see when filling out forms. Required fields will be clearly marked, and if you miss something, specific guidance will appear right below the field.

How to Add a Store Address

  1. Go to your Store Dashboard.
  2. Navigate to Settings > Address Book.
  3. Click Add New Address.
  4. Fill in the details.
    • Label: Choose "Home", "Work", or "Other".
    • Set as Default: Check this box if you want this to be your primary pickup/return address.
  5. Click Save Address.

How to Add a Personal Address

  1. Go to your Profile or Checkout page.
  2. Click on "Add Address".
  3. Fill in the details. Note that "Barangay" is marked as Optional.
  4. Click "Save Address".

Troubleshooting

Issue: "Missing required fields" error.
Solution: Check if you have filled in all fields marked with a red asterisk (*). Barangay is NOT required.
