Infrastructure as Code (CDK) & Serverless Patterns

Phase 1: API Gateway Infrastructure

Duration: 2 Weeks Stack: AWS CDK (TypeScript)
1.1. Define API Gateway (HTTP API) Est: 4 Hours

Objective: Provision a serverless HTTP API Gateway using CDK. This is your "Front Door".

File: lib/gateway-stack.ts
AWS CDK import * as apigwv2 from 'aws-cdk-lib/aws-apigatewayv2'; import { HttpLambdaIntegration } from 'aws-cdk-lib/aws-apigatewayv2-integrations'; export class GatewayStack extends cdk.Stack { constructor(scope: Construct, id: string, props?: cdk.StackProps) { super(scope, id, props); // 1. Create the API const httpApi = new apigwv2.HttpApi(this, 'BudolHttpApi', { apiName: 'BudolServiceGateway', corsPreflight: { allowMethods: [ apigwv2.CorsHttpMethod.ANY ], allowOrigins: ['*'], // Lock this down in prod }, }); // 2. Export API Endpoint for other stacks new cdk.CfnOutput(this, 'ApiEndpoint', { value: httpApi.apiEndpoint, }); } }

CLI Execution:

cdk init app --language typescript npm install aws-cdk-lib constructs cdk deploy GatewayStack

Phase 2: Auth Service (Cognito + Lambda)

2.1. Define User Pool & Client Est: 6 Hours

Objective: Create a User Pool for customer identities.

File: lib/auth-stack.ts
AWS CDK import * as cognito from 'aws-cdk-lib/aws-cognito'; const userPool = new cognito.UserPool(this, 'BudolUserPool', { selfSignUpEnabled: true, signInAliases: { email: true }, autoVerify: { email: true }, }); const client = userPool.addClient('WebClient', { authFlows: { userSrp: true, }, });
2.2. Secure Route with JWT Authorizer Est: 4 Hours

Objective: Protect the /profile route so only logged-in users can access it.

File: lib/gateway-routes.ts
AWS CDK import * as apigwv2 from 'aws-cdk-lib/aws-apigatewayv2'; import * as authorizers from 'aws-cdk-lib/aws-apigatewayv2-authorizers'; // Define the JWT Authorizer pointing to Cognito const jwtAuthorizer = new authorizers.HttpUserPoolAuthorizer( 'BudolAuthorizer', userPool, { userPoolClients: [client], } ); // Add Route httpApi.addRoutes({ path: '/profile', methods: [ apigwv2.HttpMethod.GET ], integration: profileLambdaIntegration, authorizer: jwtAuthorizer, // <--- Secured! });
