🔄 Implementation Flow

1

Store Owner Books Delivery

Action: Store owner clicks "Book Delivery" in their dashboard

Trigger: POST request to /api/shipping/lalamove/book

2

Lalamove Booking Created

Action: System creates booking with Lalamove API

Result: Booking ID and tracking link received

3

Order Status Updated

Action: Database updates order status to PROCESSING

Fields Updated:

  • status → 'PROCESSING'
  • shipping.bookingId → Lalamove booking ID
  • shipping.trackingUrl → Tracking link
  • shipping.bookedAt → Current timestamp
4

Email Notification Sent

Action: System sends "Order Processing" email to customer

Email Content:

  • Order details
  • Store information
  • Delivery tracking link
  • Estimated delivery time
5

Booking Confirmed

Action: Success response returned to store owner

Response Includes:

  • Booking ID
  • Tracking URL
  • Estimated pickup/delivery times
  • Price breakdown
