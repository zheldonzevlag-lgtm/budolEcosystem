💡 Implementation Notes

Status Guards:

Added conditional checks to prevent status updates if order is already in a later stage:

  • PICKED_UP only updates if status is not SHIPPED, DELIVERED, or COMPLETED
  • COMPLETED only updates if status is not DELIVERED or COMPLETED

This prevents race conditions and ensures status progression is one-way.

Error Handling:

Email failures don't cause webhook failures:

  • Email sending is wrapped in try-catch
  • Errors are logged but don't prevent webhook acknowledgment
  • This ensures Lalamove doesn't retry the webhook unnecessarily

Auto-Completion Calculation:

The 7-day auto-completion period is calculated using JavaScript Date:

const autoCompleteDate = new Date() autoCompleteDate.setDate(autoCompleteDate.getDate() + 7)

This correctly handles month boundaries and leap years.

Step 3: Test Delivery Completion

  2. Check Vercel logs for COMPLETED webhook
  3. Verify order status changed to DELIVERED
  4. Check deliveredAt timestamp is set
  5. Verify autoCompleteAt is set to 7 days from now
