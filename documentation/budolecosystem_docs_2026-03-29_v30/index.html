Security Hardening &
PII Data Isolation

Release v30 introduces an aggressive Zero-Trust session isolation protocol. It forcibly invalidates cached Personal Identifiable Information (PII) upon logout, sanitizes all Redux storage environments, and explicitly blocks unauthenticated cart checkout sessions.
