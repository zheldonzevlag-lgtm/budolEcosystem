User Manual: Connecting to the BudolEcosystem

This manual provides instructions for users and testers to connect their mobile devices and workstations to the BudolEcosystem via the local network.

1. Prerequisites

  • Your device must be connected to the same Wi-Fi or LAN as the server hosting the BudolEcosystem.
  • Ensure the server is running (started via start-ecosystem.js).

2. Connecting via Mobile App

1. Open the BudolPay mobile application on your Android device.
2. The app is designed to automatically detect the API Gateway using mDNS (Bonjour). If successful, you will see a "Connected to Server" notification.
3. If automatic detection fails, go to Settings > Connection and manually enter the Server IP: 192.168.1.24 and Port: 8080.

3. Accessing Web Interfaces

You can access the web-based dashboards from any device on the network using the following URLs:

  • Admin Dashboard: http://192.168.1.24:3000
  • BudolShap Marketplace: http://192.168.1.24:3001

4. Troubleshooting Connection Issues

  • Connection Timed Out: Ensure the server's firewall is not blocking the ports (8080, 3000, 3001).
  • Invalid IP: Double-check the server's current IP address by running ipconfig on the server machine.
  • Network Profile: Ensure the server's network is set to "Private".
