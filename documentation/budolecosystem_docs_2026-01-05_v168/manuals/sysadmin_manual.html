Database Infrastructure

The updates affected three separate PostgreSQL databases. Ensure these connection strings are maintained in their respective .env files.

BudolID Database

postgresql://postgres:r00t@localhost:5432/budolid

BudolPay Database

postgresql://postgres:r00t@localhost:5432/budolpay

BudolShap Database

postgresql://postgres:r00t@localhost:5432/budolshap_1db

Verification Runbook

To verify the data integrity manually, run the following command in each service directory:

npx prisma studio

Navigate to the User table and filter by the emails: reynaldomgalvez@gmail.com, tony.stark@budolshap.com, peter.parker@budolshap.com.

Risk & Mitigation

  • Risk: Duplicate phone numbers (Unique Constraint).
    Mitigation: The scripts include try-catch blocks to catch uniqueness violations before they propagate.
  • Risk: Desynchronized databases.
    Mitigation: All three databases were updated simultaneously in a single maintenance window.
