M-493-1: Realtime Connection Fallback

If Pusher or Socket.io fails to initialize, the system automatically defaults to POLLING mode to ensure the UI remains functional.

M-493-2: Bandwidth Conservation

The visibilitychange listener reduces polling frequency when the user is inactive, mitigating the risk of high cloud hosting costs.

M-493-3: Settings Caching

Admin settings are cached for 60 seconds on both frontend and backend to avoid excessive database load.
