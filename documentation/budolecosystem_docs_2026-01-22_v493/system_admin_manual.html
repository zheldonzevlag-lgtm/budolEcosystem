1. Realtime Infrastructure Configuration

Admins can control the entire ecosystem's realtime behavior through the Admin Dashboard settings. Changes take effect within 60 seconds across all nodes.

Active Provider: Choose between PUSHER, SOCKET_IO, or POLLING.

Provider Comparison

  • PUSHER: Managed service, lowest latency, highest reliability. Requires API keys.
  • SOCKET_IO: Self-hosted, zero monthly fees. Requires a dedicated Socket.io server URL.
  • POLLING: No infrastructure required. Higher bandwidth usage on AWS/Vercel.

2. Cost Controls

To keep cloud hosting bills low, v493 automatically adjusts polling rates based on user activity. No manual intervention is required, but admins should monitor "Bandwidth" in their cloud console if POLLING is active for long periods.
