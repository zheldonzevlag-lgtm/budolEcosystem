Issue: My Orders UI Flickering

The UI was flickering whenever a Pusher update was received because the entire page was being refreshed using router.refresh().

Fix: SWR Cache Patching

Replaced router.refresh() with targeted SWR cache mutation. The UI now updates only the specific order that changed, without a full reload.

Issue: High Bandwidth Usage (Polling)

Polling at 5s intervals even when the tab was hidden was consuming excessive bandwidth on AWS/Vercel.

Fix: Visibility-Aware Throttling

Implemented a listener that increases the polling interval to 30s when the tab is hidden, reducing bandwidth usage by ~83%.
