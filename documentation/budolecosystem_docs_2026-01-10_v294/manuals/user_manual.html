Accessing the System on LAN

To access the budolEcosystem from another computer or mobile device on the same network:

  1. Find the server's IP address (provided in the terminal when running npm start).
  2. Open your browser and enter the address, for example:
    http://192.168.1.24:3001 (for budolShap)
  3. When you click "Sign In", you will be redirected to the server's SSO page.
  4. Log in as usual.
Pro Tip: You can use your smartphone to scan QR codes or take selfies for identity verification by accessing the system via your phone's browser using the server's LAN IP.

Cross-Network Usage

The system is now optimized for mobile developers and admins who move between different work environments. Whether you are at home or in an office, the system adapts to your current network setup.
