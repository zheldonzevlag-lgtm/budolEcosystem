Summary of Changes

This version focuses on enabling the ecosystem to work seamlessly across different network environments (LAN/WAN) and fixing SSO redirect issues that were previously restricted to localhost/127.0.0.1.

Technical Highlights

  • Automated LAN IP Sync: New script to detect and propagate host IP across all services.
  • SSO Redirection Fixes: Removed hardcoded localhost/127.0.0.1 from redirect logic.
  • Database Synchronization: Automated updating of EcosystemApp redirect URIs in the database.
  • JWT Standardization: Unified JWT_SECRET across 21 environment files for consistent authentication.
