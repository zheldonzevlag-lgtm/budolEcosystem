SysAdmin Manual - Infrastructure & Deployment

Deployment Procedures

Mobile APK Distribution

The latest production-ready APK has been generated and moved to the centralized installer directory. System administrators should distribute this version to beta testers and production users.

  • Artifact Path: \budolEcosystem\android-installer\budolPay_v1.2.10.apk
  • Build Command: flutter build apk --release --dart-define=API_HOST=192.168.1.24

Environment Variables

The mobile application continues to use API_HOST as the primary configuration point for backend connectivity. For this release, the default host is set to 192.168.1.24.

# Build-time define
--dart-define=API_HOST=192.168.1.24

Operational Runbooks

Host Recovery

If the application loses connectivity due to a host change, use the built-in host management tools in the ApiService:

  • setHost(String newHost): Manually override the API host.
  • resetHost(): Revert to auto-discovery or environment defaults.
  • discoverAndSetHost(): Trigger a fresh UDP discovery scan for the gateway.

Risk & Mitigation

  • Risk: Hardcoded IP address in build command might fail in different network environments.
  • Mitigation: The application implements DiscoveryService for dynamic gateway lookup if the hardcoded host is unreachable.
