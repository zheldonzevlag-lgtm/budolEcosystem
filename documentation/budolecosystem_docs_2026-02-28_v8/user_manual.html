End-User Viewpoint: Hero Section Enhancements

The marketplace's home page (Hero Section) now more accurately reflects starting prices for available items. Users will no longer see "Starts from ₱0.00" if products with actual prices are available in the system.

Experience Highlights

  • Reduced Confusion: Accurate entry-level pricing for the shop.
  • Clearer Expectations: The minimum price shown represents the lowest value of buyable items.
  • Automatic Updating: As sellers add or update products with real prices, the hero section will automatically reflect the new starting price.

What to Do If ₱0.00 Still Appears?

If all items in the shop have a price of 0 (e.g., during a "Free Items" promotion), the system will display ₱0.00 for the entire shop. If you believe this is in error, please contact support.
