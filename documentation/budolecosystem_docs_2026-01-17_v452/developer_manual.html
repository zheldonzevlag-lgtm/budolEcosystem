Developer Manual - v452

Navigation Tracing & Debugging

To resolve reports of the MarketingAdsScreen being skipped, we have introduced enhanced state tracing in the SplashScreen. This helps developers understand why the app chooses a specific route.

splash_screen.dart 
debugPrint('SplashScreen: Navigation State - hasSeenAds: $hasSeenAds, isAuthenticated: $isAuthenticated');

Ads Status Reset

A new method resetAdsStatus() has been added to ApiService to allow clearing the hasSeenAds flag from SharedPreferences. A hidden debug button is now available at the bottom of the SplashScreen in debug mode to trigger this reset.

api_service.dart 
Future<void> resetAdsStatus() async {
  _hasSeenAds = false;
  final prefs = await SharedPreferences.getInstance();
  await prefs.remove(_hasSeenAdsKey);
  notifyListeners();
}
