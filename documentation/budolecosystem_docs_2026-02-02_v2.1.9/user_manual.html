Admin Interface Guide

How to use the improved navigation in the budolPay Admin console.

Improved Navigation

The Admin Sidebar has been updated to provide quicker access to frequently used settings without cluttering the main menu.

Using Submenus

1. Look for the System Settings item in the left sidebar.

2. Click the Chevron Icon (Arrow) next to the label to expand or collapse the menu.

3. Select a specific setting area to navigate directly to that section.

Pro Tip

The sidebar will automatically stay expanded if you are currently on a settings page, helping you keep track of your location within the system configuration module.
