budolPay Admin Sidebar Fix

Version 2.1.9 restores the missing System Settings submenus in the budolPay Admin sidebar, improving navigation efficiency for system administrators.

Key Changes

  • - Implemented expandable submenu for "System Settings".
  • - Added sub-navigation items: Environment Variables, Service Endpoints, and Security Config.
  • - Integrated state management to remember submenu toggle status.
  • - Enhanced NavItem component to support nested styling.

Compliance & Security

  • - NPC: Access control visibility remains restricted.
  • - BSP: Audit trails for setting changes maintained.
  • - PCI DSS: Sensitive configuration exposure minimized in UI.
