System Administration

Guide for managing system configurations via the enhanced admin sidebar.

Navigating System Settings

The "System Settings" section in the sidebar is now an expandable menu. Click the section header to reveal specific configuration areas.

Environment Variables

Access the primary configuration table where system-wide variables (e.g., API keys, provider settings) are managed. Changes here require audit log confirmation.

Service Endpoints

Quick link to the endpoints section of the settings page for managing service URLs and connection strings.

Security Config

Jump to security-specific configurations, including MFA toggles and session timeout policies.

Troubleshooting Navigation

If submenus do not appear after clicking:

  1. Ensure you are logged in with "Super Admin" or "System Admin" privileges.
  2. Clear browser cache if UI elements appear misaligned.
  3. Verify that the Sidebar.tsx component has loaded correctly by checking for the "Powered by budolEcosystem" footer.
