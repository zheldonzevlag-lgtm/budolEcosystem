Centralized Ecosystem Access

Accessing the Ecosystem

The ecosystem is now accessible via a single URL, with the BudolPay Admin as the central hub:

  • Ecosystem Admin (Command Center): https://budolecosystem.vercel.app
  • BudolShap Marketplace: https://budolecosystem.vercel.app/shap

Navigation

The Ecosystem Admin dashboard provides a centralized view of all backend services. The marketplace can be accessed by appending /shap to the main domain.
