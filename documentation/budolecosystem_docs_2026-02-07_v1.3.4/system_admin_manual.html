System Admin Manual (v1.3.4)

1. User Identification

Administrators should note that users registered via "Quick Registration" will no longer have the "User " prefix in the database. Their name field will directly contain their phone number in E.164 format (e.g., +639123456789).

Important: When searching for users in the database or admin dashboard, use the full phone number (including +63) if the user has not yet set a custom display name.

2. Configuration Settings

Setting Value Impact
Default Display Name [phoneNumber] Clean UI presentation for phone-only users.
OTP Greeting Fallback [phoneNumber] Personalized SMS/Email notifications even without a profile name.

3. Troubleshooting

  • Issue: User appears as "Guest" in Navbar.
  • Fix: Verify the user session has both name and phoneNumber fields. If both are missing, the system defaults to "Guest".
