Known Issues (v1.3.4)

  • Legacy Users: Users registered prior to v1.3.4 will still have "User " prepended to their phone numbers in the database unless they manually update their profile.
  • Search Indexing: Some admin search tools might still be optimized for name-based searches and may require the full phone number string to find v1.3.4 users.
