User Manual - budolPay v1.2.49

New Feature: Easy Setup from Login

You can now configure your BudolPay connection before even logging in! This is helpful if you are connecting to a new network or if your connection details have changed.

How to Configure API Connection

Step 1: Access Settings

On the initial login screen where you enter your mobile number, look for the Gear Icon in the top right corner of the screen.

Step 2: Enter Connection Details

Tap the icon to open the API Host Configuration window. You have two options:

  • Manual Entry: Type the IP address and port of your BudolPay Gateway (e.g., 192.168.1.24:8080).
  • Scan Network: Tap "Scan Network" to let the app automatically find the gateway for you. Make sure you are connected to the same Wi-Fi network as the gateway.

Step 3: Save and Continue

Tap Save. You will see a confirmation message "Host updated!". You can now proceed with entering your mobile number to log in.

Biometric Login Reminder

Once you are logged in, you can still enable Biometric Login (Fingerprint or Face) in the main Settings menu for even faster access next time.
