System Admin Manual - budolPay v1.2.49

Deployment Overview

This version introduces easier configuration for end-users, reducing the support burden for administrators when deploying the app in new network environments.

Build Artifacts

The latest production APK is available at:

\\budolEcosystem\\android-installer\\budolpay-1.2.49.apk

Configuration Changes

No new environment variables or database schema changes were introduced in this release. The focus was on UI-based configuration persistence.

Persistent Storage

Key Storage Method Purpose
budolpay_custom_host SharedPreferences Stores the user-defined API gateway host (IP:Port).

Operational Procedures

Network Discovery (mDNS)

Administrators should ensure that the BudolPay Gateway service is broadcasting its presence via mDNS (Bonjour/Avahi) on the local network. This allows the app's "Scan Network" feature to automatically detect the gateway without manual IP entry.

Service Name: _budolpay._tcp.local

Troubleshooting

  • "No gateway found": Ensure the mobile device is on the same subnet as the gateway and that mDNS traffic is not blocked by the router.
  • Host not updating: Clear app cache if the SharedPreferences values are not persisting correctly.
