1. Troubleshooting Connection Issues

If you encounter a message saying "Connection Timeout" or "Server not responding" during account creation or login, you can access the internal logs to help support staff diagnose the issue.

2. Accessing the Debug Console

The Debug Console is a hidden feature for troubleshooting. To access it:

Gesture: Press and hold (Long Press) on the BudolPay Logo (the large '₱' symbol) for 1 second.

Once opened, you will see a list of recent network activities. You can:

  • View Details: Tap on any log entry to see the full request and response.
  • System Info: Tap the "System Info" button to see device details (OS version, Model, etc.).
  • Clear Logs: Use the trash icon to clear the history.

3. Privacy Note

The Debug Console only stores data temporarily on your device. It does not send logs anywhere automatically. Only share screenshots with authorized BudolPay support representatives.
