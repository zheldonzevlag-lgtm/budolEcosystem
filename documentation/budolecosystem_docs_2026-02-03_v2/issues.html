📝 Known Issues - v3.0.3

I-AUTH-001: Registration Timeout

Status: Resolved in v3.0.3

Description: Users were experiencing TimeoutException (10s) during account creation because the server was waiting for slow third-party SMS providers and internal admin notifications before responding.

I-DEBUG-001: Lack of API Visibility

Status: Resolved in v3.0.3

Description: No way to inspect raw API requests and responses on physical devices without a debugger attached. This made remote troubleshooting impossible.

I-PERF-001: Slow Admin Notification

Status: Mitigated in v3.0.3

Description: The notifyAdmin helper in auth-service had a 2000ms timeout but was still blocking the main thread if the Gateway was slow.
