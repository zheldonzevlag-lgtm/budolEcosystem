User Manual: Enhanced Security

This update includes behind-the-scenes security enhancements to keep your data and transactions safe.

1. Secure Communications

BudolPay now uses "Certificate Pinning" to ensure that your mobile app only communicates with our official servers. This protects you from sophisticated hackers who might try to intercept your data on public Wi-Fi networks.

Security Tip: Always keep your BudolPay app updated to the latest version to ensure you have the latest security certificates.

2. What you need to do

Nothing! These security features work automatically. You may notice a slight increase in connection speed and stability as we optimize our network pathways for the upcoming cloud migration.

3. Troubleshooting

If you see a "Secure Connection Failed" message, please check if you are on a public Wi-Fi that requires a login (captive portal). If the problem persists, try switching to mobile data or a trusted home network.
