Using the budolPay Admin Transaction Dashboard

Transaction Visibility

The budolPay Admin dashboard provides a unified view of all money moving through the ecosystem.

1. Unified History

See transactions from GCash, PayMaya, GrabPay, and the internal budolPay Wallet in a single list.

2. Real-time Status

Transactions update from PENDING to COMPLETED as soon as the customer finishes the payment flow.

3. Detailed Metadata

Click on any transaction to see the associated Order ID from budolShap, the customer's email, and the provider used.

4. Search & Filter

Filter by status (e.g., only "COMPLETED") or type (e.g., "MERCHANT_PAYMENT") to generate financial reports.
