System Admin Manual

Configuration and maintenance guide for the Budol Ecosystem platform.

System Configuration

No new system-level configurations were introduced in this release (v2026-02-17 v1). Existing configurations for payment gateways and database connections remain valid.

Environment Variables

Variable Description Required
DATABASE_URL Connection string for the PostgreSQL database. Yes
NEXT_PUBLIC_API_URL Base URL for the public API. Yes

Maintenance Procedures

Database Integrity

The recent fix for cart item deletion relies on the `ordersService.createOrder` function correctly handling transactions. Regular database backups are recommended to ensure data safety.

Monitoring

Monitor logs for `[OrderSummary]` and `[BudolPay Webhook]` entries to verify correct cart behavior during high-traffic periods.
