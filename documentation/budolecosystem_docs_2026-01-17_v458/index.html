Release Date: 2026-01-17 | Type: Engine Recovery

Clean Build & Bootstrap Refinement

Primary Issue: DDC Module Loader Error

The browser reported Uncaught Library not defined: org-dartlang-app:/web_entrypoint.dart. This typically occurs when the Dart Development Compiler (DDC) has stale artifacts or the bootstrap configuration is not optimized for the modern Flutter loader.

Technical Resolutions

  • Flutter Clean: Performed a full flutter clean and flutter pub get to eliminate any corrupt or stale build artifacts in .dart_tool.
  • Bootstrap Optimization: Updated index.html to use the modern window.flutterConfiguration object for setting the renderer, which is the officially supported way for the new flutter_bootstrap.js loader.
  • Cache Invalidation: The clean build ensures that the server provides a fresh main.dart.js and module map to the browser.
