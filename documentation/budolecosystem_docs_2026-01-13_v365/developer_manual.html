Face Recognition Settings Implementation

This update adds a dedicated "Face Recognition Login" option in the Settings screen, allowing users to register and manage their local face profile for faster login.

Key Components

  • Face Registration: Integrated image_picker and FaceEmbeddingService into Settings.
  • State Management: Tracks _hasFaceTemplate to show registration status.
  • Security: Requires a stored PIN before allowing face registration to ensure the biometric profile is linked to a valid credential.

Code References

  • settings_screen.dart - UI and registration logic.
  • biometric_service.dart - Template storage methods.
  • face_embedding_service.dart - ML inference logic.

Technical Flow

  1. User taps "Setup face login" in Settings.
  2. App requests front-camera access via ImagePicker.
  3. FaceEmbeddingService processes the image using FaceNet TFLite model.
  4. Generated 128-dimension embedding is stored in FlutterSecureStorage.
  5. "Face Registered" status is displayed with a delete option.
