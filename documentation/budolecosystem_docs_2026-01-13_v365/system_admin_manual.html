Biometric Configuration

Admins should ensure that the following permissions are granted in the production APK:

  • android.permission.CAMERA - Required for face scanning.
  • android.permission.USE_BIOMETRIC - Required for system biometric prompts.

Troubleshooting

If users report that face recognition is not working, verify that the device has a working front camera and supports GMS (Google Mobile Services) for ML Kit functionality.
