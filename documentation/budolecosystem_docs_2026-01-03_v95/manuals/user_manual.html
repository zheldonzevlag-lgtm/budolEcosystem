User Manual - Version 95

System Stability Update

This version does not introduce new user-facing features but focuses on system availability and stability. Users may have previously experienced "Connection Refused" or "Server Error" messages during login; these issues have been resolved.

What's Changed for Users?

  • Reliable Login: The SSO (Single Sign-On) login process is now fully operational.
  • App Interconnectivity: Navigating between budolShap and budolPay is now seamless as the identity provider is active.
  • Faster Startup: The system now starts up correctly without manual intervention if the services are correctly configured.

Troubleshooting for Users

If you still cannot log in, please ensure your local internet connection is active and that the central server (budolID) is reachable at its local LAN address: http://192.168.1.24:8000.
