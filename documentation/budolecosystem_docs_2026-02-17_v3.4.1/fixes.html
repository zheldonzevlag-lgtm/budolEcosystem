Fixes & Changelog (v3.4.1)

Detailed log of changes and fixes included in this release.

v3.4.1 - 2026-02-17

Bug Fixes

  • Fixed Cart Item Persistence: Implemented a robust fallback mechanism in the PayMongo webhook handler to clear cart items upon successful payment.
    Reference: app/api/webhooks/paymongo/route.js
  • Enhanced Variation Handling: The new deletion logic strictly respects variationId, preventing accidental deletion of unpurchased product variations from the cart.

System Updates

  • Backup Script: Confirmed inclusion of Cart and CartItem tables in the database backup routine.
  • Documentation: Updated developer and system admin manuals to reflect the new webhook logic.
