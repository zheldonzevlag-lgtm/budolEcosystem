v196 Developer Manual

Subject: Technical Implementation of SMS Microservice

Architecture Overview

The sms-service acts as a wrapper around the Semaphore API. It provides an internal REST endpoint for other services to send messages.

1. Sending an OTP

Services can request an OTP by calling the /otp/send endpoint. The service handles code generation and expiration.

// POST /otp/send
{
    "phoneNumber": "09123456789",
    "purpose": "registration"
}

2. Verifying an OTP

The mobile app sends the user-provided code back for validation.

// POST /otp/verify
{
    "phoneNumber": "09123456789",
    "code": "123456"
}

Integration with Semaphore (PH)

The implementation uses the following Semaphore structure for message delivery:

const axios = require('axios');

async function sendSMS(to, message) {
    const res = await axios.post('https://api.semaphore.co/api/v4/messages', {
        apikey: process.env.SEMAPHORE_API_KEY,
        number: to,
        message: message,
        sendername: 'BUDOLPAY'
    });
    return res.data;
}
