Subject: Verifying your Phone Number with SMS OTP

What is SMS OTP?

To keep your account secure, budolPay now requires you to verify your mobile number using a One-Time Password (OTP). This ensures that only you can access your funds.

How it Works

  1. Enter your mobile number during registration.
  2. Wait for a 6-digit code to arrive via SMS.
  3. Enter the code in the app to proceed.

When will I receive an SMS?

  • During Registration: To verify your identity.
  • Sending Money: For high-value transactions.
  • Password Reset: To authorize a change to your account.

Troubleshooting

If you don't receive your code within 60 seconds, tap "Resend Code". Make sure you have a stable mobile signal and that you entered the correct number (starting with 09).
