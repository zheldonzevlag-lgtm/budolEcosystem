User Manual: Ecosystem Login Guide

How to use your centralized budolID to access budolShap.

Version: 2026-01-01_v31
Focus: Accessing the application via Single Sign-On.

Login with budolID

The "Login with budolID" button has been restored to the login forms. This allows you to use a single account for all applications in the budolEcosystem.

  1. Open the login modal or navigate to the login page.
  2. Click the Login with budolID button at the bottom of the form.
  3. You will be redirected to the secure budolID login portal.
  4. Enter your ecosystem credentials.
  5. Upon successful login, you will be automatically returned to budolShap.

Example: The SSO button is located below the standard login form.
