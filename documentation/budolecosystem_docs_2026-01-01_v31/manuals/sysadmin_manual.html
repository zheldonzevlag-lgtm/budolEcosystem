System Admin Manual: SSO Service Management

Infrastructure details and operational procedures for the budolID SSO service.

Version: 2026-01-01_v31
Focus: SSO service availability and environment configuration.

Service Management

The budolID service must be running for the ecosystem login to function. It has been restarted and configured to listen on port 8000.

budolID Status
  • Service Name: budolID
  • Port: 8000
  • Startup: npm run start
  • Location: budolID-0.1.0/

Environment Variables

The following variables control the SSO integration in budolShap:

Variable Description Default/Value
NEXT_PUBLIC_SSO_URL Base URL for the budolID service. http://[hostname]:8000
NEXT_PUBLIC_BUDOLID_API_KEY Application identifier for SSO. bs_key_2025
