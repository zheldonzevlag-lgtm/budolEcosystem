Transaction UI Standardization

The transaction list items have been standardized to improve readability. Developers should use the following pattern for any new transaction-related list items.

Colors

  • Slate 600 (#475569): Used for secondary text, descriptions, and dates.
  • Slate 700 (#334155): Used for primary section titles.
  • Slate 100 (#F1F5F9): Used for icon backgrounds.

Code Implementation

Refer to the updated helper in home_screen.dart:


Widget _buildTransactionItem(String title, String amount, String date, Color amountColor) {
  const slate600 = Color(0xFF475569);
  const fontSize = 13.0;
  
  return ListTile(
    contentPadding: EdgeInsets.zero,
    leading: CircleAvatar(
      backgroundColor: const Color(0xFFF1F5F9),
      child: const Icon(Icons.payment, color: Color(0xFFF43F5E), size: 20),
    ),
    title: UIUtils.formatBudolPayText(
      title, 
      baseStyle: const TextStyle(
        fontWeight: FontWeight.w600, 
        fontSize: fontSize,
        color: slate600,
      )
    ),
    trailing: Column(
      mainAxisAlignment: MainAxisAlignment.center,
      crossAxisAlignment: CrossAxisAlignment.end,
      children: [
        Text(amount, style: TextStyle(color: amountColor, fontWeight: FontWeight.bold, fontSize: fontSize)),
        const SizedBox(height: 2),
        Text(date, style: const TextStyle(color: slate600, fontSize: fontSize)),
      ],
    ),
  );
}
