Fixes & Improvements - v497

BUG-445-01

Shipping Status Mismatch

Problem: Return requests were not consistently updating the shipping status object, leading to orders stuck in "To Ship" state visually.

Fix: Implemented deep merge in patchOrderCache to ensure the full shipping object from the event payload is applied.

BUG-445-02

SWR Cache Staleness on Tab Switch

Problem: Switching tabs didn't always trigger a re-fetch if the data was "fresh" but incomplete due to filtering logic.

Fix: Modified patchOrderCache to return undefined when an order is removed from the current view, forcing SWR to revalidate the entire list.
