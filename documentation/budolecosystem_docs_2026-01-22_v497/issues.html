Known Issues - v497

ISSUE-497-01

Transient Duplicate Notifications

In environments where both Pusher and Socket.io are active, users may occasionally see two identical toast notifications for a single status change.

Status: Deduplication logic implemented in v497, monitoring for edge cases.

ISSUE-497-02

Stale Count on Background Tabs

If the Seller Dashboard is left open in a background tab for more than 15 minutes, the order count on the tabs may become stale until the tab is focused again.

Status: By design (battery/resource optimization), but exploring service worker persistence.
