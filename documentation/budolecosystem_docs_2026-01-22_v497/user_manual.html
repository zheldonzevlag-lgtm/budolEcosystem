Seller User Manual - v497

This guide explains how to manage returns and refunds in the updated Seller Dashboard.

1. Managing Return Requests

New Feature

Realtime Notifications: You will now receive instant pop-up notifications whenever a customer requests a return. No manual page refresh is required.

Automatic Tab Sync: Orders will automatically move to the "Returns" tab as soon as a request is submitted. The order count on the tab will update in realtime.

2. Responding to Returns

Click on an order in the Returns tab to see the customer's reason and uploaded images. You have three options:

  • Accept: Approves the return. The customer will be instructed to ship the item back.
  • Reject: Declines the request (requires a reason).
  • Dispute: Escalates the request to Budol Ecosystem support for mediation.

Escrow Protection: Once a return is requested, the funds for that order are automatically locked in escrow and cannot be withdrawn until the request is resolved.
