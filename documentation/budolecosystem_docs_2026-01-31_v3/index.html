Release Dashboard - v1.3.54

Release Overview

This release (v1.3.54) implements critical security enhancements for session management in BudolPay Mobile and fixes a state initialization bug in the web registration form. Key features include hard session timeout logout enforcement and seamless logout redirection from the lock screen UI.
