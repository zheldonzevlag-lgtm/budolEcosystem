Future Recommendations - Version 478

1. Centralized Timezone Middleware

Instead of manual Date offset calculations in each service, implement a shared middleware or utility library that automatically standardizes all outgoing timestamps to Asia/Manila.

3. Automated Compliance Scanning

Integrate a tool that scans log outputs for timestamp formats and flags any non-ISO or non-Manila timestamps as part of the CI/CD pipeline.

4. Biometric Fallback UI

Implement a "Retry Biometrics" button on the login screen for cases where the auto-refresh might fail due to OS-level delays in reporting capability changes.
