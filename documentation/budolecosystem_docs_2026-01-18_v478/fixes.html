Fixes Applied - Version 478

Fix for ISS-478-001: Asia/Manila Timezone Enforcement

Rationale: Regulatory compliance (BSP Circular 808).

Technical Explanation: Implemented a manual offset calculation (8 * 60 * 60 * 1000) for all Date objects before conversion to toISOString() in core backend services.

Files: transaction-service/index.js, settlement-service/index.js, wallet-service/index.js

Fix for ISS-478-002: Biometric Observer

Rationale: Improved User Experience and Security.

Technical Explanation: Added WidgetsBindingObserver to LoginScreen state. Triggered _checkBiometrics() within the didChangeAppLifecycleState callback when state is resumed.

Files: login_screen.dart

Fix for ISS-478-003: Code Cleanup

Rationale: Build optimization and linting compliance.

Technical Explanation: Removed unused imports in session_service.dart, add_money_screen.dart, cash_in_screen.dart, cash_out_screen.dart, and transfer_screen.dart.

Files: Multiple (see Developer Manual)
