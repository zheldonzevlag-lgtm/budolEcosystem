Executive Summary

This release provides a definitive fix for the TFLite initialization issue on Android devices. By switching from direct asset loading to a byte-buffer approach, we bypass platform-specific asset-linking issues that occasionally cause the interpreter to fail. We've also optimized the engine to use dual-thread processing for faster face scans.
