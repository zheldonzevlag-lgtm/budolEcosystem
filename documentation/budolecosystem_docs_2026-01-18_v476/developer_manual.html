Phase 8 - Dependencies & Software Rationale

  • Flutter SDK (3.x): Chosen for cross-platform performance and reactive UI.
  • Google ML Kit: Used for on-device face detection to ensure privacy and low latency.
  • TFLite (Flutter): Powers the custom face embedding model for biometric identification.
  • SecureStorage: Utilizes hardware-backed encryption (KeyStore/Keychain) for sensitive data.
