System Admin Manual

Version: v476 RELEASED

Release Date: 2026-01-18

System: BudolPay Mobile Android Build

Deployment Guide

This version marks the official production build for Android v1.3.14+110.

APK Distribution

The signed release APK is located in: \budolEcosystem\android-installer\budolPay 1.3.14.apk

Environment Requirements

  • Android SDK 21 (Lollipop) or higher.
  • Active internet connection for biometric verification and API calls.
  • Device must support Biometric API (Fingerprint/Face).

Security Configurations

  • SSL Pinning: Enabled for all production API endpoints.
  • ProGuard/R8: Enabled for code obfuscation.
  • Debuggable: Set to false in AndroidManifest.xml.

Monitoring & Maintenance

Admins should monitor the following metrics post-release:

  • Face Registration success rates via Sentry/Crashlytics.
  • API latency for authentication endpoints.
  • User session timeouts and refresh token rotations.
