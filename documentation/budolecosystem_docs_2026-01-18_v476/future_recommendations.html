Future Recommendations

REC-476-01: Multi-Face Support

Allow users to register multiple face templates (e.g., with/without glasses) for improved recognition reliability.

REC-476-02: Advanced Liveness Detection

Implement "blink detection" or "random movement" prompts during face registration to further mitigate high-quality photo spoofing.

REC-476-03: Performance Optimization

Explore TFLite GPU acceleration for faster face embedding generation on supported devices.

REC-476-04: Biometric Re-authentication

Implement biometric re-authentication for high-value transactions (e.g., transfers over PHP 10,000).
