Mitigation & Risk Management

Strategic Mitigations

Risk ID Risk Description Mitigation Strategy Owner
R-476-01 Biometric Spoofing (Face/Photo) Implemented Google ML Kit Liveness checks and TFLite embedding verification. Enforced MPIN fallback. Security Team
R-476-02 Data Breach of Biometric Templates Templates are encrypted using AES-256 via FlutterSecureStorage and stored in Hardware-backed Keystore/Keychain. Lead Developer
R-476-03 App Version Mismatch (API) Enforced strict version headers in api_service.dart. Server rejects requests from legacy builds. DevOps

Compliance Alignment

Our mitigation strategies are aligned with:

  • PCI DSS: Requirement 8 (Identify and Authenticate access).
  • NPC (Data Privacy Act): Encryption of sensitive biometric identifiers.
  • BSP: Multi-factor authentication (MFA) standards.
