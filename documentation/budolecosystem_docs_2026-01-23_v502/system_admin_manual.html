System Admin Manual

Environment Configuration

Ensure that the following environment variables are correctly set in .env for BudolShap:

  • PUSHER_APP_ID, PUSHER_KEY, PUSHER_SECRET, PUSHER_CLUSTER
  • LALAMOVE_API_KEY, LALAMOVE_API_SECRET, LALAMOVE_BASE_URL
  • DATABASE_URL (Prisma connection string)

Real-time Provider Settings

Admins can configure the real-time provider via the System Settings dashboard. Supported providers:

  • PUSHER: High-scale managed websocket service.
  • SOCKET_IO: Self-hosted websocket server.
  • POLLING: Fallback mechanism using SWR (default).

Monitoring

Check logs for [Realtime] and [Lalamove Webhook] tags to monitor synchronization health. Latency can be verified using the test_sync_latency.mjs script.
