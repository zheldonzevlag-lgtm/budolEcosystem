v481 Release

Screenshot Capability

Enabling user-driven content capture for receipts and QR sharing.

Developer Manual User Manual

Purpose

To allow users to take screenshots of the mobile application, facilitating the saving of transaction receipts and the sharing of payment QR codes.

Technical Change

Explicitly disabled FLAG_SECURE on Android via the flag_secure Flutter package in SessionService.

Risk Level

Medium

While improving UX, this allows capturing sensitive data. Mitigated by session timeouts and MPIN protection.
