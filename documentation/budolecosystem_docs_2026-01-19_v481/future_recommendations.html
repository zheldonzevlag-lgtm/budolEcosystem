Future Recommendations - v481

Selective Security

In future versions, consider re-enabling FLAG_SECURE only on specific sensitive screens (like MPIN entry) while keeping it disabled for receipts and QR codes. This would provide a better balance between security and UX.

Screenshot Detection

Implement a listener to detect when a user takes a screenshot and display a security reminder toast, similar to banking applications like BPI or GCash.
