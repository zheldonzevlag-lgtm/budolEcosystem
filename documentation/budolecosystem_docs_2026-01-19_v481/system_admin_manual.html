System Admin Manual - v481

Managing Screen Security Policies.

Policy Update: Screen Capture

As of v481, the global FLAG_SECURE policy has been relaxed to allow screenshots. This change applies to all Android devices running the budolPay mobile app.

Administrative Considerations:

  • Compliance: This change has been reviewed against internal UX guidelines. Ensure it aligns with your regional data privacy requirements (e.g., NPC for Philippines).
  • Support: Users may now submit screenshots for support tickets, which will improve troubleshooting efficiency.
  • Monitoring: No additional server-side monitoring is required for this client-side change.
