Developer Manual - v481

Disabling screen security to enable screenshot capability.

Implementation Details

The FLAG_SECURE setting on Android prevents users from taking screenshots or recording the screen. This was previously enabled for security but has been disabled to support user requirements for receipt capturing.

Key Code Reference:

Modified session_service.dart to explicitly unset the secure flag.

// In SessionService constructor if (!kIsWeb) { _disableScreenSecurity(); // Explicitly ensure screenshots are allowed } // Method implementation Future _disableScreenSecurity() async { if (kIsWeb) return; if (defaultTargetPlatform != TargetPlatform.android) return; try { await FlagSecure.unset(); if (kDebugMode) print('SessionService: Screen security (FLAG_SECURE) disabled'); } catch (e) { if (kDebugMode) print('SessionService: Failed to disable screen security: $e'); } }

Rationale

Users frequently need to screenshot QR codes for offline payments or transaction receipts for dispute resolution. Blocking this feature caused significant friction in the user experience.

Dependencies

  • flag_secure: ^2.0.2 - Used to interface with Android's WindowManager.LayoutParams.FLAG_SECURE.
