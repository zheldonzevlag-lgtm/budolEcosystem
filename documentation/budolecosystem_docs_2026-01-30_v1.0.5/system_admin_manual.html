System Admin Manual - v1.0.5

Admin Provisioning

Admin accounts must have synchronized IDs across the budolID and budolShap databases. If an admin cannot log in or see their profile correctly, run the fix-admin-id.js script.

node scripts/test_scripts/fix-admin-id.js

Order Tracking

Admins can now verify that the Order ID displayed in the budolPay mobile app matches the Job Order Number (JON) generated in the budolShap admin panel. This ensures consistency for customer support and auditing.
