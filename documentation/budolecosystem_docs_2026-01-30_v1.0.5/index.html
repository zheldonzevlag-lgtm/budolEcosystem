Release v1.0.5 - Admin ID Sync & Order Display Fix

Release Date: January 30, 2026 | Version: 1.0.5

Purpose & Rationale

This release focuses on two critical ecosystem requirements: synchronization of Admin IDs across the budolID and budolShap platforms, and ensuring accurate Order ID display in the budolPay mobile app. These changes improve system integrity and provide a better user experience for payment tracking.

Primary Changes

  • Admin ID Synchronization: Re-provisioned the admin account (reynaldomgalvez@gmail.com) with a unified ID across all services.
  • Order ID Display: Updated the wallet-service and mobile app to correctly propagate and display the budolShap-generated Order ID.
  • UI Consistency: Aligned font sizes and labels in the payment success screen for better readability.

Compliance Status

This release has been audited for compliance with:

  • PCI DSS: Audit logging for financial transactions maintained.
  • NPC (Data Privacy): User ID synchronization handled securely without exposing PII.
  • Cybersecurity Law: Secure communication between services verified.
