Viewing Order Details after Payment

Once you successfully complete a payment using the budolPay mobile app, the payment success screen will now display the Order # alongside the Reference ID.

This Order # matches your order number in the budolShap store, making it easier to track your purchases.

Troubleshooting

If you don't see the Order # on your success screen, please ensure you are scanning the latest QR codes generated by the store.
