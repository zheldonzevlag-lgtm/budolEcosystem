2. Order ID Propagation

The Wallet Service was updated to include the orderId from metadata in its response to the mobile app.

File: wallet-service/index.js

res.json({ 
    success: true, 
    transaction: {
        id: transaction.id,
        reference: transaction.referenceId,
        orderId: orderId, // Added this field
        // ...
    }
});
