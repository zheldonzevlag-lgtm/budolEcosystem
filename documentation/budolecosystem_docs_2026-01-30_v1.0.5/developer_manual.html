Admin Provisioning Script

The provisioning script ensures ID synchronization across schemas by capturing the UUID from budolid.User and injecting it into public.User.

Open File 
// scripts/test_scripts/provision-admin.js
const budolIDUser = await budolIDPrisma.user.upsert({
    where: { email },
    create: { email, role: 'ADMIN', ... }
});

const budolShapUser = await budolShapPrisma.user.upsert({
    where: { email },
    create: { 
        id: budolIDUser.id, // Syncing ID
        email, 
        isAdmin: true, 
        accountType: 'ADMIN' 
    }
});
