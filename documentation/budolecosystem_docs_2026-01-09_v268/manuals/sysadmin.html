Release Overview

This release (v1.2.29) focuses on visual branding updates for the budolPay service. No backend changes were required for this update.

Deployment Details

Artifact Location
Android APK (Release) android-installer/budolPay_1.2.29.apk
Documentation documentation/budolecosystem_docs_2026-01-09_v268/

Build Instructions

To generate the release APK, run the following command in the budolPayMobile directory:

flutter build apk --release --split-per-abi

Verification

  1. Launch the app and navigate to the marketing carousel.
  2. Verify the last slide shows "budol₱ay" and the "₱" icon.
  3. Ensure the colors match the Rose branding (#F43F5E).
