Technical Implementation | budol₱ay Branding
← Back to v268 Index

Purpose

The goal was to replace the generic wallet icon with the Philippine Peso sign (₱) and update the title "budolPay" to "budol₱ay" to better reflect the local identity of the payment service.

Code Changes

1. Slide Definition Update

In lib/screens/marketing_ads_screen.dart, the _slides list was updated to include the new title and a string-based icon.

{
  'title': 'budol₱ay',
  'description': 'Your secure digital wallet for seamless payments and transfers across the ecosystem.',
  'icon': '₱',
  'color': const Color(0xFFF43F5E), // Rose (budolPay Branding)
},

2. UI Rendering Logic

The _buildSlide method was refactored to support both IconData and String types for the icon field. This allows rendering the '₱' character as a styled Text widget when needed.

child: slide['icon'] is IconData
    ? Icon(
        slide['icon'],
        size: 100,
        color: slide['color'],
      )
    : Text(
        slide['icon'].toString(),
        style: TextStyle(
          fontSize: 100,
          fontWeight: FontWeight.bold,
          color: slide['color'],
        ),
      ),

3. Versioning

Version was incremented in pubspec.yaml and ApiService to 1.2.29.

Dependencies

  • Flutter SDK (>=3.0.0)
