SysAdmin Manual: Mobile Deployment & Rollback

Critical: Rollback Procedure

If the mobile app fails to build or results in unexpected behavior, execute these commands to restore the previous state:

Remove-Item -Recurse -Force "d:\IT Projects\budolEcosystem\budolPayMobile"
Copy-Item -Recurse "d:\IT Projects\budolEcosystem\budolPayMobile_backup_v103" "d:\IT Projects\budolEcosystem\budolPayMobile"

Build Optimization

Ensure android/gradle.properties contains the following memory limits for stable builds:

org.gradle.jvmargs=-Xmx1024m
org.gradle.daemon=true
