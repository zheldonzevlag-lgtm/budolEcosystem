Documentation v90: Memory Optimization & Stability

Published: 2026-01-02 | Version: 90

Issue Analysis

Error: Exhausted heap space during 192.168.1.24 binding.

Diagnosis: Attempting to bind the Flutter debug service (DWDS) to a LAN IP instead of localhost can significantly increase memory usage, leading to heap exhaustion on resource-constrained systems.

Resolution: Reverted to localhost binding for the web server while maintaining API_HOST as the LAN IP for cross-device communication.

Technical Fix

flutter run -d chrome --web-port=8080 --web-hostname=localhost --dart-define=API_HOST=192.168.1.24
