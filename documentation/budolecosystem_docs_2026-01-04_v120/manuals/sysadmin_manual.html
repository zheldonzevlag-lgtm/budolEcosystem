System Admin Manual: User & Database Maintenance

Infrastructure Version 120 | 2026-01-04

Purpose

To provide procedures for maintaining user credentials and handling authentication failures in the BudolEcosystem.

Seeding & Aliases

Two new/updated seeding scripts are available in budolID-0.1.0:

  • seed-tony-budolid.js: Updates the primary Stark account with correct password and required metadata.
  • seed-tony-typo.js: Creates an alias account for common domain typos (budolashap.com).

Run these scripts using node script_name.js to restore user access.

Issues & Fixes

Issue: Database seeding failed due to schema mismatch.
Fix: Updated scripts to include phoneNumber field.

Risk & Mitigation

Risk: Duplicate accounts created for the same physical user.
Mitigation: Used unique phone numbers and clear naming (e.g., "Tony (Typo)") to distinguish alias accounts in the admin panel.

Test Results

Database connectivity and user upsert operations verified successfully.
