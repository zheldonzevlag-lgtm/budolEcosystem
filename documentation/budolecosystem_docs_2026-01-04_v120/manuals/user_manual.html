User Manual: Login Support & Guidance

User Experience Version 120 | 2026-01-04

Purpose

To help users understand login error messages and correct their credential entry.

Improved Error Messages

The budolPay app now provides clearer feedback when login fails:

  • Invalid email or password: Appears when the credentials do not match our records.
  • Invalid application API key: Appears if there is a system configuration issue.

Login Tips

Ensure your email is entered correctly. For example, use @budolshap.com instead of @budolashap.com. Check that your password is at least 9 characters long if you are using your default account.

Issues & Fixes

Issue: Generic "Auth server error" message was confusing users.
Fix: Implemented specific error reporting for better self-troubleshooting.
