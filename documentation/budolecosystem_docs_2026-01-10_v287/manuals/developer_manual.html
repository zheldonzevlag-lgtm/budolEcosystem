Admin Provisioning Script

The script set-admin.js was created in the root directory to handle multi-database updates.

Key Code Logic

const prismaPay = new PrismaPay({
    datasources: { db: { url: process.env.DATABASE_URL_PAY } }
});
const prismaShap = new PrismaShap({
    datasources: { db: { url: process.env.DATABASE_URL_SHAP } }
});

// Syncing roles across platforms
await prismaPay.user.update({ ... });
await prismaShap.user.update({ ... });

Dependencies

  • @prisma/client (budolpay)
  • @prisma/client (budolshap)

Test Results

Verified that the set-admin.js script exits with code 0 and correctly reports the updated user fields in the console output.
