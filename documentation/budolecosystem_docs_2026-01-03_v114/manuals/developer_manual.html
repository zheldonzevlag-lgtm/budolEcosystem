Developer Manual - Version 114

1. Asset Configuration

The pubspec.yaml was updated to include flutter_native_splash and configure both the launcher icons and splash screen:

flutter_launcher_icons:
  android: true
  ios: true
  image_path: "assets/app_icon.png"

flutter_native_splash:
  color: "#0F172A"
  image: "assets/app_icon.png"
  android: true
  ios: true
  web: true

2. Generation Commands

The following commands were used to regenerate the native assets:

  • flutter pub get
  • flutter pub run flutter_launcher_icons:main
  • flutter pub run flutter_native_splash:create

3. Release Process

The final build continues to use the --no-tree-shake-icons flag due to the environment-specific AOT snapshotter issue:

flutter build apk --release --no-tree-shake-icons
