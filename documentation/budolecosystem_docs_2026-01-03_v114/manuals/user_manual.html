User Manual - Version 114

Visual Identification

The official budolPay 1.0 app now features the dark-themed Peso icon. When you install the app, look for the dark square icon with the pink Peso symbol.

Launch Experience

When you open the app, you will see a professional splash screen with the budolPay logo on a dark background before the main screen loads.

Verification

Ensure you are using the latest version by checking the icon on your home screen. If you still see the old white icon, please uninstall and reinstall using the latest budolPay 1.0.apk.
