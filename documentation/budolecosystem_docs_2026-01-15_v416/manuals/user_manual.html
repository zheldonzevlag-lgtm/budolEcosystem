User Manual: Enhanced Modal Interactions

1. Responsive MPIN Dialog v1.3.0

Keyboard-Friendly Modals

We've improved the way the app handles the system keyboard when entering your MPIN. The verification dialog now automatically adjusts its position so that the "Verify" and "Cancel" buttons are always visible and easy to tap, even when the keyboard is open.

2. Smooth Scrolling

Improved Accessibility

If you're using a device with a smaller screen, you can now scroll within security dialogs. This ensures that all information and action buttons are accessible, regardless of your screen size or keyboard height.

3. Version Check

Ensure your app shows v1.3.0 on the splash screen to experience these accessibility improvements.
