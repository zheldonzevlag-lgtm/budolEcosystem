Future Recommendations - v416

  • Global Modal Wrapper: Implement a reusable global wrapper or theme for all dialogs that automatically handles keyboard insets and scrollability to ensure consistency across the app.
  • Haptic Feedback: Add haptic feedback when the MPIN modal appears or when a verification is successful/failed to improve the tactile user experience.
  • Adaptive Layouts: Further explore adaptive layouts that can dynamically change based on orientation and keyboard states across a wider range of screen aspect ratios.
