Mitigation Strategies - v416

  • Keyboard Responsiveness: Used MediaQuery.of(context).viewInsets.bottom to dynamically adjust the modal's bottom padding when the keyboard is active, ensuring input fields and buttons remain above the keyboard.
  • Scrollable Dialog Content: Wrapped the dialog body in a SingleChildScrollView to handle cases where the keyboard height plus dialog content exceeds the available screen height.
  • User Experience Optimization: Focused on ensuring critical secure actions (like MPIN verification) are always accessible, preventing user frustration and potential task abandonment.
