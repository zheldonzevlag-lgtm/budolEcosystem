Issues Log

Issue I-SH-001: Incorrect Delivery Status Display

Description: When a user selects "Lalamove" as their courier, the order history page still displays "Standard Delivery".

Impact: High - User confusion and lack of trust in the system.

Status: Resolved in v2

Issue I-SH-002: Hardcoded Shipping Cost

Description: The backend ordersService.js was hardcoding shipping cost to 50 regardless of the provider selected.

Impact: Medium - Inaccurate financial records and potential loss for the platform.

Status: Resolved in v2

Issue I-SH-003: Test Script Failure (Prisma Client)

Description: Initial test scripts failed because they attempted to initialize a new PrismaClient instead of using the singleton provided by lib/prisma.js.

Impact: Low - Delayed verification process.

Status: Resolved during development
