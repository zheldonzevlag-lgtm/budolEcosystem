System Admin Manual

Database Integrity

Admins should ensure that the Order model in the database correctly handles the shipping JSON field. This field stores metadata from various providers like Lalamove and Bud Express.

Monitoring Orders

When reviewing orders in the Admin Dashboard, the shipping provider should now reflect accurately. If an order shows "Standard Delivery" for a Lalamove shipment, it indicates a failure in the checkout data submission flow.

Environment Configuration

Ensure that Lalamove credentials (CLIENT_ID, CLIENT_SECRET) are correctly set in the environment variables to allow for real-time quoting and booking, which feeds the shipping data persisted in the orders.
