User Manual - v434

What's New

1. Correct Payment Owner Display

When making a payment via QR code on budolShap, you will now see the correct owner name Jon Galvez in the payment confirmation screen. This ensures you are paying the correct merchant.

2. Improved Web Experience

The budolPay web application now correctly displays marketing advertisements after the initial splash screen. Previously, some users might have seen a black screen after the loading process.

How to Use

  1. Open the budolPay web application.
  2. Wait for the splash screen to finish loading.
  3. View the marketing ads for new services like budolLoan, budolShap, and budolExpress.
  4. Click "GET STARTED" or "Skip" to proceed to the login screen.
