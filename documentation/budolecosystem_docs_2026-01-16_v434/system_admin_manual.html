System Admin Manual - v434

Deployment Instructions

1. budolShap: Restart the budolShap service to apply changes in the payment adapter.

2. budolPay Mobile (Web): Rebuild the Flutter web app and deploy the updated files to the web server.

flutter build web --release

Verification

Ensure that the web server correctly serves the updated main.dart.js and that the splash screen JS removal logic is functioning.
