BudolPay Admin: User Guide

Welcome to the updated BudolPay Admin Dashboard. Version v339 focuses on stability, real-time insights, and secure access control.

1. Accessing the Dashboard

The Admin Dashboard is now accessible via the standard LAN IP 192.168.1.17:3000. This ensures consistent connectivity across all devices on the network, including mobile devices running the BudolPay app.

Pro Tip: Bookmark the new LAN address for quick access. The dashboard is optimized for both desktop and tablet views.

2. Key New Features

Real-time Monitoring

Track transactions as they happen. The dashboard now features a live transaction feed powered by Socket.io, providing instant visibility into ecosystem financial flows.

RBAC Integration

Role-Based Access Control ensures you only see what you need. Permissions are now granular, distinguishing between system administrators and staff members.

Environment Control

System administrators can now manage ecosystem-wide environment variables directly from the settings panel, simplifying configuration changes.

Optimized Performance

Under-the-hood improvements to the Next.js build process mean faster load times and smoother interactions across all dashboard pages.

3. Troubleshooting Connectivity

If you encounter a "Connection Refused" error, ensure that:

  • Your device is connected to the same LAN as the server.
  • The server at 192.168.1.17 is active and the admin service is running.
  • Your browser is not forcing HTTPS if the local server is running on HTTP.
