Risk Register: v339 Restoration & RBAC

Risk ID Category Description Impact Probability Severity
R-LAN-01 Network Reliance on static LAN IP (192.168.1.17). If DHCP changes the host IP, connectivity breaks. Total ecosystem communication failure across devices. High High
R-RBAC-01 Security RBAC Permission Mismatch: Token roles not aligning with defined permissions in middleware. Unauthorized access or accidental lockout of admin functions. Medium High
R-BLD-01 Stability Build-time Prisma dependency: Next.js build failing if DATABASE_URL is not accessible during CI/CD. Inability to deploy updates to the admin dashboard. Medium Medium
R-SEC-01 Security JWT_SECRET Synchronization: Secret mismatch between auth-service and other microservices. Authentication failures for valid tokens across services. Low High
R-COMP-01 Compliance Lack of audit logging for RBAC permission changes. Potential violation of PCI DSS/BSP logging requirements. Medium Medium
