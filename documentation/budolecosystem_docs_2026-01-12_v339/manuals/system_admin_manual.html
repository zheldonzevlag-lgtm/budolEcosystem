System Admin Manual: Port Isolation & Biometric Auth

Infrastructure Port Configuration

In v338, we have implemented Port Isolation to prevent service conflicts. Administrators must ensure these ports are open in the firewall and properly mapped in container configurations.

Service Internal Port Env Variable
Auth Service 8001 AUTH_PORT
Wallet Service 8002 WALLET_PORT
Transaction Service 8003 TRANSACTION_PORT
API Gateway 8080 GATEWAY_PORT

Biometric Authentication Management

Facial Recognition data is stored as encrypted embeddings in the User table. Admins should monitor the faceTemplate column for successful enrollments.

COMPLIANCE: In accordance with NPC and GDPR, raw images are NEVER stored. Only mathematical embeddings (one-way hashes) are retained in the database.

JWT Security Synchronization

All services must share the same JWT_SECRET for token verification to succeed. When rotating secrets, all .env files must be updated simultaneously followed by a service restart.
