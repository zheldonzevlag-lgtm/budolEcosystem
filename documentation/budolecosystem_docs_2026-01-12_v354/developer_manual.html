Marketing Ads Navigation

The marketing ads are designed to show once for unauthenticated users. The logic is handled in the splash screen using the hasSeenAds flag from ApiService.

Code Reference: SplashScreen Navigation

File: splash_screen.dart

if (isAuthenticated) { Navigator.pushReplacementNamed(context, Routes.home); } else if (!hasSeenAds) { Navigator.pushReplacementNamed(context, Routes.marketing); } else { Navigator.pushReplacementNamed(context, Routes.login); }

Web Stall Prevention

On web, PackageInfo.fromPlatform() can occasionally hang depending on the environment. We have introduced a 2-second timeout to ensure the app continues even if version info fails to load.

Code Reference: ApiService Init

File: api_service.dart

final packageInfo = await PackageInfo.fromPlatform().timeout(const Duration(seconds: 2));
