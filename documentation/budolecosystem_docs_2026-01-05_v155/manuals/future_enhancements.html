Consolidated Future Enhancements

1. Financial Compliance & Features

  • InstaPay/PESONet Integration: Enable real-time inter-bank transfers following Philippine standards.
  • Fee Engine: Implement a dynamic fee calculation service for different transaction types and providers.
  • QR Ph Support: Implement QR code generation and scanning compliant with BSP standards.

2. Scalability & Infrastructure

  • Message Broker (Redis/RabbitMQ): Replace direct HTTP notifications with a publish/subscribe model for better reliability.
  • Distributed Ledger: Move from a single PostgreSQL ledger table to a more scalable event-sourcing architecture.
  • Rate Limiting: Implement stricter rate limiting at the Gateway level to prevent abuse.

3. User Experience (Mobile)

  • Biometric Authentication: Add FaceID/Fingerprint support for transaction confirmation.
  • Contact Sync: Allow users to find recipients by syncing their phone contacts.
  • Push Notifications: Integrate Firebase Cloud Messaging (FCM) for off-app notifications.

4. Security

  • Transaction Signing: Implement digital signatures for every transaction to ensure non-repudiation.
  • Fraud Detection: Add a basic ML-based service to flag suspicious transaction patterns.

Conclusion

The current implementation provides a solid foundation. Focusing on compliance and scalability will be key for the next phase of growth.
