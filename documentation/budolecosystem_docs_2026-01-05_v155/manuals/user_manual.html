User Manual - Financial Operations

Introduction

This update enables core financial features: Cash In and Send Money. These features are now fully integrated across the Mobile and Admin applications.

1. Cash In (Mobile)

Step: Navigate to the "Cash In" screen from the home dashboard.

Action: Select a provider (e.g., GCash, Maya), enter the amount, and confirm.

Result: Your wallet balance updates immediately, and the transaction is recorded in the history.

2. Send Money (P2P Transfer)

Step: Navigate to the "Send Money" screen.

Action: Enter the recipient's email address and the amount.

Result: Funds are instantly transferred. Both the sender and receiver receive real-time notifications and balance updates.

3. Real-time Monitoring (Admin)

Admin users can now see all transactions appearing in the Transactions tab in real-time without refreshing the page. This allows for immediate oversight of all Cash In and Transfer activities across the ecosystem.

Dependencies

Requires an active internet connection and a verified KYC status for large transactions.
