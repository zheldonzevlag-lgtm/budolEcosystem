User Manual - v106

1. QR Code Payment Flow

The payment flow has been optimized for reliability. Users should experience the following:

  1. Scan: Open the budolPay app and scan the QR code on the checkout page.
  2. Processing: The app will validate the QR payload and communicate with the payment gateway.
  3. Redirection: Upon successful validation, the app will automatically redirect to the transaction status page.

2. Known Issues & Fixes

Resolved: Fixed an issue where the app would show a white background after scanning a QR code.

Resolved: Fixed the checkout page getting stuck on the QR code after a successful scan.

3. Installation Instructions

Once the build process completes, the app-debug.apk will be available in the build/app/outputs/flutter-apk/ directory. Install this file on your Android device to test the latest payment features.
