System Admin Manual

Transaction Monitoring

The budolPay Admin panel now displays full Reference IDs and Internal Transaction IDs. This change allows administrators to perform precise searches and reconciliation without needing to click into individual transaction details for the full ID.

User Management

User IDs are now fully visible in the User Verification module. This is critical for matching users across different sub-systems of the Budol Ecosystem.

Deployment

The Android application has been updated to version 1.3.35+132. The new APK is available in the \budolEcosystem\android-installer directory. Administrators should ensure that users are prompted to update to this version for improved payment status visibility.
