v1.3.35+132

Release Documentation - 2026-01-25

This release focuses on improving transaction traceability and UI consistency across the Budol Ecosystem, specifically addressing ID display issues and delivering a new Android build.

Task Progress

Fix Reference ID and Internal ID truncation in budolPay Admin Transactions page.
Fix User ID truncation in budolPay Admin Users page.
Ensure full display of Reference ID in budolPay Mobile app (Flutter).
Standardize Order ID and Reference ID display order in QR modals.
Build and distribute budolPay Mobile Android APK (v1.3.35+132).

Purpose & Rationale

Ensuring that critical identifiers like Reference IDs and Transaction IDs are fully visible is essential for auditability, customer support, and compliance with BSP (Bangko Sentral ng Pilipinas) regulations regarding transaction transparency.
