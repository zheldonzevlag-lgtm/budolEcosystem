Issues & Fixes

Known Issues

Issue: Transaction IDs in Admin Transactions page were truncated with "...", making reconciliation difficult.
Issue: User IDs in Admin Users page were truncated, preventing accurate identification of users across systems.
Issue: Reference IDs in Flutter payment status screen could be cut off on smaller devices.

Fixes Applied

Fix: Removed .substring(0, 8)... from transactions/page.tsx and users/page.tsx in budolPay Admin.
Fix: Added softWrap: true to the Reference ID display in payment_status_screen.dart (Flutter).
Fix: Standardized ID ordering in QRCodeModal.jsx (budolShap) to show Order ID before Reference ID.
