User Manual

Payment Status Improvements

When completing a payment via the budolPay mobile app, the Reference ID is now fully visible on the success screen. This ID can be used for any customer support inquiries.

QR Payment Flow

The QR code modal in budolShap has been updated to show the Order ID more prominently before the Reference ID, making it easier to verify which order you are paying for.

How to Update

1. Open your Android device.
2. Download the latest budolPay APK (v1.3.35+132).
3. Install the update to ensure all new features and fixes are active.
