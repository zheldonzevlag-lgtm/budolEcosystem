System Admin Manual - v244 Deployment

APK Distribution

The latest stable build of the budolPay mobile application is now available for internal distribution.

  • Build Version: 1.2.11
  • Artifact Path: \android-installer\budolpay_v1.2.11.apk
  • Build Environment: Flutter 3.27.1 / Dart 3.6.0

Deployment Procedure

To deploy the latest version to user devices:

# 1. Transfer the APK to the distribution server
# 2. Update the internal app repository link to point to:
/android-installer/budolpay_v1.2.11.apk

# 3. Notify users to upgrade to v1.2.11 for the new Cancel button feature.

Health Check

After deployment, ensure that the mobile application can connect to the API Gateway at the configured host (Default: 192.168.1.24).
