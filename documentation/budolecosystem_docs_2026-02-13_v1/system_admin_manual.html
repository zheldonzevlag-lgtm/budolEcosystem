System Admin Manual

Database Migration

Critical: After deploying this version, you must sync the database schema.

Run the following command in the project root:

npx prisma db push

Monitoring Variations

System administrators can monitor product variations through the Product table in Prisma Studio:

npx prisma studio

Ensure that the variation_matrix JSON field contains valid SKU data for all multi-variation products.

Troubleshooting

  • Empty Matrix: If a product has tiers but an empty matrix, the "Select Options" button will be disabled. Re-save the product in the seller dashboard to regenerate.
  • Cart Key Mismatch: If users report items not updating in cart, verify that variationId is being passed correctly in the Redux actions.
