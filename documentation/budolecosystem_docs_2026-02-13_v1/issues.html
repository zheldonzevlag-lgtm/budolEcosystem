Known Issues

Prisma Client Sync: The local development environment encountered an EPERM error when updating the Prisma query engine. This prevents the use of the new fields in the local @prisma/client until a successful npx prisma generate is run.
Image Uploads in Matrix: The current version of VariationMatrixManager generates SKUs and prices but does not yet support per-variation image uploads. This is scheduled for the next release.
