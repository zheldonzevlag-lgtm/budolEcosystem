System Administrator Manual

Service Monitoring

Admins should monitor the following services for payment-related errors:

  • Payment Gateway Service (Port 8004): Look for 400 errors (Invalid Amount) or logs regarding duplicate transaction attempts.
  • Wallet Service (Port 8002): Look for "Amount mismatch" logs in the /process-qr endpoint.

Security Configuration

Ensure NODE_ENV is set to production in production environments to disable the development-only auth bypass for QR processing.
