Payment Experience

Users will notice a more stable checkout experience:

  • Error Messages: If an invalid amount is entered or a technical glitch occurs during payment initiation, a clear error message will be displayed.
  • No Double Charging: The system now prevents accidental double-payments if the "Pay" button is clicked multiple times before the first request finishes.
  • Secure QR Pay: Scanning a merchant QR code now performs a background security check to ensure the amount you see on screen is exactly what is being deducted from your wallet.
