Payment Security & Integrity Update

Version: v1.8.0 Date: 2026-02-10 Author: Budol Ecosystem Orchestrator Released

Purpose

This release addresses critical security vulnerabilities in the payment flow, specifically focusing on amount validation (NaN and negative values) and transaction integrity (duplicate orders and amount mismatches during QR processing). These updates ensure compliance with BSP Circular 808 and PCI DSS v4.0 standards.

Phase 1: Architecture & Data Contracts
Phase 2: Backend NaN & Duplicate Fixes
Phase 4: Integration & Security Audit
Phase 5: End-to-End Security Testing
