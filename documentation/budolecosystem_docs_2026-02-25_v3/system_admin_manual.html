Deployment Notes

This update is purely frontend-focused. No database migrations (prisma db push) are required for this release.

Environment Variables

Existing variables like NEXT_PUBLIC_CURRENCY_SYMBOL are used. Ensure these are set in your production environment (Vercel/AWS).

Performance Monitoring

While Framer Motion is highly optimized, keep an eye on "Cumulative Layout Shift" (CLS) in Google Search Console. The new Skeleton loaders should theoretically lower your CLS scores, improving SEO.

Icons Support

The system now supports Shirt and Gamepad2 Lucide icon names. If an admin wants to change a category icon manually via the database, they can use these string values in the icon field.
