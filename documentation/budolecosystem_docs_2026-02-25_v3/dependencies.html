Software Required

  • Node.js v18+
  • Next.js 14+
  • Framer Motion 11+ (Primary animation engine)
  • Lucide React 0.300+ (Icon library)

Why These Selections?

  • Framer Motion: Chosen for its declarative API and powerful layout and AnimatePresence components which are difficult to replicate with vanilla CSS in a React environment.
  • Lucide React: Provides a professional, clean SVG-based icon set that is highly customizable and tree-shakable.
