Developer Manual budolPay budolShap

Documentation Scope: This manual covers the budolPay monorepo (fintech core) and its integration with budolShap (e-commerce platform).

System Breakdown

budolPay Core

Handles the financial infrastructure: Wallets, Transactions, and Accounting. Located in /budolpay-0.1.0.

budolShap Integration

The e-commerce engine that consumes budolPay services for checkout and seller payouts. Located in /budolshap-0.1.0.

Technical Details & Shared Dependencies

  • Shared Database: Both apps connect to PostgreSQL via the @budolpay/database package.
  • SSO Mechanism: budolPay acts as the Identity Provider (IdP) for budolShap.
  • Shared Types: Transaction types and User roles are synchronized across both ecosystems.

Developer Recommendations

  • Cross-App Testing: When updating the database schema in budolPay, always run npm install in budolShap to ensure types are updated.
  • Environment Sync: Maintain consistent DATABASE_URL and JWT_SECRET across both directories.
