System Administrator Manual

Ecosystem Overview

Admins manage two primary environments that share a database cluster:

  • budolPay Admin Portal: Financial monitoring, dispute resolution, and system settings.
  • budolShap Store Management: Product moderation, order tracking, and payouts.

Critical Infrastructure

ComponentTarget AppResponsibility
PostgreSQL (budolpay_db)BothSchema sync, backups, and user data.
Auth ServicebudolPayEcosystem-wide single sign-on.
Payment GatewayBothProcessing external payments (PayMongo/Xendit).
Critical Admin Note: The budolpay_db is the single point of truth for both apps. Corruption in this DB will take down both the payment system and the marketplace.

Maintenance Steps

  1. Database Sync: Run npx prisma db push in /budolpay-0.1.0/packages/database.
  2. App Linking: Ensure budolShap API keys are registered in budolPay System Settings via the seed script.
