v22 User Manual: Accessing the Vercel Mirror

For End Users

The Budol Ecosystem is now accessible via Vercel for the main E-commerce storefront. Follow these steps to access and test the mirrored environment.

1 Visit the Vercel App: Navigate to https://budolshap.vercel.app.
2 Login via SSO: Click the "Login" button. You will be redirected to the production SSO endpoint budolid.duckdns.org. Use your existing credentials.
3 Verify Transactions: Any transactions made on the Vercel mirror will be processed by the budolPay gateway mirror on DuckDNS and recorded in the 13-service shared database.
Note: Performance may vary due to serverless cold starts compared to the local persistent dev environment.
