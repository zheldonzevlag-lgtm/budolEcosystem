Bug: Biometric Login Disabled after Enablement

Status: RESOLVED

Description: User successfully enabled biometric and face recognition login in Settings. However, upon returning to the login screen, tapping the biometric login icon resulted in an error message stating "Biometric login is disabled."

Root Cause: The LoginScreen component was checking the biometric enabled status only once during initState(). It did not refresh this status when the user actually tried to use the feature, leading to a state mismatch if the status changed in another screen (Settings).
