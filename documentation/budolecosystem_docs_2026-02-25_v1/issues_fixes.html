Issues & Fixes

Issue Disappearing Icons on Search

During testing, the CategorySelector in the product wizard lost icons when searching for categories.

Fix Updated the search filtering logic in CategorySelector.jsx to re-resolve the icon component using getCategoryLucideIcon for matched search items.

Issue Unsafe Import Aliases

Detected Wrench as AutoWrench which caused naming collisions with standard library imports.

Fix Refactored CategoryIcons.jsx to use precise named imports without confusing aliases.
