System Admin Manual: Category Management

Admin Interface Changes

The Category Management section (/admin/categories) has been updated to provide a visual preview of the Lucide icons associated with each category.

Important: Icons are now resolved automatically based on the category slug or name. If you create a new category, ensure the name contains relevant keywords (e.g., "Electronics", "Fashions") to trigger the correct icon matching.

Configuration

While the system uses intelligent slug-matching, Admins can still manually override icons in the Category Editor if the database model supports custom icon strings. Currently, the system prioritizes the automated mapping for stability.

Maintenance

No special maintenance is required. The icons are bundled with the application via the lucide-react dependency. If an icon appears as a "Package" (fallback), it means the category slug does not match any known professional icon keys.
