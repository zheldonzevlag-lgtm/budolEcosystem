Future Enhancements & Recommendations

1. Database-Driven Icon Customization

Recommendation: Extend the Category model in Prisma with an iconName field (String). This would allow Admins to choose a specific Lucide icon from the UI rather than relying on automated slug matching.

2. Dynamic Icon Colors

Recommendation: Currently, icon colors are cycled from a static list based on index. Implementing a color picker in the Admin category dashboard would allow for better brand-specific categorization (e.g., Red for Sales, Blue for Tech).

3. Animated Icon Transitions

Recommendation: Use framer-motion to add subtle entry/hover animations to category icons on the homepage and sidebar to increase user engagement and "feel" of premium quality.

4. L3-Specific Icons

Recommendation: Create an even more granular mapping for Level-3 categories (Leaf categories). Currently, many share the same icon as their Level-1 parent.
