Executive Summary

This release targets the "null embedding" issue by adding several layers of robustness to the image processing pipeline. We now explicitly convert images to RGB8 format, add padding to face crops to prevent edge artifacts, and provide a fallback that processes the entire image if the specialized crop fails. Error reporting has been enhanced to capture specific TFLite or Image library exceptions.
