Biometric Stability Update II

This version further improves the face registration process:

  • Enhanced Image Processing: The app now handles different camera image formats more effectively.
  • Auto-Correction: If the initial face scan fails, the app automatically tries an alternative scanning method to ensure success.
  • Detailed Error Messages: If a scan fails, you will now see more specific information to help resolve the issue.
