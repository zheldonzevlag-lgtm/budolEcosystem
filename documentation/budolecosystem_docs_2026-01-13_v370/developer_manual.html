Enhanced AI Preprocessing

1. Format Normalization

In face_embedding_service.dart, we now ensure the image is in RGB8 format before passing it to the interpreter.

if (originalImage.format != img.Format.uint8 || originalImage.numChannels != 3) {
  originalImage = originalImage.convert(format: img.Format.uint8, numChannels: 3);
}

2. Robust Cropping

Added 10% padding to the face bounding box and clamped values to prevent out-of-bounds exceptions when the face is near the edge of the frame.

3. Fallback Mechanism

If the cropped inference fails, settings_screen.dart now attempts a full-image inference as a last resort.

try {
  template = await faceService.generateEmbedding(file, faceArea: face.boundingBox);
} catch (e) {
  template = await faceService.generateEmbedding(file); // Fallback
}
