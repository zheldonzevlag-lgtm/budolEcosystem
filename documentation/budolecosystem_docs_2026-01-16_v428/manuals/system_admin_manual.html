System Admin Manual - v428

Overview

This version updates the frontend UI components to correctly synchronize payment status for BudolPay transactions. No server-side or database migrations are required.

Verification Steps

  1. Navigate to an order paid via BudolPay.
  2. Ensure the "Order Items" tab shows "Paid".
  3. Switch to the "Tracking History" tab.
  4. Verify the "Order Progress" tracker shows the payment step as completed (green).
  5. Verify the "Tracking Timeline" shows the "Paid" event with a correct description.
