System Admin Manual - v69

Release APK Installation

A new release APK has been built with updated connectivity settings for physical devices.

  • APK Location: D:\IT Projects\budolEcosystem\budolPayMobile\build\app\outputs\flutter-apk\app-release.apk
  • Command to Install (via ADB): 
    & "d:\android\platform-tools\adb.exe" install -r "D:\IT Projects\budolEcosystem\budolPayMobile\build\app\outputs\flutter-apk\app-release.apk"
Important: Ensure your phone is connected via USB and USB Debugging is enabled. The device ID used for testing is RR8M60AYXKF (Samsung A50).

Network Configuration

The app is configured to connect to the host machine at 192.168.1.24. If your host IP changes, you must update lib/services/api_service.dart and rebuild the APK.

Current Ports used by the ecosystem:

  • budolID (Auth): Port 8000
  • budolPay Gateway: Port 8004

Gradle Build Optimizations

To avoid "Out of Memory" or "Paging file" errors during build, the android/gradle.properties has been configured as follows:

org.gradle.jvmargs=-Xmx2048m -XX:MaxMetaspaceSize=512m
org.gradle.daemon=false
android.useAndroidX=true
android.enableJetifier=true
