Ecosystem Port Mapping

The following services must be active for the universal payment modal to function correctly:

  • budolID (SSO): Port 8000
  • budolPay Gateway: Port 8004
  • budolShap (App): Port 3001

Environment Variables

Ensure BUDOLPAY_GATEWAY_URL is correctly set in budolshap-0.1.0/.env:

BUDOLPAY_GATEWAY_URL="http://localhost:8004"

Verification

Run the following command to verify service availability:

netstat -ano | findstr :8004
