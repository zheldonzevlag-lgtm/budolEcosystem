User Manual - v281

1. Accessing the Ecosystem on LAN

You can now access the budolEcosystem web apps from any device (laptop, tablet, phone) connected to the same Wi-Fi.

Web Access URLs:

  • budolShap (Main App): http://[HOST_IP]:3001
  • budolID (SSO): http://[HOST_IP]:8000

Replace [HOST_IP] with the IP address shown in the terminal when starting the ecosystem.

2. Mobile App Update v1.2.36

The new version of the budolPay mobile app is now available for installation.

What's New:

  • LAN Support: The app is now optimized to connect to the ecosystem server over local Wi-Fi.
  • Splash Screen Versioning: The correct version (1.2.36) is now displayed on the splash screen for easier verification.
  • Improved Stability: Fixed several internal errors that caused the app to crash during startup.

Installation:

The installer can be found at: \budolEcosystem\android-installer\budolPayMobile_1.2.36.apk

3. Troubleshooting Connectivity

If you cannot reach the services from another device:

  1. Check that both devices are on the same Wi-Fi network.
  2. Verify the Host IP using the sync-network.js tool.
  3. Ensure the host computer is not blocking the connection via Firewall.
