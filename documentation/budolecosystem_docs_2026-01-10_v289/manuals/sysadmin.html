SysAdmin Manual Update

APK Management & Distribution Procedures

Infrastructure Changes

1. APK Storage Location

The latest production APK is stored in the centralized installer directory:

D:\IT Projects\budolEcosystem\android-installer\budolPay_1.2.39.apk

2. Versioning Standard

All mobile builds must follow the budolPay_X.X.X.apk naming convention. The version number must match the pubspec.yaml version exactly.

Deployment Procedure

  1. Cleanup: Run flutter clean to ensure no cached artifacts interfere with the build.
  2. Build: Execute the release build command with the correct API_HOST injection.
  3. Distribution: Copy the generated app-release.apk to the android-installer directory and rename it.

Risk & Mitigation

Risk Mitigation
IP Address Mismatch Hardcoded fallback to 192.168.1.24 in ApiService.
Version Drift Mandatory update of both pubspec and ApiService.
