Lazada-Style Authentication & KYC Strategy

This version focuses on aligning the Budol Ecosystem with global e-commerce standards (Lazada/Shopee) while maintaining strict compliance with Philippine regulations (BSP, NPC, AMLA).

Executive Summary

Following the Lazada Blueprint, Budol Ecosystem will implement a tiered verification system:

  • Level 1 (Unverified): Login via Phone/Email. Basic shopping/cash-in features enabled.
  • Level 2 (Fully Verified): Government ID + Liveness Check (Selfie). Unlocks full wallet features (Withdrawal, Transfers, Loans).

This implementation ensures compliance with BSP Circular 1022 and the Philippine Identification System Act (PhilSys).
