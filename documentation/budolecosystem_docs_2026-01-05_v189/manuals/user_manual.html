User Manual - v189

Tiered KYC System Active

The Tiered KYC system is now fully operational. Users are automatically assigned tiers based on their verification progress.

Tier Definitions

  • BASIC Basic: Default tier for all new users. Limited transaction capabilities.
  • SEMI_VERIFIED Semi-Verified: Automatically assigned when a user uploads their first verification document.
  • FULLY_VERIFIED Fully Verified: Assigned by an administrator after successful review of all documents.

What's New for Admins

In the User Management screen, you will now see a "KYC Tier" column. This allows you to quickly identify the verification status of any user at a glance.
