Release Date: 2026-01-19

Purpose & Rationale

This release (v488) focuses on standardizing the payment experience across the Budol Ecosystem. Specifically, it implements a universal in-page modal for QR-based payments, eliminating unnecessary redirections and providing a consistent UI for both "Place Order" and "Pay Now" actions.
