Configuration Management

Admin must ensure that the PAYMENT_GATEWAY_URL is correctly set in the .env of budolshap to point to the updated budolpay service.

Parameter Value Description
BUDOLPAY_INTERNAL_QR ENABLED Allows generation of internal QR codes for budolPay transactions.
MODAL_TIMEOUT_MS 600000 Payment modal timeout (10 minutes).

Monitoring & Logs

Check payment-gateway-service logs for "QR_GENERATED" events to verify that raw QR strings are being correctly sent to the frontend. 

tail -f logs/payment-gateway.log | grep "QR_GENERATED"

Compliance Audit

Verification of PCI DSS compliance is automated via scripts/test_scripts/budolPay/test-forensic-audit-v406.js. Admins should run this weekly.
