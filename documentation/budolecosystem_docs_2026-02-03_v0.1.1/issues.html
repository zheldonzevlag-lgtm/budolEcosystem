Known Issues & Technical Debt

ISS-011-01: Socket.io Room Leak

Clients are not always explicitly leaving rooms on component unmount, which may lead to memory accumulation in the API Gateway over long periods.

ISS-011-02: Pusher Latency in Free Tier

When using the Pusher Free tier, there is a noticeable 1-2s delay in event propagation compared to local Socket.io.
