Quick Summary

This release addresses a critical failure in the SSO login flow where the budolID service was returning a 500 Internal Server Error. The root cause was identified as a database schema mismatch in the `budolid` PostgreSQL database. We successfully synchronized the schema and verified the fix through end-to-end testing.
