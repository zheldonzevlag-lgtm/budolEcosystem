User Manual

Guide for ecosystem users on authentication and security.

SSO Login Process

1

Access Login

Navigate to any Budol application (Shap, Pay, or Express) and click "Login with budolID".

2

Enter Credentials

Provide your registered email address and secure password on the budolID portal.

3

Secure Redirect

Upon successful authentication, you will be securely redirected back to your target application with an active session.

Troubleshooting Login Issues

💡 Internal Server Error?

If you encounter an "Internal Server Error" message during login, please wait a moment and refresh. This usually indicates a temporary synchronization issue that our automated systems resolve within seconds. If the issue persists, contact support.
