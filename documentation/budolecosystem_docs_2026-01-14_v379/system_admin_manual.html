System Admin Manual – v379

Installation Requirements

Ensure that the mobile device has at least 50MB of free space for the FaceNet TFLite model and temporary files. The app requires READ_EXTERNAL_STORAGE and WRITE_EXTERNAL_STORAGE (or appropriate scoped storage permissions) to handle model loading on certain Android versions.

Monitoring

Admins should monitor logs for "AI Inference failed to initialize" errors. High frequency of these errors may indicate corrupted app installations or incompatible hardware (missing NPU/GPU support for specific TFLite kernels).
