Fixes and Resolutions v467

Issue: Recurring Blank Screen on Web

Symptom: App hangs after splash screen with "Flutter engine still initializing" error in console.

Fix: Implemented 15s fallback timer in index.html and explicit signaling from SplashScreen.dart. Removed explicit renderer: "html" to prevent engine selection conflicts.

Issue: Zero-Size Layout Constraints

Symptom: Marketing Ads screen occasionally renders as a blank white/black screen due to parent constraints being zero.

Fix: Added Positioned.fill to SessionOverlay and zero-size guards/logging in MarketingAdsScreen to prevent silent rendering failures.

Issue: JS Interop Crashes

Symptom: Errors when calling JS methods that haven't loaded yet.

Fix: Updated js_helper_web.dart to use hasProperty checks before invocation.
