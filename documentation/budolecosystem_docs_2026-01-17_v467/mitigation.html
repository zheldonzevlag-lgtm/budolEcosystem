Mitigation Strategy v467

M-467-01: Engine Initialization Delay

Implemented a 15-second fallback timer in index.html. If the engine takes longer than 15 seconds to bootstrap, the splash screen is forcefully removed to prevent a permanent blank screen, allowing the user to at least see the underlying "Booting BudolPay Engine..." UI in main.dart.

M-467-02: Zero-Size Layout Guard

The MarketingAdsScreen now checks for zero width or height in its LayoutBuilder. If detected, it logs a warning and returns a SizedBox.shrink(), preventing a layout crash and providing diagnostic data for further optimization.
