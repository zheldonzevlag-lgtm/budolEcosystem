System Admin Manual - v152

Infrastructure Updates

Database Schema Alignment: The Prisma schema has been updated to align with the existing database state. The STAFF role is now officially supported by the application layer.

Deployment Procedure:

  1. Navigate to budolpay-0.1.0/packages/database.
  2. Run npm install if dependencies are missing.
  3. Run npx prisma generate to update the local Prisma Client.
  4. Restart the Admin dashboard application (budolpay-0.1.0/apps/admin).

Operational Runbook

Issue: Dashboard shows "Unhandled Runtime Error" with Prisma validation messages.

Fix: This is typically caused by the Prisma Client being out of sync with the database schema. Verify the UserRole enum in schema.prisma and regenerate the client.

Verification: Access the Admin dashboard at http://192.168.1.24:3000 and verify that the "Operations Staff" count is displayed correctly without errors.
