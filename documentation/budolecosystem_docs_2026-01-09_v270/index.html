Overview

This version enhances the Transaction Details UI by reordering recipient information and standardizing button layouts. It also refines the marketing branding solution using a dynamic RegExp-based parser for all ecosystem services and increments the app version to 1.2.31.

  • Developer Manual - Technical details on UI refactoring and RegExp branding.
  • System Admin Manual - Release details for v1.2.31 and APK distribution.
  • User Manual - Overview of UI changes and enhanced branding.

Change Log

  • Transaction UI: Reordered recipient full name to appear before Transaction ID for better readability.
  • Button Layout: Standardized "Favorite/Share" and "Receipt" buttons into a single row with consistent Rose branding.
  • Dynamic Branding: Implemented a scalable RegExp-based solution in marketing_ads_screen.dart to colorize all budol services (Pay, Shap, Loan, Express).
  • Bug Fixes: Fixed typos in marketing slide descriptions and standardized service naming.
  • Release: Version incremented to 1.2.31+31.
