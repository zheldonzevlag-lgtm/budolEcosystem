Release v188 (2026-01-08)

Release Summary

This release (v188) focuses on Core Ecosystem Connectivity. We have implemented a centralized environment synchronization strategy, switched to Prisma Accelerate for database reliability, and optimized the API Gateway proxy logic to handle serverless deployment nuances on Vercel.

  • Fixed trailing character issues in environment variables.
  • Transitioned to Prisma Accelerate connection pooling.
  • Automated Vercel environment synchronization across 6 core services.
