System Admin Manual

Configuration

To enable Google Maps integration, the following environment variable must be set:

NEXT_PUBLIC_GOOGLE_MAPS_API_KEY

This key must be enabled for the Google Maps JavaScript API and the Places API.

Provider Switching

The map provider can be switched dynamically in the Admin Settings panel. Available options:

  • Google Maps: Requires API Key. Provides best accuracy and satellite imagery.
  • Geoapify: Requires API Key. Good alternative with OSM data.
  • Radar: Requires API Key. Location services focused.
  • OpenStreetMap (Default): Free, no API key required.

Troubleshooting

If the map displays "Loading Google Maps..." indefinitely, check the browser console for API key errors (e.g., Referer restrictions).
