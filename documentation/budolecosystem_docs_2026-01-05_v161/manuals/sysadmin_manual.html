System Admin Manual - Transaction Infrastructure

Infrastructure Requirements

This update relies on the budolPay Gateway and budolPay Transaction Service being operational.

Service Endpoint Required Port
Gateway API /transactions/history/:userId 8080
Socket.io Root Gateway URL 8080

Operational Notes

  • Traffic Impact: Each Home screen load now performs two API calls (Balance and History). This should be monitored for high-traffic environments.
  • Data Retention: The history endpoint returns the most recent transactions stored in the PostgreSQL database.
  • Socket Stability: Ensure the socket server is not dropping connections, as real-time updates depend on a persistent WebSocket link.
