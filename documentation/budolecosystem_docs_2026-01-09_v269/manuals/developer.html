Purpose

The objective was to apply branding colors (Slate for "budol" and Rose for "Pay/₱ay") within the description text of marketing slides, ensuring a cohesive visual identity across the entire marketing carousel.

Code Changes

1. RichText Description Helper

A new _buildDescription method was added to lib/screens/marketing_ads_screen.dart. This method parses the description string for branding keywords and returns a RichText widget with appropriate TextSpans.

Widget _buildDescription(String description) {
  // ... parsing logic ...
  spans.add(const TextSpan(
    text: 'budol',
    style: TextStyle(color: Color(0xFF94A3B8)), // Slate 400
  ));
  spans.add(const TextSpan(
    text: '₱ay',
    style: TextStyle(color: Color(0xFFF43F5E), fontWeight: FontWeight.bold), // Rose
  ));
  // ...
}

2. UI Integration

The _buildSlide method was updated to call _buildDescription instead of using a standard Text widget for the description field.

// Before
Text(slide['description'], ...)

// After
_buildDescription(slide['description'])

3. Versioning

Version incremented to 1.2.30 in pubspec.yaml and ApiService.

Future Enhancements

  • Move branding keywords and colors to a centralized ThemeConstants file.
  • Implement a more robust regex-based parser for handling multiple branding keywords (Loan, Shap, Express).
