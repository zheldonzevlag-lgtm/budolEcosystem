System Admin Manual

Operational guide for managing the Ecosystem Configuration Engine v2.2.1.

1. Maintenance Mode

Variable: SYSTEM_MAINTENANCE_MODE

When enabled, this restricts non-administrative access to the ecosystem. Use this during scheduled updates or security incidents.

Caution

Enabling maintenance mode will disconnect active user sessions and block new transactions.

2. Notification Providers

Email Configuration

Select between Resend, SendGrid, or SMTP.

API keys must be updated in the NOTIFICATION section of the settings page.

SMS Configuration

Select between Twilio, Infobip, or Vonage.

Ensure Twilio SID and Auth Token are correctly masked as secret data.

3. Compliance: Data Retention

Variable: COMPLIANCE_DATA_RETENTION_DAYS

Defines how many days transaction logs and audit traces are kept in the primary database before being moved to cold storage (archived). Default is 365 days.
