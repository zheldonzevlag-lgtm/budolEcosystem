What's New in v109

We've updated the app's visual identity to give it a more professional and polished look. Here’s what you’ll see when you update:

✨ New Professional App Icon

Look for the high-definition budolPay icon on your home screen. It’s clearer, sharper, and matches our official brand assets.

🚀 New Splash Screen

When you open the app, you'll be greeted by a sleek new loading screen that seamlessly transitions into the login page.

Installation Guide

Since this is a production release, we recommend uninstalling any previous "debug" versions of the app before installing this new version to ensure a clean setup.

Support

If you encounter any issues with the new interface or branding, please contact the development team with a screenshot of the issue.
