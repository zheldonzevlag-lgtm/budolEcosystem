SysAdmin Manual - v109

Production Deployment

The application is now ready for production distribution. The release APK is optimized for performance and size.

APK Location

budolPayMobile/build/app/outputs/flutter-apk/app-release.apk

Installation Instructions

  1. Transfer the app-release.apk to the target Android device.
  2. Enable "Install from Unknown Sources" if prompted.
  3. Open the APK file to begin the installation process.
  4. Verify the new app icon appears on the home screen.
  5. Launch the app to verify the new splash screen.

Build Troubleshooting

If the build fails with an "Out of Memory" error, ensure that the --no-tree-shake-icons flag is used, and consider increasing the JVM heap size for Gradle if necessary.
