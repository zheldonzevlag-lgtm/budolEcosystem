User Manual Update (v25)

Switching Networks (Portable Ecosystem)

If you take your laptop to a different location (office, coffee shop, or home) and want to continue using the budolPay Ecosystem on other devices, follow these steps:

Step 1: Connect to the new Wi-Fi.
Ensure all devices (laptop, phone, tablet) are on the same network.
Step 2: Sync the Network.
Open your terminal in the root directory and run:
node sync-network.js
Step 3: Access your Apps.
The script will tell you the new IP address. You can now access the apps on other devices using that IP (e.g., http://192.168.x.x:3000).

Accessing via Localhost

If you are only working on your laptop and don't need LAN access, you can still use http://localhost:3000. The ecosystem is now smart enough to handle both localhost and your IP address simultaneously.
