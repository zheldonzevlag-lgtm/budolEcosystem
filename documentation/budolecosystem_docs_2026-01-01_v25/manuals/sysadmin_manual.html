System Admin Manual Update (v25)

Network Configuration Utility

A new utility sync-network.js has been added to the root directory to handle network environment changes.

Usage Instructions

Run this command whenever you switch networks or your IP address changes:

node sync-network.js

Automatic Updates

  • Database: Automatically updates redirectUri in the budolID database to match the new host IP.
  • Environment: Updates budolshap-0.1.0/.env variables (NEXT_PUBLIC_SSO_URL, NEXT_PUBLIC_APP_URL, etc.).

Troubleshooting IP Detection

If the script detects the wrong IP (e.g., a virtual adapter), it will print the name of the adapter it skipped. You can manually set the IP in the .env files if needed, but the script is designed to prioritize physical adapters.
