Release Overview

This version focuses on completing the end-to-end integration between the mobile application and the backend services. We have replaced all mock logic with real production API calls and implemented proper state management.

  • Real Payment Processing: Replaced mock payment success with real calls to Wallet Service /process-qr.
  • State Management: Integrated Provider for consistent user session and API state.
  • Dynamic Data: Implemented real balance fetching and automatic refresh in HomeScreen.
  • Auth Integration: Fixed SSO login flow to use API Gateway and centralized authentication.
  • QR Parsing: Validated and fixed QR data parsing to match Payment Gateway standards.
