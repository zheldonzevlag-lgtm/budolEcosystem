Topic: Release 1.1.4 Build Management

1. Release Overview

Version 1.1.4 is a maintenance and feature release that includes the backend-ready phone number P2P support.

2. Build Verification

Admin should verify the following before distributing the APK:

  • APK size is approximately 63-65MB.
  • Version code is 1.1.4 (build 1).

3. Distribution

The release APK is located at: budolPayMobile/build/app/outputs/flutter-apk/app-release.apk
