STABLE RELEASE

Local SSO Stabilization Suite

Version 2.2.0 • Updated: April 10, 2026 • Status: Verified
📌 Purpose & Rationale

During the transition from Vercel to local LAN development, it was identified that the SSO (Single Sign-On) redirection logic was incorrectly defaulting to production URLs. This occurred because several .env files across the ecosystem were hardcoded with https://budolshap.vercel.app, causing callbacks to fail in the local environment.

This release aligns all environment variables to the Local LAN IP (192.168.1.2) and ensures consistent redirection across BudolID, BudolShap, and BudolPay.
