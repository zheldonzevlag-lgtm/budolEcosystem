System Admin Manual: Environment Synchronization

Global Environment Sync

To ensure all ecosystem services (BudolID, BudolShap, BudolPay, Accounting) are aligned with the local development IP, follow these steps:

Step 1: Configure Template

Ensure budolshap-0.1.0/.env.connection.local contains the target LAN IP:

LOCAL_IP=192.168.1.2

Step 2: Run Synchronization

Execute the environment manager script from the BudolShap directory:

cd budolshap-0.1.0
node scripts/env-manager.cjs local

Step 3: Service Restart

Restart the following services to apply the new redirect_uri logic:

  • BudolID: (Port 8000)
  • BudolShap: (Port 3000)
  • BudolPay: (Port 3002/monorepo)
