System Admin Manual: LAN & mDNS

Network Requirements

  • Port 5353 (UDP): Must be open for mDNS/Bonjour traffic.
  • Port 8000-8080 (TCP): Must be open for ecosystem services.
  • Same Subnet: Devices must be on the same LAN subnet for mDNS to work.

mDNS Implementation

We use bonjour-service to advertise services. This is compatible with Apple Bonjour (iOS/macOS) and Avahi (Linux).

Note: Windows users may need to ensure the "mDNS" feature is enabled in their network settings or have iTunes/Bonjour Print Services installed for legacy support.

Automatic IP Detection

The system automatically detects the primary LAN IP (e.g., 192.168.1.24). If multiple network interfaces are present, ensure the correct one is prioritized in scripts/network-util.js.

Troubleshooting

If apps are not accessible via LAN:

  1. Check if the server is binding to 0.0.0.0 (use netstat -ano | findstr 8000).
  2. Verify the Windows Firewall allows traffic on the relevant ports.
  3. Run npm run sync-network to refresh IP configurations.
