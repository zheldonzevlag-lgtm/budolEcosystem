Developer Manual - v1.3.53

Technical Purpose

The goal of this update is to ensure that the session lock UI is perfectly centered on all devices. Previously, the branding was tied to the bottom of a Column, which could lead to inconsistent spacing on different screen aspect ratios.

Code Changes

1. Session Overlay Centering

Refactored the SessionOverlay to use a Stack with a Center widget for the main content and a Positioned widget for the branding, ensuring perfect centering regardless of content height.

File: session_overlay.dart

child: Stack(
  children: [
    Center(
      child: Column(
        mainAxisAlignment: MainAxisAlignment.center,
        crossAxisAlignment: CrossAxisAlignment.center,
        children: [
          // Icon, Text, Buttons...
        ],
      ),
    ),
    Positioned(
      bottom: 24,
      left: 0,
      right: 0,
      child: Center(
        child: UIUtils.formatBudolPayText('budol₱ay', ...),
      ),
    ),
  ],
),

Rationale

Using Stack + Center + Positioned provides a more robust centering mechanism than Column + Spacer, especially when the branding element needs to be fixed at the bottom while the main content remains centered.

Dependencies

  • Flutter SDK >=3.0.0
  • Provider 6.0.5
