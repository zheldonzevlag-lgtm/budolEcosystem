v423 - Admin User Management Enhancement

This version introduces enhanced user management for administrators, allowing the capture and management of user phone numbers and addresses using the universal address component.

Release Information

  • Release Date: 2026-01-15
  • Version: 423 (v4.2.3)
  • Scope: Admin Dashboard, API Routes, User Profiles

Purpose & Rationale

The previous admin user management was limited to basic fields (name, email, password, role). To support logistical operations and verified user identities, capturing phone numbers and physical addresses is critical.

By integrating the Universal Address Component, we ensure data consistency across the ecosystem, utilizing the same geocoding and validation logic used in store creation and checkout flows.

Compliance Status

  • NPC (Data Privacy): Compliant - New fields are collected for legitimate business purposes (logistics/verification).
  • BSP/Cybersecurity: Compliant - Admin actions are protected by RBAC and logged.
