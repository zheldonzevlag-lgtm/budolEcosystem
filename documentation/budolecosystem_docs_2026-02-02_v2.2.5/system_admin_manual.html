Admin Guide: Card Interface

Navigating the Card View

The settings page is now organized into functional sections, with each setting presented as a standalone card. This improves focus when managing complex parameters.

  • Identification: The mono-spaced key at the top left of each card is the internal system variable name.
  • Status: Use the "Active/Inactive" badge to quickly see if a setting is being applied.
  • Updating: Enter the new value in the input field and click "Save Changes" at the bottom of the card.

Search & Filtering

The search bar at the top remains the primary tool for finding settings. It filters cards by their key, description, or group name in real-time.

Audit Logs

Every change made via a setting card is recorded in the Audit Logs. The log entry will include the ID of the setting modified and the new value provided.
