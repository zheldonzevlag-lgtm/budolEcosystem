User Interface Guide: Settings

Using the Card View

We've redesigned the settings interface to make it more intuitive and easier to navigate on all devices.

Better Focus

Each setting has its own dedicated card, so you're never confused about which description or save button belongs to which parameter.

Mobile Friendly

The cards stack vertically on small screens, making it much easier to manage your ecosystem from a mobile device compared to a large table.

Clear Descriptions

Descriptions are now centered and given more space, helping you understand exactly what each configuration parameter does.

Immediate Feedback

The "Save Changes" button is right below the input field, providing a clear and direct path to committing your updates.

Pro Tip

Use the search bar at the top to filter cards by name or description. This is the fastest way to find a specific setting in a large list.
