Ecosystem Documentation v45

Release Date: April 05, 2026

This release focuses on resolving a critical security forensic gap where staff identities were missing from authentication audit logs. This update ensures 100% traceability for all administrative actions in compliance with PCI-DSS 10.2.

Release Highlights

  • Audit Identity Fix: Resolved an issue where non-UUID staff IDs were being nullified during logging.
  • Schema Resilience: Implemented UI fallbacks to support both name and firstName/lastName data models.
  • Forensic Accuracy: Guaranteed that USER_LOGOUT and LOGIN events correctly link to the staff member's profile.
