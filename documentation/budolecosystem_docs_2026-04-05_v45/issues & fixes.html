Issues & Fixes

Issue: "Staff Identity" column shows dot or "Anonymous" for LOGIN/LOGOUT events.
Fix: Relaxed userId regex in lib/audit.ts. The regex was strictly matching UUIDs with hyphens, rejecting valid custom IDs like admin-reynaldo-id.
Issue: Name appears as "undefined undefined" in some database schemas.
Fix: Added fallback for log.user.name in the Audit Trail UI. Some schemas use a single name field instead of split firstName/lastName.
