User Manual (Staff)

This manual covers the visibility of audit logs for staff members and managers.

Employee Management -> Forensic Audit Trail

The Forensic Audit Trail allows managers to view the history of actions taken on the platform. The "Staff Identity" column is the primary way to identify who performed an action.

Identifying the Actor

  • Full Name: If the staff member's profile is complete and correctly linked, their full name will appear.
  • Role Tag: A small tag (e.g., ADMIN, STAFF) appears below the name for quick role identification.
  • System / Anonymous: If an action was performed by an automated process or an unauthenticated user, this default label will appear.

This ensures accountability across all security-critical events like logins, transfers, and role changes.
