Deployment & Routing Configuration

Infrastructure Overview

The Budol Ecosystem is composed of multiple Vercel projects coordinated via a central "Ecosystem" project that handles global routing.

Routing Configuration (Root vercel.json)


{
  "version": 2,
  "routes": [
    { "src": "/api/(.*)", "dest": "budolpay-0.1.0/services/api-gateway/index.js" },
    { "src": "/admin/(.*)", "dest": "https://budolpay-admin.vercel.app/$1" },
    { "src": "/shap/?(.*)", "dest": "https://budolshap-five.vercel.app/$1" },
    { "src": "/beta/?(.*)", "dest": "https://budolshap-five.vercel.app/beta/$1" },
    { "src": "/(.*)", "dest": "https://budolpay-admin.vercel.app/$1" }
  ]
}

Deployment Procedures

  1. BudolPay Admin: Deployed from budolpay-0.1.0/apps/admin.
  2. BudolShap Marketplace: Deployed from budolshap-0.1.0. Headers managed in next.config.mjs.
  3. Ecosystem Root: Handles API Gateway and routing.

Maintenance Tasks

  • Ensure budolshap-five.vercel.app is active and points to the latest v0.1.0 build.
  • Monitor API Gateway health via /api/gateway/health.
