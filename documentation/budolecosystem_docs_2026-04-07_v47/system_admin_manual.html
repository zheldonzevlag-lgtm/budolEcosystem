System Admin Manual

To deploy budolid-sso effectively in Vercel:

  • Change the Framework Preset to Other (Do NOT use Next.js, as this is an Express API).
  • Ensure the Root Directory correctly points to budolID-0.1.0.
  • Set the install command to npm install and the build command to npm run build.

To deploy budolpay-0.1.0 effectively in Vercel:

  • Keep the Framework Preset to Next.js for the apps/admin module if deploying individually. If deploying the monorepo from root, use Next.js configuration but ensure `Root Directory` is `budolpay-0.1.0/apps/admin`.
