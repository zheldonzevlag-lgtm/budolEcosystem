System Admin Manual - Environment Management (v192)

Infrastructure Changes

Centralized source connection files now reside in the budolpay-0.1.0 directory. These files act as the "Source of Truth" for the entire ecosystem when switching environments locally.

Operational Procedures

Switching Environments

  1. Run npm start from the root directory.
  2. Select 1 for Local, 2 for Production/Vercel.
  3. The system will automatically propagate the correct DATABASE_URL to all services.

Risks & Mitigation

  • Risk: Incorrect database credentials in .env.connection.prod could lead to cross-service data corruption if services share a database.
    Mitigation: Services use distinct databases (budolpay, budolid, budolshap, budolaccounting). The mapping logic ensures each service only receives its own credentials.

Dependencies

  • Node.js 18+
  • npm Workspaces
  • Write access to all service .env files.
