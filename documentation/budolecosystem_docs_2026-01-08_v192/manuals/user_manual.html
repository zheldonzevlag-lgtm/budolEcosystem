User Manual - Simplified Startup (v192)

This update simplifies how you start the Budol Ecosystem for development or testing.

One-Command Startup

You no longer need to manually edit .env files for different services. Everything is handled via the root startup command.

How to use:

  1. Open your terminal in the budolEcosystem root folder.
  2. Type npm start.
  3. Follow the on-screen prompt to choose your environment.

The system will automatically configure and start all services (SSO, Admin, Wallet, Accounting, etc.) with the correct settings for your chosen environment.
