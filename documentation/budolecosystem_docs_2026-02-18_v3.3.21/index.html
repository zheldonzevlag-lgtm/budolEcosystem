Version 3.3.21 - Cart Logic & Profile Fixes

Purpose & Rationale

This release addresses a critical bug in the Checkout Process where items remained in the cart after successful payment. It also finalizes the Profile Edit enhancements from v3.3.20 with verified persistence logic.

Technical Summary

  • Cart Cleanup Logic: Fixed strict equality check failure for `variationId` (null vs empty string) in `ordersService` and `paymongo` webhook handler.
  • Unit Testing: Added dedicated unit tests for cart cleanup condition generation.
  • (Inherited) Profile Enhancements: Visibility toggles and strict validation for user profile fields.
