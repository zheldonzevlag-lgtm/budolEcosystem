v278: Mobile Release v1.2.35 Latest

Comprehensive mobile app stabilization, static analysis cleanup, and production release.

Developer Manual System Admin Manual User Manual Recommendations

Purpose

The primary goal of this update was to resolve all static analysis issues in the budolPayMobile application and prepare a production-ready release (v1.2.35). This ensures a clean codebase for future development and a stable experience for end-users.

Key Changes

  • Static Analysis: Removed 16 unused lucide_icons imports and redundant provider imports across multiple screens.
  • Version Increment: Updated pubspec.yaml to version 1.2.35+35.
  • Release Build: Successfully generated production APK budolPay_1.2.35.apk.
  • Splash Screen: Verified that the splash screen correctly displays the current app version dynamically.
  • Distribution: Deployed the release APK to the android-installer directory.

Technical Details

The build was executed using flutter build apk --release. Tree-shaking was enabled for icons, significantly reducing the final APK size while maintaining all required functionality.
