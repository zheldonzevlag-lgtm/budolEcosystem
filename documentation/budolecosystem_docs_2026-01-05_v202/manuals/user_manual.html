User Manual - v202

Saving Feedback

When you click any "Save Settings" button, the button text will change to "Saving..." and a spinning icon will appear. This indicates that your changes are being transmitted to the system.

Provider Activation Feedback

When switching between different email or SMS providers using the "Use This" button, the button will display "Activating..." to confirm the switch is in progress.

Prevention of Multiple Submissions

While the saving or activation process is active, the buttons are temporarily disabled. This prevents accidental duplicate submissions and ensures a smooth configuration experience.
