Implementation Plan - v202

Notification Settings UX Enhancement

Purpose

Improve the user experience of the Notification Settings page by providing visual feedback during the save process. This ensures users are aware that their settings are being processed and prevents multiple submissions.

Objectives

  • Implement a reusable SubmitButton component using useFormStatus.
  • Add loading indicators (spinner and text) to all provider save buttons.
  • Add loading states to provider activation buttons ("Use This").
  • Ensure visual consistency across Email and SMS sections.

Technical Details

Utilized React's useFormStatus hook from react-dom to detect form submission state without requiring complex client-side state management in the main page component.

Test Results

  • Save buttons show "Saving..." and a spinner when clicked.
  • "Use This" buttons show "Activating..." and a spinner when clicked.
  • Buttons are disabled during submission to prevent duplicate actions.
  • Settings are correctly persisted to the database.
