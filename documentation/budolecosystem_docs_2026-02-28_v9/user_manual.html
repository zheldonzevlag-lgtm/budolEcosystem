User Manual

Improved Professional Layout

The marketplace has been visually polished with standardized spacing. Each section (Hero, Categories, Products, Specifications, and Newsletter) is now consistently spaced, providing a smoother scrolling experience. On mobile devices, text sizes have been adjusted to ensure better legibility on small screens without overwhelming the interface.

System Admin Manual

Pure UI Polish Release

This release (v5.3.9) is purely focused on the frontend and user experience (UI/UX). There are no changes to the database schema, API logic, or core backend services. System administrators do not need to perform any data migrations or security audits for this update.

Affected Components: Homepage components (Hero, LatestProducts, BestSelling, OurSpec, Newsletter).
