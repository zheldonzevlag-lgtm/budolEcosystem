v5.3.9 - Developer Manual

Standardized UI Gaps & Typography Polish

Overview

This release fixes layout inconsistencies that made the homepage feel disjointed. We have standardized on a mb-12 (48px) vertical gap between all major landing page sections.

Code References

  • Hero.jsx: Updated my-10 to mb-12 (48px) and adjusted font size from text-2xl up to text-xl for mobile.
  • CategoriesSection.jsx: Updated padding to standardized mb-12 and increased label font size to text-[11px] for mobile clarity.
  • LatestProducts.jsx: Drastically decreased gap from my-30 (120px) to mb-12 (48px).
  • BestSelling.jsx: Decreased gap from my-30 to mb-12.
  • OurSpec.jsx: Decreased gap from my-20 to mb-12.
  • Newsletter.jsx: Decreased gap from my-36 (144px) to mb-12 (48px). Added mx-6 padding.
  • Title.jsx: Standardized h2 title typography to text-xl sm:text-2xl for responsive scaling.

Styling Changes

The 48px gap (mb-12) was chosen as it provides sufficient breath between sections while maintaining a tight, professional e-commerce feel. Spacing was removed from top margins (mt) to avoid doubling up when sections are stacked.
