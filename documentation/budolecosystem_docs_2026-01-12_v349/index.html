Version 349

Fixed Web Splash Screen hanging and improved cross-platform compatibility.

Release Overview

This version addresses a critical issue where the application would hang on the splash screen when running on the Web. It improves the reliability of the transition from the native HTML splash to the Flutter application UI by implementing manual removal triggers and safety fallbacks.
