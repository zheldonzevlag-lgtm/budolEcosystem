Mitigation Plan - v349

R-349-01: Web Splash Screen Hang

Mitigation: Removed the FlutterNativeSplash.preserve() call in main.dart and implemented manual removal via dart:js in the initState of SplashScreen and LoginScreen. This ensures that the splash screen is removed as soon as the Flutter app is ready to render.

R-349-02: JS Interop Safety

Mitigation: All dart:js calls are wrapped in if (kIsWeb) blocks and try-catch statements to prevent crashes on mobile platforms (Android/iOS) where dart:js is not available.
