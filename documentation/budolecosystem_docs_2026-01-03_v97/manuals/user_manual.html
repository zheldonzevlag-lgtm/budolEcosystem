Understanding the QRPH Payment Experience

QRPH Payment Flow

This guide explains how a customer (e.g., Clark Kent) completes a purchase using budolPay QRPH.

1 Checkout: The user selects items and proceeds to checkout in budolShap, choosing "QRPH" as the payment method.

2 QR Code Generation: The system generates a unique QR code via budolPay. The user is redirected to a payment page showing the QR code.

3 Mobile Scan: The user opens their budolPay mobile app and scans the QRPH code.

4 Confirmation: After scanning and confirming the amount, the mobile app deducts the balance from the user's wallet.

5 Order Update: budolPay sends a secure confirmation to budolShap. The order status is automatically updated to "PAID" and the user sees a success message.

Verification

Users can verify their successful payment by:

  • Checking "My Orders" in budolShap for the updated status.
  • Viewing the "Transaction History" in the budolPay app to see the wallet deduction.
