budolEcosystem v457

Release Date: 2026-01-17 | Type: CLI Hotfix

Renderer Configuration Sync

Primary Issue: Flutter CLI Option Error

The previous fix attempted to use --web-renderer as a CLI flag for flutter run. In this environment, that flag is not recognized by the web-server device, causing a startup crash (Exit Code 64).

Technical Resolution

  • In-App Configuration: Moved the renderer selection from the CLI to the application layer. Added window.flutterWebRenderer = "html"; to index.html.
  • CLI Cleanup: Reverted start-ecosystem.js to use standard flutter run arguments, ensuring the ecosystem starts without error.
  • Verified Path: This approach ensures the HTML renderer is used regardless of how the app is launched (CLI, IDE, or direct build).
