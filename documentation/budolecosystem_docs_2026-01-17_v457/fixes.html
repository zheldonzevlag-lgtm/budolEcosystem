CLI & Renderer Fixes (v457)

1. Unknown Option "--web-renderer"

Problem: The Flutter run -d web-server command does not support the --web-renderer flag directly in the user's environment, causing the app to exit with code 64.

Solution:

  • Removed the flag from start-ecosystem.js.
  • Moved the renderer setting to the application layer in index.html using window.flutterWebRenderer = "html";.
