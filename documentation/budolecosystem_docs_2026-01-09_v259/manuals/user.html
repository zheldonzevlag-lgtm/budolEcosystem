User Manual Updates - v259

1. New Features & Improvements

Real-Time Balance Updates: Your balance will now update instantly after a transfer, deposit, or payment without needing to refresh the screen.
Transaction Notifications: Receive immediate confirmation of successful transfers directly in the app.

2. App Version Verification

To ensure you are on the latest version:

  1. Open the budolPay app.
  2. On the Splash Screen, look for budolEcosystem 1.2.23 at the bottom.
  3. If you see an older version, please install the latest APK from the installer folder.

3. Known Issues & Fixes

Fixed an issue where transfers would complete but the balance wouldn't update on the home screen until the app was restarted.
