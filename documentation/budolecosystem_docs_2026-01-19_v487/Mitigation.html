Mitigation Plan - v487

MITIGATION FOR R-487-01

Manual Closing Protection

Strategy: Implemented a "Pay Now" button in the Orders page. If a user closes the modal accidentally, they can re-trigger it from their order history without starting a new checkout.

MITIGATION FOR R-487-02

Network Interruption Resilience

Strategy: The polling logic in QRCodeModal.jsx includes automatic retry mechanisms and a clear error state. Future versions will implement LocalStorage persistence to recover the modal state after a page refresh.

MITIGATION FOR R-487-03

Data Normalization Layer

Strategy: Created a normalization function in OrderSummary.jsx that maps diverse gateway responses into a standard qrData object used by the modal.
