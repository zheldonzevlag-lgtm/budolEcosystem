Identified Issues - v487

ISSUE-487-01

Modal State Persistence

Description: If the user refreshes the page while the payment modal is open, the modal closes and the session is lost. The user must navigate back to the Orders page to re-open the modal.

Severity: Medium

ISSUE-487-02

Mobile Viewport Constraints

Description: On very small mobile devices, the step-by-step instructions in the modal may require scrolling, which could be overlooked by some users.

Severity: Low
