Applied Fixes - v487

FIX-487-01

Redirection Loop Avoidance

Description: Fixed an issue where the system would sometimes trigger a redirection to the legacy payment page while the modal was trying to open.

FIX-487-02

QR Data Normalization

Description: Resolved a bug where qrCode.imageUrl was missing in some PayMongo responses, causing the modal to show a broken image icon.

FIX-487-03

Timer Synchronization

Description: Synchronized the modal's internal 10-minute timer with the actual payment gateway's expiration time to avoid premature timeout messages.

FIX-487-04

budolPay Redirect Suppression

Description: Modified the backend payment-gateway-service to return raw QR data for internal budolPay transactions, preventing the frontend from being forced to redirect to an external checkout page.
