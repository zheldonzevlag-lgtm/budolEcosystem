RBAC UI & User Management Integration v230

Date: 2026-01-08

Purpose

To provide a visual interface for managing Role-Based Access Control (RBAC) in BudolShap and integrate granular role management into the User Management administrative workflow. This completes the end-to-end implementation of the unified ecosystem RBAC standards established in v229.

Technical Details

1. RBAC Permissions Matrix UI

Created a dedicated PermissionsMatrix component in BudolShap that visualizes the relationship between roles and granular permissions, categorized by functional groups (User Management, System Settings, Financial Operations).

  • Location: /admin/settings/rbac
  • Component: components/admin/PermissionsMatrix.jsx

2. User Management Integration

Enhanced the AdminUsers page to allow administrators to assign specific RBAC roles to ADMIN account types.

  • Modified: app/admin/users/page.jsx (UI updates for role display and selection)
  • Modified: app/api/admin/users/route.js (POST handler for role persistence)
  • Modified: app/api/admin/users/[userId]/route.js (PUT handler for role updates)

Issues & Fixes

Issue Fix
Missing RBAC Menu Added "RBAC & Permissions" to the Settings submenu in AdminSidebar.jsx.
Role Visibility Added RBAC role badges to the user cards in the User Management listing.
API Payload Mismatch Updated Next.js API routes to accept and persist the role field in Prisma.

Test Results

VERIFIED
  • Sidebar: RBAC menu is visible and correctly navigates to the settings page.
  • Matrix: Correctly displays permissions for all roles (ADMIN, STAFF, etc.).
  • Provisioning: Successfully created and updated users with specific RBAC roles via the UI.
  • API: Verified that the role field is correctly stored in the database.

Recommendation

  • Implement dynamic permission toggles (overrides) per user in the User Management modal.
  • Add visual indicators in the Sidebar for menu items restricted by the current user's RBAC role.
  • Port the RBAC audit log viewer to the /admin/settings/rbac page for specialized security monitoring.
