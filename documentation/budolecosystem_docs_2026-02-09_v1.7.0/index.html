AI Security Release v1.7.0

Release Summary

This release introduces the AI Anti-Spam Engine, a critical security enhancement designed to prevent fraudulent account creation. By utilizing pattern recognition and risk scoring, the system proactively blocks suspicious registration attempts, ensuring ecosystem integrity and compliance with BSP Circular 808 anti-fraud requirements.

Key Changes

  • AI Anti-Spam Engine: Implemented a risk-scoring module that analyzes registration data (email randomness, name consistency, disposable domains).
  • Real-time Security Telemetry: Integrated spam block metrics into the Admin Security Dashboard.
  • Forensic Auditing: Enhanced `auth-service` to log blocked registration attempts with detailed risk analysis.
  • UI Enhancements: Added interactive hover states and AI Spam Guard status to the Security dashboard.
  • Compliance: Fully aligned with BSP Circular 808 guidelines on Cyber-Resiliency and Fraud Management.
