User Experience & Workflow

Proposed User Flow

Step 1: First-time Opt-in
After a successful login with PIN, a bottom sheet appears: "Enable Biometric Login? Use your fingerprint or face for faster access next time."
Step 2: Verification
User confirms with their biometric sensor to link the device.
Step 3: Subsequent Logins
On the Login screen (PIN stage), the biometric prompt automatically appears. If cancelled, the user can still enter their PIN manually.

Interface Changes

  • Login Screen: Add a biometric icon button near the PIN entry field.
  • Settings Screen: Add a toggle for "Biometric Login" under the Security section.
  • Onboarding: Optional slide explaining the benefits of biometric security.

User FAQ

Is it safe?
Yes, budolPay does not store your actual biometric data. We only use the device's secure system to verify your identity.

What if my fingerprint scanner is broken?
You can always use your 6-digit PIN as a fallback.
