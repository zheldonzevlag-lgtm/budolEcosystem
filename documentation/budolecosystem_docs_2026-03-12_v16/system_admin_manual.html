System Admin Manual - v16

Environment Configuration

Admin must ensure that the DATABASE_URL in AWS Secrets Manager (budol/db-url) correctly includes the budolshap_1db database name.

CRITICAL: Connecting to the default budolshap database instead of budolshap_1db will cause 500 errors as the SystemSettings table resides in the latter.

Deployment Infrastructure

  • ECR Repository: budol-shap-repo
  • Image Tag: v17
  • ECS Cluster: budol-ecosystem-cluster
  • ECS Service: budol-shap-service

Monitoring Logs

Admins should monitor CloudWatch Log Group /ecs/budol-shap for any recurring P2021 (Table missing) or P2024 (Connection limit) errors.

Rollback Procedure

To rollback, update the ECS Service to use the previous image tag (e.g., v16) and revert the Secrets Manager budol/db-url to the previous value if necessary.
