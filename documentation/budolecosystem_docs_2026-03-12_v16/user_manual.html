User Manual - v16

What's Fixed?

Users should no longer see "Internal Server Error" when trying to log in via mobile or browse products. The application stability has been greatly improved by ensuring all background configurations are correctly synchronized.

OTP Login

The login process using mobile numbers and OTP (One-Time Password) is now fully restored. Users will receive their 6-digit codes via SMS as expected once the new update propagates.

Product Catalog

Product categories and individual items are now correctly displayed across all devices. If you encounter any empty pages, please try refreshing your browser or clearing your app cache.
