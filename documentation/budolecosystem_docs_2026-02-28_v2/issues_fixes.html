Issues & Fixes Log (v5.3.2)

1. Runtime ReferenceError: PROFESSIONAL_ICONS is not defined

Issue: The Category Management modal crashed immediately upon opening because it referenced an array that was removed during the Lucide Icon standardization.

Fix: Replaced the reference with Object.entries(PROFESSIONAL_ICON_MAP) and updated the filtering logic to handle entry arrays.

2. Dead Search Field (Grid Picker)

Issue: The "Search Lucide icons..." input in the modern grid picker had an empty onChange handler, making it non-functional.

Fix: Connected the input to the setIconSearch state, instantly enabling search for the grid view.
