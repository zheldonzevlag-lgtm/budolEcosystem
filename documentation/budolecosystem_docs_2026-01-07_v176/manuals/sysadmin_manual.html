Vercel Service Setup & Configuration

Administrative Checklist

1. Provision Vercel Postgres

Navigate to Storage → Create Database → Postgres. Link the database to both the budolpay-admin and budolshap projects.

2. Provision Vercel KV (Redis)

Navigate to Storage → Create Database → KV. Ensure the REDIS_URL is added to the budolshap environment variables.

3. Enable Vercel Blob

Navigate to Storage → Create Database → Blob. This generates the BLOB_READ_WRITE_TOKEN required for document uploads.

4. External Service Configuration

Manually add the following keys to the Vercel Project Settings:

  • CLOUDINARY_URL
  • PUSHER_APP_ID, PUSHER_KEY, PUSHER_SECRET, PUSHER_CLUSTER
  • PAYMONGO_SECRET_KEY, PAYMONGO_PUBLIC_KEY
  • JWT_SECRET (Must be consistent across all services)
