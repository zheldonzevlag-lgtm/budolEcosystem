User Manual

1. Phone Number Input Guidelines

Consistent Formatting

The system now automatically normalizes your phone number. You can enter it in any of these formats:

  • 0917 654 3281
  • +63 917 654 3281
  • 639176543281
  • 9176543281
Note: If you receive an "Account not found" error, ensure you are using the same phone number you registered with.

2. Seller Workflow Improvements

Breadcrumb Navigation

When adding or editing products, you will now see a clear navigation path: Dashboard > Manage Product > Add/Edit Product. This allows you to easily return to the product list.

Consistent UI Layout

The product description and variations sections now share a consistent card-based layout and width, providing a cleaner and more professional workspace.
