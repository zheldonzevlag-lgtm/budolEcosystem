System Admin Manual

1. User Data Consistency

Admins should be aware that all phone numbers are now stored in E.164 format (+639XXXXXXXXX). Any manual database updates must follow this standard to avoid login issues.

Important: If a user reports "Account not found", verify their phoneNumber field in the database. It should start with +63 and contain 10 digits after that.

2. Verification Code Debugging

OTP verification codes are now tracked using the normalized phone number or lowercase email. Check the VerificationCode table for active codes if users report delivery issues.
