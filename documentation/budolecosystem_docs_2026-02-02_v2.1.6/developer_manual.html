SSPR Architecture

The Self-Service Password Reset (SSPR) is implemented centrally in budolID to provide ecosystem-wide recovery. It uses a three-step verification process:

  1. Identification: User provides email. System generates 6-digit OTP.
  2. Verification: User provides OTP. System generates a secure Reset Token.
  3. Execution: User provides Reset Token and New Password. System updates the hash.
