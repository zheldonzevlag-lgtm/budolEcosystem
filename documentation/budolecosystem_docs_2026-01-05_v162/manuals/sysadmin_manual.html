Build Environment Notes

  • Flutter SDK: 3.x.x
  • Gradle Version: 8.9
  • Memory Limits: The build requires at least 2GB of JVM heap space (-Xmx2048m). In case of Gradle daemon crashes, use flutter clean before retrying.
