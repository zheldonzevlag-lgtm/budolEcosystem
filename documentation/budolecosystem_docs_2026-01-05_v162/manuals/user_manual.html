User Manual - Installing budolPay 1.1.5

Version 1.1.5 is Ready!

This update includes real-time transaction history and smarter recipient validation.

Filename: budolPay 1.1.5.apk

Installation Steps

  1. Locate the budolPay 1.1.5.apk file in the android-installer folder.
  2. Transfer the file to your Android device or open it directly if you are using an emulator.
  3. If prompted, allow "Installation from Unknown Sources" in your device settings.
  4. Follow the on-screen prompts to complete the installation.
  5. Open the app and log in to see your new Home screen!

What's New?

  • Live Activity: Your transactions now update in real-time.
  • Smart Entry: Instant feedback when typing phone numbers or emails.
  • Performance: Optimized build for faster startup.
