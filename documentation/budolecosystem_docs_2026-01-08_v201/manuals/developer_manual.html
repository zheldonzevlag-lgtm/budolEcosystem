Maintenance: Resetting User PINs

During development or testing, you may need to reset a user's PIN to re-test the onboarding/recovery flow. We have provided a utility script for this.

Reset Script Location

budolpay-0.1.0/packages/database/reset_pin.js

Usage

node reset_pin.js

Technical Logic

The script sets pinHash: null for a specific phone number. This state triggers the !user.pinHash condition in the identify route, forcing the mobile app to go through the OTP verification flow and then the setup-pin phase.
