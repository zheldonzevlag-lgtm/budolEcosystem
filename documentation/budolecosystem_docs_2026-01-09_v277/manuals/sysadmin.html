System Admin Manual - v277

Infrastructure & Configuration

Configuration Changes

The next.config.js file has been modified to include the Cloudinary remote pattern. This change is required for the application to serve images from the Cloudinary CDN.

Environment Impact

  • File: next.config.js
  • New Settings: images.remotePatterns
  • Deployment: A restart of the Next.js server is required for these configuration changes to take effect.

Risk & Mitigation

  • Risk: Misconfigured remote patterns could block images.
  • Mitigation: Used a broad glob pattern (/**) for the pathname to ensure all Cloudinary assets are allowed while restricting the hostname to res.cloudinary.com.
