Biometric Authentication Setup

Step 1: Initial Setup

  1. Open the BudolPay app
  2. Navigate to Settings → Security
  3. Tap "Enable Face Recognition"
  4. Grant camera permissions when prompted

Step 2: Face Registration

  1. Position your face in the camera frame
  2. Look directly at the camera
  3. Ensure good lighting conditions
  4. Follow the on-screen instructions for multiple angles
  5. Wait for the "Face Registered Successfully" message
⚠️ Important: The app now includes automatic model validation. If you encounter any issues during setup, the app will guide you through troubleshooting steps.

Using Face Recognition

Authentication Process

  1. Open the BudolPay app
  2. Tap "Login with Face Recognition"
  3. Position your face in the camera frame
  4. Look directly at the camera
  5. Wait for authentication to complete
✅ Tips for Best Results:
  • Use the app in well-lit environments
  • Hold the phone at eye level
  • Look directly at the camera
  • Remove glasses or hats if authentication fails
  • Ensure your face is clearly visible

Enhanced Security Features

Model Integrity Validation

The app now automatically validates the face recognition model to ensure it hasn't been corrupted or tampered with. This happens transparently during app startup.

What This Means for You:

  • Enhanced protection against security threats
  • Automatic detection of corrupted authentication data
  • Improved reliability of face recognition
  • Compliance with banking security standards
🛡️ Security Note: If the app detects any issues with the face recognition model, it will automatically re-download the correct model. This is a normal security feature.

Full-Screen Debug Mode

For troubleshooting face recognition issues, the app now includes an enhanced debug mode with full-screen display.

Accessing Debug Mode:

  1. Go to Settings → Face Recognition
  2. Tap "Debug Mode" 7 times rapidly
  3. The debug window will open in full-screen mode
  4. View real-time camera feed and model status
🔧 Debug Mode Features:
  • Full-screen camera preview
  • Real-time model loading status
  • Error highlighting for issues
  • Model file information display
  • Detailed authentication logs

Troubleshooting Common Issues

❌ Face Recognition Not Working

Try these solutions:

  1. Check Lighting: Move to a better-lit area
  2. Clean Camera: Wipe your phone's camera lens
  3. Remove Obstructions: Take off glasses, hats, or masks
  4. Restart App: Close and reopen the BudolPay app
  5. Use Debug Mode: Check for error messages in debug mode

❌ App Crashes During Face Recognition

Try these solutions:

  1. Update App: Ensure you have the latest version
  2. Check Storage: Make sure you have enough free space
  3. Restart Device: Turn your phone off and on again
  4. Clear Cache: Go to Settings → Apps → BudolPay → Clear Cache
