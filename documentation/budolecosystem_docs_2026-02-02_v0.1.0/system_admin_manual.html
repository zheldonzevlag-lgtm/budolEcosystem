Audit Trail Management

The forensic audit trail is automated and requires minimal manual intervention. However, administrators should be aware of the following:

  • Retention Policy: Audit logs are kept indefinitely in the primary PostgreSQL database as per BSP Circular 808.
  • Real-time Monitoring: Ensure the API Gateway (Port 8080) and Socket.io server are operational for live updates.
  • Compliance Exports: Use the "Download Audit Report" feature in the Security page for BIR/BSP audit submissions.

Troubleshooting Real-time Updates

If live updates are not appearing:

  1. Check Gateway logs for [Notification] Emitted AUDIT_LOG_CREATED.
  2. Verify the admin user has joined the admin room in Socket.io.
  3. Ensure the REALTIME_METHOD in System Settings is correctly set (SOCKET_IO or PUSHER).
