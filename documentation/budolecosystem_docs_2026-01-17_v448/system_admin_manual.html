Deployment Note: Color Synchronization

This release synchronizes branding colors between the Flutter mobile application and the backend email service. Ensure both components are deployed together to maintain visual consistency for the end user.

1. Backend (Email Service)

The email branding logic is located in budolshap-0.1.0/lib/email.js. No environment variable changes are required for this update as the colors are baked into the HTML template logic.

2. Frontend (Flutter Web/Mobile)

The Flutter branding logic is centralized in lib/utils/ui_utils.dart. After deployment, users on web may need to refresh their browser cache to see the updated colors in the marketing ads screen.
