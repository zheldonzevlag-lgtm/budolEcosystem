User Manual

Guide for viewing and interacting with the restored Budolshap storefront.

Product Catalog

Users can now browse the full product catalog. All items, including categories like "Electronics", "Fashion", and "Socks", are now successfully indexing and displaying their respective images.

Searching & Filtering

The search filter in the sidebar is fully functional. Users can filter by Category or Shipping Location. The "Budol Premium Socks" items should now appear correctly when searching or selecting relevant categories.

Checkout Data Consistency

Address validation has been optimized. Users with incomplete legacy address data (missing barangay) can now access the checkout flow without errors, as the system provides graceful fallbacks for these fields.
