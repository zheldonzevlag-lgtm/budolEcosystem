v26 Issues & Fixes

Issue: Missing Login Bot Protection

While registration was protected by MathCaptcha in v25, the Login flow remained vulnerable to automated brute-force attempts.

Fix: Unified Shield Challenge

Integrated `MathCaptcha` into the `AuthForm` as a top-level gatekeeper that applies to all modes (Login, Quick Reg, Standard Reg).

Issue: HTML Validation Warning (Nested Forms)

The original `MathCaptcha` used an internal `

`, which when rendered inside `AuthForm`, created a nested form structure invalid in HTML5.

Fix: Div-based Verification

Refactored the captcha to use a standard `div` and manual event listeners, eliminating the nesting error.
