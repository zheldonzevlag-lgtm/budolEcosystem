Fail-Safe Initialization Pattern

The FaceEmbeddingService now implements a defensive health check before every inference operation.

// Health Check Logic in generateEmbedding
try {
  _interpreter!.getInputTensors(); // Proactive check for native validity
} catch (e) {
  _isModelLoaded = false;
  _interpreter = null;
  await _initModel(); // Self-healing re-initialization
}

Synchronization Improvements

The _initModel method now uses a finally block to reset the _loadingCompleter, ensuring that transient failures do not permanently block the service.
