Budol Ecosystem v374

Release Notes

This version implements a "Fail-Safe" architecture for Face Recognition:

  • Fix: Eliminated initialization race conditions using optimized Completer synchronization.
  • Robustness: Added a Health Check system that detects invalid native interpreters and triggers automatic re-initialization.
  • Compatibility: Disabled NNAPI by default to prevent native driver crashes on specific hardware (Samsung/MediaTek).
  • Stability: Switched to a versioned model file path (facenet_v2.tflite) to ensure clean extraction on upgrades.
