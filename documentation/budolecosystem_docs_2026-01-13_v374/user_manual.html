Self-Healing Face Registration

We have introduced a "Self-Healing" mechanism for the Face Login feature. If the internal scanner encounters an issue, it will now automatically attempt to restart itself without crashing the app.

Updates in this version:

  • Fix: Eliminated the "Interpreter is null" error seen on some devices.
  • Performance: Faster and more stable face detection.
  • Compatibility: Improved support for older Android hardware.
