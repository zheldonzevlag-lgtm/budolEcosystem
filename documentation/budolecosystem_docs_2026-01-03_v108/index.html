Mobile UI Restoration & Brand Consistency Update

Release Overview - 2026-01-03

Version 108 focuses on restoring the original brand identity of the budolPay Mobile app. After identifying discrepancies in the UI and app icons, we have successfully restored the signature Rose Red theme and the Philippine Peso (₱) branding as per historical specifications.

  • UI Restoration: Re-implemented the Rose Red (#F43F5E) and Slate Gray (#94A3B8) theme.
  • Icon Recovery: Extracted original app icons from backup build intermediates.
  • Brand Alignment: Restored the circular Peso badge logo on the login screen.
  • Backend Sync: Maintained end-to-end payment flow fixes from v107.
