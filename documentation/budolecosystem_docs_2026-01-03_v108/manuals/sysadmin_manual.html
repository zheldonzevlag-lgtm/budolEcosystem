SysAdmin Manual - v108

1. APK Deployment Guide

The mobile app icons have been restored. To generate the updated APK for distribution:

cd budolPayMobile
flutter pub get
flutter build apk --debug

The resulting file will be located at: build/app/outputs/flutter-apk/app-debug.apk

2. Network Configuration

Ensure the API Gateway is running on the host machine to allow mobile app connectivity:

  • API Gateway IP: 192.168.1.24 (Internal LAN)
  • API Gateway Port: 8080
  • Services Proxied:
    • Auth Service: /auth/*
    • Wallet Service: /wallet/*
    • Payment Gateway: /payment/*

3. Icon Verification

System administrators should verify that the app icon appears correctly on the Android home screen after installation. The icons were restored to the android/app/src/main/res/mipmap-* directories.
