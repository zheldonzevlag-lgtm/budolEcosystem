Purpose & Rationale

Standardize branding across all system-generated identifiers (Order IDs, Booking IDs, etc.) and ecosystem references (budolPay, budolShap, etc.) to ensure a unified user experience and brand identity across React, Flutter, and Email modules.

Technical Implementation

1. React Branding Component

The BudolPayText component uses a regex pattern to identify and style ecosystem keywords.

File: BudolPayText.jsx

const pattern = /(budol)(_?)([a-zA-Z₱]+)/gi;
// Styles "budol" with slate-700 and suffix with specific ecosystem colors

2. HTML Email Branding Helper

A new helper function formatBrandedText was added to handle branding in server-side generated HTML emails.

File: email.js

const formatBrandedText = (text) => {
    const pattern = /(budol)(_?)([a-zA-Z₱]+)/gi;
    return text.replace(pattern, (match, p1, p2, p3) => {
        // Returns <span> with inline styles matching React colors
    });
};

3. Flutter Branding Utility

Flutter uses UIUtils.formatBudolPayText to render RichText with brand styling.

Changes

  • Wrapped Order IDs in OrderItem.jsx, OrderHeader.jsx, and Admin views with BudolPayText.

Compliance
