v23: Vercel Production Restoration & Data Parity

Release Date: 2026-03-26 STABLE

Purpose

This release documents the successful emergency restoration of production data to the Vercel-hosted BudolShap ecosystem. It marks the transition from "0 products" to a fully populated storefront with functional user authentication across all 13 microservices.

Summary of Achievements

  • Data Restoration: Synchronized 14 products and 97 categories from local backups to Vercel Postgres schemas using targeted Node.js sync scripts.
  • Environment Parity: Configured 7+ service-specific environment variables for `budolid`, `budolpay`, and `budolaccounting` schemas.
  • Auth Recovery: Restored `User` and `Wallet` records, enabling secure login and payment ledger functionality.
  • Live Verification: Confirmed 100% storefront visibility on https://budolshap.vercel.app.
