Release v7.0.0 ECS Fargate

ECS Fargate Container Migration

Migration of budolEcosystem backend microservices to AWS ECS Fargate for scalable, managed container orchestration.

Release Summary

This major release focuses on deploying all 9 backend microservices to AWS ECS Fargate. Key features include:

Key Objectives

  • Deploy 9 microservices to ECS Fargate
  • Configure Application Load Balancers (Internal & External)
  • Set up container-to-container communication
  • Integrate with existing RDS PostgreSQL and Redis
  • Implement SSM Parameter Store for configuration

Microservices Deployed

  • budolID (SSO) - Port 8000
  • budolAccounting - Port 8005
  • budolPay Gateway - Port 8080
  • budolPay Auth - Port 8001
  • budolPay Wallet - Port 8002
  • budolPay Transaction - Port 8003
  • budolPay Payment - Port 8004
  • budolPay KYC - Port 8006
  • budolPay Settlement - Port 8007

Cloud Architecture Overview (v7.0.0)

        |
        +--> budolid.duckdns.org (Port 8000)

        +--> accounting.duckdns.org (Port 8005)

        [Internal ALB] -> [ECS Services (Private)]
               |
