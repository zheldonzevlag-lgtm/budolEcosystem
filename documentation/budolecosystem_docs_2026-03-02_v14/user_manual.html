User Manual: Accessing budolEcosystem

Guide for end users accessing the budolEcosystem services deployed on ECS Fargate.

Accessing Services

External URLs

Access budolEcosystem services through these public URLs:

https://budolid.duckdns.org - SSO/Authentication Service
https://budolpay.duckdns.org - Payment Gateway API
https://accounting.duckdns.org - Accounting Service
https://budolshap.duckdns.org - Main Application (Amplify)

Mobile App Access

The mobile application connects to the following endpoints:

  • API Base URL: https://budolpay.duckdns.org/api
  • Auth Endpoint: https://budolid.duckdns.org
  • WebSocket: wss://budolws.duckdns.org

Authentication

Login Flow

  1. User opens the application or mobile app
  2. Redirected to https://budolid.duckdns.org/login
  3. User enters credentials
  4. SSO service validates credentials against database
  5. JWT token returned and stored
  6. Token used for subsequent API calls

API Authentication

Include the JWT token in the Authorization header:

Authorization: Bearer

Payment Services

Payment Gateway API

The budolPay Gateway handles all payment operations:

Endpoint Method Description
/api/payment/create POST Create new payment
/api/payment/:id GET Get payment details
/api/payment/:id/cancel POST Cancel payment
/api/wallet/balance GET Get wallet balance
/api/wallet/topup POST Top up wallet

KYC Verification

Verification Process

  1. User submits KYC documents via the app
  2. Documents uploaded to S3
  3. KYC service processes the submission
  4. Admin reviews and approves/rejects
  5. User notified of verification status

KYC API Endpoints

Endpoint Method Description
/api/kyc/submit POST Submit KYC documents
/api/kyc/status GET Get verification status

Settlement Services

Settlement API Endpoints

Endpoint Method Description
/api/settlement/create POST Create settlement
/api/settlement/:id GET Get settlement details
/api/settlement/history GET Get settlement history

Troubleshooting

Common Issues

Login Failed

  • Check your internet connection
  • Verify credentials are correct
  • Clear browser/app cache and try again

Payment Failed

  • Ensure sufficient wallet balance
  • Check card details are correct
  • Verify network connection

KYC Verification Pending

  • Ensure all required documents are uploaded
  • Documents must be clear and readable
  • Wait 24-48 hours for verification
