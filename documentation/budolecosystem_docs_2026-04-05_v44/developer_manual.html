Developer Manual (v1.3.81) - Release v44

This manual details the technical implementation of the currency parsing fix and phone normalization.

1. Centralized Currency Parser

To avoid double.tryParse failures due to localized formatting (commas), we implemented a safe parser in AmountUtils.

View formatters.dart 
static double parse(String text) {
  final cleanText = text.replaceAll(',', '');
  return double.tryParse(cleanText) ?? 0.0;
}

2. Phone Number Normalization

Added normalization to ensure local number inputs match international formats stored in the backend.

View send_money_screen.dart 
final recipient = PhoneUtils.normalizePhoneNumber(recipientInput) 
    ?? recipientInput.replaceAll('-', '');
