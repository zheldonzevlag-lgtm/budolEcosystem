System Admin Manual (v1.3.81)

Build Instructions

  1. Navigate to budolPayMobile directory.
  2. Run flutter build apk --release.
  3. Copy the generated APK from build/app/outputs/flutter-apk/app-release.apk to android-installer/budolPay 1.3.81.apk.

Version Verification

Ensure the splash screen and About section show version 1.3.81+182.
