1. Forensic Audit Testing (Phase 10)

Developers must now use Jest to verify any changes to the audit logging system. The test suite ensures that metadata structures remain compliant with PCI DSS and BSP standards.

Running Tests

# Run all forensic tests npm test scripts/test_scripts/forensic-audit.test.ts

Reference: forensic-audit.test.ts

2. Bootstrap Auditing

Any script that modifies user accounts or security settings must implement forensic logging. The `create-admin-account.js` script serves as the reference implementation.

Implementation Example

await prisma.auditLog.create({ data: { action: 'ADMIN_ACCOUNT_UPDATED', userId: updatedUser.id, entity: 'Security', ipAddress: '127.0.0.1', userAgent: 'Budol Bootstrap Engine/1.0', metadata: { method: 'create-admin-account.js', compliance: { pci_dss: '10.2.2', bsp: 'Circular 808' } } } });

Reference: create-admin-account.js
