Monitoring Cancelled Orders

Administrators can monitor the orders table for orders with status: 'CANCELLED'. A high volume of cancelled orders may indicate issues with the payment gateway or user interface friction.

Database Integrity

The system uses Prisma transactions to ensure that when an order is cancelled, the cart items are restored. Admins should verify that no 'zombie' orders exist in the system (orders with status: 'ORDER_PLACED' but no corresponding payment activity for > 24 hours).
