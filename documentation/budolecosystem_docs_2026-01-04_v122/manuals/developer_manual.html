Developer Manual - Session Persistence Implementation

Overview

Authentication tokens are now persistent. The ApiService now uses SharedPreferences to store the JWT token and user profile locally.

Implementation Details

Modified lib/services/api_service.dart:

// Session loading in constructor
ApiService({this.token, this.user}) {
  if (token == null || user == null) {
    _loadSession();
  }
}
