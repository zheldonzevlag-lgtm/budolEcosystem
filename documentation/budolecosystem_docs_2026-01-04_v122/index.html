budolEcosystem Documentation v122

Date: 2026-01-04

Focus: Session Persistence & Token Lifecycle Management

Purpose

To resolve the "User session expired" issue where users are logged out upon app restart or backgrounding. This implementation introduces local storage persistence for authentication tokens and user data.

Section

Technical Details

  • Package: shared_preferences for encrypted-at-rest-equivalent local storage.
  • Persistence Logic: Tokens are saved during login and loaded automatically during ApiService initialization.
  • Lifecycle Management: Implemented _loadSession, _saveSession, and _clearSession helpers.

Test Results

Automated tests in test/api_service_test.dart confirm that sessions are correctly loaded from mocked storage and cleared upon logout.
