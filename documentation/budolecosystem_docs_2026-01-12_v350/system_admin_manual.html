System Admin Manual - v350

Infrastructure Compatibility

The v350 update introduces enhanced web deployment stability. Admins should ensure that the index.html file in the web build directory contains the latest removeSplashFromWeb function to prevent deployment-related white screen issues.

Monitoring

Monitor application startup times. The 10-second splash removal fallback is a safety measure; if it triggers frequently, investigate potential bottlenecks in the API discovery or initialization process.
