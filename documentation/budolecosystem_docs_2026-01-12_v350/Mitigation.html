Mitigation Strategies - v350

Platform-Specific Stalling

To mitigate the risk of the application hanging on startup, we have implemented a multi-layered removal strategy for the splash screen:

  1. Application-Level: Flutter code triggers removal via dart:js.
  2. Platform-Level: index.html includes a hard-coded fallback timeout.
  3. Logic-Level: ApiService.init() includes a 10-second timeout to prevent network issues from blocking the UI indefinitely.
