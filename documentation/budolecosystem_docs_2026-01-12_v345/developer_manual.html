Logout Messaging Implementation

This update focuses on improving the user experience during logout in the Flutter application. The key change is the addition of SnackBar notifications and proper await handling for the logout process.

Modified Code: Settings Screen

File: settings_screen.dart

The logout button's onPressed handler was updated to be async and include a try-catch block.


Expanded(
  child: ElevatedButton(
    onPressed: () async {
      try {
        final apiService = Provider.of<ApiService>(context, listen: false);
        await apiService.logout();
        if (context.mounted) {
          Navigator.pushNamedAndRemoveUntil(context, Routes.login, (route) => false);
          ScaffoldMessenger.of(context).showSnackBar(
            const SnackBar(
              content: Text('Logged out successfully'),
              backgroundColor: Colors.green,
            ),
          );
        }
      } catch (e) {
        if (context.mounted) {
          ScaffoldMessenger.of(context).showSnackBar(
            SnackBar(
              content: Text('Logout failed: $e'),
              backgroundColor: Colors.red,
            ),
          );
        }
      }
    },
    // ... styling
  ),
),

Core Service: ApiService

File: api_service.dart

The logout function ensures that all session-related data is cleared from SharedPreferences.


Future<void> logout() async {
  token = null;
  user = null;
  await _clearSession();
  notifyListeners();
}

Future<void> _clearSession() async {
  final prefs = await SharedPreferences.getInstance();
  await prefs.remove(_tokenKey);
  await prefs.remove(_userKey);
}
