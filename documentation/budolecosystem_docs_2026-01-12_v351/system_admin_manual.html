System Admin Manual - v351

Persistence Management

The v351 update utilizes SharedPreferences for tracking marketing ad views. If a user needs to see the marketing slides again for support or testing purposes, clearing the application data/cache will reset the budolpay_has_seen_ads flag.

Deployment Verification

Admins should verify that the navigation flow Splash -> Marketing -> Login works correctly on fresh installations. For web deployments, ensure the SharedPreferences interop (via local storage) is functional.
