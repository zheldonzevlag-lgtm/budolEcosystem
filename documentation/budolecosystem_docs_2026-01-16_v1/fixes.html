Issues & Fixes

Issue: Post-Splash Black Screen on Web

Description: After the splash screen, the app displays a dark slate background instead of the marketing ads.

Fix: Added Positioned.fill to PageView.builder in MarketingAdsScreen and the app Navigator in main.dart to ensure they expand to fill the web container.

Issue: Rapid Splash Removal Race Condition

Description: The HTML splash screen was being removed before the Flutter UI was ready to render the next frame.

Fix: Introduced a 100ms delay before calling removeSplashFromWeb and a 50ms delay after, providing a buffer for UI synchronization.
