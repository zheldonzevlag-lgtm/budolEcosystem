Mitigation Plan

M-UI-001: Rendering Fallback

Risk: UI widgets fail to render due to missing constraints.

Mitigation: Implement a global ErrorWidget.builder that redirects users to the login screen if a rendering error occurs, preventing a permanent "black screen" state.

M-UI-002: Transition Synchronization

Risk: Race condition between HTML splash removal and Flutter rendering.

Mitigation: Added Future.delayed calls in the splash screen navigation logic to ensure the browser has time to process the JS call before the next route is pushed.
