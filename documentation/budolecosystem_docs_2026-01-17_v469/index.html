BudolEcosystem Documentation v469

Release Date: 2026-01-17

Status: Compliance Verified

Purpose

This version resolves the issue where Marketing Ads were being skipped due to persistent "hasSeenAds" flags in the browser's local storage. It also adds diagnostic logging to verify navigation decisions.

Implementation Phases

  • Phase 0: Analyzed navigation logic in SplashScreen and identified dependency on hasSeenAds flag.
  • Phase 1: Implemented a "Force Reset" for the hasSeenAds flag in Web Debug environments within main.dart.
  • Phase 2: Enhanced diagnostic logging in SplashScreen and MarketingAdsScreen to track navigation flow.
