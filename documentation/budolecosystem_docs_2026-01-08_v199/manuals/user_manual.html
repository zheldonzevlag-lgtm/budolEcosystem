Setting Up Your Security PIN

New: Secure PIN Setup

If you haven't set up a security PIN yet, budolPay will now guide you through the setup process immediately after you verify your phone number. This ensures your account is always protected by a code known only to you.

How to Set Your PIN

  1. Verify your mobile number using the 6-digit OTP sent via SMS.
  2. If you don't have a PIN, you'll be automatically asked to "Create a Secure PIN".
  3. Enter a 6-digit number that is hard to guess (avoid using birthdays or simple sequences like 123456).
  4. Confirm your PIN.
  5. You're all set! Use this PIN for future logins and to authorize transactions.

Security Tip

We recommend changing your PIN every 90 days. You can do this from the Security Settings menu in the app.
