Release Summary: v199 (Missing PIN Recovery & Setup Flow)

This version addresses the "stuck" state where users on the PIN screen could not proceed because their account lacked a PIN. It introduces a dedicated PIN setup flow that triggers after successful OTP verification if a PIN is missing.

Key Improvements

  • PIN Setup Endpoint: Added /login/mobile/setup-pin to allow secure PIN creation for existing accounts.
  • Intelligent Routing: The /verify-otp endpoint now returns a needsPinSetup flag, signaling the mobile app to redirect the user to the PIN creation screen.
  • Security: PIN setup is only accessible after a successful identity verification (OTP) and provides 6-digit validation.
