User Manual v187

Purpose

This manual guides end-users and administrators on how to access and use the Budol Ecosystem services in the production environment.

Accessing the Services

Important: All services are currently protected by Vercel Deployment Protection. You must be logged into your Vercel account to view the applications.

Service Links

  • Budol Marketplace (Shap): budolshap-010.vercel.app
  • Admin Dashboard: budolpay-admin.vercel.app

User Workflows

Marketplace Access

  1. Navigate to the Budol Marketplace link.
  2. If prompted, log in with your Vercel credentials to bypass Deployment Protection.
  3. Browse products and manage your cart.

Admin Dashboard

  1. Access the Admin Dashboard via the provided link.
  2. Manage transactions, wallet balances, and system settings.

Issues & Troubleshooting

If you encounter a "508 Infinite Loop Detected" error, please use the direct deployment URLs provided in the System Admin manual instead of the aliased links.
