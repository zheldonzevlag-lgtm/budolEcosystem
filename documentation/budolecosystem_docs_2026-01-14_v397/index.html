Release v1.2.88 v397

Release Summary

This release (v1.2.88) introduces the Zero-Action Face Login, which eliminates manual photo capture for users with registered face profiles, opting instead for seamless OS-level biometric verification.

  • ✔ Phase 0: Baseline Analysis of fallback chain.
  • ✔ Phase 1: Implemented App Face -> OS Face -> OS Biometric fallback.
  • ✔ Phase 2: Eliminated manual capture for registered users.
  • ✔ Phase 7: Built and deployed APK v1.2.88.

Zero-Action Philosophy

  • No Manual Capture: Users no longer need to click a shutter button if they have an app-level or OS-level face profile.
  • Smart Fallback: Automatically scales down to Fingerprint if Face recognition is unavailable but generic biometrics are enabled.
  • Seamless UX: The Face Icon now triggers the most secure and convenient biometric available on the device.
