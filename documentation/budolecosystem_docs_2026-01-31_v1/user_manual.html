User Manual - v1.3.52

What's New?

We've updated the Transaction Details screen to make it easier to read your payment history.

New Alignment: The Store Name is now centered and displayed in a clear, readable font.

How to View Transaction Details

  1. Open the BudolPay app.
  2. Go to your Transaction History.
  3. Tap on any transaction to see the full details, including the merchant name, amount, and reference ID.
