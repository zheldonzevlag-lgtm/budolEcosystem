System Admin Manual - v1.3.52

Deployment Instructions

This update is a client-side UI change for the BudolPay Mobile application. No backend or database migrations are required.

  1. Ensure the latest Flutter environment is configured.
  2. Run flutter clean and flutter pub get.
  3. Build the release APK using flutter build apk --release.
  4. Deploy the APK to the android-installer directory.

Verification

Administrators can verify the update by opening any transaction record in the mobile app. The merchant/store name should be perfectly centered below the payment icon.
