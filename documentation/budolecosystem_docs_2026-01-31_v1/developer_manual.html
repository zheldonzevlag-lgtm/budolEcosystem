Developer Manual - v1.3.52

Technical Purpose

The goal of this update is to improve the visual hierarchy and density of the Transaction Details screen. Specifically, the store name (merchant) is now centered and uses a standardized 12px font size.

Code Changes

1. Transaction Details Screen UI

Modified the UIUtils.formatBudolPayText call to include textAlign: TextAlign.center and verified fontSize: 12.

File: transaction_details_screen.dart

UIUtils.formatBudolPayText(
  isIncome 
    ? ... 
    : (widget.transaction['storeName'] ?? widget.transaction['description'] ?? 'Money Sent'),
  baseStyle: const TextStyle(fontSize: 12, fontWeight: FontWeight.bold, color: Color(0xFF1E293B)),
  textAlign: TextAlign.center,
),

Rationale

Centering the merchant name provides a more professional, "bank-grade" appearance similar to modern fintech apps. The 12px font size ensures clarity while maintaining high information density.

Dependencies

  • Flutter SDK >=3.0.0
  • Provider 6.0.5
  • Custom UIUtils helper in lib/utils/ui_utils.dart
