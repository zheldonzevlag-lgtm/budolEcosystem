User Manual - v153

System Environment Management

The System Environment page has been restored. Admin users can now manage ecosystem-wide settings.

Restored Features

  • Add New Setting: Create new environment variables for specific apps (BudolPay, BudolID, etc.) or Global scope.
  • Secret Management: Mark sensitive values as "Secret" to mask them in the UI.
  • Sync to Vercel: Push local configurations to the Vercel production environment.

How to access

Navigate to Settings in the sidebar or go directly to /settings. You should see the "Ecosystem Control Center" section with your active settings.
