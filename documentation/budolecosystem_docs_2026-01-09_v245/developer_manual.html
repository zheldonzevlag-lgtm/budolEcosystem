Developer Manual: Marketing Integration

1. New MarketingAdsScreen

A new screen MarketingAdsScreen was implemented in lib/screens/marketing_ads_screen.dart. It uses a PageView to display three marketing slides with the following specifications:

  • Slide 1 (budolLoan): Quick loans focus (Emerald theme).
  • Slide 2 (budolShap): Ecosystem shopping focus (Blue theme).
  • Slide 3 (budolExpress): Logistics/Remittance focus (Amber theme).

2. Routing Logic

The Routes constant in lib/constants/routes.dart was updated to include /marketing. The MaterialApp in lib/main.dart was also updated to register the new screen.

// lib/constants/routes.dart
static const String marketing = '/marketing';

// lib/main.dart
Routes.marketing: (context) => const MarketingAdsScreen(),

3. Splash Screen Redirection

The SplashScreen navigation logic was modified to always transition to the marketing slides after initialization.

// lib/screens/splash_screen.dart
Navigator.pushReplacementNamed(context, Routes.marketing);

4. Navigation Post-Marketing

The MarketingAdsScreen handles the final transition based on authentication status:

void _proceedToApp() {
    final apiService = context.read<ApiService>();
    if (apiService.isAuthenticated) {
      Navigator.pushReplacementNamed(context, Routes.home);
    } else {
      Navigator.pushReplacementNamed(context, Routes.login);
    }
}
