Developer Manual - v455

Splash Navigation Status

Added a reactive _navStatus string to the SplashScreen to provide visual feedback during the asynchronous initialization process. This helps diagnose exactly which stage of the initialization is currently active.

splash_screen.dart 
setState(() => _navStatus = 'Checking configuration...');
await apiService.init();
setState(() => _navStatus = 'Navigating to Ads...');

Persistent State Logging

Added detailed logging in ApiService to track the value of hasSeenAds immediately after it is loaded from SharedPreferences. This is critical for identifying if the state is being incorrectly persisted or retrieved.

api_service.dart 
_hasSeenAds = prefs.getBool(_hasSeenAdsKey) ?? false;
if (kDebugMode) print('ApiService: Loaded hasSeenAds: $_hasSeenAds');

Version Increment

Incremented the app version to 1.3.7. This serves as a "canary" change to ensure the user is seeing the latest build in their browser/environment.
