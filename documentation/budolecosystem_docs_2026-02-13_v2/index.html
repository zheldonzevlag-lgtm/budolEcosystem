This release focuses on unifying phone number handling across the ecosystem, refining the seller workflow navigation, and ensuring UI consistency in the product management dashboard.

Compliance Statement

All changes have been reviewed for compliance with:

  • NPC (National Privacy Commission): Secure handling of PII (Phone numbers).
  • BSP (Bangko Sentral ng Pilipinas): Standardized formatting for financial identifiers.
  • Cybersecurity Law: Reduced attack surface by sanitizing user inputs.
