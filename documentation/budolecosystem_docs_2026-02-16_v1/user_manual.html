User Manual

This guide explains how to use the Budol Ecosystem Admin Dashboard.

1. Dashboard Overview

The Admin Dashboard is your central hub for monitoring system activity and security. It provides a real-time view of what's happening in the system, including user actions and security alerts.

Key Features

  • Audit Logs: View a detailed history of all user actions.
  • Security Alerts: Get notified of potential security threats.
  • User Activity: Track user logins and interactions.

2. Understanding Audit Logs

Audit logs are a record of every significant action taken within the system. They are used for security and compliance purposes.

What's in a Log Entry?

  • Who: The user who performed the action.
  • What: The action that was performed (e.g., "Created User").
  • When: The date and time the action occurred.

3. Viewing Security Compliance

The Security Compliance Dashboard shows the system's current security posture. It highlights any vulnerabilities or threats that need attention.

Interpreting the Data

  • Green: Secure/Compliant.
  • Yellow: Warning/Review Needed.
  • Red: Critical/Action Required.
