Using the Collapsible Sidebar

To access the menu options, simply hover over the icons in the sidebar. The menu names will appear in a tooltip.

If you prefer to see the full menu names, click the arrow icon at the top of the sidebar to expand it.

To collapse the sidebar and save space, click the arrow icon again.

Your profile initials are displayed below your avatar when the sidebar is collapsed.
