System Admin Manual - v3.0.3

Dashboard Navigation Updates

The admin dashboard now features a collapsible sidebar to maximize workspace area.

  • Toggle: Use the arrow icon at the top-right of the sidebar to expand/collapse.
  • Default State: The sidebar is collapsed by default upon login/refresh.
  • Collapsed View: Displays icons only. Hover over an icon to see the menu name in a tooltip.
  • User Profile: Shows user initials below the avatar when collapsed.
