Risk Register

Identification of risks associated with the dynamic realtime provider system.

Risk ID Description Impact Severity
R-495-01 Invalid credentials for Pusher/Socket.io saved in admin panel. Realtime updates fail for all users. HIGH
R-495-02 External provider (Pusher) downtime. Loss of instant updates. MEDIUM
R-495-03 Cache stale data (Settings cached for 60s). Delayed switching of providers after update. LOW
R-495-04 Socket.io microservice resource exhaustion. Slow or failing triggers. MEDIUM
