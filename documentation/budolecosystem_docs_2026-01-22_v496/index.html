Store Notification Fix & Tab Routing Optimization

v496 | Released: 2026-01-22

Purpose

This version addresses critical issues where sellers were not receiving realtime order update notifications and orders were not syncing to the correct status tabs in the UI. Key improvements include session-based store context optimization, realtime event normalization, and intelligent SWR cache patching for seamless tab transitions.
