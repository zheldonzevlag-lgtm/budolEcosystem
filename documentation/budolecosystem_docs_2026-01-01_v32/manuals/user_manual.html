User Manual: Enhanced Login Experience

Getting started with the new ecosystem-prioritized login interface.

1. Using "Login with budolID"

We've made it easier to access your ecosystem account. The "Login with budolID" button is now located at the very top of the login window.

1

Open the login modal or page.

2

Look for the Quick Access section at the top.

3

Click the Login with budolID button to use your unified account.

2. Why use budolID?

Using your budolID account provides several benefits:

  • One Account: Access budolShap, budolpay, and budolAccounting with a single login.
  • Unified Profile: Your information and settings are synchronized across all apps.
  • Secure Access: Enhanced security features powered by the core ecosystem authority.

3. Compact Dashboard View

The budolPay admin dashboard has been optimized for better visibility. The new layout is more compact, allowing you to see more information without excessive scrolling.

  • Consolidated Stats: Key performance indicators are now presented in a streamlined grid.
  • Improved Information Density: Critical transaction and compliance data are now more accessible.
  • Optimized Layout: The "Compliance Shield" has been narrowed to prioritize the "Transaction Pulse" view, providing a better balance of information.
  • Better on Small Screens: The layout adapts more gracefully to different window sizes.
