Deployment Configuration

Ensure the following environment variables are correctly set in budolshap-0.1.0/.env to allow seamless communication across the ecosystem:

NEXT_PUBLIC_SSO_URL="http://192.168.1.24:8000"
NEXT_PUBLIC_APP_URL="http://192.168.1.24:3001"
NEXT_PUBLIC_BUDOLID_API_KEY="bs_key_2025"

Common Issues

If you see FATAL: the database system is in recovery mode, please wait 30-60 seconds for PostgreSQL to complete its internal consistency checks. This usually occurs after a hard restart of the service.
