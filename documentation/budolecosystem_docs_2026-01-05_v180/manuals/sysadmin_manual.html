System Admin Manual: KYC Image Processing

Infrastructure Changes

None. No database migrations or environment variable changes were required for this update.

Data Integrity

Images are stored as BYTEA (PostgreSQL) or BLOB (MySQL/SQLite) in the VerificationDocument table. This update fixes the presentation layer without altering how data is stored or received.

Troubleshooting

If an administrator reports that images are still not showing:

  1. Ask for a screenshot of the "Debug" text shown in the document viewer.
  2. Verify that the verification-service is running and successfully processing uploads (check pm2 logs or container logs).
