User Manual: Admin KYC Review

Feature Update

The "Users" section in the Admin Dashboard now correctly displays identity documents submitted by users for KYC verification.

How to Review KYC Documents

  1. Navigate to the Users page.
  2. Locate a user with a "PENDING" KYC status.
  3. Click the Eye (View Documents) icon in the actions column.
  4. The document viewer will open, displaying the uploaded ID photos.
  5. You can now visually verify the user's identity and click Approve KYC or Reject KYC accordingly.

Common Issues

  • Image is slow to load: High-resolution images may take a few seconds to process depending on your computer's performance.
  • Placeholder showing: If you see a file icon with "Debug" text, the image data might be corrupt or in an unsupported format. Please contact technical support.
