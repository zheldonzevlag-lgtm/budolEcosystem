Deployment Guide

Mobile App Deployment

Version: 1.3.70

Path: \budolEcosystem\android-installer\budolPay 1.3.70.apk

  1. Upload the APK to the public download site.
  2. Update the download link in the mobile splash redirect service.
  3. Notify users of the new version for critical bug fixes.

Backend Server Update

Ensure that the latest Prisma client is generated after any schema changes.

npx prisma generate

Verify that DATABASE_URL is correctly set in .env for the production PostgreSQL instance.

Maintenance Tasks

Database Health

Regularly check for orphaned items in the CartItem table. Although the API now filters these out, it is recommended to run a periodic cleanup script for old abandoned carts with non-existent products.
