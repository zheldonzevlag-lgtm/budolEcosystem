Issues & Fixes - 2026-02-26 v1

ISSUE: 500 Error when syncing cart with deleted products.

Impact: Buyers with products in their cart that were deleted by a seller were unable to checkout or view their cart.

FIX: Added a server-side filter to /api/cart PUT route to remove non-existent product IDs before database insertion.

ISSUE: NaN% display on Hero section discounts.

Impact: Incorrect calculation of discount percentage when a product had 0 or missing SRP (mrp).

FIX: Added a filter in Hero.jsx to only include products with valid discounts (mrp > price) and a fallback to 0%.

ISSUE: Variation Matrix "Argument 1 could not be converted" crash.

Impact: The VariationMatrixManager would crash when trying to preview an image that was not a valid Blob or File.

FIX: Updated VariationMatrixManager to perform strict type-checking on the image object before calling URL.createObjectURL().

ISSUE: Prisma "Unknown argument 'hidden_combos'" error.

Impact: Failed product creation/updates because the cached server client didn't recognize the new field.

FIX: Updated API routes to build data objects dynamically and removed custom client output paths to ensure better synchronization.
