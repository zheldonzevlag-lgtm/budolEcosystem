System Admin Manual - v347

Release Version: 1.2.55

Platform: Flutter Web / Mobile

Web Deployment Notes

This release fixes a critical UI hang on web browsers. Ensure that the budolPay web server is started using the npm start command which includes the --web-hostname 0.0.0.0 flag for LAN accessibility.

Verification Steps

  1. Access the app via http://[IP_ADDRESS]:9000.
  2. Observe the Splash Screen (v1.2.55).
  3. Verify that the "Page Not Responding" dialog does not appear.
  4. Confirm navigation to Login or Home within 5-10 seconds.
