System Admin Manual - v433

System Configuration

This update is a code-level change in the BudolPay adapter. No configuration changes are required in the environment variables or database for this release.

Verification Steps

Administrators can verify the change by:

  1. Navigating to the checkout page on budolShap.
  2. Selecting "BudolPay QR" as the payment method.
  3. Clicking "Place Order".
  4. In the resulting QR Modal, confirm that "budolShap Owner" displays Jon Galvez.

Dependencies

  • Node.js environment
  • BudolPay API access
  • QR Code generation service (api.qrserver.com)

Monitoring

Monitor the payment-checkout API logs for any anomalies in QR generation, although none are expected as this is a string value change only.
