budolEcosystem Docs v257

Date: 2026-01-09 | Version: v257

Overview

This release fixes critical real-time communication issues. The Admin Panel's provider switching logic was fixed using a Client Component. The mobile app's Socket.io initialization was corrected to use the server-provided URL, and the transaction service was updated to ensure notifications are sent for cash-in events.

Key Changes

  • Admin Panel: Introduced RealtimeMethodSelector Client Component to allow seamless switching between SWR, Socket.io, and Pusher.
  • Mobile Realtime Service: Updated RealtimeService.dart to dynamically read the SOCKETIO_URL from system settings instead of defaulting to the base URL.
  • Transaction Service: Fixed cash-in route to call notifyUpdate, ensuring users receive live updates when topping up.
  • Mobile Build: Incremented version to 1.2.22+22.
