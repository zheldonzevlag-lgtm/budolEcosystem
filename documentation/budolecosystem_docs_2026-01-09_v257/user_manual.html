User Manual v257

Improved Real-time Updates

You no longer need to manually refresh the transaction history page to see your latest activities. New transactions and cash-in confirmations will now appear automatically at the top of the list.

Cash-In Confirmation

When you successfully top up your wallet, a confirmation notification will appear instantly within the app, and your balance will update without requiring a restart.

App Version

The current app version is 1.2.22. You can verify this on the splash screen during app startup.
