Developer Manual - BudolEcosystem v1.0.0

System Architecture

The BudolEcosystem is a distributed microservices platform designed for high availability and modular scalability. Component interactions are facilitated through an API Gateway with mDNS service discovery for local environments.

Core Components

  • budolID (SSO): Centralized identity provider using JWT and Prisma.
  • budolShap: The primary e-commerce / marketplace storefront (Next.js).
  • budolPay: Financial service mesh (Monorepo) comprising multiple sub-services (Auth, Wallet, TX, Gateway, Settlement).
  • budolAccounting: Financial record-keeping and ledgering service.

Data Contracts & Prisma Schemas

Shared database structures are managed via Prisma. Primary databases include budolpay, budolid, budolshap, and budolaccounting.

Transaction Entity Example

model Transaction {
  id           String            @id @default(uuid())
  amount       Decimal           @db.Decimal(18, 2)
  type         TransactionType
  status       TransactionStatus @default(PENDING)
  referenceId  String            @unique
  createdAt    DateTime          @default(now())
}
