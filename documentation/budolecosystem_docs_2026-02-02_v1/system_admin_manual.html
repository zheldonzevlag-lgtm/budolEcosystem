System Admin Manual

Administrative Role Management

To promote a user to ADMIN role, use the provided scripts in scripts/test_scripts/budolPay/. Ensure you target the correct database environment.

Warning: Direct database updates bypass application-level validation. Verify changes using the verify_admin_role.test.ts script.

Database Connectivity

All services connect via the following connection string pattern:

postgresql://postgres:r00t@localhost:5432/[DATABASE_NAME]?schema=[SCHEMA_NAME]
