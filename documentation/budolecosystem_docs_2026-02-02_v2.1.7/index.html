Release v2.1.7: Dual-Channel OTP & SSO Resiliency

This release enhances the Self-Service Password Reset (SSPR) flow with dual-channel delivery (SMS + Email) and fixes critical SSO redirection errors.

Key Improvements

  • Dual-Channel OTP: OTPs are now simultaneously delivered via SMS and Email to ensure out-of-band verification redundancy (BSP Circular 808 compliant).
  • SSO Resiliency: Fixed "Unauthorized Application" error by implementing `apiKey` fallbacks for internal ecosystem transitions.
  • Context Persistence: The `apiKey` is now carried through the entire SSPR flow, ensuring users are returned to the correct application after reset.
