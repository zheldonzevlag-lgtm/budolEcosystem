System Admin Manual: SSPR & SSO Configuration

1. OTP Expiration Monitoring

OTPs are valid for 5 minutes. Administrators can monitor delivery logs in the budolID server console to verify that both Email and SMS channels are being triggered correctly.

2. Default Ecosystem Application

The system now defaults to bp_key_2025 (budolPay) for any SSO requests that do not specify an apiKey. This ensures that administrative access remains available even if external links are broken.

3. Auditing SSPR Events

Password reset attempts are logged on the server. In future releases, these will be pushed to the centralized audit log (PCI DSS requirement).
