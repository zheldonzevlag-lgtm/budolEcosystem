User Manual Updates - v276

Product Management Improvements

Reliable Image Uploads

We have improved the stability of image uploads when adding or editing products. The system now automatically compresses large images before uploading them, ensuring a faster and more reliable experience even on slower connections.

Clearer Error Messages

If an upload fails, you will now see a more specific error message instead of a generic "Failed to fetch". This will help you understand if the issue is related to your internet connection, the image size, or a server issue.

Tip for Store Owners: For the best results, use images in JPG or PNG format. The system will handle the conversion and optimization for you.

Workflow Changes

No changes to the existing workflow. Users should continue using the "Add Product" form as usual.
