Developer Manual - Release 1.0.6

Overview

Release 1.0.6 focuses on UI clarity and session stability. The transaction flow now clearly identifies the merchant at the top of the details list.

Release Assets

The release APK is located at: android-installer/budolPay 1.0.6.apk

Build Instructions

# Update version in pubspec.yaml
version: 1.0.6+1

# Build command used
flutter build apk --release --no-tree-shake-icons
