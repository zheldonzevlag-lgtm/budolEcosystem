System Administrator Manual

Accessing the Security Dashboard

Navigate to /admin/security/dashboard. Access is restricted to users with `ADMIN` or `SUPER_ADMIN` roles.

Monitoring Audit Logs

Go to /admin/forensic-audit-trails. The list shows all system activities.

  • Green rows: Normal activity.
  • Yellow rows: Warnings (e.g., failed logins).
  • Red rows: Critical alerts (e.g., integrity check failure).

Database Backups

Automated backups are stored in \backup-db\db-[date]. Ensure these are rotated weekly.
