Mitigation Strategies

Audit Integrity

Threat: Database administrator modifying logs.

Mitigation: Application-level hashing ensures that any modification to the database record invalidates the `integrityHash`. Regular integrity checks (planned feature) will scan for mismatches.

Availability (DoS)

Threat: Botnet flooding the login endpoint.

Mitigation: Middleware tracks IP hits. If > 100 req/min, IP is temporarily banned (429 Too Many Requests).
