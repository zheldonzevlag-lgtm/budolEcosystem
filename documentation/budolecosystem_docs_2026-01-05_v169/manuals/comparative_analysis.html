UX Research Insights (GCash)

  • Trust through Transparency: GCash clearly displays verification status badges (e.g., "Fully Verified") on the profile to build user trust.
  • Accessibility: High-contrast icons and clear labels for financial actions (Cash In, Send, Pay).
  • Security Friction: Purposeful friction (OTP, Biometrics) during high-value transactions or settings changes.

Achievements (v169)

  • ✅ Integrated ML Kit: On-device OCR and Face Detection for KYC flow.
  • ✅ Wallet Quick Actions: Full implementation of Cash In, Cash Out, and Transfer.
  • ✅ Verification Service: Dedicated microservice for document handling and optimization.

Strategic Next Steps

  1. Biometric Integration: Implement Fingerprint/FaceID for wallet actions to match market standards.
  2. Tiered Limits Backend: Link KYC status directly to transaction limits in the accounting service.
  3. MPIN Recovery: Add secure flows for resetting PIN via SMS/Email.
