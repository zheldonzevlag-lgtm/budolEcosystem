User Manual: Provider Setup

Step-by-step instructions for administrators to configure ecosystem providers.

Email Configuration (Gmail Example)

  1. Log in to the Admin Dashboard.
  2. Click on System Settings in the sidebar.
  3. Scroll down to the Notification Settings section.
  4. Find NOTIFICATION_EMAIL_PROVIDER and select Gmail (Google Workspace).
  5. Enter your full Gmail address in the NOTIFICATION_EMAIL_USER field.
  6. Enter your App Password in the NOTIFICATION_EMAIL_PASSWORD field.
  7. Click the Save icon on the right side of the row.

Payment Gateway Configuration

  1. Navigate to the Payment Gateways section (or use the sidebar link).
  2. Select your active provider (e.g., PayMongo) in ACTIVE_PAYMENT_PROVIDER.
  3. Locate your provider's specific key fields (e.g., PAYMONGO_SECRET_KEY).
  4. Paste your API keys from your provider's dashboard.
  5. Click Save. The keys will be automatically masked for security.
