Consolidated Future Recommendations

A unified repository of all proposed enhancements, architectural improvements, and feature roadmaps across the Budol Ecosystem.

Authentication & Security
WhatsApp OTP Delivery Channel v4

Integrate WhatsApp as an additional delivery channel for verification codes to improve reliability and reduce SMS costs.

IP-Based Geofencing for OTP v4

Implement automated IP-based geofencing to block OTP requests from high-risk regions or non-target markets (e.g., outside PH).

OTP Resend UI Countdown v4

Add a visual countdown timer for the "Resend Code" button to prevent spam and guide user expectations.

OTP (One-Time Password) Support v560

Implement SMS-based OTP login for the mobile number method to enable passwordless entry and improve security for mobile-first users.

Biometric Authentication Integration v560

Integrate FaceID/Fingerprint login for the BudolPay mobile app, mapping biometrics to either email or phone identifiers.

Social Login Integration v560

Add Google and Facebook login options to further streamline onboarding and reduce friction during registration.

AI-Driven Selfie Validation v559

Integrate real-time face detection and liveness checks within the camera preview to improve KYC data quality.

KYC & Compliance
Government API Integration (BIR/DTI) v1.3.7

Automate verification of TIN and Business Registration numbers via official government portals to reduce manual audit workload.

AML/Sanction List Screening v1.3.7

Cross-reference new seller applications against global sanction lists to ensure compliance with international AML regulations.

Just-In-Time KYC Triggers v1.3.7

Prompt for KYC completion when users attempt to activate financial features (Refund to Wallet, P2P Transfer) rather than waiting for withdrawal.

UI / UX Excellence
Enhanced Animation Orchestration v560

Utilize Framer Motion for complex staggered animations and dynamic form resizing to create a more premium experience.

Progressive Onboarding Wizard v559

Transition long registration forms into a multi-step wizard to reduce cognitive load and improve conversion rates.

Skeleton Loading States v559

Implement skeleton loaders for camera and media containers to provide better visual feedback during initialization.

Dark Mode Support v559

Standardize component coloring using CSS variables for full light/dark mode compatibility across all ecosystem applications.
