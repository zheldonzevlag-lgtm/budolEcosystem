Developer Manual

Technical implementation details for the Flutter Web build fix.

Technical Explanation

The tflite_flutter package provides a reshape extension method on the List class. However, in the Flutter Web build context where tflite_flutter is substituted by a stub, this extension was missing. Additionally, Dart's extension resolution can sometimes fail for Float32List unless it is explicitly cast to a List type.

// Fix in tflite_web_stub.dart
extension Reshape on List {
  List reshape(List<int> shape) {
    return this; // Web implementation returns the flat list
  }
}

Phase 8: Dependencies & Software Choices

Dart Conditional Imports

Used to decouple the native TFLite dependency from the Web build, preventing compilation errors in environments where native libraries are unavailable.

TFLite Flutter

Chosen for its efficient on-device AI inference capabilities. The 'reshape' method is critical for structuring image data into the 4D tensors expected by the FaceNet model.
