End-user experience improvements in v401.

Mobile Web Access

The budolPay web application is now fully accessible via mobile browsers. This release fixes a critical stability issue that prevented the web version from loading on some devices.

Note on Face Login:

Face recognition is currently optimized for native Android/iOS apps. Web users should continue using MPIN for authentication.
