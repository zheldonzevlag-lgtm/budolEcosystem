User Manual - v329

Stable Biometric Login

Version 1.2.53 is the final stable version with the biometric login fix. The troubleshooting window has been removed for a cleaner look.

Update to 1.2.53

Please download and install version 1.2.53. This version contains the permanent fix for the fingerprint login issue.

Login with Biometrics

  1. Open the app and enter your phone number.
  2. On the PIN screen, tap the fingerprint icon.
  3. Your phone's security prompt (fingerprint or face) will appear.
  4. Authenticate to instantly log in to your account.
