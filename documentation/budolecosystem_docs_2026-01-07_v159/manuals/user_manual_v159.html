Interface Changes & New Features

The migration to the cloud is a backend-heavy update, but users will notice significant improvements in app stability and performance.

Modified Workflows

  • KYC Verification: Document uploads are now handled via AWS S3, ensuring faster processing and better security for personal data.
  • Real-time Notifications: The new persistent AWS EC2 Gateway ensures that transaction alerts are received instantly, even when the app is in the background.

User-Facing Impacts

Users no longer need to be on the same local network as the developer's PC. The app is now truly global, accessible via api.budolpay.com.
