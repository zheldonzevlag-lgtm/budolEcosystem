Operational Runbook

Environment Variable Update

If the domain changes in the future, ensure the following variables are updated in .env.local and Vercel dashboard:

  • NEXT_PUBLIC_APP_URL
  • NEXT_PUBLIC_BASE_URL
  • NEXT_PUBLIC_SITE_URL

Verification Procedure

1. Start the budolShap service.

2. Perform a test registration.

3. Check the console logs for the "Verification Link".

4. Ensure the domain is budolshap.vercel.app (with an 'o').

Rollback Plan

Revert the changes in .env.local if the previous incorrect domain is still required for legacy reasons (unlikely).
