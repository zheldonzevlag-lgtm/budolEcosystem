System Admin Manual - v101

Infrastructure Changes

The payment-gateway-service now serves as both a REST API and a web server for the checkout UI. No additional ports need to be opened, as it continues to run on port 8004.

Environment Variables

The following environment variable can be configured in services/payment-gateway-service/.env:

Variable Default Description
BASE_URL http://localhost:8004 The base URL used to construct checkout links. Should be set to the server's IP if accessing from a physical mobile device (e.g., http://192.168.1.24:8004).

Service Ports

Service Port Role
Payment Gateway 8004 API + Checkout UI
API Gateway 8080 Central Entry Point

Deployment Notes

Ensure that the qrcode.js CDN (cdn.jsdelivr.net) is accessible from the client's browser to render the checkout page correctly.
