System Admin Manual - v59

Asset Management

No new environment variables or infrastructure changes were required for this version. The GCash icon was switched to a CDN-hosted asset to reduce server load and eliminate 404 errors during local development and production.

Monitoring

Admins should monitor server logs for any 404 errors related to assets. The current fix addresses the most prominent one identified in recent logs.

Rollback Procedures

To rollback the GCash icon change, revert the changes in OrderSummary.jsx to point back to the local path. However, ensure that the public/assets/icons/gcash.png file exists before doing so.
