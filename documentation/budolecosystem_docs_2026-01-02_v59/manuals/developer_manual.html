GCash Icon Fix (OrderSummary.jsx)

Issue: The application was attempting to load a local icon at /assets/icons/gcash.png which did not exist, resulting in a 404 error in the console and server logs.

Fix: Updated the OrderSummary.jsx component to use a reliable external CDN link for the GCash logo directly. This eliminates the initial 404 attempt and ensures the icon is always visible without hosting local assets for every payment method.

// Before
<img src="/assets/icons/gcash.png" onError={(e) => e.target.src = "https://..."} ... />

// After
<img src="https://upload.wikimedia.org/wikipedia/commons/thumb/5/52/GCash_logo.svg/100px-GCash_logo.svg.png" ... />
