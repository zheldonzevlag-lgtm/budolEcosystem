User Manual - v59

Checkout Interface

The checkout interface has been improved to ensure that payment method logos (specifically GCash) load reliably and quickly. Users will now see the GCash logo immediately upon opening the payment summary.

Shipping Quotations

The Lalamove shipping quotation service is fully operational. When placing an order, users will receive accurate and real-time shipping quotes based on their delivery address and the store's location.
