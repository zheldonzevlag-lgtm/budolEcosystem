User Manual - Admin Login Security

Instructions for administrative users regarding the updated BudolPay login security flow.

1

Budol Shield (Math Challenge)

Upon arriving at the login page, you will be presented with a security math challenge. Solve the simple equation (e.g., 5 + 3) and enter the result in the corresponding field.

[Image: Shield Challenge UI Animation]
2

Unlocking Credential Entry

Once the math challenge is cleared, the system will automatically unlock the email, password fields, and the "Login with budolID" button. If the challenge result is incorrect, the system will provide a new math problem.

3

Authentication Selection

Choose your preferred login method: Email/Password or budolID SSO. Follow the standard authentication procedures from there.

Important Notice:

This security measure is mandatory for all production environments to prevent automated bot access. If you experience persistent issues, please contact the System Administrator.
