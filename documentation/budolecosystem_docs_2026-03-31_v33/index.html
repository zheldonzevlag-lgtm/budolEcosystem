BudolPay v33 - Build & Security Stabilization

Version: 33.0.0 Date: 2026-03-31 RELEASED / VERIFIED

Purpose & Rationale

This release addresses the persistent Vercel production build failures related to internal workspace resolution and restores the mandatory Shield Challenge (MathCaptcha) for the Admin Login flow. It ensures the monorepo structure is correctly navigated by Vercel's build pipeline and reinforces bot protection for administrative access.

Key Fixes

3 Internal Workspaces Fixed
1 Security Shield Restored
0 Webpack Failures

Compliance Status (NPC & BSP)

  • Anti-Bot Compliance: Restored MathCaptcha shield challenge for admin login flow.
  • Data Integrity: Verified that `@budolpay/audit` logs are correctly captured during authentication events.
  • BSP Circular 1019: Enhanced authentication security with mandatory automated challenge before credential entry.
