System Admin Manual - Build Deployment v33

1. Vercel Deployment Strategy

Ensure the following environment variables are set in the Vercel Production Environment for correct internal workspace resolution and Prisma engine generation:

PRISMA_GENERATE_SKIP=true (if using remote generation)
NEXT_PUBLIC_SSO_URL=https://budol-id-sso.onrender.com
NEXT_PUBLIC_APP_URL=https://budolpay-admin.vercel.app

2. Cache Management

When deploying v33 changes, a Redeploy with Clean Cache is required in the Vercel dashboard. This forces the Webpack compiler to pick up the updated `package.json` mappings for the internal workspace packages (`@budolpay/*`).

  1. Navigate to Project Dashboard
  2. Go to Deployments tab
  3. Select the latest deployment
  4. Click "Redeploy" and select "Redeploy with Clean Cache"

3. Security Challenge Maintenance

The MathCaptcha challenge is mandatory for production. If bot monitoring shows high fail rates, the challenge is working as expected. Ensure `apps/admin/components/MathCaptcha.tsx` is always present in the build artifacts.
