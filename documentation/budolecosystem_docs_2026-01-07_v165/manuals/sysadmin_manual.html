Deployment Strategy

To deploy any service in the ecosystem, navigate to its respective directory and run the production deployment command.

Example: Deploying budolID

cd "d:\IT Projects\budolEcosystem\budolID-0.1.0"
vercel deploy --prod

Example: Deploying Admin Portal

cd "d:\IT Projects\budolEcosystem\budolpay-0.1.0\apps\admin"
vercel deploy --prod
