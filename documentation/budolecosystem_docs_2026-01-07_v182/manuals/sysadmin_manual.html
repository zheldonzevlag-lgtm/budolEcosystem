Operational Guidelines for Environment Management

Admins must ensure that environment configurations are synchronized across the team without breaking local workflows.

Switching Environments Safely

To switch to the local development environment:

cd budolpay-0.1.0
node scripts/env-manager.js local

To switch to the Vercel production environment:

cd budolpay-0.1.0
node scripts/env-manager.js vercel

Maintenance of Connection Source Files

  • .env.connection.local: Never contains production credentials. Only local docker/localhost URLs.
  • .env.vercel: Contains Vercel Postgres URLs. Managed via Vercel CLI or Dashboard.
