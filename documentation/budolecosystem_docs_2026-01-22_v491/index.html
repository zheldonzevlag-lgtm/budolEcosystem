Release Overview - v491 (2026-01-22)

Release Date: January 22, 2026 | Version: 491

Primary Objective

Automated Post-Payment Status Transition & Realtime Updates. Ensuring seamless order flow from payment to fulfillment with zero-latency UI updates for both buyers and sellers.

Key Changes

  • Realtime Provider Expansion: Added user-channel subscriptions to RealtimeProvider.jsx.
  • Webhook Orchestration: Integrated triggerRealtimeEvent in PayMongo, BudolPay, and Lalamove webhooks.
  • Automated UI Navigation: Implemented setActiveTab('TO_SHIP') in (public)/orders/page.jsx post-payment.
  • Store Dashboard Rebranding: Updated store/orders/page.jsx labels from "Needs Booking" to "To Ship (Pending)".
  • Service Layer Enhancements: Added realtime notifications in shippingService.js during booking operations.

Compliance Status

Verified: BSP (Payment Transparency), NPC (Data Privacy), Cybersecurity Law. Automated status updates reduce manual entry errors and improve financial audit trails.
