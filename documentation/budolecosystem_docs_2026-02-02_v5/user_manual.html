For Admin Users

This manual covers the new features and updates in the Admin Dashboard v2.4.0.

1. Enhanced Employee Management

When updating employee profiles, all changes are now recorded in the forensic audit trail. This ensures that every phone number change, role update, or department reassignment is traceable.

2. Real-time Dashboard

The dashboard now automatically connects to the real-time event stream. You will see updates to transactions and audit logs instantly without needing to refresh the page.

3. Dispute Resolution

Resolving disputes is now a audited action. When you resolve a dispute, the system logs the resolution details and the admin who performed the action.

4. KYC Verification

Updating a user's KYC status (e.g., from Pending to Verified) now triggers a forensic log entry. This is critical for regulatory compliance and fraud prevention.
