Mitigation Strategies

Addressing R-344-01 (User Confusion)

Strategy: Clear user communication.

  • Update the user manual and FAQ to explain that unpaid orders are saved in "My Orders".
  • Future enhancement: Add a toast message or modal on return from payment failure: "Payment not completed. Your order is saved in My Orders."

Addressing R-344-02 (Data Loss)

Strategy: Transactional Integrity.

  • The code clears the cart only if response.ok is true (meaning the order is successfully created in the backend).
  • This ensures that items are never removed unless they are safely stored in an Order record.
