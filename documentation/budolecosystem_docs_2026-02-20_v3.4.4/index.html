v3.4.4: Cart Clearance & Multi-Store Logic

Date: 2026-02-20 Version: v3.4.4 Status: Released

Implementation Summary

This release focuses on fixing the cart retention issue after order placement and enforcing multi-store delivery restrictions.

  • Cart Clearance: Removed the restriction that prevented cart clearance for asynchronous payments (GCash, Maya, etc.). Now, all successful order creations trigger local cart cleanup.
  • Multi-Store Logic: Verified and enforced restriction on "Same Day Delivery" (Lalamove) for multi-store orders, while allowing Standard Delivery for such cases.
