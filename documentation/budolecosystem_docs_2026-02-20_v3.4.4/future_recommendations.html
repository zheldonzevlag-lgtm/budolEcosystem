Future Recommendations

1. "Restore Cart" Feature

If a payment fails, allow users to one-click "Restore Items to Cart" from their Pending Orders list. This improves UX without risking duplicate orders.

2. Advanced Multi-Store Logistics

Implement a split-booking system for Same Day Delivery where a multi-store order automatically generates multiple Lalamove bookings (one per store pickup). This would remove the restriction on Same Day Delivery for multi-store carts.

3. Automated Integration Tests

Add Cypress or Playwright end-to-end tests that simulate the full checkout flow, including mock payment success/failure, to verify cart state automatically in CI/CD.
