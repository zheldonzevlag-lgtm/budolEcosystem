Developer Manual - v214

Implementation Details

The email system uses nodemailer with a Google SMTP transport. We identified that environment variables in .env.local were wrapped in quotes, which caused authentication issues.

Code Logic Explanations

The sendPasswordResetEmail function in lib/email.js pulls configuration from process.env or the database. We prioritized cleaning both sources.

// Correct formatting in .env.local
SMTP_PASS=ajwppodmhoqbhahkd (No quotes, no spaces)

Impact Analysis

Existing systems were not negatively impacted as these changes only affect the forgot-password and onboarding email flows. However, the app must be restarted to clear cached environment variables.
