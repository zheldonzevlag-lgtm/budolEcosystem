Budol Ecosystem v388

Release Date: 2026-01-14

Release Summary

This release fixes a critical TFLite shape mismatch error in the Face Recognition subsystem. The facenet.tflite model was found to produce 512-dimensional embeddings while the app logic was hardcoded for 128 dimensions. This version updates the embedding generation flow to support the 512-dimensional output, restoring biometric login functionality.
