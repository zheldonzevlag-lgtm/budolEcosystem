System Admin Manual - v3 (2026-02-22)

New Features Overview

Version 3 introduces AI-powered background removal for product variant images. This feature leverages Cloudinary's "Background Removal" add-on.

Cloudinary Configuration

Ensure that the Cloudinary account linked to the application has the Cloudinary AI Background Removal add-on enabled. This is a paid add-on or has usage limits on free tiers.

Configuration Key: CLOUDINARY_URL in environment variables.

Monitoring Usage

Monitor Cloudinary usage via the Cloudinary Dashboard to ensure credit limits are not exceeded by frequent background removal operations.

Backup Procedures

No changes to database schema in this release. Regular database backups should continue as scheduled.

Backup Location: \backup-db\db-date
