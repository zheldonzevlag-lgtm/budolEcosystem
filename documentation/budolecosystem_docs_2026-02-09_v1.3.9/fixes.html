Resolved Fixes in v1.3.9

The following issues were identified and resolved in this release.

F-UX-103 Black Document Preview in Smart Crop

Fixed an issue where the document preview would show as a black box when the smart crop utility failed to detect an ID or encountered invalid dimensions. Added safety checks and an automatic fallback to the original image.

F-AI-201 UMID/Driver's License OCR Extraction Failures

Improved regex patterns for Philippine IDs (UMID, Driver's License) to better handle variations in OCR text quality and layout.

F-PERF-301 KYC Processing Latency

Optimized the smart crop pixel analysis by skipping every 4th pixel, reducing client-side CPU usage by ~50% without sacrificing accuracy.
