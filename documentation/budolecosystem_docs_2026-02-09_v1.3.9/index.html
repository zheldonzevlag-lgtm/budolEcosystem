v1.3.9

Release Overview: AI Smart Crop & Enhanced Data Extraction

Version 1.3.9 focuses on automating document preparation and improving OCR accuracy through advanced image processing and refined extraction logic.

Key Improvements

  • AI Smart Crop: Automatically detects ID cards in uploaded photos, removes backgrounds, and crops to the ID boundaries using client-side canvas processing.
  • Robust Image Handling: Implemented multi-stage fallbacks in the smartCrop utility to prevent "black image" errors and handle poor lighting or cluttered backgrounds gracefully.
  • Advanced OCR Extraction: Refined regex patterns and sequence-based logic to accurately extract Names and ID Numbers for UMID, Driver's License, and Voter's ID.
  • Background Removal Transparency: Users see the final cropped and processed ID, increasing confidence that the AI is focusing on the correct document area.
  • Security & Privacy: All image processing happens client-side, ensuring raw photos with potentially sensitive background data are not transmitted to the server.
