System Admin Manual: KYC Processing

Admin instructions for reviewing applications in v1.3.9.

Reviewing AI-Processed Documents

The KYC system now provides pre-cropped images. Administrators should follow these steps:

  • Verify Crop Integrity: Ensure the AI didn't cut off important details (Name, Photo, ID Number). If it did, request a resubmission.
  • Cross-Check Extraction: Compare the auto-filled Name and ID Number fields with the visual data in the photo.
  • Transparency Audit: Use the "Raw Scan Lines" in the user's submission metadata to see exactly what the AI read if there is a discrepancy.
Important Note: All images are processed client-side. If a user reports "black images," it is likely due to their device's GPU or Browser compatibility with Canvas. Encourage them to update their browser or try another device.
