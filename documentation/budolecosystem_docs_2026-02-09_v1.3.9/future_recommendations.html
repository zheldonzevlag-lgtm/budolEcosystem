Future Recommendations & Enhancements

  • TensorFlow.js Integration: Move from simple pixel analysis to a lightweight object detection model for 100% accurate ID isolation.
  • Auto-Rotate: Implement skew correction to straighten IDs that were photographed at an angle.
  • Server-Side Verification: Add a second pass of OCR on the server using more powerful engines (e.g., Google Vision) for high-tier accounts.
  • Identity Cross-Check: Automatically compare extracted names with bank account names or social profiles to prevent fraud.
