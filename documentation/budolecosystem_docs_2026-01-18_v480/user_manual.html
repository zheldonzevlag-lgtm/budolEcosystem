User Manual

How to use the Budol Ecosystem across your network.

Connecting from Multiple Devices

With v480, you can now access BudolPay and BudolShap from any device on your Wi-Fi network.

  1. Open your browser on your phone or another laptop.
  2. Enter the IP address of the host machine (e.g., http://192.168.1.2:3000 for BudolShap).
  3. Log in as usual. All services are automatically linked.

Mobile App Setup

The BudolPay mobile app (budolPay 1.3.18) now supports dynamic host discovery. If the app cannot find the server automatically, you can manually enter the IP address in the Settings screen.
