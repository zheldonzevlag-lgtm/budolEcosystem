System Admin Manual

Managing the Network-Aware Budol Ecosystem.

1. Automatic Network Configuration

To update the ecosystem with your current local IP address, run the following command from the root directory:

node scripts/run-network-update.js

This will automatically detect your IP and update all .env files and mobile configurations.

2. Starting the Ecosystem

Launch all services using the central runner:

npm start

Select Option 3 (SKIP) if you have already run the network update script manually, or Option 1 (LOCAL) for a fresh setup.

3. Verifying System Health

Check the status of all services across the network:

node scripts/test_scripts/verify-ecosystem-health.cjs

4. Mobile App Connectivity

For physical device testing, refer to scripts/test_scripts/global/MOBILE_CONFIG.txt for the correct --dart-define flags to use during flutter run.
