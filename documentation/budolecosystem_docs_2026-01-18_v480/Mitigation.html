Mitigation Plan - v480

Strategies to address identified risks.

M-01: Auto-detection on Startup

The start-ecosystem.js script now runs network detection at the beginning of the boot sequence, ensuring the LOCAL_IP is always current before services start.

M-02: Network Hardening

Administrators are advised to use the .env file to restrict ALLOWED_HOSTS and use VPNs for remote access instead of opening router ports.

M-03: Manual Host Override

The BudolPay mobile app includes a Host Configuration dialog allowing users to manually enter the server IP if automatic discovery fails.
