System Overview v480

Release Date: 2026-01-18

Purpose & Rationale

This release (v480) focuses on Network Awareness and Environment Portability. We have eliminated hardcoded static IP addresses (specifically 192.168.1.17 and 192.168.1.24) from all services, adapters, test scripts, and mobile configurations.

This ensures that the Budol Ecosystem can be deployed in any local or cloud environment without manual IP reconfiguration, using the LOCAL_IP environment variable and dynamic network detection.
