Future Enhancements & Recommendations

Next steps for improving network awareness and scalability.

1. Dockerization

Containerize all services to further isolate network configurations and use Docker Compose for service orchestration.

2. mDNS Discovery

Implement Multicast DNS (mDNS) so the mobile app and admin dashboard can find services using hostnames like budolpay.local instead of raw IP addresses.

3. SSL/TLS for Local

Introduce mkcert or similar tools to provide HTTPS for local development, which is required for certain modern browser APIs.
