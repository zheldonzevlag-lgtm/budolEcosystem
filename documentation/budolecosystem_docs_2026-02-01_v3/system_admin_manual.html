System Admin Manual: v2.1.0

Service Configuration

Ensure auth-service is running on port 8001. The environment variable IS_LOCAL=true should be set for local testing to use the default OTP 123456.

Database Migration

No schema changes required for this release as kycTier and kycStatus already existed in the Prisma schema. Verify KYCTier enum supports BASIC.

Monitoring

Monitor logs for [Quick Reg Error] and SECURITY_QUICK_REG_INIT audit events.
