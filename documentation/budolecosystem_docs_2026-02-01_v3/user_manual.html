User Manual: Quick Registration

Step 1: Open Registration

Navigate to the Registration screen. By default, "Quick Registration" is selected.

Step 2: Enter Phone Number

Enter your 11-digit mobile number (e.g., 09171234567). Tap "Register with Phone".

Step 3: Verify OTP

You will receive a 6-digit verification code via SMS. Enter it on the next screen.

Step 4: Set PIN

After verification, set a secure 6-digit PIN to protect your account.

Note: You will be assigned a "BASIC" account. Full features like Send Money may require additional identity verification (KYC) later.
