SSO Login & Environment Isolation Fix

Resolved a critical issue where environment variables leaked from the Central Runner to sub-services, causing Prisma initialization failures during SSO login.

← Back to Index

Issue Overview

After introducing the Interactive Environment Selector, the Central Runner's pre-flight validation loop was polluting the parent process's process.env with app-specific variables like DATABASE_URL and NODE_ENV. This caused the first app (budolID) to inherit incorrect settings, leading to Prisma protocol errors.
