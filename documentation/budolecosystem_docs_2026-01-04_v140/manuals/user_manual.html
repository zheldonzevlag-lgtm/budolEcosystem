User Manual: Login Stability & Verification

1. Overview

This update fixes the "Login Error" issues encountered when using the budolPay mobile app or other ecosystem applications.

2. How to Verify

Step 1: Restart the Ecosystem
Stop any running terminal processes and run npm start in the root folder.
Step 2: Select Environment
Choose "1" for LOCAL or "2" for PRODUCTION when prompted.
Step 3: Attempt Login
Open the budolPay app and attempt to log in. You should no longer see "Internal Server Error" or Prisma initialization errors.

3. Issues Resolved

  • Fixed login failure in budolPay mobile app.
  • Resolved "Unauthorized Application" errors caused by incorrect API key lookups in the database.

4. Troubleshooting

If you still see errors, please ensure you have run npm run db:local (or npm run db:vercel) at least once to ensure your databases are initialized correctly.
