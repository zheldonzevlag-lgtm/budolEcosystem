User Manual: Installing the App

Installation Steps

Follow these steps to install the latest budolPay Mobile App:

  1. Download the app-debug.apk from the build outputs.
  2. Transfer the file to your Android device.
  3. Open the file on your device and follow the prompts to install (you may need to enable "Install from Unknown Sources").
  4. Launch the app and login to start scanning QR codes for payment.
