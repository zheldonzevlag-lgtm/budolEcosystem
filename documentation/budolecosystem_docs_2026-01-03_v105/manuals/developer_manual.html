Developer Manual: APK & API Gateway

API Gateway: Proxy Body Fix

The http-proxy-middleware in the API Gateway was dropping POST request bodies. This was fixed by adding an onProxyReq hook to manually write the body data to the proxy request.

onProxyReq: (proxyReq, req, res) => {
    if (req.body && Object.keys(req.body).length) {
        const bodyData = JSON.stringify(req.body);
        proxyReq.setHeader('Content-Type', 'application/json');
        proxyReq.setHeader('Content-Length', Buffer.byteLength(bodyData));
        proxyReq.write(bodyData);
    }
}
