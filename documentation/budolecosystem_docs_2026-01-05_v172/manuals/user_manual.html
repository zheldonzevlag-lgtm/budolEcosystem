User Manual: What's New in 1.1.9

Guide to the latest features and improvements in BudolPay Mobile 1.1.9.

1. Profile & Security Updates

  • Hyphenated Birthdate Entry: Your date of birth is now automatically formatted as MM-DD-YYYY using a convenient calendar picker.
  • Pin Your Address: Easily pin your location during registration for accurate address entry.
  • Secure Phone Change: If you need to update your registered number, you can now submit a request along with a photo of your ID for secure verification.

2. General Improvements

  • Improved app stability and reduced startup time.
  • Smoother transitions between KYC verification steps.
