Roadmap & Recommendations: 2026 Strategy

Consolidated future enhancements for the BudolPay identity and security ecosystem.

1. Identity Evolution

  • Phase 1: Dual-OTP Handshake (Q1 2026) - Automate number changes for users with access to both SIMs.
  • Phase 2: Automated Face Matching (Q2 2026) - Integrate ML-based biometric verification for lost SIM scenarios.
  • Phase 3: FIDO2 Passkeys (Q4 2026) - Transition to passwordless, device-bound identity standards.

2. Compliance & Infrastructure

  • S3 Storage Migration: Offload KYC document blobs from the main database to object storage.
  • NIST Compliance: Align all identity protocols with NIST Digital Identity Guidelines (800-63).
