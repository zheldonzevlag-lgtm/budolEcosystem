Infrastructure Overview

The real-time balance update system relies on the API Gateway (Port 8080) acting as a WebSocket hub. Ensure that firewall rules allow both WebSocket and long-polling traffic on this port.

Environment Configuration

Variable Required Value Purpose
LOCAL_IP 192.168.1.24 LAN accessibility for mobile devices
GATEWAY_URL http://192.168.1.24:8080 Target for internal notifications
Important: If the LOCAL_IP changes, all mobile apps must be updated via the "Network Settings" dialog on the login screen, and backend services must be restarted.

Connectivity Troubleshooting

  • Symptom: Mobile app shows "Disconnected" dot (Orange).
    Check: Ensure the mobile device is on the same Wi-Fi network as the server.
    Check: Verify api-gateway is running: node services/api-gateway/index.js
  • Symptom: Balance updates work on web but not on mobile.
    Check: Verify if 192.168.1.24:8080 is reachable from the mobile browser.
