User Guide for v1.2.0

Logging In

When you log in to BudolShap, you will be asked to verify your identity using a 6-digit code.

How to get your code

  1. Enter your registered Mobile Number or Email address.
  2. Click "Send Code".
  3. Check your SMS messages and your Email inbox.
  4. The code will be sent to both locations for your convenience.
  5. Enter the code in the "Verification Code" field and click "Login".

Note: The code is valid for 10 minutes. If you don't receive it, please check your Spam folder or wait 60 seconds to request a new one.
