System Admin Manual

Administration and Configuration for v1.2.0

OTP Provider Configuration

Administrators can configure SMS and Email providers via the Admin Settings dashboard.

Email Settings

  • Provider: BREVO or SMTP (Google/Custom)
  • Credentials: API Key or SMTP User/Pass

SMS Settings

  • Provider: ZERIX, ITEXTMO, or VIBER
  • Credentials: API Key and Client Code (if applicable)

Monitoring

Check server logs for "Dual-Channel Delivery" entries to verify successful multi-channel dispatch.
