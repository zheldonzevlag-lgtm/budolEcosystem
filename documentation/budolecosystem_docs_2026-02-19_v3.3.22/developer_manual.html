Marketing Ads Screen Layout Logic

File: lib/screens/marketing_ads_screen.dart

The MarketingAdsScreen uses a LayoutBuilder to determine available space. In v3.3.22, we enhanced the robustness of this layout to prevent black screens when constraints are temporarily zero.

Implementation Details

We introduced a fallback mechanism using MediaQuery and hardcoded defaults (393x852) to ensure the screen always renders a container, even if layout constraints are invalid.

// Fallback logic for layout constraints double width = maxWidth; double height = maxHeight; if (width <= 0 || height <= 0) { debugPrint('MarketingAdsScreen: WARNING - Zero size detected! Using fallback.'); try { final size = MediaQuery.of(context).size; width = size.width; height = size.height; } catch (e) { width = 393; height = 852; } }
