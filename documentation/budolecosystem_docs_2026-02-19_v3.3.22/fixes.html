Fixes & Resolutions

This document details the critical fixes implemented in version 3.3.22.

1. Post-Splash Black Screen Fix FIXED

Issue: Users reported a black screen after the splash screen. Debugging revealed that MarketingAdsScreen was receiving zero width/height constraints in some scenarios, causing it to return SizedBox.shrink().

Resolution: Modified MarketingAdsScreen to handle zero constraints by falling back to MediaQuery.of(context).size and ensuring minimum dimensions.

Code Reference: MarketingAdsScreen.dart

// Before
if (maxWidth == 0 || maxHeight == 0) {
    return const SizedBox.shrink();
}

// After
if (width <= 0 || height <= 0) {
    try {
        final size = MediaQuery.of(context).size;
        width = size.width;
        height = size.height;
    } catch (e) {
        width = 393; height = 852;
    }
}

2. Registration "Continue" Button Visibility FIXED

Issue: The "Continue" button on the registration profile step was invisible when disabled, confusing users.

Resolution: Added explicit disabledBackgroundColor and disabledForegroundColor to the ElevatedButton style.

Code Reference: RegistrationScreen.dart

style: ElevatedButton.styleFrom(
  // ...
  disabledBackgroundColor: const Color(0xFF334155),
  disabledForegroundColor: Colors.white38,
),
