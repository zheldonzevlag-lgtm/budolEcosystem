User Manual: Understanding Your Reference ID

What is a Reference ID?

A Reference ID is a unique number assigned to every transaction you make in the BudolEcosystem. It helps our support team and payment providers identify your payment quickly and securely.

The New Secure Format

Starting from January 4, 2026, all new transactions will use an improved Reference ID format that looks like this:

JON-20260104123456-ABC123XY

Why the change? This new format is more secure and helps us process your payments faster by ensuring no two transactions ever have the same ID.

Where to find it?

  • Checkout Page: Displayed after you initiate a payment.
  • Transaction History: Listed next to each payment in your dashboard.
  • Email Receipts: Included in the confirmation email sent after a successful purchase.

When to use it?

Please provide this Reference ID whenever you contact our customer support regarding a specific payment. It is the fastest way for us to help you!
