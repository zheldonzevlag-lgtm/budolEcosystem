Developer Manual: Navigation Guard

Technical Explanation

The implementation uses the PopScope widget introduced in Flutter 3.12+ (replacing WillPopScope). It wraps the main Scaffold of the Home Screen.

Implementation Details

File: home_screen.dart

return PopScope(
  canPop: false, // Prevents popping the route
  onPopInvokedWithResult: (bool didPop, dynamic result) {
    if (didPop) return;
    // Log intercepted back button attempt
    if (kDebugMode) {
      print('HomeScreen: Back button pressed, but pop is disabled.');
    }
  },
  child: Scaffold(...)
);

Navigation Flow Compliance

All authentication screens (Login, Registration, Splash) must use pushReplacementNamed or pushNamedAndRemoveUntil when navigating to the Home Screen to ensure the stack is clean. This PopScope acts as a final safety layer.

Dependencies

  • Flutter SDK >= 3.12.0
  • Material Design Package
