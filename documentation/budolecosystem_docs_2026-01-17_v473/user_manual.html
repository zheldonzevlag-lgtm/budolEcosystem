User Manual: Back Button Behavior

What's New?

We've improved the security and stability of the app. Starting with version v473, the native back button on your Android device will no longer take you back to the login screen once you are logged in.

Why?

This prevents you from accidentally leaving your account dashboard. To switch accounts or log out, please use the Logout option in the Settings menu.

Summary of Changes

  • Home Screen back button disabled.
  • Improved session stability.
