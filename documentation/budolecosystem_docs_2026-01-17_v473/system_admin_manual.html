System Admin Manual: Navigation Policy

Deployment Impact

This update affects only the mobile application frontend behavior. No backend changes or API migrations are required.

Compliance Verification

  • PCI DSS: Ensures that authenticated sessions are not accidentally exposed by "backing out" into cached login states.
  • UX Standard: Prevents app termination or blank screens on accidental back button taps.

Troubleshooting

If users report being "stuck" on the Home Screen, ensure that the Logout functionality is working correctly, as it is the only intended way to return to the Login Screen.
