v473: Navigation Guard Implementation

Version: v473 RELEASED

Date: 2026-01-17

Lead AI: Budol Ecosystem Master Orchestrator

Scope: Mobile Navigation Security

Purpose

To prevent unauthorized or accidental navigation from the Home Screen back to the Login Screen using the native device back button, ensuring session integrity and improved UX.

Rationale

In financial applications, once a user is authenticated and on the dashboard, the back button should not return them to a sensitive login interface unless they explicitly log out. This implementation enforces the "Home as Root" principle.

Key Changes

  • Implemented PopScope in home_screen.dart.
  • Disabled canPop property to intercept native back events.
  • Added diagnostic logging for intercepted pop attempts.
