SysAdmin Manual v94: Mobile Deployment

Branch Management

The mobile application code is hosted on a clean branch:

git checkout feature/flutter-budolpay-app-qr-scanner

This branch is tracked remotely at origin/feature/flutter-budolpay-app-qr-scanner.

Build Requirements

To build the APK successfully on development machines with limited resources:

  • Java Home: Ensure JAVA_HOME points to a valid JDK (Android Studio's bundled JBR recommended).
  • Gradle Memory: Do not exceed 1024m for -Xmx to avoid OS-level memory allocation failures.
