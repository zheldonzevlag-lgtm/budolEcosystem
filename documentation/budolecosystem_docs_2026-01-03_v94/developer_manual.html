Developer Manual v94: Flutter QR Implementation

Technical Specification

The budolPayMobile app leverages Flutter's high-performance rendering and the mobile_scanner package for real-time QR code detection.

Core Components:

  • QR Scanner Screen: lib/screens/qr_scanner_screen.dart
  • Camera Logic: Uses MobileScannerController with optimized resolution settings.
  • Build Config: android/gradle.properties optimized with -Xmx1024m to prevent JVM crashes on 4GB systems.

Issues & Fixes

Issue: JVM crash during Android build due to insufficient memory.

Fix: Modified org.gradle.jvmargs in gradle.properties to limit memory usage while enabling org.gradle.daemon=true for performance.

Future Enhancements

  • Implementation of QR code generation for user profiles.
  • Deep linking support for scanned payment URLs.
  • Advanced error handling for low-light camera conditions.
