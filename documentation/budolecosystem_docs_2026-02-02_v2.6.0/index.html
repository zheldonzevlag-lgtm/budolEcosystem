Release Summary

This release addresses the critical requirement for staff identity attribution in the Forensic Audit Trail. It resolves issues where system setting changes were logged without associating the acting administrator, ensuring full compliance with PCI DSS 10.2.2 and BSP Circular 808. Additionally, audit coverage has been extended to all system configuration modules.
