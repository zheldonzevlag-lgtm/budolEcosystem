Developer Manual - v68

1. Branding Implementation

The app branding has been updated to use the Philippine Peso sign (₱) for better local identity. This was implemented using RichText in the AppBar to allow for multi-colored text styling.

// home_screen.dart
title: RichText(
  text: const TextSpan(
    style: TextStyle(
      fontSize: 20,
      fontWeight: FontWeight.bold,
      color: Colors.white,
    ),
    children: [
      TextSpan(text: 'budol'),
      TextSpan(
        text: '₱ay',
        style: TextStyle(color: Colors.white),
      ),
    ],
  ),
),

2. App Icon Configuration

The flutter_launcher_icons package has been added and configured in pubspec.yaml. To generate new icons for all platforms (Android, iOS, Web) using the Philippine Peso logo, follow these steps:

  1. Place your 1024x1024 logo in assets/images/logo.png.
  2. Run the following command:
flutter pub get
flutter pub run flutter_launcher_icons

3. User Seeding and Login Verification

A new user diana.prince@budolpay.com was created in the budolID database using a custom script create-diana-user.js to resolve reported login failures.

// create-diana-user.js snippet
await prisma.user.upsert({
  where: { email: 'diana.prince@budolpay.com' },
  update: { passwordHash: hashedPassword },
  create: {
    email: 'diana.prince@budolpay.com',
    passwordHash: hashedPassword,
    firstName: 'Diana',
    lastName: 'Prince',
    role: 'USER',
    isVerified: true
  }
});
