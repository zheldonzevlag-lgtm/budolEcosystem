Known Issues

Version: Login Toggle & Animation | Release Date: 2026-02-06

Toggle Animation Flicker

On extremely slow network connections, the toggle animation might appear to "snap" rather than slide smoothly if the component is re-rendering due to external state changes.

Autofill Overlap

Browser autofill may occasionally populate both the hidden and visible fields simultaneously, though this does not affect login functionality as only the active field's value is submitted.
