Mitigation Plan

Version: Login Toggle & Animation | Release Date: 2026-02-06

Security Mitigations

Identifier Normalization (R-005)

To prevent account enumeration, the system normalizes both email and phone numbers before querying. Additionally, error messages are kept generic (e.g., "Invalid credentials") regardless of whether the identifier or password was incorrect.

Unified Rate Limiting (R-007)

The rate-limiting logic has been updated to track attempts across both identifiers for the same user account. This prevents attackers from bypassing limits by switching between email and phone number attempts.

Hardware-Accelerated Animations (R-006)

CSS animations use transform (translate-x) and opacity, which are hardware-accelerated on most modern browsers and mobile devices, ensuring smooth performance even on lower-end hardware.
