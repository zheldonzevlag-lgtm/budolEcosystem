Future Recommendations

Version: Login Toggle & Animation | Release Date: 2026-02-06

OTP (One-Time Password) Support

Implement SMS-based OTP login for the mobile number method. This would allow users to log in securely without needing to remember their password, which is highly preferred in mobile-first markets.

Biometric Authentication Integration

For the mobile application (BudolPay), integrate biometric (FaceID/Fingerprint) login that maps to the user's selected primary identifier (Email or Phone).

Enhanced Animation Orchestration

Explore using Framer Motion for more complex state-driven animations, such as staggered field entry and dynamic form resizing, to further elevate the premium feel of the application.

Social Login Integration

Add Google and Facebook login options as additional toggles or buttons below the main form to further streamline the onboarding process.
