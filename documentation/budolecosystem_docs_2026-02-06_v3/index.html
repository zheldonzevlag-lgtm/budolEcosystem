Release: 2026-02-06 v3 | Login Toggle & Animation

Release Summary

This release introduces a togglable login interface for BudolShap, allowing users to switch between Email and Mobile Number login modes. The transition is enhanced with smooth CSS animations, providing a modern and responsive user experience. The backend has been updated to support authentication via both identifiers, ensuring seamless cross-channel access.
