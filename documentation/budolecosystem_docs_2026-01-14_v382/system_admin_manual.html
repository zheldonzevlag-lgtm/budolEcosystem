Service Management

To start the entire ecosystem, use the following command from the root directory:

npm start

This will launch the API Gateway, Auth Service, Wallet Service, Transaction Service, and Admin Dashboard concurrently.

Port Configuration

  • API Gateway: 8080
  • Admin Dashboard: 3000
  • Auth Service: 8001
  • Wallet Service: 8002
  • Transaction Service: 8003
Critical Maintenance: If port 8080 is occupied, identify and terminate the process using:
netstat -ano | findstr :8080
taskkill /F /PID <PID>

Database Synchronization

If the schema drifts from the database, run:

npx prisma db pull
npx prisma generate
