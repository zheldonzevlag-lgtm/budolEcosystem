Developer Manual - v451

Technical Fix: Missing JS Helper Import

During the optimization of the web splash screen removal in v449 and the UI polish in v450, the MarketingAdsScreen was updated to call callJsMethod('removeSplashFromWeb'). However, the necessary utility import was missing, leading to a compilation error.

Core Changes

1. Import Correction

Added the missing js_helper.dart import to marketing_ads_screen.dart to resolve the undefined method error.

marketing_ads_screen.dart 
import '../utils/ui_utils.dart';
import '../utils/js_helper.dart'; // Added this line

2. Verification of Usage

Confirmed that callJsMethod is correctly invoked within the addPostFrameCallback to ensure it runs only after the initial frame is rendered, preventing the black screen issue on web.

marketing_ads_screen.dart 
WidgetsBinding.instance.addPostFrameCallback((_) {
  if (kIsWeb) {
    callJsMethod('removeSplashFromWeb');
  }
});

Files Impacted

  • marketing_ads_screen.dart: Added missing import.
