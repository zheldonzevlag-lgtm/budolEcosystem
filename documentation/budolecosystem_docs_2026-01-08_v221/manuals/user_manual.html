User Manual: Clean Slate for Testing

1. Transaction History Reset

All previous transaction history (CASH_IN, P2P_TRANSFER, etc.) has been cleared. This allows you to start testing with a clean ledger and ensures that no old data interferes with new verification tests.

2. Wallet Balance

Your wallet balance has been reset to 0.00 PHP. To test "Send Money" or "Payment" features, you will first need to perform a "Cash In" operation or have an administrator update your balance.

3. Account Availability

Only verified test accounts remain active. If you are unable to log in, please ensure you are using one of the authorized test emails (Tony Stark, Peter Parker, or Reynaldo Galvez).
