v38 Walkthrough: Guided Capture Integration

This walkthrough demonstrates the new standardized capture frames for ID and Selfie used in the BudolPay v38 release.

Standardized Capture Frames

ID Card Capture (Rectangle)

Features a 1:0.63 aspect ratio rounded-rectangle cutout with a pink guideline border.

Selfie Capture (Oval)

Features a center-aligned oval cutout with a pink guideline border for face centering.

QR Code Scanner (Square Techno)

Features neon corner accents and an animated vertical scanning laser.

User Experience Flow

  1. Access: User navigates to Edit Profile and taps their avatar.
  2. Capture: The KYCCaptureScreen launches with the oval overlay.
  3. Feedback: The pink glowing border pulses to indicate the system is ready for capture.
  4. Upload: Tapping the shutter button triggers an immediate multipart upload to verification-service.
  5. Completion: The user is returned to the profile screen, which now displays the new high-resolution selfie.
