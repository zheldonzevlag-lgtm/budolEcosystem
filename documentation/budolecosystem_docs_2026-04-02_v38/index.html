v38: Profile Selfie Integration

Compliance: BSP / NPC / CyberSec Law

Release Date: 2026-04-02

Purpose

This release integrates the standardized, guided camera capture flow into the user profile editing process. By leveraging the KYCCaptureScreen for profile selfies, we ensure that user profile pictures meet the same high standards of clarity and framing as KYC documents, while also providing a "wow" factor through high-fidelity UI overlays.

Feature Overview

  • Guided Profile Selfie: Real-time circular overlay in EditProfileScreen for consistent headshot framing.
  • Standardized Backend: Unified verification-service upload endpoint for both KYC and profile media.
  • Live Preview: Instant update of the profile avatar upon successful upload.
  • Mock Graphics: High-fidelity capture mockup frames integrated into documentation.
