System Admin Manual: Profile Selfie v38

Storage Management

Profile selfies are stored in the /verification/uploads/ directory of the verification-service. Admins should ensure that this directory is backed up regularly and has sufficient disk space.

Maintenance: It is recommended to run a periodic script to purge orphaned profile images if a user changes their picture multiple times.

Environment Variables

No new environment variables were introduced in v38. The system continues to use the DATABASE_URL and UPLOAD_PATH configured in v37.

Troubleshooting Uploads

  • 413 Payload Too Large: The Nginx or Load Balancer limit is lower than the 5MB app limit. Adjust client_max_body_size.
  • 403 Forbidden: Ensure the uploads/ directory has 0755 permissions for the web user.
