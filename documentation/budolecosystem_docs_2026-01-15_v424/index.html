v424 - Product Details Runtime Stability Fix

This version addresses a critical runtime error in the Product Details page where non-array image data caused the application to crash. It introduces robust data handling for product images and ratings.

Release Information

  • Version: 424 (v4.2.4)

Purpose & Rationale

Users reported a "product.images.map is not a function" error when viewing certain products. This was caused by inconsistent data in the database where image fields were not properly formatted as JSON arrays.

The fix involves multi-layered defensive programming: sanitizing data at the API level and adding safety checks in the React components to ensure the UI remains stable even with legacy or malformed data.
