User Manual - v102

Cross-Device Checkout

You can now seamlessly transition from your laptop to your mobile phone. When you place an order on budolShap using your local IP, the checkout page will correctly load the budolPay secure interface, even if the services are hosted on different ports.

What to Expect

After clicking "Place Order", you should see the budolPay Secure Checkout screen with a QR code. If you see a generic "404 - Page Not Found" screen, please contact support as it indicates a service routing issue.
