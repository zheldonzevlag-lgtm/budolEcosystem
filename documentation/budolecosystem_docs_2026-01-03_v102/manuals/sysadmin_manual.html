System Admin Manual - v102

Network Connectivity

Ensure that port 8004 is open on the host firewall. While port 3001 (budolShap) might be accessible, the checkout process depends on direct communication with port 8004.

Troubleshooting 404 Errors

If you still see a 404 page:

  1. Check the browser URL. It should end with :8004/checkout/....
  2. If it ends with :3001/checkout/..., the budolshap cache or build might need a refresh.
  3. Verify the budolPay (Gateway) service is running in the central runner logs.
