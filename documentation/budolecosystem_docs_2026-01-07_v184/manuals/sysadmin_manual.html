System Admin Manual v184

Operational Guide: Phase 2 Implementation

1. Deploying budolID

Admins should link the budolID-0.1.0 directory to a Vercel project named budolid.

cd budolID-0.1.0
vercel link
vercel env pull .env.vercel
vercel deploy --prod

2. Verifying Connectivity

Once deployed, verify that the health check endpoint is accessible:

curl https://budolid.vercel.app/api/id/login

Risk & Mitigation

  • Risk: Redirect URI mismatch in production.
  • Mitigation: Ensure that all consuming apps (Shap, Pay) have their ID_SERVICE_URL set to https://budolid.vercel.app.
