System Admin Manual: budolID Redesign Deployment

Version: v40.0.0 | Date: 2026-01-02

Deployment Procedures

The changes were applied to the budolID SSO service. Since this service runs as a standalone Node.js process managed by the central runner, a service restart was performed.

1. File Updates

Modified budolID-0.1.0/index.js with the new UI template.

2. Service Restart

To apply the changes, the budolEcosystem central runner was restarted:

node start-ecosystem.js

This ensures the budolID (SSO) process is running the latest code on port 8000.

Monitoring: Verify that the login page is accessible at http://[LOCAL_IP]:8000/login. Ensure that the apiKey parameter is correctly passed from the consuming applications (e.g., budolShap).

Environment Variables

No changes were made to environment variables (.env) in this version.
