Future Enhancements: Fintech Roadmap

1. QR Ph Interoperability

Purpose: Allow budolPay users to pay at any merchant using the national QR Ph standard.

Technical Detail: Implement EMVCo QR code generation and parsing. Integrate with a BSP-licensed clearing house.

Priority: High

2. Biometric Authentication

Purpose: Increase security for "Send Money" and "Cash Out" operations.

Technical Detail: Use Flutter's local_auth package for Fingerprint/FaceID integration before final transaction commit.

Priority: Medium

3. Micro-Lending (budolLoan) Sync

Purpose: Allow users to "Cash In" via a pre-approved credit line.

Technical Detail: Integrate loan-service with transaction-service to handle "Credit-to-Wallet" transactions.

Priority: High

4. AI Fraud Detection

Purpose: Flag suspicious transaction patterns in real-time for Admin review.

Technical Detail: Implement a streaming analysis worker that monitors the Socket.io transaction feed and runs anomaly detection models.

Priority: Low
