Release Summary: v200 (Authentication Flow Completion & Final Verification)

This version marks the successful resolution of the mobile authentication issues. We have finalized the GoTyme-aligned security flow, ensuring that all users—even those without initial PINs—can securely access their accounts via OTP and PIN setup.

Key Achievements

  • Full Authentication Cycle: Verified the end-to-end flow from Identification to PIN login, including the new OTP-to-PIN setup recovery path.
  • User State Synchronization: Successfully updated accounts missing pinHash values, allowing them to bypass the "Invalid authentication request" error.
  • Operational Stability: Confirmed all services (Auth, Wallet, Gateway) are communicating correctly after the logic refinements.
