Login Issue Resolved

We have fixed the issue where some users were unable to log in because their security PIN was not recognized. Your account is now fully protected and accessible.

Final Steps for Stuck Users

If you see an error message saying "PIN not set for this account", please follow these simple steps:

  1. Tap the "Back to Phone Number" button at the bottom of the screen.
  2. Enter your mobile number again.
  3. You will now receive a 6-digit code (OTP) via SMS.
  4. Enter the OTP, and you will be prompted to Create your 6-digit PIN.
  5. Once set, you can log in instantly!

Thank you for your patience as we upgraded our security systems!
