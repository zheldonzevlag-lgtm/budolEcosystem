Developer Manual - v450

Technical Implementation: Smooth Page Indicator

In this version, we replaced the manual AnimatedContainer based page indicators with the smooth_page_indicator package. This package provides highly customizable and smoothly animated dots that react to the PageController's scroll offset.

Core Changes

1. Dependency Addition

The package smooth_page_indicator: ^1.1.0 was added to the dependencies in pubspec.yaml.

pubspec.yaml 
dependencies:
  ...
  smooth_page_indicator: ^1.1.0

2. UI Implementation

The MarketingAdsScreen was updated to use SmoothPageIndicator with the ExpandingDotsEffect, which mimics the behavior of the previous custom implementation but with smoother animations.

marketing_ads_screen.dart 
SmoothPageIndicator(
  controller: _pageController,
  count: _slides.length,
  effect: const ExpandingDotsEffect(
    dotHeight: 8,
    dotWidth: 8,
    expansionFactor: 3,
    spacing: 8,
    dotColor: Colors.white24,
    activeDotColor: Color(0xFFF43F5E),
  ),
)

Dependencies

  • smooth_page_indicator: Used for animated page indicators.
  • PageController: Essential for linking the indicator to the PageView.
