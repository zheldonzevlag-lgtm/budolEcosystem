Issues & Fixes - v450

Issue: Manual Indicator Lag

Symptoms: The custom page dots in the Marketing Ads screen had a slight delay in their animation when swiping between pages.

Root Cause: AnimatedContainer was triggering on onPageChanged, which only fires after the page transition is mostly complete.

Fix: Smooth Page Indicator

  • Direct Controller Binding: SmoothPageIndicator binds directly to the PageController, allowing dots to animate in real-time as the user swipes.
  • ExpandingDotsEffect: Implemented a 3x expansion factor for the active dot to maintain visual consistency with the previous design.
