Known Issues: v1.3.8

No critical issues identified in this release. All functional tests passed.

  • Browser Compatibility: Camera access for liveness check requires HTTPS. Users on HTTP environments will see a camera error prompt.
  • OCR Speed: Tesseract.js may take 3-7 seconds to process high-resolution images depending on user hardware.
