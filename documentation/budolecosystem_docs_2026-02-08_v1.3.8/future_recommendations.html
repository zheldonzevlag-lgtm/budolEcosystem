Future Recommendations & Enhancements

1. Server-Side OCR Validation

While client-side OCR is great for UX, server-side OCR (using Google Cloud Vision or AWS Textract) should be implemented to provide a high-trust secondary validation of the data submitted by the user.

2. Automated Liveness Score

Integrate an AI service to calculate a "liveness score" for the captured selfie, automatically flagging suspicious submissions for high-priority admin review.

3. Multi-Language OCR

Expand Tesseract.js configuration to better handle Tagalog or other regional dialects that may appear on some local government documents.

4. Admin Dashboard UI Overhaul

Create a specialized "KYC Review Room" for admin staff, featuring a split-screen view with the uploaded document on one side and the editable fields/OCR lines on the other.
