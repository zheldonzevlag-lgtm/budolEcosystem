v1.3.8

Release Overview: Simplified 3-Step KYC & OCR Transparency

Version 1.3.8 introduces a major UX overhaul for the KYC process, reducing friction while increasing transparency. The system now features a logical 3-step progression (Choose Tier -> Upload Documents -> Submit), shifting the verification burden to administrative staff.

Key Improvements

  • 3-Step KYC Flow: Structured the process into three clear phases: Tier Selection, Document Upload, and Final Review/Submission.
  • OCR Transparency Panel: Real-time display of raw OCR scan lines, providing users with visibility into the AI's data extraction process.
  • Admin-Centric Verification: Removed mandatory user-side data correction, allowing users to submit OCR-extracted data directly for admin review.
  • Interactive AI Feedback: Added an "Extraction in Progress" state with animated feedback for better user engagement during document processing.
  • Enhanced UI Previews: Increased ID and document preview sizes by 100% (from 128px to 256px) for improved clarity and user confidence.
  • Review & Submit Step: A dedicated final step for users to confirm their uploaded documents and extracted data before submission.
