Accessing from Other Devices

To access the ecosystem from a different computer or mobile device on the same Wi-Fi:

  1. Find the LAN IP of the host computer (e.g., 192.168.1.24).
  2. Open a web browser on your other device.
  3. Enter one of the following URLs:
    • budolShap: http://192.168.1.24:3001
    • budolID: http://192.168.1.24:8000
    • budolPay Admin: http://192.168.1.24:3000

Verifying Connectivity

If you cannot reach a service, ensure that the host computer's firewall allows incoming traffic on the respective ports (3000, 3001, 8000, 8080, 8005).
