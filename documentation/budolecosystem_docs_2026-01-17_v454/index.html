Release Documentation v454

Release Date: 2026-01-17

Focus: Navigation Diagnostics & Forced Ads Visibility

Purpose & Rationale

This release provides advanced diagnostic tools to resolve reports of the Marketing Ads screen not displaying. It introduces a "Forced Ads" debug feature and enhanced logging in the ads screen to pinpoint rendering or navigation failures.

Task List (v454)

  • ✅ Phase 0: Add enhanced logging to MarketingAdsScreen to verify entry and rendering.
  • ✅ Phase 1: Add a more prominent debug reset button and "FORCE ADS" button to SplashScreen.
  • ✅ Phase 2: Implement auto-scroll stability checks for Web platform.
  • ✅ Phase 3: Update documentation and knowledgebase to v454.
