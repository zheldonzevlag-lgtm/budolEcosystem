Developer Manual - v67

1. UI Clipping Fix

Resolved an issue where the app's internal background color would overflow the rounded corners of the web preview container.

1.1 Implementation of ClipRRect

The MaterialApp child in main.dart is now wrapped in a ClipRRect with a border radius slightly smaller than the outer bezel. This ensures that all screen transitions and backgrounds are strictly contained within the "device" frame.

// Fixed in main.dart
child: ClipRRect(
  borderRadius: BorderRadius.circular(32), // Fits inside the 40px outer radius
  child: child,
),

1.2 Visual Consistency

This fix prevents "square corners" from appearing at the edges of the rounded device frame, providing a more polished and professional web preview experience.
