Release Overview

This version introduces a more compact and readable Transaction Details screen in the BudolPay Mobile app, inspired by GCash and GoTyme. It also implements backend support for storing and displaying the Store Name associated with transactions.

Key Changes

  • Backend: Updated Prisma schema to include storeName and storeId in the Transaction model.
  • Wallet Service: Modified /process-qr endpoint to propagate store information from QR data to transactions.
  • Mobile App: Updated TransactionDetailsScreen to display Store Name and use a 12px font size for a cleaner look.
