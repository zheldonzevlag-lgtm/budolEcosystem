Mitigation Plan - v512

M-512-01: Schema Integrity

Ensure npx prisma generate is run successfully before deployment. Use prisma db push in a staging environment first.

M-512-02: Data Failsafe

The backend uses || null for store fields, and the frontend checks for null before rendering the row, preventing UI crashes.

M-512-03: Backward Compatibility

Legacy transactions will simply not show the "Store Name" field, which is acceptable behavior for historical data.
