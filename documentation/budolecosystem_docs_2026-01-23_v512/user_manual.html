User Manual - v512

What's New?

Enhanced Transaction History

You can now see the Store Name directly in your transaction details for all QR payments. This makes it easier to track where you spent your money.

Improved Layout

The Transaction Details screen has been redesigned with a cleaner, more compact look, similar to GCash and GoTyme. Text is now easier to read at a glance.

How to view?

1. Go to your Transaction History.
2. Tap on any QR Payment transaction.
3. The Store Name will be displayed right before the Transaction ID.
