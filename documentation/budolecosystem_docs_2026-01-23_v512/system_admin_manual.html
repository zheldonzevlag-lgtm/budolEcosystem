System Admin Manual - v512

Deployment Instructions

  1. Database Migration: Run npx prisma generate and npx prisma db push in the packages/database directory to apply the new storeId and storeName fields.
  2. Service Restart: Restart the wallet-service and transaction-service to ensure they pick up the schema changes.
  3. Mobile App: Deploy the latest APK (v1.3.27+124) to users.
Note: Old transactions will have null for storeName. The mobile app handles this gracefully by not showing the row.

Monitoring

Monitor the wallet-service logs for any issues during the QR payment process, specifically looking for storeId propagation errors.
