System Admin Manual

Guide for managing notification providers and system settings.

1. Configuring Notification Providers

Administrators can configure providers via the Admin Settings Dashboard.

Step 1: Access Settings

Navigate to Admin > Settings > Notifications.

Step 2: Select Providers

Choose the preferred Email Provider (Google, Brevo, GMASS) and SMS Provider (Zerix, iTextMo, Brevo, Viber, Console).

Step 3: Enter Credentials

Input the API Keys and Client Codes provided by the respective service providers.

2. Provider Settings Reference

3. Troubleshooting

  • OTP Not Received: Check the "Console" logs in the backend to see if the provider returned an error.
  • Cache Invalidation: Settings take up to 60 seconds to propagate. Use the "Clear Cache" button if immediate update is required.
