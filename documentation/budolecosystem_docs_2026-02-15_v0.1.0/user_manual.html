User Manual

Welcome to BudolPay. This guide covers account security and OTP verification.

1. Logging In with OTP

For your security, BudolPay requires a One-Time Password (OTP) when logging in from a new device or using your mobile number.

  1. Enter your mobile number on the login screen.
  2. Wait for the 6-digit code to arrive via SMS or Email.
  3. Enter the code in the app.

Note: If you don't receive the code within 60 seconds, you can tap "Resend Now".

2. Setting Up Your PIN

After your first login, you will be prompted to set up a 6-digit Security PIN. This PIN will be used for fast authentication and transaction approval.

3. Support

If you encounter issues with OTP delivery, please ensure your mobile number is correctly formatted (e.g., 0917XXXXXXX) and you have a stable network connection.
