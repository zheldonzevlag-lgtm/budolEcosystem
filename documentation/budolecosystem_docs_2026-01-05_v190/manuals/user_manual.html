User Manual - v190

System Stability Update

We have implemented a system stability update to resolve issues with logging into the Admin Panel. The system is now fully synchronized and operational.

What was Fixed?

  • Login Issues: Users who were previously unable to log in due to "SSO Callback Errors" should now be able to access the system normally.
  • Data Visibility: All user information, including the new KYC Tiers, is now correctly stored and displayed in the management interface.

No changes to the user interface were made in this version. All existing features remain the same.
