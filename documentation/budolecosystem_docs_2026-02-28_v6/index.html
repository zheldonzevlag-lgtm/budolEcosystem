v5.3.6 Release Notes

Critical API Restoration

Date: 2026-02-28

Executive Summary

This hotfix restores the NextResponse import in the public categories API. A regression in v5.3.5 caused the /api/categories endpoint to return 500 Internal Server Errors, which blocked the category picker from loading new data. We have verified recovery via local HTTP testing.
