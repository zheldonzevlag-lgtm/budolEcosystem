Forensic Audit Trails

The forensic audit trail system has been updated to include account registration events.

New Event Types

  • SECURITY_OTP_SENT: Logged when an OTP is sent for Quick Registration (Phone Only). Includes phone number and email in metadata.
  • SECURITY_OTP_VERIFIED: Logged when an OTP is successfully verified.
  • USER_REGISTERED: Logged when a new user account is created.

Viewing Logs

Go to Admin Dashboard > Settings > Forensic Audit Trails to view these logs. Use the filters to search for "Auth" or "User" entities.
