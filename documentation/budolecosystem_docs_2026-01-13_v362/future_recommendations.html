Future Recommendations: Biometric Roadmap

Version: v362

Strategic Recommendations

1. Backend Cross-Device Sync

Implement server-side face template matching. While we store templates on the server, matching currently happens locally for speed. Server-side matching will enable "Face Login" on a new device without needing a fresh scan if the user is already fully verified.

2. 3D Liveness Detection

Upgrade from 2D Blink/Smile detection to 3D depth-sensing liveness detection for devices equipped with IR/Depth cameras (e.g., iPhone FaceID or high-end Androids).

3. Transaction-Level Biometrics

Require a fresh face scan for high-value transactions (> PHP 50,000) even if the session is already active, to ensure the authorized user is still the one holding the device.

4. Anti-Injection Protection

Implement secure camera pipelines to prevent "virtual camera" or "video injection" attacks where attackers try to feed recorded video into the biometric service.
