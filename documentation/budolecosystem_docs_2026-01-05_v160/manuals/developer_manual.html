Developer Manual - Real-time Recipient Validation

Purpose

To improve user experience and data integrity by providing real-time validation for recipient identifiers (Email and Phone Number) in the Send Money screen.

Technical Implementation

The validation is implemented in the SendMoneyScreen using Flutter's TextField onChanged callback and state management.

1. State Management

A new state variable _recipientError was added to track validation status:

String? _recipientError;

2. Validation Logic

The _validateRecipient method handles dual-format validation:

  • Email: Uses a standard email regex. Triggered if the input contains '@'.
  • Phone Number: Specifically targets Philippine mobile formats:
    • 09XXXXXXXXX (11 digits)
    • +639XXXXXXXXX (13 characters)

Code Details

File: lib/screens/send_money_screen.dart


void _validateRecipient(String value) {
  if (value.isEmpty) {
    setState(() => _recipientError = null);
    return;
  }

  if (value.contains('@')) {
    final emailRegex = RegExp(r'^[\w-\.]+@([\w-]+\.)+[\w-]{2,4}$');
    setState(() => _recipientError = emailRegex.hasMatch(value) ? null : 'Invalid email format');
    return;
  }

  if (RegExp(r'^[\d+]').hasMatch(value)) {
    if (value.startsWith('+')) {
      if (!value.startsWith('+639')) {
        setState(() => _recipientError = 'Use +639XXXXXXXXX format');
      } else if (value.length != 13) {
        setState(() => _recipientError = 'Must be 13 characters');
      } else {
        setState(() => _recipientError = RegExp(r'^\+639\d{9}$').hasMatch(value) ? null : 'Invalid phone number');
      }
    } else if (value.startsWith('09')) {
      if (value.length != 11) {
        setState(() => _recipientError = 'Must be 11 digits (09XXXXXXXXX)');
      } else {
        setState(() => _recipientError = RegExp(r'^09\d{9}$').hasMatch(value) ? null : 'Invalid phone number');
      }
    } else {
      setState(() => _recipientError = 'Phone must start with 09 or +639');
    }
    return;
  }

  setState(() => _recipientError = 'Invalid email or phone number');
}
