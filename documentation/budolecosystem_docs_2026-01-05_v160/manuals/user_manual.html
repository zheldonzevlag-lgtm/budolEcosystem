User Manual - Smart Recipient Entry

New Feature: The "Send Money" screen now automatically checks if you've entered a valid email or phone number as you type!

How to Use

  1. Go to the Send Money screen.
  2. In the Recipient Information field, start typing the email or phone number.
  3. If the format is incorrect (e.g., too few digits for a phone number), a red message will appear below the field to guide you.

Supported Formats

  • Email: e.g., user@example.com
  • Phone (Local): 09123456789 (11 digits starting with 09)
  • Phone (International): +639123456789 (13 characters starting with +639)

Common Errors

  • "Must be 11 digits (09XXXXXXXXX)": You typed a local number but missed some digits.
  • "Phone must start with 09 or +639": Please use standard Philippine mobile prefixes.
  • "Invalid email format": The email address is missing a domain or '@' symbol.
