Registration Flow Restoration

Restored the universal registration endpoint in budolID to enable ecosystem-wide user onboarding.

! Implementation Summary

Issue

The budolID service was missing the /auth/register endpoint, causing budolShap registration to fail with a JSON parsing error when receiving an HTML 404 response.

Fix

Implemented the /auth/register endpoint in budolID with full support for user creation, password hashing, and mock fraud analysis responses.

Risk

Direct registration in budolID bypasses local application validations if not synchronized correctly.

Mitigation

The endpoint includes basic fraud analysis flags to allow downstream applications to perform additional security checks.
