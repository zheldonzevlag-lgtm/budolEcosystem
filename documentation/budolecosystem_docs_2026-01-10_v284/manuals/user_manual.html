User Manual - v284

Seamless Ecosystem Integration

Users can now experience a seamless transition between budolShap and budolPay. Once logged in via budolID, your identity is recognized across all applications on the local network.

Stable LAN Access

Performance and stability improvements have been made for users accessing the ecosystem from multiple devices. Login sessions are now more reliable, even when switching between different network-connected computers.

Payment Reliability

Checkout links generated by budolShap are now fully compatible with LAN access, allowing you to complete transactions from your mobile device or another computer connected to the same network.
