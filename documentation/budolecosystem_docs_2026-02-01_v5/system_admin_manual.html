System Admin Manual - v2.1.2

1. Database Synchronization

After any schema changes or when setting up a new environment, run the following to sync ecosystem applications:

node fix-apps.js

2. Provisioning Admin Accounts

To ensure admin access for the ecosystem, use the sync script:

node sync-admins.js

3. Mobile APK Distribution

New builds are stored in /android-installer/. Ensure the latest version budolPay 1.3.60.apk is served to users.
