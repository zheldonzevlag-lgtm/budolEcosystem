Fixes & Resolutions - v2.1.2

Issue: "Unauthorized Application" (403 Forbidden) on SSO Login

Cause: The ecosystemApp table in budolID did not contain the correct apiKey mapping for budolPay Admin (bp_key_2025) or the correct redirect URI.

Fix: Re-seeded the ecosystem apps using fix-apps.js to ensure bp_key_2025 is mapped to the correct local IP and callback URL.

Issue: Admin Login Failure (401 Unauthorized)

Cause: Admin accounts were missing from the User table in budolID after schema migrations.

Fix: Implemented sync-admins.js to provision admin@budolpay.com and reynaldomgalvez@gmail.com with the ADMIN role and verified password hashes.

Issue: Flutter Build version mismatch

Cause: api_service.dart was trailing behind pubspec.yaml.

Fix: Synchronized both to 1.3.60 and verified via flutter analyze.

Issue: Empty Workforce Registry in budolPay Admin

Cause: Missing /api/employees and /api/audit-logs endpoints in the Next.js admin app.

Fix: Implemented GET and POST routes for employee management and audit trail visualization, ensuring full visibility for administrators.

Issue: Quick Registration Failure (Mobile)

Cause: Missing /auth/register/quick endpoint in budolID service.

Fix: Implemented Shopee-style quick registration endpoint in budolID, allowing phone-based onboarding with temporary profile generation.
