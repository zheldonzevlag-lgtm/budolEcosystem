System Admin Manual: Mobile Camera Stabilization

Deployment Overview

Version v37 of the BudolPay Mobile App focuses on hardware bridge stabilization for KYC steps. Admins should ensure that deployment builds follow the updated SDK constraints.

Build Configuration

  • Compile SDK: 35 (Required for NoxPlayer camera bridge stability).
  • Min SDK: 21.
  • Target SDK: 35.
Emulator Note: While production devices support high-resolution capture, virtual devices (NoxPlayer, BlueStacks) require requestLegacyExternalStorage="true" in the manifest to prevent storage access crashes on Android 9/10/11.

Environment Settings

Ensure usesCleartextTraffic="true" is maintained for local development API connectivity if non-HTTPS endpoints are used during testing.
