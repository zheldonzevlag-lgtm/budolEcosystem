v37: Mobile Camera Stabilization & Guided KYC Capture

Compliance: BSP / NPC / CyberSec Law

Release Date: 2026-04-02

Purpose

This release resolves critical camera orientation and stability issues in the BudolPay mobile application's KYC verification flow. By replacing the generic system ImagePicker with a custom KYCCaptureScreen, we provide guided circular and rectangular frames, ensuring high-quality biometric and document captures even on emulators like NoxPlayer.

Feature Overview

  • Guided KYC Experience: Real-time oval (face) and rounded-rectangle (ID) overlays for precise user alignment.
  • Orientation Stability: Hardened camera controller logic to prevent "sideways" captures on virtual devices.
  • Optimized Memory Footprint: Controlled capture resolution (1024x1024) to eliminate "Camera keeps stopping" crashes.
  • Android Compatibility: Support for legacy storage permissions and optimized SDK level (35).
