Future Recommendations: KYC v37

1. Automatic ID Edge Detection

Integrate computer vision (e.g., OpenCV or ML Kit Object Detection) to automatically detect ID card boundaries and trigger capture only when the card is perfectly aligned and stable.

2. Liveness Detection

Enhance the face capture flow with liveness checks (smiling, blinking, or following a dot) to prevent spoofing attacks using static photos.

3. Real-time OCR Preview

Implement real-time text recognition during the ID scan to alert the user if details like Name or ID Number are unreadable before they hit capture.

4. Biometric Fallback for OTP

Implement biometric authentication (FaceID/Fingerprint) as a fallback for OTP in the mobile app to improve user experience while maintaining high security standards.
