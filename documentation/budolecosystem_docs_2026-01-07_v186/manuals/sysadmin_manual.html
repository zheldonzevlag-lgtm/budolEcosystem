Operational Guide: Phase 4 Commerce & Accounting

1. Deploying budolShap (Next.js)

Ensure all environment variables from .env.vercel are uploaded to the Vercel project settings.

cd budolshap-0.1.0
vercel env pull .env.vercel
vercel deploy --prod

2. Deploying budolAccounting (Express)

This service uses vercel.json to route all requests to index.js.

cd budolAccounting-0.1.0
vercel deploy --prod

3. Environment Variables Audit

Crucial variables for this phase:

  • NEXT_PUBLIC_SSO_URL: https://budolid.vercel.app
  • SSO_URL: https://budolid.vercel.app
  • JWT_SECRET: Ecosystem-wide shared secret.

Risk & Mitigation

  • Risk: Database schema mismatch between local development and Vercel Postgres.
  • Mitigation: Always run prisma generate and prisma db push (or migrate deploy) before final production deployment.
