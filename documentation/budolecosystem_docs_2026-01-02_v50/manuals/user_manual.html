Topic: Persistent Admin Profile Edits

Managing Admin User Profiles

Administrators can now reliably edit user names within the budolShap and budolPay Admin Panels. These changes will persist even after logging out and logging back in via SSO.

1. Access User Management

In budolShap, navigate to Admin Dashboard -> Users. In budolPay, profile edits are handled via the user settings section.

2. Edit User Details

Modify the name fields (First Name/Last Name) and click Save. The system will now automatically synchronize this change with the central budolID service.

3. Update Name and Save

Modify the name field and click Save. The system will now automatically synchronize this change with the central budolID service.

4. Verification

The next time the user logs in, their updated name will be correctly displayed, confirming that the change has been successfully persisted across the ecosystem.
