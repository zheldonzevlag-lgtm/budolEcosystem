System Admin Manual Updates

Infrastructure and Configuration

Environment Variable Modifications

Ensure the following environment variables are correctly set in the transaction-service and api-gateway:

  • WALLET_SERVICE_URL: URL of the wallet service (e.g., http://localhost:8002).
  • GATEWAY_URL: URL of the API gateway (e.g., http://localhost:8080).

RBAC Configuration Updates

The rbac-config.js in @budolpay/database has been updated to include:

  • TRANSACTION_CREATE_P2P: Required for sending money.
  • WALLET_DEBIT: Required for deducting funds.
  • SYSTEM_SETTINGS_READ: Required for fetching app configuration.

Verify that USER and MERCHANT roles have these permissions assigned.

Deployment Procedures

  1. Update the @budolpay/database package in all microservices.
  2. Restart api-gateway, wallet-service, and transaction-service.
  3. Deploy the updated mobile app (v1.2.13).

Database Schema Changes

No new schema migrations were required for this update, but ensure that the Transaction and LedgerEntry tables are healthy.

Operational Runbooks

  • Failed Transactions: Monitor logs for "Transfer Error" in the transaction service. Check if rollback was successful.
  • Real-time Notifications: Verify Socket.io connectivity at /health on the Gateway.
