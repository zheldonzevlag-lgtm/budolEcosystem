Future Recommendations

R-001: Biometric Authentication

Implement Passkeys (WebAuthn) for mobile devices to reduce reliance on OTP and improve user experience.

R-002: Real-time Phone Validation

Integrate a third-party service (like Twilio Lookup) to verify if the mobile number is active and reachable before sending OTP.

R-003: Regional Support

While currently focused on PH (+63), the normalization logic should be expanded to support international formats if global expansion is planned.
