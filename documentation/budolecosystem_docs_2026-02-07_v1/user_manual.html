User Manual

Logging in with Mobile Number

The login process has been simplified and standardized. Follow these steps:

  1. Click on the Mobile toggle in the login form.
  2. Enter your 10-digit mobile number (starting with 9). Do not include '0' or '+63' as these are handled automatically.
  3. Click Send Code via Email/SMS.
  4. Enter the 6-digit code received on your device.
  5. Click Sign In.

Registration

During registration, ensure your phone number is correct. If the number is already registered in the Budol Ecosystem, you will see a notification to log in instead.
