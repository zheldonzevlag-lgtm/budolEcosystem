Developer Manual v83: Database Auditing

The check_users.js Script

A new utility script was added to budolID-0.1.0/check_users.js to help developers verify the state of the user database without using a full GUI client.

// Usage
node check_users.js

This script connects via Prisma and lists all users, their roles, and verified status, which is essential for debugging SSO integration issues.
