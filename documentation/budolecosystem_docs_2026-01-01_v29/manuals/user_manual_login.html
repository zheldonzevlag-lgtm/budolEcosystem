User Manual: Account Registration & Login

1. Standard Login (BudolShap Only)

If you only have a shop account, use the standard login form on the BudolShap website.

Local Only Works only on BudolShap.

2. Login with budolID (SSO)

To use one account for everything (Shop, Payments, Delivery):

  1. Go to the login page of any Budol app.
  2. Click "Login with budolID".
  3. You will be redirected to the secure portal at 192.168.1.24:8000/login.
  4. Enter your global credentials.

Ecosystem Works across all apps.

3. Creating a budolID

Currently, global accounts are created via the BudolPay registration flow or assigned by an administrator.

Once created, you can use the same email/password everywhere.
