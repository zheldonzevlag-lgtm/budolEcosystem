budolID UI Redesign

Aligning budolID with the budolPay Admin design system while maintaining brand identity with the blue shield.

Purpose

Standardize the user experience across the ecosystem by unifying the login, forgot password, and reset password interfaces under a modern, cohesive design language.

Key Changes

  • Implemented Tailwind CSS for styling
  • Adopted Inter font family
  • Updated color scheme to slate-900/blue-500
  • Refined card-based layout with soft shadows
  • Added budolShield footer branding
