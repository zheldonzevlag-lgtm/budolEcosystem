Risk Register

Purpose

Identify and track risks introduced by product gallery video tiles.

Rationale

Video-rich galleries can introduce performance and usability risks.

Technical Explanation

Gallery media includes video elements, seller previews render video tiles with play button overlays, and uploads are stored in user-scoped Cloudinary folders.

Changes

  • Added video thumbnails and main viewer playback.
  • Added seller preview play buttons for uploaded videos.
  • Created Cloudinary folders per user and timestamp before upload.
  • Updated test scripts and lint configuration.

Issues

No production incidents reported.

Fixes

Test and lint failures resolved to stabilize CI.

Risks

  • Increased load time for galleries with multiple videos.
  • Potential seller confusion if preview controls are not visible.
  • Folder naming relies on user full name normalization.

Mitigations

  • Use metadata preload and lightweight overlays.
  • Maintain consistent play icon overlays on thumbnails and seller previews.
  • Normalize names and guard against unsupported characters.

Dependencies

React, Next.js, HTML5 video, Cloudinary API.

Code References

ProductDetails.jsx, DragDropVideoUpload.jsx, upload/route.js

Tests

Jest and lint checks passed.

Compliance

No compliance-impacting changes identified.
