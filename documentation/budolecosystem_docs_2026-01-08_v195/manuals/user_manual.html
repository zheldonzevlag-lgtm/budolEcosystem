User Manual: Mobile App Identification Restoration

Overview

This update addresses the "identification failed" error that some users encountered when logging into or accessing their balance in the mobile application.

Current Status

Service Status: RESTORED

What Changed?

A technical glitch in our backend wallet system was preventing the app from correctly identifying your account during balance checks. This has been resolved, and the system now correctly processes your identification requests.

User Actions Required

  • No Update Needed: This was a backend fix. You do not need to download a new version of the app.
  • Refresh App: If you still see an error, please close the app completely and reopen it.
  • Re-login: In some cases, logging out and logging back in may be necessary to refresh your session data.

Support

If you continue to experience "identification failed" errors, please contact support with the following information:

  • Your registered phone number or email.
  • The specific action you were performing (e.g., checking balance, sending money).
  • A screenshot of the error message if possible.
