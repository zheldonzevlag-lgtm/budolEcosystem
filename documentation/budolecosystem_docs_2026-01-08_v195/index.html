Release Summary: v195 (Wallet Service Scoping Fix)

This version documents the resolution of a critical "identification failed" error in the mobile app. The issue was traced to a JavaScript ReferenceError in the Wallet Service, where the userId variable was incorrectly scoped within a try block, making it inaccessible to the error handler in the catch block.
