Admin Logout Implementation

Date: 2025-12-31 | Version: v20

Developer Manual Updates

Technical Implementation

Added a logout mechanism to the budolPay Admin application sidebar. This involved creating a new API endpoint and updating the Sidebar component.

1. API Logout Route

Created a new route at apps/admin/app/api/auth/logout/route.ts that clears the authentication token cookie.

import { NextResponse } from 'next/server'; export async function POST() { const response = NextResponse.json({ success: true }); response.cookies.set('token', '', { httpOnly: true, expires: new Date(0), path: '/', }); return response; }

2. Sidebar Component Update

Modified apps/admin/components/Sidebar.tsx to include:

  • State management for the logout menu visibility.
  • A logout handler that calls the new API endpoint and redirects to the login page.
  • UI updates to show a "Sign Out" button when the admin profile section is clicked.
System Admin Manual Updates

Infrastructure & Security

No infrastructure changes required. The logout mechanism uses standard HTTP cookies.

  • Environment Variables: No new environment variables.
  • Security: The logout route clears the token cookie with httpOnly: true and sets its expiration to the past.
User Manual Updates

How to Logout

Admins can now securely sign out of the dashboard:

  1. Navigate to the bottom of the sidebar.
  2. Click on the Admin User section (the card showing your name and email).
  3. A Sign Out button will appear above the card.
  4. Click Sign Out to terminate your session and return to the login screen.
