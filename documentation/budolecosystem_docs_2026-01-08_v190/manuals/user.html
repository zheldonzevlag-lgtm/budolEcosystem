User Manual - Connecting to BudolPay Gateway

Setting up your Connection

1 Open the BudolPay Mobile app and go to the Settings screen (gear icon).
2 Tap on Gateway Host under the Connectivity section.
3 In the dialog that appears, tap Scan Network. The app will search for the BudolPay Gateway on your local Wi-Fi.
4 If the gateway is found, the address field will be updated automatically. Tap Save to apply the changes.
5 If scanning fails, you can manually type the IP address and port of your gateway (e.g., 192.168.1.15:8080) and tap Save.

Troubleshooting

  • Ensure your phone is connected to the same Wi-Fi network as the BudolPay Gateway.
  • Try turning Wi-Fi off and back on if scanning fails multiple times.
  • Check if your router's "AP Isolation" or "Guest Network" feature is disabled, as these can block discovery.
