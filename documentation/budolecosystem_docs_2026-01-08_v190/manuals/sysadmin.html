System Administrator Manual - Gateway Configuration

mDNS Advertisement

The API Gateway now automatically advertises itself on the local network. This requires the bonjour-service dependency.

To ensure advertisement works:

  • Ensure the gateway host machine allows UDP traffic on port 5353 (standard mDNS port).
  • The gateway must be running on a network interface that supports multicast.

Deployment Procedures

  1. Run npm install in the api-gateway directory to install bonjour-service.
  2. Start the gateway normally using npm start or node index.js.
  3. The gateway will log mDNS: Advertising BudolPay Gateway on port [PORT] upon successful startup.
Important: mDNS typically does not cross subnet boundaries. Ensure the mobile device and the API Gateway are on the same local subnet for auto-discovery to function correctly.
