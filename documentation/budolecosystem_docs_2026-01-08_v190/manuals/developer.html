Developer Manual - Mobile Improvements

Route Centralization

Instead of hardcoding route names in Navigator.push, developers must use the Routes class in lib/constants/routes.dart.

// Correct way to navigate
Navigator.pushNamed(context, Routes.sendMoney);

// Navigating with arguments
Navigator.pushNamed(
  context, 
  Routes.transactionDetails, 
  arguments: transactionData
);

mDNS Discovery Implementation

The DiscoveryService handles the mDNS lookup. It specifically looks for the _budolpay._tcp.local service type.

// In lib/services/discovery_service.dart
final MDnsClient client = MDnsClient();
await client.start();
await for (final PtrResourceRecord ptr in client.lookup(
    ResourceRecordQuery.serverPointer('_budolpay._tcp.local'))) {
  // Extract SRV and A records to get IP and Port
}

API Host Management

The ApiService now maintains the host configuration. It uses SharedPreferences to persist manual overrides and discovered hosts.

  • apiService.host: Returns the current host (e.g., 192.168.1.5:8080).
  • apiService.setHost(String newHost): Manually updates and persists the host.
  • apiService.discoverAndSetHost(): Triggers the mDNS scan and updates if found.
