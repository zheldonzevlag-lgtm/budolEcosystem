System Admin Manual: Ecosystem Configuration

Management guide for the budolID SSO service and budolShap integration.

Current Status: Ecosystem Online ●

Ecosystem App Registration

Applications must be registered in the budolID database to use SSO features. The following entry is required for budolShap:

Table: EcosystemApp
apiKey: bs_key_2025
redirectUri: http://192.168.1.24:3001/api/auth/sso/callback
name: budolShap

Note: Use check_app.cjs in the budolID directory to verify current registrations.

Critical Environment Variables

budolID (.env)

PORT=8000
JWT_SECRET=budolid-super-secret-key
DATABASE_URL=postgresql://...

budolShap (.env.local)

NEXT_PUBLIC_SSO_URL=http://192.168.1.24:8000
NEXT_PUBLIC_APP_URL=http://192.168.1.24:3001
JWT_SECRET=your-super-secret-jwt-key...

LAN Connectivity Guide

To allow other devices on your local network to access the apps:

  1. Ensure budolID is listening on 0.0.0.0 (port 8000).
  2. Ensure budolShap is started with next start -H 0.0.0.0 (port 3001).
  3. Update your NEXT_PUBLIC_SSO_URL and NEXT_PUBLIC_APP_URL to use your machine's LAN IP (e.g., 192.168.1.24).
