Developer Manual: Extended Currency Formatting

1. Implementation Overview

This update extends the use of CurrencyInputFormatter to the CashInScreen and CashOutScreen components. This ensures that all amount inputs in the application follow the same rules: mandatory 2-decimal precision and thousand separators (commas).
