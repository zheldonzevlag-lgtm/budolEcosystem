Recommendations Roadmap

1. CI/CD Pipeline Automation

Purpose: Automate the building and testing of the APK whenever code is pushed to the repository.

Benefit: Reduces manual errors and ensures that every release is consistently built and verified.

Technical Detail: Use GitHub Actions or GitLab CI to trigger flutter build apk and flutter analyze on every pull request.

2. Automated Release Distribution

Purpose: Automatically upload the generated APK to the android-installer folder or a cloud storage service.

Benefit: Faster delivery of updates to stakeholders and testers.

Technical Detail: Integrate a distribution step in the CI/CD pipeline using tools like Fastlane.

3. Real-time Transaction Notifications

Purpose: Notify users instantly when they receive or send money.

Benefit: Improves user engagement and security awareness.

Technical Detail: Implement Firebase Cloud Messaging (FCM) to push notifications from the backend services to the mobile app.

4. Advanced Transaction Search

Purpose: Allow users to search for transactions by reference number or counterparty name.

Benefit: Better usability for power users with high transaction volumes.

Technical Detail: Update the History screen to include a search bar and optimize backend APIs for search queries.
