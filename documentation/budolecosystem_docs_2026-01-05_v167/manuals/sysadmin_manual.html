System Deployment Overview

This release (v1.1.7) includes the production-ready APK for the budolPay mobile application, focusing on full transaction details and navigation.

Distribution Details

  • Artifact Path: android-installer/budolPay 1.1.7.apk
  • Version: 1.1.7+1
  • Environment: Production

Infrastructure Requirements

No changes were made to the backend infrastructure in this release. The existing microservices ecosystem (v163+) is fully compatible with this mobile update.

Installation Verification

  1. Distribute the APK to the test devices.
  2. Verify that the "History" tab in the bottom navigation opens the transaction history.
  3. Verify that clicking "See All" on the Home screen opens the transaction history.
  4. Ensure that tapping any transaction item correctly displays the detailed view.

Rollback Plan

If the 1.1.7 release exhibits critical issues, rollback to budolPay 1.1.6.apk is recommended. No database migrations are required for this rollback.
