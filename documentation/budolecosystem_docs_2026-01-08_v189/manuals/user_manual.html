User Manual: Restored Platform Access

Information regarding the resolution of login and navigation issues.

Resolved

Access Restored to Admin & E-commerce

Users who previously encountered "Infinite Loop" or "508" errors when trying to access BudolPay Admin or BudolShap should now be able to log in and navigate normally.

What was the issue?

A technical configuration error caused the websites to repeatedly redirect users between pages, preventing them from ever reaching the dashboard or store. This was primarily affecting the production (live) versions of the platforms.

What has changed?

BudolPay Admin

Login authentication is now correctly synchronized with the production servers. You can now access the admin dashboard at budol-pay-admin.vercel.app.

BudolShap

Navigation and store access have been restored. The platform is now live and stable at budolshap-0.10.vercel.app.

Instructions for Users

If you are still experiencing issues, please follow these steps:

  • Clear Browser Cache: Sometimes your browser "remembers" the old redirect loop. Clearing your cache ensures you load the latest configuration.
  • Check Your Connection: Ensure you are connected to the internet and not behind a restrictive firewall that might block authentication requests.
  • Re-login: If you were previously logged in, log out completely and log back in to refresh your session.

Support

If the problem persists, please contact the IT support team with a screenshot of any error messages and the URL you are trying to access.
