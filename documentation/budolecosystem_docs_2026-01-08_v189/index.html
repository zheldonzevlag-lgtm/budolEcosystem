Executive Summary

This update addresses critical production issues where users encountered infinite redirection loops on the BudolPay Admin and BudolShap platforms. The root cause was identified as a mismatch between public URLs in environment variables (local IPs vs. production URLs) and a typo in the BudolShap production URL.
