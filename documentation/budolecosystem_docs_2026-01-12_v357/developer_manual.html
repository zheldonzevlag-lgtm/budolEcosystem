Developer Manual - v357

Biometric UI Enhancement

The PIN input field on the login screen has been updated to include biometric icons within the text field itself, rather than as external buttons.

Code References:

  • login_screen.dart: Modified _inputDecoration and _buildPinStep.

Implementation Details

1. Modified _inputDecoration: Added a suffixIcon parameter to allow custom widgets at the end of input fields.

InputDecoration _inputDecoration(String label, IconData icon, {Color? iconColor, String? errorText, Widget? suffixIcon}) {
  return InputDecoration(
    // ...
    suffixIcon: suffixIcon,
    // ...
  );
}

2. Updated _buildPinStep: Replaced the Row layout with a direct TextField using a suffixIcon containing both Fingerprint and Face icons.

suffixIcon: _isBiometricAvailable ? Row(
  mainAxisSize: MainAxisSize.min,
  children: [
    IconButton(
      icon: Icon(Icons.fingerprint, ...),
      onPressed: _handleBiometricLogin,
    ),
    IconButton(
      icon: Icon(Icons.face, ...),
      onPressed: _handleBiometricLogin,
    ),
  ],
) : null
