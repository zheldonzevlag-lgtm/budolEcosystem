System Admin Manual - v193

Tiered Compliance Oversight

Administrators must monitor user tiers to ensure system-wide compliance with BSP regulations. The following limits are enforced at the system level for budolPay.

BASIC (Unverified) Limits

  • Max Wallet Balance: ₱5,000
  • Daily Outgoing Limit: ₱1,000 (Merchant only)
  • Monthly Outgoing Limit: ₱5,000
  • Restricted Features: P2P Transfers, Bank Cash-out, GSave/GInvest equivalents.

FULLY VERIFIED Limits

  • Max Wallet Balance: ₱500,000
  • Daily Outgoing Limit: ₱100,000
  • Monthly Outgoing Limit: No strict cap (Monitoring based)
  • Restricted Features: None. Full access to the ecosystem.

Risk & Mitigation

  • Risk: Users attempting to bypass limits via multiple accounts.
  • Mitigation: KYC documents are validated against a central database to prevent duplicate identity submissions across different mobile numbers.
