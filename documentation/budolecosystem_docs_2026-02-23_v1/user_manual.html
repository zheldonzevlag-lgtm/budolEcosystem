User Manual: Address Management

1. Adding/Editing an Address

When you add or edit an address in your profile or store settings, you can use the interactive map or the search bar to automatically fill in the details.

2. Using the Map Picker

Step 1: Click on the map to pin your exact location.
Step 2: The system will automatically detect the address details (City, Barangay, Province, Street, Zip) and fill the form fields below.
Step 3: You can manually edit any field if the automatic detection is slightly off.

3. Using Search

Step 1: Type your street name, building, or landmark in the search bar above the map.
Step 2: Select the correct location from the suggestions list.
Step 3: The map will move to that location, and the form fields will be automatically updated.

4. Troubleshooting

If the address fields do not update when you move the pin:

  • Ensure you have a stable internet connection.
  • Try searching for a nearby landmark instead.
  • You can always manually type the address details.
