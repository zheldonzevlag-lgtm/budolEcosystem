User Manual

Guide for end-users on how to use the new features in v503.

1. Dynamic Global Search

The search bar in the top navigation is now context-aware. It changes its behavior based on the page you are currently viewing.

Contextual Search Examples:

  • Orders Page: Search by Order ID, Customer Name, or Product Name.
  • Shop Page: Search for products by name or category.
  • Users Page (Admin): Search by User Name or Email.
  • Stores Page (Admin): Search by Store Name or Owner Username.

Simply start typing in the search bar, and the results on your current page will update in real-time.

2. Instant Order Status Updates

Order status updates are now more reliable. Whether you are in the "All Orders" view or a specific tab like "Pending" or "Cancelled", status changes (e.g., from Processing to Shipped) will appear instantly without requiring a page refresh.
