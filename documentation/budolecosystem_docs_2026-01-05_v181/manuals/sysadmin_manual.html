Build Infrastructure

The build environment for BudolPay Mobile requires specific configuration due to the 3GB RAM constraint and JDK path requirements.

Warning: Always ensure JAVA_HOME is set before running gradlew.

Build Commands

$env:JAVA_HOME = "D:\Android\Android Studio\jbr"
./gradlew assembleRelease --no-daemon

Deployment

Release APKs are automatically moved to the android-installer directory after a successful build.

  • Location: D:\IT Projects\budolEcosystem\android-installer\budolPay 1.2.0.apk

Troubleshooting

If images are not appearing in the Admin panel, check the blobData field in the database. Ensure it contains valid image data and is not empty. The Admin panel debug overlay (v180) can help identify the data type being received.
