Admin User Manual - KYC Review

Reviewing User Documents

1. Access User Management

Navigate to the "Users" section in the Admin Panel sidebar.

2. Locate Pending KYC

Users with pending verification will have a status badge. Click on the user to expand their details.

3. View Documents

The KYC documents (ID Front, Selfie, etc.) will now render as high-resolution images in the review modal. You can click on an image to zoom in.

[Image: High-resolution KYC document rendering in Admin Panel]

4. Verification Action

After reviewing the documents and verifying they match the user's profile, use the "Approve" or "Reject" buttons to update the KYC status.
