Provider Configuration UI Enhancement

RELEASED Last updated: 2026-02-02

Overview

Version 2.2.4 introduces a dedicated Provider Quick Setup Dashboard within the System Configuration module. This enhancement addresses user requirements for a more intuitive and accessible way to manage critical integrations for Email, SMS, and Payment Gateways.

Purpose: To consolidate high-impact service configurations into a visual status dashboard, reducing navigation time and improving system visibility.

Key Features

  • Provider Status Cards: Real-time visibility into active Email, SMS, and Payment providers directly at the top of the settings page.
  • One-Click Configuration: Anchor links that jump directly to the relevant settings group (Notification/Payment).
  • Expanded Provider Support: Pre-seeded configuration keys for Google Workspace (Gmail), Resend, Twilio, and PayMongo.
  • Responsive Design: Dashboard layout that adapts seamlessly to desktop and mobile views.
