Architecture & Data Contracts

The provider configuration UI leverages the existing SystemSetting model in Prisma. We utilize the group field to logically partition settings into functional domains.

// Grouping Logic in page.tsx
const groupedSettings = filteredSettings.reduce((acc, setting) => {
  const group = setting.group || 'OTHER';
  if (!acc[group]) acc[group] = [];
  acc[group].push(setting);
  return acc;
}, {} as Record<string, typeof settings>);

Compliance Verification

This implementation adheres to the following standards:

  • PCI DSS: Sensitive provider keys (API keys, passwords) are marked as isSecret: true and masked in the UI.
  • Cybersecurity Law: Audit logs are generated for every change to system settings, including the entity modified and the new value.
