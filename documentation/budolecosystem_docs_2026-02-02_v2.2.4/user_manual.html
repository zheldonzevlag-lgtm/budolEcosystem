User Configuration Guide

Configuring Providers

The Budol Admin portal provides a streamlined interface for configuring your core services.

1

Check Provider Status

At the top of the Settings page, you can see which providers are currently active for Email, SMS, and Payments. This gives you an immediate overview of your system's integration state.

2

Configure Integration

Click the "Configure Integration" link on any status card. The page will automatically scroll to the detailed settings table for that service.

3

Enter Credentials

Locate the specific keys for your chosen provider (e.g., API keys, passwords). Enter the values and click the "Save" icon on the right side of the input field.

Important Note on Security

Passwords and API keys are hidden after saving. If you need to update them, simply type the new value into the masked field and save. The system will never show you the original secret once it has been secured.
