System Administrator Manual

Managing Providers

Administrators can now manage Email, SMS, and Payment providers through a high-level dashboard. To access this:

  1. Navigate to Settings in the Admin Sidebar.
  2. Observe the Provider Quick Setup dashboard at the top.
  3. Click Configure Integration on any card to jump to the detailed settings.

Database Seeding

If new environment variables are required, administrators can run the following command from the database package:

node seed-providers.js

Note: This script is idempotent and will not overwrite existing configuration values unless explicitly modified.

Security Masking

Settings marked as "Sensitive Data" will appear as dots in the UI. When updating these settings, ensure you have the correct API key as the previous value cannot be read from the UI for security reasons.
