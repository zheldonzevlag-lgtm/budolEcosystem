Web Build Process

The web application was built using the following command to ensure all latest fixes for splash screen and marketing ads are included in the release artifacts.

flutter build web --release --dart-define=API_HOST=192.168.1.17

Build artifacts are located in: budolPayMobile/build/web

Key Artifacts

  • index.html: Contains the fixed splash removal script and background styling.
  • main.dart.js: Optimized production JavaScript bundle.
  • assets/: Contains all branding and transparent logo assets.
