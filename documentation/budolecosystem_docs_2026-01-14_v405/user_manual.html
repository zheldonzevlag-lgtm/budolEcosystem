User Manual: Navigating the Heartbeat Dashboard

📈 Transaction Volume

This metric represents the total liquidity within the Budol Ecosystem. It sums the balances of all company assets (Cash, Bank, and Digital Wallets). It is updated in real-time as transactions are processed.

⏱️ System Uptime

This percentage indicates the current health of the entire ecosystem. A value of 100% means all core services (Payments, Security, and Database) are fully operational. If this number drops, it indicates a technical issue is being addressed by the engineering team.

👥 User & Staff Counts

These counters reflect the active population of the ecosystem. The User Count includes all registered customers, while the Staff Count reflects internal administrators and support personnel.

💓 Heartbeat Center

The Heartbeat Center provides a live view of system pulses. Events like "New User Registration" or "High-Value Transaction" will pulse in real-time on the dashboard to alert administrators of significant ecosystem activity.
