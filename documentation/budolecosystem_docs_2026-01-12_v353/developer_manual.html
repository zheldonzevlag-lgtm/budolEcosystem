HTML/CSS Changes

The web splash screen is defined in web/index.html. The previous implementation used images from the splash/img/ directory which contained white backgrounds.

Modified Code: CSS Styling

File: index.html


.center {
  margin: 0;
  position: absolute;
  top: 50%;
  left: 50%;
  -ms-transform: translate(-50%, -50%);
  transform: translate(-50%, -50%);
  border-radius: 20px;
  background-color: transparent;
}

#splash img {
  filter: drop-shadow(0 0 10px rgba(0,0,0,0.5));
  max-width: 200px;
}

Modified Code: Asset Replacement

Switched from splash/img/light-1x.png to icons/Icon-192.png, which is the standard transparent app icon.
