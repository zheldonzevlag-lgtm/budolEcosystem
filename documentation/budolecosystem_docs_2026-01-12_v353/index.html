Release Overview

This release (v353) addresses a branding inconsistency in the web version of BudolPay. The initial loading screen (splash screen) was displaying a logo with a sharp white background, which did not align with the application's dark theme.

Key Improvements

  • Logo Transparency: Replaced the non-transparent splash images with the standard transparent app icons (Icon-192/Icon-512).
  • Visual Styling: Added border-radius and drop-shadow to the splash logo for a more modern, integrated look.
  • CSS Optimization: Explicitly set background-color: transparent for the logo container in the HTML splash.
