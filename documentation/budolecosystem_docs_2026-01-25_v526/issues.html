Issue Log: v526

This log tracks the issues discovered during the synchronization audit and their current status.

ISS-526-01 RESOLVED
Event Name Mismatch in Orders Service

The ordersService.js was triggering order-update but the frontend was listening for order-updated, causing sync failures on the budolShap web platform.

ISS-526-02 RESOLVED
Missing Mobile Status Listeners

The PaymentStatusScreen on the mobile app relied on manual polling instead of real-time events, leading to a "frozen" UI during high-latency transactions.

ISS-526-03 RESOLVED
Authorization Failure in Test Scripts

The end-to-end sync test script failed with a 401 Unauthorized because it didn't include a valid JWT token required by the wallet-service.
