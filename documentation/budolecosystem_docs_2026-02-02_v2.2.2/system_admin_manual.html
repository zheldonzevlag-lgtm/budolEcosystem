System Admin Manual

Operational guide for Gmail provider configuration in v2.2.2.

1. Configuring Gmail Provider

Step-by-Step Instructions:

  1. Navigate to System Settings > Notification Settings.
  2. Locate the NOTIFICATION_EMAIL_PROVIDER variable.
  3. Select Gmail (Google Workspace) from the dropdown menu.
  4. Click the Save icon to apply changes.
  5. Ensure the NOTIFICATION_EMAIL_USER and NOTIFICATION_EMAIL_PASSWORD (or App Password) are correctly set in the configuration.

Pro Tip

Use a Google "App Password" if 2-Step Verification is enabled on the account for improved security.
