User Manual: Image Rotation Persistence

Overview

Administrators can now permanently fix the orientation of uploaded KYC documents. If a user uploads an ID sideways or upside down, the admin can rotate it once, and the system will remember that orientation.

How to Use

1
Open User Documents: In the User Verification table, click the "View Documents" (eye icon) for a pending user.
2
Rotate Document: Click the rotate icon (circular arrow) next to the document type label. Each click rotates the image 90 degrees clockwise.
3
Automatic Save: The system automatically saves the new orientation. You don't need to click any separate save button.
4
Verify: Close the modal and reopen it. The document will maintain the orientation you set.

Benefits

  • Consistency: Future reviews by different administrators will see the same corrected orientation.
  • Efficiency: No need to download and rotate images locally for analysis.
  • Accuracy: Better readability of OCR data and visual details.
