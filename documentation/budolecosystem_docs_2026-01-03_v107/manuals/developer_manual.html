1. Flutter Project Reset

Due to configuration drift and unresolved references in GeneratedPluginRegistrant.java, the project structure was reset using:

This regenerated the android directory while preserving the lib and pubspec.yaml files. Key changes included:

  • Switching to Kotlin-based MainActivity (default for new Flutter projects).
  • Updating app/build.gradle to match modern Flutter/Gradle standards.
  • Renaming the project to budol_pay_mobile in pubspec.yaml to ensure consistency.
