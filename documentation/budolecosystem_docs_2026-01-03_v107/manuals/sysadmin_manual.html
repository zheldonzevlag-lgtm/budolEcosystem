System Admin Manual - v107

1. Building the APK

To generate a fresh APK for testing, run the following command from the budolPayMobile directory:

flutter build apk --debug

The output will be located at: build\app\outputs\flutter-apk\app-debug.apk

2. Service Troubleshooting

If the payment flow fails, check the logs for the following services:

  • Wallet Service (Port 8002): Look for [Wallet] prefixed logs to verify if the QR scan reached the backend.
  • API Gateway (Port 8080): Check for proxy errors if the mobile app cannot reach the wallet service.
  • Payment Gateway (Port 8004): Verify if the webhook from the wallet service was received.

3. Network Configuration

Ensure the mobile device/emulator can reach the API Gateway IP (currently configured as 192.168.1.24 in the app). If the IP changes, update the apiUrl in qr_scanner_screen.dart and rebuild the APK.
