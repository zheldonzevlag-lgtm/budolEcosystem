User Manual - v3.3.27

End-User Guide

What's New?

This release focuses on internal system improvements and reliability. End-users will not see visual changes in the app interface, but may experience improved reliability in receiving OTPs via SMS and Email.

Key Improvements

  • Reliable OTP Delivery: Issues preventing OTPs from being sent via email or SMS have been resolved.
  • Faster Registration: Optimizations in the backend ensure smoother signup flows.
