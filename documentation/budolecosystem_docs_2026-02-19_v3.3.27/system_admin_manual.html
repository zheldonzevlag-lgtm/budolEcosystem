System Administrator Manual - v3.3.27

Operations & Maintenance Guide

Logging Updates

OTP Visibility: The Authentication Service now explicitly logs OTP codes to the console in Yellow Text (`\x1b[33m`). This is intended for development and troubleshooting.

Monitoring: Admins monitoring the `auth-service` logs should expect to see these entries. If log aggregation tools (like Splunk or ELK) are used, ensure they can handle or strip ANSI color codes if necessary for search indexing.

Environment Variables

Ensure the `.env` file is present in the project root. The service now strictly enforces loading this file at startup. Missing `.env` or missing credentials within it will result in startup errors or functional failures (e.g., email sending).
