Risk Register - v3.3.27

Risk Assessment & Mitigation Log
Risk ID Risk Description Severity Status Mitigation / Acceptance
RSK-001 OTP Codes logged in plain text in console logs. Medium Accepted Accepted for Development: Required by developers for testing flows without accessing SMS/Email gateways. In Production, ensure logs are restricted or the specific logging line is commented out if strict security is required. Currently enabled to facilitate debugging.
RSK-002 Environment variables loaded via relative path in `auth-service`. Low Mitigated Using `path.resolve` to reliably locate `.env` file regardless of execution directory.
