User Manual - v328

Biometric Prompt Fixed

In version 1.2.52, we have fixed the issue where the fingerprint prompt was not showing up when you tapped the icon.

Step 1: Update to 1.2.52

Download and install the latest APK. You should see version 1.2.52 at the bottom of the login screen.

Step 2: Authenticate

  1. Tap the fingerprint icon.
  2. The system fingerprint or face recognition prompt should now appear instantly.
  3. Complete the authentication to log in.

Step 3: Still having issues?

If the prompt still doesn't show, check the Debug Logs at the bottom of the screen. It will now show specific Android error messages (e.g., "Hardware Busy", "Too many attempts", etc.) that will help us fix it for your specific phone model.
