Biometric Prompt Fix

This version fixes a critical issue where the Android biometric prompt would not appear even if hardware was available. This was a system-level integration issue between the app's Main Activity and the Android OS security layer.

Verification: Tapping the biometric icon should now correctly trigger the OS fingerprint/face prompt.
