v1.3.6

Release Overview: Registration UX & OTP Flow Optimization

Released: 2026-02-07 | Security Status: Verified

Version 1.3.6 focuses on optimizing the registration-to-login transition, specifically ensuring that users who register via phone number are immediately presented with the OTP verification input without having to re-enter their details.

Key Improvements

  • OTP Flow Continuity: Automatically displays the verification code input after successful phone registration.
  • State Preservation: Persists the registered phone number into the login form state to eliminate redundant input.
  • Transition UX: Smooth transition from "Create Account" to "Welcome Back" with active verification state.
