Known Issues: v1.3.5

1. UI: Mobile Menu Identifiers

While the desktop Navbar dropdown hides temporary emails, the mobile drawer implementation requires further testing across all screen sizes to ensure consistent truncation and display logic.

2. Auth: Browser Password Managers

Despite autoComplete="off", some password managers (Chrome, LastPass) may still attempt to offer password generation or autofill for new account forms. This is a browser-level behavior that is difficult to override completely.
