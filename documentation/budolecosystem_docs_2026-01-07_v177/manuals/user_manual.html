Local Development Data

Note to Users/Developers: The data stored in your local environment is completely separate from the production (Vercel) database. Actions taken locally will not affect live user accounts or transactions.

Key Benefits of Local Setup

  • Testing Safety: Create and delete test users or transactions without risk to real data.
  • Performance: Fast, offline access to database features.
  • Experimentation: Try new features or database changes in a safe sandbox.

Accessing Your Local Data

Developers can use the Prisma Studio tool to visually browse and edit local data. This is typically accessed by running npm run prisma:studio in the project folder.
