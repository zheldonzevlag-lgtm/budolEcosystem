Local Database Service Management

Service Status (Verified 2026-01-07)

PostgreSQL (postgresql-x64-18)

Status: Running

Role: Primary local database provider for all ecosystem services.

WAMP (MySQL/MariaDB)

Status: Stopped

Note: WAMP is currently not in use for this ecosystem setup but remains installed as a legacy/alternative option.

Verification Steps

  1. Open PowerShell as Administrator.
  2. Run Get-Service -Name postgresql* to confirm the service is active.
  3. Check port availability using netstat -ano | findstr :5432.
