Local Database Configuration

Connection Overview

Both core projects are currently aligned to use a local PostgreSQL instance. This ensures schema compatibility with the production Vercel environment.

1. budolpay (Monorepo)

Configured via budolpay-0.1.0/packages/database/.env:

DATABASE_URL="postgresql://postgres:r00t@localhost:5432/budolpay?schema=public"

2. budolshap (App)

Configured via budolshap-0.1.0/.env:

DATABASE_URL="postgresql://postgres:r00t@localhost:5432/budolshap_1db?schema=public"

Common Credentials

  • User: postgres
  • Password: r00t
  • Host: localhost
  • Port: 5432

Prisma Commands

To sync local changes, run these in the respective project roots:

  • npx prisma db push (Fast sync)
  • npx prisma studio (Data browser)
