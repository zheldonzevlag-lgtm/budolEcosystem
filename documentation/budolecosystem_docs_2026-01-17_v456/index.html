budolEcosystem v456

Release Date: 2026-01-17 | Type: Emergency Hotfix

MIME Type & Web Renderer Fix

Primary Issue: Web Black Screen

The Flutter Web application was failing to initialize due to a MIME type mismatch (text/html) when loading DDC module scripts. This occurred because the development server was misidentifying script requests as page requests.

Technical Fixes

  • Web Renderer (HTML): Forced the Flutter web renderer to html in start-ecosystem.js. This renderer is more stable for basic web servers and avoids complex CanvasKit module loading.
  • Base HREF: Explicitly set <base href="/"> in index.html to ensure consistent path resolution across all environments.
  • Sync Bootstrap: Removed the async attribute from flutter_bootstrap.js to ensure synchronous engine initialization and better error catching.
  • Enhanced Fallback UI: Improved the builder in main.dart to show a "Booting BudolPay Engine..." UI if the Flutter engine takes longer than expected to render the first frame.
