System Admin Manual - v4

OTP Configuration

1. Navigate to System Settings in the Admin Dashboard.
2. Configure the SMS Provider (ZERIX, ITEXTMO, or VIBER).
3. Enter the respective API Keys and Client Codes.
4. Set the OTP Expiry (default 10 minutes) and Rate Limit (default 5 attempts/10 mins).

Monitoring

Check the AuditLog table for OTP_SEND and OTP_VERIFY actions to track delivery success rates and potential abuse.
