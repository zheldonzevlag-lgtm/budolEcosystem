Release Summary: OTP Login

This release introduces One-Time Password (OTP) authentication for mobile number logins, enhancing security and user experience by eliminating the need for passwords for mobile users.

Key Features

  • Passwordless Mobile Login: Users can now login using a 6-digit verification code sent via SMS/Email.
  • Dynamic UI Transitions: Seamless switching between Email (Password) and Mobile (OTP) login methods.
  • Secure OTP Generation: 6-digit codes with 10-minute expiry and rate limiting.
  • Unified Backend: API support for both traditional password and OTP-based authentication.
