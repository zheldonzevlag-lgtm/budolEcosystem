Developer Manual: Android Build v1.2.66

Version: v363 (Internal Build Reference)

This manual covers the technical changes and build process for the 1.2.66 release of budolPay Mobile.

1. Versioning Changes

The application version has been incremented to 1.2.66+66. This is reflected in the following files:

  • pubspec.yaml - version: 1.2.66+66
  • api_service.dart - _appVersion = '1.2.66'

2. Build Process

To generate the production APK, the following command is executed:

flutter build apk --release

The resulting artifact is renamed to budolPay 1.2.66.apk and moved to the /android-installer directory.

3. Key Feature Integration: Face Recognition

This build includes the finalized Phase 4 integration of Face Recognition.

  • Liveness Detection: Uses ML Kit in kyc_verification_screen.dart
  • Face Embedding: FaceNet TFLite implementation in face_embedding_service.dart
  • Biometric Login: Secure template matching in login_screen.dart
