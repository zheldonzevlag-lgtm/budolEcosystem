System Admin Manual - v363

Deployment & Release Management

This manual provides instructions for deploying and managing the budolPay 1.2.66 release.

1. Artifact Management

The Android application installer is stored at:

\budolEcosystem\android-installer\budolPay 1.2.66.apk

Admins should ensure the previous version (1.2.65) is archived before distributing the new build.

2. Phase 8: Software Rationale & Dependencies

Rationale for Choice of Software:

  • Flutter 3.x: Chosen for cross-platform consistency and high-performance UI rendering on low-end Android devices common in the local market.
  • FaceNet TFLite: Selected for its lightweight footprint (approx. 20MB) and high accuracy for 1:1 biometric verification on-device.
  • Package Info Plus: Used to ensure the splash screen versioning matches the physical APK version for compliance auditing.
