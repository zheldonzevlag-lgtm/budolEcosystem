v234: Real-time Compliance Monitoring

Live Monitoring Enabled

Purpose

To transition the Compliance Shield from a static UI component to a proactive, real-time monitoring system that detects regulatory violations as they occur in the development and production environments.

Technical Implementation Details

1. Telemetry API (Backend)

A new internal endpoint /api/admin/compliance/telemetry was created to perform health checks on critical compliance vectors:

  • Database Security (DPA 2012): Verifies SSL/TLS encryption for database connections.
  • Gateway Heartbeat (PCI DSS): Validates the connectivity and health of the budolPay gateway.
  • Audit Integrity (BSP 808): Checks for recent administrative activity logs to ensure auditability.
  • Receipt Synchronization (E-Commerce Act): Monitors the backlog of digital receipts awaiting synchronization.

2. Reactive Shield (Frontend)

The ComplianceShield.jsx component now implements a polling mechanism (30-second intervals) to fetch telemetry data. It features:

  • Dynamic Status Colors: Visual feedback ranging from emerald (SECURED) to rose (RISK/ERROR).
  • Warning Tooltips: Inline explanations for "amber" or "rose" states to guide administrators on fixes.
  • Connection Resiliency: Automatic "Telemetry Link Severed" warnings if the API becomes unreachable.

Risks & Mitigations

Risk: High polling frequency could impact database performance.
Mitigation: Polling is restricted to 30-second intervals and utilizes indexed queries for audit log counts.
Risk: False positives in development due to local DB configurations.
Mitigation: Status logic accounts for process.env.NODE_ENV, downgrading SSL requirements in development while maintaining warnings for production.

Future Enhancements

  • WebSocket Integration: Move from polling to real-time pushes for instant violation alerts.
  • Automatic Remediation: Scripts to automatically restart services or rotate keys upon certain compliance failures.
  • Executive Alerts: Email/SMS notifications for high-severity (RISK) status changes.
