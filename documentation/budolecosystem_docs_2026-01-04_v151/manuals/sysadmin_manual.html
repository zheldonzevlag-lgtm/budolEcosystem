System Admin Manual - Version 151

Release 1.1.3 Overview

This release contains critical stability improvements for the mobile application. All previous 1.1.x versions should be replaced with this build.

New Installer Location

The new APK has been deployed to the following path:

D:\IT Projects\budolEcosystem\android-installer\budolPay 1.1.3.apk

Deployment Steps

  1. Distribute the budolPay 1.1.3.apk to users.
  2. Ensure all backend services (API Gateway, Wallet Service) are running on the correct LOCAL_IP (currently 192.168.1.24).
  3. Users may need to uninstall previous versions if there are signature conflicts (though standard updates should work if keystores match).
