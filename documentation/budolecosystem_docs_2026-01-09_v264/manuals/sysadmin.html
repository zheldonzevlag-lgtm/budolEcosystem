Branding Guidelines

This version implements a new branding rule for the marketing splash screens. All services within the budolEcosystem should follow the "budol[Service]" naming convention, where "[Service]" is highlighted in the service's primary brand color.

Service Color Map

Service Color Hex Visual Group
budolLoan #3B82F6 Financial / Blue
budolShap #10B981 Commerce / Emerald
budolExpress #F59E0B Logistics / Amber

Operational Impact

  • UI/UX: The splash screens now have higher visual contrast, making it easier for users to distinguish between different ecosystem services during onboarding.
  • No Backend Impact: These changes are purely cosmetic and reside within the mobile application's frontend logic.
