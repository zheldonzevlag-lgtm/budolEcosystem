Purpose

This update enhances the MarketingAdsScreen to allow for more granular styling of service titles. Specifically, it enables the "Loan" part of "budolLoan" (and other service suffixes) to be rendered in their respective theme colors while keeping the "budol" prefix consistent.

Technical Details

The Text widget in _buildSlide was replaced with a RichText widget to allow multiple TextSpans with different styles.

Code Implementation

File: lib/screens/marketing_ads_screen.dart

// Implementation of multi-colored titles
RichText(
  text: TextSpan(
    style: const TextStyle(
      color: Colors.white,
      fontSize: 32,
      fontWeight: FontWeight.bold,
      letterSpacing: -0.5,
    ),
    children: [
      TextSpan(
        text: slide['title'].toString().startsWith('budol') ? 'budol' : slide['title'].toString(),
      ),
      if (slide['title'].toString().startsWith('budol'))
        TextSpan(
          text: slide['title'].toString().substring(5),
          style: TextStyle(color: slide['color']),
        ),
    ],
  ),
)

Logic Explanation

  • String Splitting: The logic checks if the title starts with "budol".
  • Prefix Styling: The first 5 characters ("budol") are rendered using the default white color.
  • Suffix Styling: The remaining characters (from index 5 onwards) are rendered using the slide['color'] defined in the _slides list.
  • Thematic Consistency: For "budolLoan", the color is blue (0xFF3B82F6). For "budolShap", it is emerald green, and for "budolExpress", it is amber.
