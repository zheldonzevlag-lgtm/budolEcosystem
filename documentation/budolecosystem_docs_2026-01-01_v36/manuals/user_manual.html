User Manual Updates (2026-01-01 v36)

Quick Registration Fix

We have fixed an issue where the "Quick (Phone Only)" registration was showing an error about passwords even though no password was required.

Improved Workflow:

  • Users can now use the "Quick (Phone Only)" option by simply providing their phone number.
  • The system will no longer ask for or validate a password for this specific registration type.

Next Steps for Users:

After registering with just your phone number, you will have a temporary profile. You can set up your permanent password and other security details later in your Account Settings.
