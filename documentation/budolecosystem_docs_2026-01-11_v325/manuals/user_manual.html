User Manual - v325

System Availability & Connection

This update ensures the backend services are correctly initialized and available for use by the BudolPay mobile application.

Backend Services Restored

A technical issue preventing the services from starting on Windows systems has been resolved. You should now be able to:

  • Log in using your mobile number and PIN.
  • Perform wallet transactions (Cash In, Cash Out, Transfer).
  • Manage account settings including Biometric Login.

Troubleshooting Connection Issues

If the mobile app shows a "Connection Error" or "Gateway Timeout":

  1. Ensure your mobile device is on the same Wi-Fi network as the gateway.
  2. Use the Scan Network feature in the Login screen settings (Gear icon) to automatically find the active gateway.
  3. Verify the gateway IP is set to 192.168.1.17:8080 (or the current IP displayed in the server logs).
