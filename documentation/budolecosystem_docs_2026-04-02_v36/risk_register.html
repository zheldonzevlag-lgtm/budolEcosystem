Compliance Risk Register: v36 Stabilization

Analysis of potential threats, security vulnerabilities, and compliance gaps related to mobile authentication and transaction processing.

Risk ID Conflict/Description Impact Severity Response
R-AUTH-006 PIN Reset Loop (Vercel Branch Mismatch) Denial of Service for legitimate users trying to reset credentials. High Fixed: Monorepo consolidated into main branch with correct route mapping.
R-AML-021 P2P Transfer KYC Bypass Fines from BSP (Circular 920) for allowing unverified money laundering transfers. Critical Mitigated: Global BASIC bypass removed. Manual intervention only for test accounts.
R-SEC-012 Invisible "Forgot PIN" Button Administrative Confusion & User frustration leading to high support ticket volume. Medium Fixed: Material Theme CSS applied with Colors.white foreground.
R-SEC-014 Stale OTP Persistence Attacker re-using previous OTP codes to fulfill authentication steps. Medium Mitigated: Backend now explicitly clears otpCode on reset-pin call.
