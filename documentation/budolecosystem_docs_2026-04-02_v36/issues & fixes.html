Bug Report: v36 Release Issues & Fixes

Invisible "Forgot PIN" Button Resolved

ISSUE: Android Dark Theme defaulted button text to a dark color against a dark background, making the reset link completely invisible on some devices.

FIX: Applied explicit foregroundColor: Colors.white and textStyle: TextStyle(fontWeight: FontWeight.bold) to the Flutter TextButton component.

PIN not set (Vercel API Mismatch) Resolved

ISSUE: Mobile app was stuck in a loop during PIN setup. The backend was hitting a 404 on the preview branch during the final step due to monorepo branch syncing failure.

FIX: Merged feature/auth-bridge-v32 into main and updated Flutter api_service.dart to target budolpay.vercel.app as the default fallback.

Invalid recipient resolving (Self-Transfer) Resolved

ISSUE: The P2P transfer route allowed users to input their own phone number as the recipient, which leads to ledger duplication and account balance instability.

FIX: Added server-side validation: if (senderId === resolvedReceiverId) throw new Error("Cannot send money to yourself");
