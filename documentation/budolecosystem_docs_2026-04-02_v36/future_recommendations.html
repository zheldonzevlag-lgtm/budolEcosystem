Future Enhancements & Roadmap: BudolPay Mobile

Strategic directions for elevating the user experience and security post-migration to Vercel.

🛡️ PIN Security: Biometric Integration (BSP Compliance)

The current 6-digit PIN as a sole second-factor (besides OTP) is sufficient for current volume. However, as the user base scales, integrating Biometrics (FaceID/Fingerprint) into the verification step will improve security and user convenience. local_auth in Flutter should be priority #1 for v37.

🏗️ Automated KYC Workflow

Manual intervention is currently required to move users from BASIC to FULLY_VERIFIED tiers. We recommend integrating a third-party Verification-as-a-Service (like Netverify or Jumio) to automatically scan government IDs and perform "liveness checks" directly in the Flutter app.

🔐 Real-Time Push Notifications for P2P

Transactions currently rely on polling or simple Pusher bridge. Implementing Firebase Cloud Messaging (FCM) or native Android/iOS push notifications will allow users to receive real-time "Money Received" alerts, closing the trust gap for face-to-face QR merchant transactions.
