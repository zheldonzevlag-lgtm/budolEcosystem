Deployment Checklist

  1. Verify the APK size is approximately 63.9MB.
  2. Distribute the APK to authorized users via the internal download portal.
  3. Ensure the backend services (Gateway, Transaction, Auth) are running at least v163 or higher for compatibility with history features.
  4. Monitor the Gateway logs for increased traffic following the update.

Operational Runbook

In case of issues with the 1.1.6 release:

  1. Retain the previous v162 APK as a fallback.
  2. If users report crashes, verify they are seeing the v164 "Oops" screen, which indicates the recovery system is working.
  3. Collect logs from the Gateway API to diagnose communication errors.
