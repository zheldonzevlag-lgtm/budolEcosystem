Web Performance Tuning

V447 Update: The transition delay from Splash to Marketing Ads is now 800ms. This is intentional to prevent the "black screen" glitch on certain browsers (Chrome/Safari) when using the CanvasKit renderer.

1. Browser Compatibility

Ensure that client browsers support WebGL for optimal rendering. If users report sluggishness, verify that hardware acceleration is enabled in their browser settings.

2. Viewport Constraints

The app now implements a hard fallback for viewport dimensions (393x780). If the app is embedded in an iframe, ensure the iframe width/height are explicitly set to avoid the fallback dimensions taking precedence over the actual frame size.
