System Admin Manual v46

Backup & Recovery

All system states are now backed up to the from-local-backup branch. In case of local failure, administrators should clone the repository and switch to this branch to restore the latest known stable state.

Manual Synchronization

To manually synchronize local changes, run:

git checkout -b from-local-backup
git add .
git commit -m "Manual sync v46"
git push origin from-local-backup
