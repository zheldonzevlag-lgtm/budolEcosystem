Release Summary

This version addresses a critical issue where the budolID SSO service was overriding production redirect URIs with local IP addresses during startup. This caused production users on budolpay.vercel.app to be incorrectly redirected to 192.168.1.2 after successful authentication.

Key Changes:

  • Dynamic Redirect URIs: SSO now respects the redirect_uri parameter passed during login.
  • Security Whitelisting: Implemented domain validation to prevent Open Redirect attacks (PCI DSS Compliance).
  • Non-destructive Seeding: Core app configuration is no longer reset in production environments.
