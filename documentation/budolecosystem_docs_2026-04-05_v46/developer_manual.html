Developer Manual v46

Technical Explanation

This version includes diagnostic utilities for Prisma and Database connectivity.

Code References

  • tmp_check_db.ts: Utility for checking raw database connection via Prisma.
  • tmp_check_prisma.ts: Utility for inspecting Prisma Client properties.

Rationale

Providing standardized diagnostic tools reduces troubleshooting time during environment setup or after database migrations.

Dependencies

  • Prisma Client v5.10.x
  • Node.js v18+
