Future Enhancements & Recommendations: SSO Security v46

Updated: 2026-04-05 | Release: v46

1. MFA Integration

Priority: High Implementation of Multi-Factor Authentication (MFA) for administrative accounts to further secure the BudolPay ecosystem in accordance with BSP and NPC recommendations.

Proposed Action: Integrate TOTP (Time-based One-Time Password) or SMS-based verification into the budolID login flow.

2. PKCE (Proof Key for Code Exchange) Support

Priority: Medium Adoption of the PKCE flow for all SSO redirections to prevent authorization code interception attacks, especially for mobile and single-page applications.

Proposed Action: Update the SSO handshake to support code_challenge and code_verifier parameters.

3. Enhanced Session Monitoring

Priority: Low Real-time monitoring and alerting for anomalous login patterns (e.g., multiple failed attempts from different IPs) to proactively identify brute-force or credential stuffing attacks.

Proposed Action: Integrate with budolShield's real-time event tracking for security alerting.
