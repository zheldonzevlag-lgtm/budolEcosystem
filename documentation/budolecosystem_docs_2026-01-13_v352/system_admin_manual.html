Deployment and Monitoring for v1.2.61

← Back to Index

1. Build Requirements

To deploy the updated mobile app, use the following command to generate the APK:

flutter build apk --release --version 1.2.61 --build-number 61

2. Web Deployment

For Flutter Web, ensure the build artifacts are updated in the web server root:

flutter build web --release --base-href /

The output in build/web/ should be synced to the production environment.

3. Verification

After deployment, verify that the splash screen shows budolEcosystem 1.2.61 at the bottom. This confirms that the ApiService and pubspec.yaml are in sync.

4. Troubleshooting

If marketing ads still do not show, clear the application cache or local storage (Web) to reset the budolpay_has_seen_ads key in SharedPreferences.
