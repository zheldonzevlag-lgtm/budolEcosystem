Endpoint Aliasing

Action: Implemented endpoint aliasing in the verification-service to support both /verify and /upload routes.

Status: Resolved

Ref: R-KYC-002 (Admin URL Resolution)

Status: Resolved

Audit Logging & Access Control

Status: In Progress
