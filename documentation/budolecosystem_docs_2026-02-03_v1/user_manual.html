For Mobile Users: KYC Verification

1 Open the BudolPay app and navigate to Profile > Identity Verification.
2 Select Verify Identity. You will need to provide:
  • Personal Information (Legal Name, Birthday)
  • A clear photo of a Government-issued ID
  • A Selfie for liveness detection
3 Once submitted, your status will change to PENDING. You will receive a notification once verified.

For Admin Users: Reviewing KYC

1 Log in to the BudolPay Admin Dashboard.
2 Navigate to Compliance & Trust > User Verification.
3 Select a user with PENDING status to review their documents.
4 Click Approve or Reject based on the document quality.
