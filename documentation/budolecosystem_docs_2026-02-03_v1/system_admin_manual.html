Service Configuration

Ensure the Verification Service is running on port 8006 and the API Gateway is correctly routing /verification traffic.

# API Gateway Routing
/verification -> http://localhost:8006

KYC Document Storage

Documents uploaded by users are stored in the /uploads directory of the verification service. This directory is served statically to allow the Admin Panel to display documents.

Important: Ensure the uploads/ directory has the correct write permissions for the node process.

Monitoring Audit Logs

Admins can monitor KYC status changes in the AuditLog table. Each entry includes compliance metadata (PCI DSS, BSP).
