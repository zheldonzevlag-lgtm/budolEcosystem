📝 Known Issues v3.0.2

High

Mobile KYC Upload 404

Description: The mobile app was calling /verification/upload while the verification service only exposed /verify. This caused KYC submissions to fail with a 404 error.

Workaround: Use the updated verification-service with endpoint aliasing.

Medium

Broken KYC Images in Admin Panel

Description: Admin dashboard was unable to resolve relative remoteUrl paths for KYC documents, resulting in broken image icons.

Workaround: Ensure NEXT_PUBLIC_GATEWAY_URL is correctly set in the admin's .env file.
