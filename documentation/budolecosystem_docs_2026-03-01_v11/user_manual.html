User Manual v5.4.0

Guide for Admin staff on verified profile editing.

1. Editing a Staff Account

To edit a staff account, follow the standard procedure in the "Internal Workforce" registry:

  1. Click the "Edit" icon (the pencil icon in the Actions column) next to the user.
  2. A "Security Authorization" modal will appear.
  3. An OTP (One-Time PIN) will be sent via both **SMS** and **Email** to the staff member's registered contact details.
  4. Ask the staff member for the code. This ensures they are aware and authorize the modification of their profile.
  5. Enter the 6-digit code in the verification modal.
  6. If the code is correct, the "Edit User Profile" modal will be displayed.

2. Phone Number Formatting

When entering a phone number (e.g., 0917-XXX-XXXX), the system will automatically format it to the international standard (+63917...) when you move to the next field or click update. This ensures consistency across the ecosystem.
