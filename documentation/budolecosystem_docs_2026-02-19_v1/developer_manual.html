1. Authentication Service: Unmasking User Data

Modified Response Handling

The auth-service API endpoints for OTP verification, PIN verification, and PIN setup were modified to return unmasked `firstName` and `lastName` fields. This allows the mobile application to display the user's real name upon successful authentication, enabling the "Show/Hide Name" feature on the Home Screen.

// services/auth-service/index.js // In /verify-otp endpoint user: { id: updatedUser.id, phoneNumber: maskPII(updatedUser.phoneNumber), firstName: updatedUser.firstName, // Changed from maskPII() lastName: updatedUser.lastName // Changed from maskPII() }, // In /login/mobile/verify-pin endpoint user: { id: user.id, phoneNumber: maskPII(user.phoneNumber), firstName: user.firstName, // Changed from maskPII() lastName: user.lastName // Changed from maskPII() }

2. Mobile App: Home Screen Visibility Toggle

UI Implementation

The `HomeScreen` widget was updated to include a visibility toggle state (`_isNameVisible`). The welcome message now dynamically switches between the user's first name and a masked string (`........`) based on this state. The default state is now masked (false). An eye icon button controls the toggle. The font size was reduced to 14sp to prevent overflow, and `Flexible` widget was added to handle long names gracefully.

// lib/screens/home_screen.dart bool _isNameVisible = false; // State variable (Default: Masked) // Updated Logic for First Name Only final firstName = user['firstName']?.toString() ?? ''; userName = firstName.trim(); // Widget build method Row( children: [ Flexible( child: Text( 'Welcome back, ${_isNameVisible ? userName : '........'}', style: const TextStyle(color: Colors.white, fontSize: 14, fontWeight: FontWeight.w500), overflow: TextOverflow.ellipsis, ), ), const SizedBox(width: 8), GestureDetector( onTap: () { setState(() { _isNameVisible = !_isNameVisible; }); }, child: Icon( _isNameVisible ? Icons.visibility : Icons.visibility_off, color: Colors.white70, size: 16, ), ), ], ),

Enhanced OTP Logging
