v5.0.2: Checkout Hotfix & Schema Migration

Date: 2026-03-02 Version: v5.0.2 Status: Released

Implementation Summary

This release addresses critical regressions discovered after v5.0.1. It resolves undefined variable references in checkout services and implements a logical separation of databases using PostgreSQL schemas to avoid model conflicts between budolShap and budolPay.

  • Prisma Client: Fixed prisma reference in checkoutService.js.
  • Variable Scoping: Corrected activeCheckoutId scope in API routes.
  • Schema Migration: Migrated budolPay to a dedicated budolpay schema.
  • Environment: Synchronized all .env files to use the new schema-aware DATABASE_URL.
