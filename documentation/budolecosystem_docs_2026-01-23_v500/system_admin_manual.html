System Admin Manual - v500

Lalamove Webhook Monitoring

Admins should monitor the /api/webhooks/lalamove endpoint logs for any mapping failures. If a "null" status is logged, it indicates a new Lalamove status that needs to be added to statusMapper.js.

Configuration

No changes to environment variables are required for this release. The system continues to use the standard Lalamove API keys and Webhook secrets.

Auto-Completion Policy

Orders are now automatically scheduled for completion 3 days after the deliveredAt timestamp is set. This can be adjusted in the webhook handler if business requirements change.
