Known Issues

Issues identified during the development and testing of version 1.3.65.

Minor

Hardcoded Timeout Constants

The 3-minute idle timeout and grace period are currently hardcoded in SessionService. Changes require a re-build of the mobile app rather than a server-side setting update.

Minor

Blur Background Performance

On older Android devices, the BackdropFilter blur effect might cause a slight frame rate drop when the lock screen appears.

Minor

Unit Test Plugin Mocking

Automated tests show MissingPluginException for shared_preferences. While logic is verified, the full integration test requires a physical device or emulator.
