User Manual – v378 Face Recognition Reliability

What Changed

Face registration is now more reliable on Android phones. If the camera capture fails with a technical error, the app automatically retries once before asking you to try again.

Steps Unchanged

  1. Open Settings → Security → Face Recognition Login.
  2. Tap Register Face and allow camera access.
  3. Position your face inside the oval and blink when prompted.
  4. Wait for the success check-mark.

What You’ll Notice

  • Fewer “Face registration failed” pop-ups.
  • Slightly longer first-time setup (extra safety check).

Need Help?

If registration still fails twice, ensure strong lighting and clean camera lens, then retry. Contact support with screenshot if issue persists.
