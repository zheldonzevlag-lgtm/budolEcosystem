System Admin Manual – v378 Face Recognition Reliability

Deployment Checklist

  1. Verify assets/models/facenet.tflite exists in repo.
  2. Ensure path_provider permission for temp directory (Android).
  3. Deploy APK version 1.2.78+78.
  4. Monitor crash-free rate for face_embedding_service in Crashlytics.

Monitoring

  • Logcat keyword: FaceEmbeddingService – filter for retry attempts.
  • Metric: face_registration_null_interpreter should trend to zero.

Rollback

Revert to v377 APK; no server-side changes required.

Compliance

No new data retention or network endpoints; biometric accuracy improvement only.
