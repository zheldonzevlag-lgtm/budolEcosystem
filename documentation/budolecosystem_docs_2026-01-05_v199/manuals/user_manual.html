User Manual - v199: Notification Dashboard Guide

Overview

The Notification Settings page allows administrators to configure how the system sends emails and SMS messages to users and staff.

How to Configure Providers

  1. Select Provider: Choose between Email and SMS sections.
  2. Fill Credentials: Enter the API keys, usernames, and passwords provided by your service provider.
  3. Save Changes: Click the "Save Settings" button at the bottom of the provider's card.
  4. Activate: If the provider is not already active, click "USE THIS" at the top right of the card to make it the primary gateway.

Active Indicators

An ACTIVE badge in blue (for Email) or green (for SMS) indicates which provider is currently being used by the system to send real-time notifications.
