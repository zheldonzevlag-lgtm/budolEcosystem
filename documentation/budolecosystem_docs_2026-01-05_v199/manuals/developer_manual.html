Data Model

Settings are stored in the SystemSetting table. The group field is used to categorize settings for efficient retrieval:

  • group: 'EMAIL': Contains SMTP, Brevo, and GMass configurations.
  • group: 'SMS': Contains Viber, ITexMo, Brevo SMS, Zerix, and existing gateways.

UI Architecture

The dashboard is built using a responsive Tailwind CSS grid (grid-cols-1 lg:grid-cols-2). Each provider card handles its own form submission, allowing for modular updates without affecting other configurations.

Recommendation: When adding new providers, ensure the group and provider hidden inputs are correctly set in the form to maintain compatibility with the server action.
