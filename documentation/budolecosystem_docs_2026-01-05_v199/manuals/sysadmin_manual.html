System Admin Manual - v199: Managing Notifications

Environment Variables / Database Keys

Key Category Description
EMAIL_PROVIDER EMAIL Active email gateway (GOOGLE, BREVO, GMASS)
SMS_PROVIDER SMS Active SMS gateway (VIBER, ITEXMO, BREVO_SMS, ZERIX)
SMTP_HOST EMAIL SMTP Server address
ITEXMO_API_KEY SMS API Key for ITexMo services
Security Notice: Ensure all API keys are kept confidential. The system masks these values in the UI after saving for enhanced security.

Deployment Steps

  1. Navigate to Settings > Notification in the Admin Panel.
  2. Enter the required credentials for your preferred providers.
  3. Click Save [Provider] Settings.
  4. Click USE THIS on the provider you wish to activate for the ecosystem.
