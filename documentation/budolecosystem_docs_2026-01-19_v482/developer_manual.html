Implementing Balance Masking on Home Screen.

Implementation Details

The account balance is now masked by default using a boolean state variable _isBalanceVisible. An IconButton with dynamic icons (visibility / visibility_off) allows the user to toggle this state.

Key Code Reference:

Modified home_screen.dart to include the masking logic.

// State variable bool _isBalanceVisible = false; // UI Implementation Row( crossAxisAlignment: CrossAxisAlignment.center, children: [ Text( _isBalanceVisible ? _balance : '₱ ••••••', style: const TextStyle( color: Colors.white, fontSize: 36, fontWeight: FontWeight.bold, ), ), const SizedBox(width: 8), IconButton( constraints: const BoxConstraints(), padding: EdgeInsets.zero, iconSize: 20, icon: Icon( _isBalanceVisible ? Icons.visibility : Icons.visibility_off, color: Colors.white70, ), onPressed: () { setState(() { _isBalanceVisible = !_isBalanceVisible; }); }, ), ], )

UX Rationale

Following fintech best practices (e.g., GCash, Maya), hiding sensitive financial information by default reduces the risk of visual hacking in public places. The eye icon is placed immediately after the amount for intuitive access.
