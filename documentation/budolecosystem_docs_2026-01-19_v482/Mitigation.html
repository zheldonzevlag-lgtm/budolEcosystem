Mitigation Strategies (v482)

M-482-01: User Education

Related Risk: R-482-01 (User Confusion)

Action: Updated the User Manual with clear instructions on how to use the eye icon toggle. The masking is intuitive (₱ ••••••) to signal hidden content.

Related Risk: R-482-02 (Accidental Reveal)
