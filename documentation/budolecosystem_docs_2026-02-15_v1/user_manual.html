User Manual - Forensic Audit Trails (Internal Use)

Date: 2026-02-15

1. Overview

The Forensic Audit Trails system is designed for system administrators and compliance officers. It provides a detailed record of all actions taken within the Budol Ecosystem.

2. Access Requirements

To access the Forensic Audit Trails, you must have the Admin or Compliance Officer role. If you believe you should have access but do not, please contact your system administrator.

3. Viewing Logs

  1. Navigate to the Admin Dashboard.
  2. Select Settings > Forensic Audit Trails.
  3. Use the filters to find relevant logs.

4. Reporting Issues

If you encounter any discrepancies or errors in the audit logs, please report them immediately via the Issues page or contact the IT support team.
