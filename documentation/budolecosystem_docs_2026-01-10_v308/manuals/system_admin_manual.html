1. Environment Configuration

IP Address Synchronization

The entire ecosystem has been synchronized to use the static IP 192.168.1.17. This is critical for cross-service communication and mobile app connectivity.

2. Deployment Procedures

Mobile App Deployment

New mobile app builds are stored in the centralized /android-installer/ directory. Admins should ensure that users are notified of version 1.2.43.

Steps:

  1. Verify IP configuration in api_service.dart.
  2. Run flutter build apk --release.
  3. Move the generated APK to /android-installer/budolPayMobile_1.2.43.apk.

3. Monitoring & Maintenance

Monitor the budolID logs for any SSO token verification failures. The recent fix in budolshap ensures that tokens from budolID are correctly processed.
