Future Enhancements

Strategic roadmap and recommended improvements for the budolEcosystem.

← Back to Index

1. Payment Integration (PayMongo Webhooks)

Goal

Fully transition from client-side redirect verification to server-side webhook processing for all PayMongo transactions.

Reason

To improve reliability and prevent "ghost" payments where the user pays but the order status isn't updated due to browser closure before redirection.

Mitigation

Implement a robust webhook listener with idempotency checks to handle duplicate events from PayMongo.

2. Real-time Notifications

Goal

Expand the use of Pusher/Socket.io for real-time admin notifications when new users register or orders are placed.

3. Automated IP Discovery

Goal

Implement a dynamic IP discovery service for the mobile app to automatically find the local gateway without manual .env updates.
