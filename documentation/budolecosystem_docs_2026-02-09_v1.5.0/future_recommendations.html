Future Recommendations - v1.5.0

1. Automated Document Comparison

Integrate an AI-based comparison tool to automatically score the match percentage between the ID document name and the user's profile name, flagging high-risk discrepancies for manual lead review.

2. KYC Tier Upgrades

Implement a workflow for users to upgrade their KYC tier (e.g., from Individual to Business) without re-submitting existing verified documents.

3. Expiry Notifications

Add a system for tracking document expiry dates (extracted via OCR) and automatically notifying users 30 days before their verification status expires.

4. Centralized API Fetch Wrapper

To avoid repeated boilerplate for authentication headers, implement a custom useApi hook or a centralized fetchWrapper utility that automatically injects JWT tokens and handles common error responses for all admin components.
