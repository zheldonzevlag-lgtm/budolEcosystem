Known Issues - v1.5.0

The following issues have been identified in this release and are slated for resolution in future updates.

ISS-1.5.0-01

Prisma Include Limitations on KYC Scalars

Description: Attempting to include kycDetails or kycStatus in Prisma's include block for the User model results in an error because they are scalar fields, not relations.

Workaround: These fields are included by default in the selection. Use select if granular field control is needed, or rely on the default model return.

Status: Documented / Logic Adjusted

ISS-1.5.0-02

Inconsistent Auth Headers in Legacy Admin Pages

Description: Some older admin pages may still rely on cookies without explicit Authorization headers, which can fail in certain cross-origin scenarios.

Workaround: Standardized auth pattern has been applied to most critical pages. Remaining pages are being updated sequentially.

Status: Ongoing Cleanup

ISS-1.5.0-03

Sidebar Vertical Truncation

Description: The admin sidebar settings submenu can be very long, potentially cutting off links on smaller screens.

Workaround: Use the scrollbar within the sidebar or expand the viewport.

Status: UI Optimization Required
