User Manual - Identity Verification

How to verify your account

  1. Go to your Profile.
  2. Click Verify Account.
  3. Follow the KYC Wizard steps:
    • Select Tier (Individual/Business).
    • Upload ID Document.
    • Perform Biometric Liveness Check.
  4. Submit and wait for admin approval (1-3 business days).

Checking Verification Status

You can track the progress of your application in the Verification Center within your profile. Statuses include:

  • Pending: Application received and awaiting review.
  • Verified: Identity confirmed; all features unlocked.
  • Rejected: Application failed compliance checks. Please see the rejection reason and resubmit.
