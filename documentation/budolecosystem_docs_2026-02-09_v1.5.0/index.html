Release Notes - v1.5.0

v1.5.0

Release Date: 2026-02-09

Title: Advanced KYC Approval System & Extended RBAC

Purpose

This release introduces a centralized KYC (Know Your Customer) approval dashboard for administrators, managers, and leads. It expands the Role-Based Access Control (RBAC) system to include a 'LEAD' role and granular permissions for KYC verification, ensuring compliance with BSP and NPC standards while enabling efficient identity verification workflows.

Key Changes

  • Centralized KYC Dashboard: New admin page at /admin/kyc for reviewing and approving user identity submissions.
  • Extended RBAC: Introduced LEAD role and USERS_KYC_APPROVE permission.
  • KYC Admin API: Implemented secure endpoints for fetching filtered KYC applications and updating verification statuses with audit logging.
  • Sidebar Integration: Added "KYC Submission & Approval" submenu under "User & Member Management" in the admin sidebar.
  • Build Stability: Fixed Prisma import errors and Turbopack duplicate variable issues.
