KYC Approval Workflow

  1. Navigate to User & Member Management > KYC Submission & Approval.
  2. Review the list of pending applications.
  3. Click View Details to inspect uploaded documents (ID, Selfie).
  4. Click Approve Application to verify the user or Reject Application with a reason.

Role-Based Access Control (RBAC)

Access to KYC approval is restricted to the following roles:

  • ADMIN: Full access to all KYC and system settings.
  • MANAGER: Can review and approve/reject KYC applications.
  • LEAD: Specialized role for KYC verification and document review.

Permission required: users:kyc_approve

Audit Logs

All KYC actions are logged in the AuditLog table for compliance (BSP/AMLA).
