Developer Manual v82: SSO & Binding Logic

Binding Conflict Resolution

Developers encountered EADDRINUSE errors during service startup. This was resolved by implementing a cleanup script to kill processes occupying ports 3000, 3001, and 8080.

# Example cleanup command
netstat -ano | findstr :3000
taskkill /F /PID <PID>

Explicit IP Binding

To support multi-device testing, the ApiService and budolID server were updated to use the host machine's LAN IP instead of localhost.
