Documentation v82: SSO Service Recovery & Binding Fix

Published: 2026-01-02 | Version: 82

Purpose

This version addresses critical application startup failures caused by port binding conflicts and SSO service unavailability.

Technical Summary

  • Fixed EADDRINUSE errors by identifying and stopping orphaned node processes.
  • Explicitly bound the budolID service to the LAN IP (192.168.1.24) to ensure cross-device accessibility.
  • Restored SSO session persistence by aligning the API_HOST environment variables across the ecosystem.
