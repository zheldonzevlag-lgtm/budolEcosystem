SysAdmin Manual v82: Service Management

SSO Service Recovery

If the budolID service becomes unresponsive or returns 401 errors, follow these steps:

  1. Navigate to budolID-0.1.0 directory.
  2. Stop any running instances of npm run dev.
  3. Verify database connectivity in .env.
  4. Restart the service using npm run dev.
