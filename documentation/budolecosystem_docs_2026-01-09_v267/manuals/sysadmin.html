Release v1.2.28

This release includes UI improvements to the transaction details screen and a horizontal button layout optimization.

Artifact Details

  • Version: 1.2.28+28
  • APK Name: budolPay_1.2.28.apk
  • Build Environment: Android arm64-v8a

Deployment Instructions

  1. Download the latest APK from the android-installer directory.
  2. Distribute to testing or production channels as required.

UI Changes

The Transaction Details screen now includes the recipient's name at the top of the list, providing immediate context for the transaction. The action buttons are now side-by-side, improving the layout's efficiency.
