Improved Transaction Details

Clearer Recipient Info

You can now see the full name of the person you sent money to (or received money from) right at the top of your transaction details. No more guessing who the transaction was with!

Optimized Button Layout

The "Favorite" and "Receipt" buttons are now placed side-by-side at the bottom of the screen. This new layout makes the screen more compact and easier to navigate on all devices.

How to use

  • View Recipient: Open any transaction from your history to see the recipient's name listed first in the details section.
  • Action Buttons: Use the "Favorite" button to save the recipient for future transfers, and the "Receipt" button to download a copy of the transaction details.
