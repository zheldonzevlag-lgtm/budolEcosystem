User Manual: Map Picker Updates

We have improved the reliability of the Map Picker component used during address selection.

What's New

  • Improved Address Finding: When you pin a location on the map, the address lookup is now faster and more reliable.
  • Fewer Errors: Resolved issues where the address would fail to load ("Failed to fetch").

How to Use

No changes to usage. Simply drag the map or search for a location as usual. The address will update automatically.
