System Admin Manual - v358

Deployment Overview

This version focuses on the production build of the Android APK (v1.2.60). Admins should ensure the APK is distributed to the appropriate testing or production channels.

Build Information

  • Version: 1.2.60+60
  • Environment: Production
  • API Host: 192.168.1.17 (Configured via --dart-define)
  • Output: build/app/outputs/flutter-apk/app-release.apk

Configuration

Ensure the backend services are running and accessible at the configured API host. The biometric login features require devices with compatible hardware (Fingerprint/Face Recognition).
