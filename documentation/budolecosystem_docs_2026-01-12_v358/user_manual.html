New Features

Enhanced Login Experience

The login screen now features integrated biometric icons directly within the PIN input field. This makes it easier to access Fingerprint and Face Recognition options.

How to Use Biometrics

  1. On the Login screen, look at the right side of the PIN input field.
  2. Tap the Fingerprint icon for fingerprint authentication.
  3. Tap the Face icon for face recognition authentication.
  4. If biometrics are enabled and recognized, you will be logged in immediately.

App Version

Ensure you are using version 1.2.60. You can verify this on the splash screen or in the app settings.
