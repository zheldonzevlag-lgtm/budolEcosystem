Documentation v91: Camera Access & Browser Security

Published: 2026-01-02 | Version: 91

Issue Analysis

Error: No camera found or permission denied on HTTP origins.

Diagnosis: Modern browsers block hardware access (camera, microphone) on non-secure (HTTP) origins. localhost is generally exempt, but LAN IP access is not.

Resolution: Documented the chrome://flags workaround to treat insecure origins as secure for local development.

Technical Fix

Enable unsafely-treat-insecure-origin-as-secure in chrome://flags and add http://localhost:8080 to the list.
