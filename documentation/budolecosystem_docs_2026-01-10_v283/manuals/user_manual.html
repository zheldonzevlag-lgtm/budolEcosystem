User Manual - v283

Unified Login (budolID)

You can now use your budolID credentials to log in to both the budolShap marketplace and the budolPay Admin portal. This eliminates the need to remember multiple passwords for different services within the ecosystem.

Logging in to budolPay Admin

On the budolPay Admin login page, you will now see a new button: "Login with budolID". Clicking this will redirect you to the secure budolID login screen. Once authenticated, you will be automatically redirected back to the Admin dashboard.

LAN Testing Support

The ecosystem now supports access from other computers on the same local network. This allows for real-world testing of features like payment scanning and KYC verification across multiple devices.
