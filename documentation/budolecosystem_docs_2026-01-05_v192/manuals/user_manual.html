User Manual - v192

Real-time KYC Monitoring

Administrators can now monitor KYC submissions in real-time. The User Verification screen will automatically update every 5 seconds with the latest data from the mobile application.

Key Benefits

  • Instant Visibility: See new KYC applications as soon as they are submitted by users on their mobile devices.
  • No Page Reloads: The list of users and their statuses update automatically without you having to click anything.
  • Live Modal Updates: If you are reviewing a user's documents, any new uploads or changes will appear automatically in the viewer.
