Biometric Debugging

Biometric authentication is a client-side OS operation. If the mobile app is "not responding" to the biometric icon:

  1. Verify the user is running v1.2.50 or higher (shown at bottom of Login screen).
  2. Ensure the user has enrolled at least one fingerprint or face in their Android/iOS security settings.
  3. Check if the user has enabled "Biometric Login" in the app's Settings menu after their first PIN login.
