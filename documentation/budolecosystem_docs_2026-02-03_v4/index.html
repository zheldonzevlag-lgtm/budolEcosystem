Release v2026-02-03_v4 (v3.0.5)

Purpose: Resolve 'bonjour-service' missing module error and fix dependency issues across all ecosystem services.

Implementation Task List

  • Fix missing 'bonjour-service' dependency in root package.json
  • Resolve missing 'next' dependency in budolshap-0.1.0
  • Resolve missing 'express' dependency in budolID-0.1.0
  • Fix missing '@budolpay/database' link in budolAccounting-0.1.0
  • Fix missing dependencies (socket.io, axios, pusher) in api-gateway
  • Run full ecosystem startup and verify all services (Ports 8000-8007, 8080, 3000)
  • Documentation update for system fixes

Technical Explanation

This release addresses critical system-level issues that prevented the Budol Ecosystem from starting correctly. The primary fix involved adding bonjour-service to the root package.json for mDNS discovery. Additionally, npm install was executed across all service directories (Shap, ID, Accounting) to restore missing node_modules. A local file reference for @budolpay/database was added to the Accounting service to ensure proper database integration.

Compliance

  • Cybersecurity Law: Ensured system availability and service discovery integrity.
  • DTI/NPC: Maintained service reliability for data processing.
