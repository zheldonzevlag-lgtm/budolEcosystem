Future Recommendations - Version 410

1. QR Code Integration

Implement QR code generation for the formatted phone numbers to allow "Scan to Pay" functionality, further reducing manual entry errors.

2. Contact List Sync

Allow users to pick recipients directly from their phone's contact list, automatically applying the PhoneNumberFormatter logic to the selected contact.

3. Dynamic Currency Localization

If the ecosystem expands internationally, evolve the CurrencyInputFormatter to support different locales and currency symbols dynamically based on the user's region.

4. Backend Validation Sync

Mirror the UI validation (09XX-XXX-XXXX) in the backend API to ensure that even non-app clients (if any) adhere to the same data quality standards.
