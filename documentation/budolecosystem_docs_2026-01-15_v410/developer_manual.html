Developer Manual - Version 410

Rationale

To ensure data integrity and a consistent user experience, financial inputs must be strictly validated. This release introduces custom formatters to handle Philippine phone numbers and currency decimals at the UI level.

Technical Implementation

1. Custom Formatters

We created a new utility file formatters.dart to house global input formatters.

PhoneNumberFormatter

Automatically inserts hyphens at the 4th and 7th positions of an 11-digit phone number.

class PhoneNumberFormatter extends TextInputFormatter {
  @override
  TextEditingValue formatEditUpdate(...) {
    final text = newValue.text.replaceAll(RegExp(r'\D'), '');
    if (text.length > 11) return oldValue;
    // Formatting logic: 09XX-XXX-XXXX
  }
}

CurrencyInputFormatter

Restricts input to numbers and a maximum of 2 decimal places.

class CurrencyInputFormatter extends TextInputFormatter {
  @override
  TextEditingValue formatEditUpdate(...) {
    if (RegExp(r'^\d*\.?\d{0,2}$').hasMatch(newValue.text)) {
      return newValue;
    }
    return oldValue;
  }
}
