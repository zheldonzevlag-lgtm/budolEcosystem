System Admin Manual - Version 410

Compliance Verification

This release aligns with the following regulatory standards:

  • BSP (Bangko Sentral ng Pilipinas): Enforces strict transaction traceability and valid mobile number identification.
  • PCI DSS: Ensures that no sensitive data (like PINs or full card numbers) are captured in logs during formatting.
  • Cybersecurity Law: Enhances user protection by preventing accidental transfers to malformed identifiers.

Audit Logging

Admins can monitor transaction logs to ensure that clean numbers (without hyphens) are correctly recorded in the database, while the UI maintains user-friendly formatting.
