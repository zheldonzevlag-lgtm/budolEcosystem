Version 410

Transaction Formatting & Validation Release

Release Date: 2026-01-15

Executive Summary

Version 410 focuses on standardizing financial transaction inputs and improving the user experience for P2P and bank transfers. Key changes include restricting Send Money to phone numbers only, implementing real-time hyphenated phone formatting (09XX-XXX-XXXX), and enforcing 2 decimal digits for all monetary amounts.

Core Objectives

  • Phone-Only Transactions: Standardized P2P transfers to use Philippine mobile numbers only.
  • Hyphenated Formatting: Improved readability with 09XX-XXX-XXXX format during input.
  • Financial Precision: Enforced 2 decimal digits for all currency inputs and displays.
  • Compliance: Alignment with local financial reporting standards (BSP/BIR).
