User Manual - Version 410

What's New for You?

We've made transactions easier and more secure with improved formatting.

1. Sending Money via Phone

You can now only send money using a mobile number. As you type, the app will automatically add hyphens (e.g., 0917-123-4567) to help you check the number easily.

2. Clearer Amounts

All amounts now show exactly 2 decimal places (e.g., ₱100.00). This ensures you always know exactly how much you are sending or receiving.

3. Automatic Validation

The app will tell you immediately if a phone number is invalid or doesn't start with 09, preventing accidental transfers to wrong numbers.
