System Admin Manual - v504

Order Management & COD Liquidity

Administrators should monitor the Escrow service logs to ensure that Cash on Delivery (COD) funds are correctly credited to seller wallets upon delivery.

Escrow Integration:

  • COD orders trigger a creditPendingBalance call when status is set to DELIVERED.
  • Ensure the lib/escrow.js service is reachable and environment variables for wallet integration are set correctly.
