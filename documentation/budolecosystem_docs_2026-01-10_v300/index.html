Purpose

This release focuses on standardizing the Role-Based Access Control (RBAC) system across the BudolShap platform. It resolves critical access issues for administrative accounts, secures unprotected API endpoints, and ensures consistent permission enforcement based on the isAdmin flag and granular permissions JSON.
