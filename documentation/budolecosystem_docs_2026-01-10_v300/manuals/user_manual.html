User Manual - Version v300

Admin Dashboard Access

Users elevated to the Administrator role now have seamless access to the BudolShap Admin Dashboard. If you previously encountered "Access Denied" or "Unauthorized" errors, please ensure you are logged in with the correct credentials.

Accessing the Dashboard
  1. Log in to BudolShap using your administrative email.
  2. Navigate to the /admin URL or select "Admin Dashboard" from your user menu.
  3. You should now have full access to all administrative features, including store approvals, product management, and system settings.

Role-Specific Features

The system now strictly enforces feature availability based on your assigned role:

  • Admin: Full system access, including sensitive settings and financial data.
  • Seller: Access to store management, product creation, and order fulfillment.
  • Buyer: Access to product browsing, purchasing, and personal order history.

Support

If you still encounter access issues, please contact your System Administrator to verify your isAdmin status in the user profile settings.
