SysAdmin Manual: Ecosystem Portfolio (v163)

Portfolio Overview

The budolEcosystem Vercel portfolio now consists of 8 projects. 1 central ecosystem project and 7 microservice projects.

Project List

  • budolecosystem: Main root project.
  • budolpay-api-gateway: Core routing engine.
  • budolpay-auth: Identity provider.
  • budolpay-wallet: Ledger and balance service.
  • budolpay-tx: Transaction processing.
  • budolpay-payment-gw: External gateway connector.
  • budolpay-settlement: Reconciliation service.
  • budolpay-kyc: Identity verification service.

Domain Management

Administrators should ensure that no other projects in the Vercel organization use these names, as they are reserved for the budolEcosystem production environment. Custom domains (e.g., budolpay.com) can be added to the budolecosystem or budolpay-api-gateway projects as needed.

Post-Linking Cleanup

The previous projects (e.g., budolecosystem-api-gateway) have been superseded by the new budolpay-api-gateway naming convention. Administrators may choose to delete the old projects from the Vercel Dashboard to keep the workspace clean.
