QR Code Payment Process

When you choose to pay via QR Code (e.g., GCash, Maya, QRPh), a modal will appear with the following information:

  • Amount: The total amount to be paid.
  • Store Name: The name of the store you are purchasing from.
  • Owner: The name of the person responsible for the transaction (e.g., Jon Galvez).
  • Payment ID: A unique identifier for your transaction.

Scan the QR code using your chosen payment app to complete the transaction.
