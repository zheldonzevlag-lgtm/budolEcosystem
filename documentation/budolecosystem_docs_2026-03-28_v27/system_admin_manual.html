System Admin Manual

Version: 27.0.0 Component: Vercel Postgres Infrastructure

Infrastructure Connection

Admin access to the consolidated Vercel PostgreSQL instance is restricted to authenticated psql clients using SSL mode. The following environment variables must be configured in Vercel to maintain mirroring integrity:

DATABASE_URL="postgres://[USER]:[PASS]@[HOST]:5432/postgres?sslmode=require"
PRISMA_DATABASE_URL="prisma+postgres://[API_KEY]@[PROXY_HOST]/"

Standard Mirroring Procedure

1
Wipe Schemas: Clear existing tables in public, budolid, budolpay, and budolaccounting using the wipe.sql utility script.
2
Import Base: Execute psql -f budolshap_1db.sql which reconstructs the public, id, and pay schemas.
3
Import Accounting: Execute psql -f budolAccounting.sql ensuring the search path is set to the budolaccounting schema.

Backup Location

Manual SQL backups are stored on the localized server: G:\IT\database-backup\03-29-2026\
