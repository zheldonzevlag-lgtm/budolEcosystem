Issues & Fixes Log

Issue: "0 of 0 products" displayed on Vercel homepage.

Root Cause: The Prisma client was querying a clean database that hadn't yet received the 03-29-2026 mirror import.

Fix: Executed 4-schema SQL restoration.

Verified current total of 14 products in the public."Product" table.

Issue: Prisma connection string corruption ("Invalid URL").

Root Cause: CRLF (\r\n) characters in Vercel environment variables.

Fix: Sanitized PRISMA_DATABASE_URL in lib/prisma.js.
Issue: pg_restore "restricted superuser" errors.

Root Cause: SQL dumps containing REVOKE commands on system schemas restricted in Vercel managed Postgres.

Fix: Targeted data-only restoration.

Filtered the dump to focus on table content and custom types within user schemas.
