Budol Ecosystem v27.0 - Database Mirroring

Version: 27.0.0 Date: 2026-03-28 RELEASED

Purpose & Rationale

The primary objective of this version was to unify the disconnected local PostgreSQL databases into a single, high-performance Vercel PostgreSQL instance. This migration ensures data consistency across the ecosystem and resolves the "0 products" display issue caused by schema isolation mismatches.

Mirroring Results

Verification confirms that all 4 local databases were successfully mirrored into isolated schemas on Vercel.

14 Products (Public)
41 Transactions (Pay)
2 Users (ID)
7 Accounts (Acct)

Compliance Status

  • BSP: Data localized to Singapore (sin1) for lower latency and proximity to regional nodes.
  • NPC: Schema isolation maintained (budolid, budolpay) to prevent cross-service data leakage.
  • PCI DSS: Transaction data mirrored with existing integrity constraints intact.
