Future Enhancements & Recommendations

Automated Synchronization Pipeline

Recommendation: Transition from manual SQL dump mirroring to an automated CI/CD pipeline that leverages GitHub Actions and Vercel CLI to sync the database schemas upon every successful local backup completion.

Real-time Socket Gateway Fix

Recommendation: Update the SOCKET_SERVER_URL environment variable in Vercel to reflect the production Render.com URL instead of the local dev IP (192.168.1.10). This will resolve the ERR_CONNECTION_REFUSED observed in browser audits.

Regional Expansion

Recommendation: Implement Read Replicas for the Vercel PostgreSQL cluster across other regions (e.g., US West) to support low-latency requirements for international customers while maintaining Singapore as the primary write node.
