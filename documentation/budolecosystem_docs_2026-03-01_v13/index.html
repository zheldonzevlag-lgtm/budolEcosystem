Release v6.0.0 Cost-Optimized

AWS Free Tier Cloud Migration

Transitioning the budolEcosystem to a scalable, zero-cost AWS infrastructure for development and testing.

Release Summary

This major release focuses on migrating our storage from Cloudinary to AWS S3 and hosting our core services on AWS Lambda and EC2 within the Free Tier limits. Key features include an automated image background removal process triggered via S3.

Key Objectives

  • Zero-cost hosting using AWS Free Tier
  • S3 storage for all ecosystem assets
  • Automated background removal via Lambda
  • Centralized RDS for all services

Compliance & Standards

  • PCI DSS (Secure Data Transit)
  • NPC (Encrypted S3 Storage)
  • Budol standard documentation
