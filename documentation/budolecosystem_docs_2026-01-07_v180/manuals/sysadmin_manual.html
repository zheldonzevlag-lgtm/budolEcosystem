System Admin Manual v180

Infrastructure Changes

The budolpay production environment now points to a Vercel Postgres instance optimized with Prisma Accelerate.

Environment Variables

Variable Purpose Status
DATABASE_URL Main connection string for Prisma Updated
DIRECT_URL Non-pooling connection for migrations Updated
PRISMA_DATABASE_URL Accelerate connection for edge functions Updated

Operational Runbook

Database Verification

# Run this to check connection status
npx prisma db pull --schema=prisma/schema.prisma

Schema Deployment

# Run this to sync schema changes
npx prisma db push --schema=prisma/schema.prisma

Risk & Mitigation

  • Risk: Connection latency in edge functions.
  • Mitigation: Enabled Prisma Accelerate for connection pooling and global distribution.
