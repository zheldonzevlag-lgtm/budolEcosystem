Applied Fixes

FIX-492-001: SWR Cache Patching (for ISSUE-492-001)

Replaced router.refresh() with targeted SWR cache mutation. The UI now updates only the specific order that changed, maintaining smooth interaction and eliminating flickering.

FIX-492-002: Realtime Chat Events (for ISSUE-492-002)

Removed 3-second polling in ChatWindow.jsx. Implemented message-received events in the backend that trigger immediate SWR cache updates on the frontend.

FIX-492-003: Dashboard Refresh Triggers (for ISSUE-492-003)

Added global event handlers in RealtimeProvider.jsx that automatically trigger SWR mutation for dashboard stats whenever an order or rating event occurs.

FIX-492-004: Visibility-Aware Polling (for ISSUE-492-004)

Implemented visibilitychange listener to adjust polling intervals. Bandwidth consumption is reduced by ~83% for inactive tabs.
