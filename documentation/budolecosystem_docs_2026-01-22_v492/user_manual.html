What's New in v492

This update brings instant feedback across the entire Budol Ecosystem. You no longer need to refresh the page to see the latest updates.

📦 Instant Order Status

When a seller updates your order status (e.g., to "Shipping"), you will see the update immediately on your "My Orders" page without any screen flicker.

💬 Realtime Chat

Messages now arrive instantly. A notification toast will appear if you receive a message while browsing other parts of the site.

💰 Wallet Balance Sync

Sellers will see their wallet balance update automatically as soon as an order is completed or delivered, reflecting the released funds in realtime.

📊 Seller Dashboard

The store dashboard now refreshes its statistics (Total Orders, Revenue, Ratings) automatically whenever relevant events occur.
