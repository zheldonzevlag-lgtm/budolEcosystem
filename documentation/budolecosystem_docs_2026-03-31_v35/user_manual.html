User Manual: Advanced Admin Security Flow

This manual covers the updated security protocols for the BudolPay Admin Ecosystem (v35).

Security Constraint
🛡️

MathCaptcha: The Shield Challenge

Admin users must now complete the Shield Challenge (MathCaptcha) before reaching the login fields. This prevents automated bot attempts and complies with PCI DSS 10.2.1 for authentication protection.

[UI Reference: MathCaptcha Challenge displayed as first interaction on Login Page]
Access Management

Dual-Stage Login Flow

1. Solve the Shield Challenge correctly.
2. Enter your authorized admin credentials (Email & Password).
3. On success, you'll reach the Dashboard (rendered dynamically via Vercel Postgres).

Optimization

Dynamic Dashboard Loading

The dashboard and all its financial modules (Accounting, Transactions) load in real-time. No more cached stale data; every view is a direct feed from the Vercel Nexus database at request-time.
