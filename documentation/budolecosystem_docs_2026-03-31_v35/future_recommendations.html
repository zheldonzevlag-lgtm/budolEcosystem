Roadmap: BudolPay Ecosystem Stabilization (v35)

REC-35-01: Full Database Mirror to AWS RDS

Goal: Multi-region redundancy for PCI DSS Level 1 compliance.

Action: Set up a replication task from Vercel Postgres to an AWS RDS instance. Use the RDS instance as a DIRECT_URL for read-only analytical loads.

REC-35-02: Advanced WAF Integration

Goal: Bot mitigation beyond MathCaptcha.

Action: Configure Vercel Firewall (WAF) to challenge any IP address hitting /api/auth more than 10 times per minute, reducing overhead on the Shield Challenge.

REC-35-03: Real-time Audit Dashboard

Goal: Real-time visibility into admin actions.

Action: Transition /api/audit-logs from polling to a Pusher-based event subscription model to further reduce database read frequency.
