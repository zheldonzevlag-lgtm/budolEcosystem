Mitigation Strategy (v35)

MIT-35-01: Build-time Persistence

Goal: Prevent any build machine failure due to database unavailability.

Solution: Implementation of a resilient fallback for DATABASE_URL and DIRECT_URL in all PrismaClient entry points.

MIT-35-02: Compliance & Security

Goal: Maintain PCI DSS 10.2.1 compliance for admin dashboards.

Solution: Re-integration of the MathCaptcha (Shield Challenge). This forces a bot challenge before sensitive fields are interactive.

MIT-35-03: Storage Isolation

Goal: Scale to Vercel production without shared connection collisions.

Solution: Forced schema isolation using getIsolatingUrl in the Vercel connection string.
