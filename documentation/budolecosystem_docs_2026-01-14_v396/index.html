Release v1.2.87 v396

Release Summary

This release (v1.2.87) optimizes the face login flow by prioritizing the custom TFLite-based facial recognition while providing a seamless fallback to native OS biometrics.

  • ✔ Phase 0: Baseline & Risk Analysis completed.
  • ✔ Phase 1: Incremented version to 1.2.87 in pubspec.yaml and ApiService.
  • ✔ Phase 5: Validated build with app_smoke_test.dart.
  • ✔ Phase 7: Built and deployed APK v1.2.87.

Key Changes

  • Biometric Logic Reversal: Tapping the face icon now triggers the custom ML flow if a face is registered.
  • Smart Fallback: Automatically falls back to system Face ID / Android Face Unlock if no app-level profile exists.
  • Security Hardening: Maintained 0.8 similarity threshold for custom recognition.
