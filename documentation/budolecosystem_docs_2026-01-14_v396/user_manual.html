Using Face Login

Version 1.2.87 introduces a "Smart Face Icon" that adapts to your configuration.

1
Custom Face Profile (Default)

If you have registered your face in Settings > Biometrics, tapping the face icon will open the camera for a quick selfie verification. This provides the highest level of security with liveness detection.

2
System Biometrics (Fallback)

If you haven't registered a custom profile but have Face ID (iOS) or Face Unlock (Android) enabled on your phone, the face icon will trigger your phone's native verification prompt instead.

Updating to 1.2.87

Download the latest installer from the android-installer directory. You can install it directly over your existing version to keep your settings and face profiles intact.
