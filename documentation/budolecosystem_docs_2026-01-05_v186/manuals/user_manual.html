User Manual: KYC Document Rotation

This guide explains how to use the new image rotation feature in the Admin Panel to assist with KYC document reviews.

How to Rotate Documents

  1. Navigate to the User Verification section.
  2. Click the Eye (View) icon on a user with pending documents.
  3. Locate the Rotate (clockwise) icon next to the document type label.
  4. Click the icon to rotate the image by 90 degrees.
Automatic Saving: You don't need to click a "Save" button. The system automatically remembers the orientation after you stop clicking for 1 second. A small loading spinner will appear briefly to confirm the save.

Benefits

  • Persistence: Once you rotate a document, it will stay in that orientation for you and all other administrators.
  • Efficiency: No need to download and rotate images locally for review.
  • Speed: Real-time rotation with optimized background saving.
