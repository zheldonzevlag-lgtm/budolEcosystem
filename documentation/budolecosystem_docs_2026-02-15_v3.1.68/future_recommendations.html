Future Recommendations

Roadmap Enhancements

Centralized Environment Config Package

Move the isLocal logic and other environment-specific helpers into a shared package (e.g., @budolpay/config) to avoid duplication and ReferenceErrors across services.

Automated Mobile Build Pipeline

Integrate Fastlane or GitHub Actions to automate the APK build process, ensuring that version increments in pubspec.yaml automatically trigger a new release build.

Biometric Fallback for OTP

Implement biometric authentication (FaceID/Fingerprint) as a fallback for OTP in the mobile app to improve UX while maintaining high security (GoTyme standard).
