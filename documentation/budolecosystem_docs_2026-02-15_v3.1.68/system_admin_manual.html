System Administration Manual

Environment Configuration

The following environment variables must be correctly configured in auth-service/.env:

VariableDescriptionRequired Value
NODE_ENVEnvironment modeproduction, development, or test
LOCAL_IPLocal IP for dev detectionSystem IP (e.g., 127.0.0.1)
DATABASE_URLPrisma connection stringValid PostgreSQL URL

Deployment Steps

  1. Run npm install in the root directory.
  2. Run npx prisma generate in packages/database.
  3. Run npm run build in services/auth-service.
  4. Deploy the mobile app using the generated APK in budolPayMobile/build/app/outputs/flutter-apk/app-release.apk.

Maintenance

Monitor logs for [AuthService] and [Notification] prefixes to verify OTP delivery. Ensure the otpUpdatedAt field in the User table is being updated to enforce rate limiting.
