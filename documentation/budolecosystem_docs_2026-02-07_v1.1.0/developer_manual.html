User Synchronization Logic

To support SSO for users registered in legacy app schemas, we implemented a synchronization script that copies missing users from the public schema to the budolid schema.

Code Reference: sync-users.js

// Snippet of sync logic const isDuplicate = budolidEmails.has(user.email) || (user.phoneNumber && budolidPhones.has(user.phoneNumber)); if (!isDuplicate) { await prisma.user.create({ data: { email: user.email, password: user.password, phoneNumber: user.phoneNumber, // ... } }); }

Frontend Phone Validation UX

The AuthForm.jsx component was updated to prevent the "NOT FOUND" message from flickering while the user is typing. We now set checkingPhone to true immediately when the input length reaches 10 digits, before the debounce timer fires.
