System Admin Manual - v1.1.0

User Data Synchronization

If users report that they cannot log in even though they are registered in a specific app, run the synchronization script to import them into the SSO schema.

Run Sync Script:
cd budolID-0.1.0
node scripts/sync-users.js

Database Maintenance

Regularly check for unstandardized phone numbers. The system now automatically normalizes them on login/registration, but legacy data may still exist.

Standardize All Phones:
cd budolID-0.1.0
node scripts/standardize-phones.js

Service Monitoring

Ensure both services are running for the ecosystem to function:

  • budolID (SSO): Port 8000
  • budolShap (App): Port 3000/3001
