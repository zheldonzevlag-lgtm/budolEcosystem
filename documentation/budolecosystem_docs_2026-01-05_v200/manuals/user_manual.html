User Manual - v200: Using Advanced SMS Providers

Semaphore: The Local Standard

Use Semaphore for high-reliability OTP delivery. It is specifically optimized for local carriers like Globe, Smart, and DITO.

engageSPARK: Professional Branding

Use engageSPARK when you want your messages to appear as "BUDOLPAY" instead of a random mobile number. This is ideal for customer support and marketing alerts.

Configuration Steps

  1. Navigate to Settings > Notification.
  2. Scroll to the SMS Providers column.
  3. Locate Semaphore or engageSPARK.
  4. Enter your API credentials.
  5. Click Save Settings to store the information.
  6. Click USE THIS to activate the provider for all system notifications.
