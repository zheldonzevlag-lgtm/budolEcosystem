Implementation Plan - v200: Enhanced SMS Provider Integration

1. SMS Provider Expansion

Added Semaphore and engageSPARK to the SMS provider selection in /settings/notifications. Semaphore is designated as the primary choice for local carrier optimization, while engageSPARK is introduced for branded SMS and 2-way support.

2. UI Components Addition

Created dedicated configuration cards for both providers. Semaphore includes fields for API Key and Sender Name. engageSPARK includes API Key and Organization ID.

3. Data Model Persistence

Integrated new setting keys: SEMAPHORE_API_KEY, SEMAPHORE_SENDER_NAME, ENGAGESPARK_API_KEY, and ENGAGESPARK_ORG_ID into the existing SystemSetting upsert logic.

  • engageSPARK card rendering: Success

  • Configuration save functionality for engageSPARK: Verified
