Developer Manual - v200: SMS Gateway Integration Details

Semaphore Integration

Semaphore is optimized for Philippine carriers. The integration uses the SEMAPHORE provider key.

// Provider Keys
SEMAPHORE_API_KEY
SEMAPHORE_SENDER_NAME

engageSPARK Integration

engageSPARK is used for branded messaging. The integration uses the ENGAGESPARK provider key.

// Provider Keys
ENGAGESPARK_API_KEY
ENGAGESPARK_ORG_ID

Unified Server Action

The updateSetting server action automatically handles these new keys by iterating through the formData and upserting into the SystemSetting model with the SMS group.

Implementation Detail: The activeSMSProvider constant now supports SEMAPHORE and ENGAGESPARK as valid values for the SMS_PROVIDER setting key.
