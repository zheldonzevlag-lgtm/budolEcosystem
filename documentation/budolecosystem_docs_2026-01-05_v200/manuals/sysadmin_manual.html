System Admin Manual - v200: Configuring Advanced SMS Gateways

Semaphore (Primary SMS)

Ideal for OTP and transactional alerts within the Philippines.

Setting Key Description
SEMAPHORE_API_KEY Your Semaphore API credential.
SEMAPHORE_SENDER_NAME The registered sender name (e.g., BUDOLPAY).

engageSPARK (Branded & 2-Way)

Used for professional branding and interactive customer support.

Setting Key Description
ENGAGESPARK_API_KEY Your engageSPARK API credential.
ENGAGESPARK_ORG_ID Your unique Organization ID from the engageSPARK dashboard.

Activation

To switch to these providers, click "USE THIS" on their respective cards. The ACTIVE badge will update to confirm the selection.
