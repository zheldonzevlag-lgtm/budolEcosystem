2. Monitoring KYC Success

Admins should monitor the verification-service logs for successful document uploads. A successful selfie upload should now return a 201 Created status and correctly update the user's kycStatus to VERIFIED.

Verification Query

Run the following query in the database console to verify the status of recent KYC attempts:

SELECT email, kycStatus, kycTier FROM "User" WHERE updatedAt > NOW() - INTERVAL '1 hour';
