System Admin Manual Update (v24)

Database Schema & Data Changes

The redirectUri for all ecosystem applications in the budolID database has been updated.

Prisma Update Script

// Applied to budolID database
await prisma.ecosystemApp.update({
    where: { apiKey: 'bp_key_2025' },
    data: { redirectUri: 'http://localhost:3000/api/auth/callback' }
});

await prisma.ecosystemApp.update({
    where: { apiKey: 'bs_key_2025' },
    data: { redirectUri: 'http://localhost:3001/api/auth/sso/callback' }
});

Environment Variable Modifications

The following variables in budolshap-0.1.0/.env were updated:

  • BUDOLPAY_GATEWAY_URL: http://localhost:8000
  • NEXT_PUBLIC_SSO_URL: http://localhost:8000
  • NEXT_PUBLIC_APP_URL: http://localhost:3001

Operational Runbook

  1. Ensure budolID is running on port 8000.
  2. Verify that the DATABASE_URL in budolID-0.1.0/.env is pointing to the correct PostgreSQL instance.
  3. Run node check-apps.js in the budolID-0.1.0 directory to verify current redirection URIs.
