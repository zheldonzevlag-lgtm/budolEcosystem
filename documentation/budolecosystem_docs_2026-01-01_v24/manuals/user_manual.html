User Manual Update (v24)

Login Workflow Changes

The login process for the budolPay Ecosystem has been simplified to consistently use localhost for all users working on the primary development machine.

Standard Login Procedure

  1. Open your browser and navigate to http://localhost:3000/login (for Admin) or http://localhost:3001/login (for App).
  2. Click the "Sign in with budolID" button.
  3. You will be taken to the centralized login page at localhost:8000.
  4. Enter your credentials and click "Sign In".
  5. You will be automatically redirected back to the original application on localhost.

What to expect

  • No more IP jumping: You will no longer see 192.168.1.24 in your address bar unless you specifically configure it for external LAN testing.
  • Consistent Session: Your login session will now persist correctly across all localhost ports.
