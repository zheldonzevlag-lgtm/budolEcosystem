Version 2026-02-21 v1 Release Notes

Overview

This release focuses on stability improvements for the Store Management module, specifically the Product Creation Wizard.

  • Fix: Resolved unresponsiveness in "Publish Product" and "Save as Draft" buttons.
  • Feature: Implemented local storage draft persistence for product creation.
  • Documentation: Restored full knowledgebase version history.
  • FIX-20260221-06: Resolved "Unique Key Prop" error in DragDropImageUpload component by implementing internal image normalization and robust fallback ID generation, ensuring reliable image display and drag-and-drop functionality.
  • FIX-20260221-07: Fixed "Product images still not showing" and 404 error by correcting redirects, API image validation, and list view rendering logic.
  • FIX-20260221-08: Fixed data persistence issue where Shipping Info (Weight, Dimensions) and Condition/Pre-Order status were not being saved or updated.

Change Log

  • AddProductWizard.jsx: Added storeId prop, implemented draft saving, improved validation, fixed image normalization for drag-and-drop.
  • knowledgebase.html: Reconstructed version history sidebar.
