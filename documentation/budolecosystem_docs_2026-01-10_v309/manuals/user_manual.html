User Guide & Features - v309

Ecosystem Access

Users can now access all services reliably via the unified SSO provider (budolID). The stability of the backend services has been improved to ensure seamless transitions between the marketplace, wallet, and admin dashboard.

Service Discovery

The ecosystem now utilizes mDNS for service discovery, allowing local network devices to find and connect to services automatically without manual IP configuration.

  • budolID: Authentication and Identity
  • budolShap: Marketplace and Shopping
  • budolPay: Payments and Wallet management
