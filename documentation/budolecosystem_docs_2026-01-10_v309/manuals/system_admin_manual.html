Service Orchestration

The central runner (npm start) has been stabilized. It now performs a pre-flight check to ensure all microservices have the required environment variables.

Important Check:

Ensure JWT_SECRET and LOCAL_IP are defined in each service's .env file. The runner will block startup if these are missing for critical services.
