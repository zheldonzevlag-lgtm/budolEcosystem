Client-Side Image Rotation

Rotation is managed via a local React state mapping document IDs to rotation degrees:

const [rotations, setRotations] = useState<Record<string, number>>({});

const handleRotate = (docId: string) => {
  setRotations(prev => ({
    ...prev,
    [docId]: (prev[docId] || 0) + 90
  }));
};

The rotation is applied using a CSS transform on the img element within the aspect-video container.

UI Styling Adjustments

  • Used object-contain instead of object-cover for images to prevent cropping during rotation.
  • Added transition-transform for smooth 300ms rotation animations.
  • Implemented a "pointer-events-none" overlay to prevent gradient interaction from blocking image clicks (if zoom features are added later).
