Admin User Manual - v182 Enhancements

Rotating Documents

If a user uploads an ID or selfie in the wrong orientation, you can now rotate it directly within the verification modal.

  • Click the Rotate icon (clockwise arrow) next to the document type tag.
  • Each click rotates the image by 90 degrees.
  • The orientation is only changed for your current viewing session and is not saved to the database.

Reading OCR Results

The "OCR Analysis Result" section has been cleaned up. Instead of raw code, you will now see:

  • A grid of extracted fields (e.g., Name, ID Number) if available.
  • A status message if data is still being processed or could not be read.
  • The engine/source of the extraction listed at the bottom (e.g., On-Device ML Kit).
