SysAdmin Manual - v182

Frontend Updates

The Admin panel has been updated with client-side image manipulation and OCR data parsing. No changes were made to the backend or database schema.

Deployment Notes

  • Dependencies: Ensure lucide-react is up to date as new icons (RotateCw, Info) were added.
  • Build: Run npm run build in the apps/admin directory to apply these changes.

Data Integrity

The image rotation feature is purely client-side. No rotation metadata is stored in the VerificationDocument table. OCR results are parsed dynamically from the existing ocrData JSONB column.
