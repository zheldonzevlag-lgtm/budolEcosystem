User Manual

Purpose

Explain how shoppers experience inline video playback on product listings.

Rationale

Users gain quick product previews while retaining control of audio.

Technical Explanation

Video previews play on hover and unmute on hover after a user gesture is detected on the page.

Changes

  • Hovering over a product video starts playback without audio.
  • After any click or keypress on the page, hover playback includes audio.

Issues

Audio remains muted until a user gesture is detected.

Fixes

Hover audio is enabled after user interaction.

Risks

Users may not click to enable sound.

Mitigations

Provide UX cues if user feedback indicates confusion.

Dependencies

Browser autoplay policy compliance.
