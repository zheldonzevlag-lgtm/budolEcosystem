User Manual: Edit Profile Feature

Version: v3.3.26

Date: 2026-02-19

Overview

This guide explains how to use the updated "Edit Profile" feature in the BudolPay mobile application. We have enhanced security and data visibility to ensure your information is safe and accurate.

How to Edit Your Profile

1. Accessing the Profile Screen

Step 1: Log in to the BudolPay app.

Step 2: Tap on the Profile icon in the bottom navigation bar or the menu.

Step 3: Select Edit Profile.

2. Viewing Your Information

By default, your First Name and Last Name may be hidden (shown as dots or asterisks) for privacy.

To view your name: Tap the Eye Icon 👁️ next to the name field. The real value will be revealed.

Note: If you see asterisks (e.g., M*****) even after tapping the eye icon, please wait a moment for the data to refresh from the server. The app prevents you from saving if the data is not fully loaded to protect your profile.

3. Updating Your Information

Step 1: Ensure your name is visible (not masked with asterisks).

Step 2: Tap on the field you want to change (First Name, Last Name, etc.).

Step 3: Enter the new information.

Step 4: Tap the Save Changes button at the bottom of the screen.

Troubleshooting

"Cannot save masked data" Error

If you see a message saying "Cannot save masked data. Please wait for profile to fully load," it means the app is still fetching your secure details from the server.

Solution:

  • Wait 5-10 seconds.
  • Pull down to refresh the screen (if available) or go back and re-enter the Edit Profile screen.
  • Check your internet connection.

Email Address is Empty

If your email address appears empty but you know you have one registered:

Solution: The app is designed to fetch this automatically. Ensure you are connected to the internet. If it persists, try logging out and logging back in.
