Risk Register

Version: v3.3.26

Date: 2026-02-19

Risk ID Description Impact Likelihood Severity Status
R-2026-001 Data Overwrite with Masked Values
Users might accidentally save "M*****" or similar masked strings back to the database if the UI doesn't prevent it or if the backend accepts it.		 Data Corruption (Permanent loss of name data) Low (after fix) High Mitigated
R-2026-002 PII Exposure in Logs
Debug logs added for troubleshooting might accidentally record unmasked PII (names, emails) in production logs.		 Privacy Violation / Compliance Breach Medium Medium Monitored
R-2026-003 Endpoint Confusion
Having multiple endpoints returning user data (some masked, some unmasked) could lead to developers using the wrong one in future features.		 Inconsistent UI / Security Gaps Medium Medium Mitigated
R-2026-004 Network Latency on /verify
The /verify endpoint now performs multiple checks. High latency could cause the "Edit Profile" screen to load slowly or time out.		 Poor User Experience Medium Low Accepted
