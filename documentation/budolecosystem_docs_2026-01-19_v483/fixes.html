Applied Fixes v483

F-483-01: BudolPay Button Restriction

Problem: BudolPay orders were excluded from the repayment UI, preventing users from paying for unpaid orders after checkout.

Fix: Removed the order.paymentMethod !== 'BUDOL_PAY' restriction in OrderItem.jsx.

F-483-02: Web Crypto TypeError Crash

Problem: The app crashed with TypeError: generateKey when running in non-secure web environments due to crypto.subtle being undefined.

Fix: Implemented isWebCryptoSupported() check and configured FlutterSecureStorage to disable encryption on Web if the API is unavailable.
