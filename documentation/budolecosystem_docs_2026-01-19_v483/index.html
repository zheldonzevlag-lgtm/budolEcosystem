System Overview v483

Release Date: 2026-01-19

Purpose & Rationale

This release (v483) addresses two critical areas: BudolPay Repayment Capability and Web Security Compatibility.

  • BudolPay Repayment: Standardized the UI in budolshap to allow users to "Pay Now" for unpaid BudolPay orders, matching the QRPH flow.
  • Web Security Fix: Resolved a TypeError: generateKey crash in the Flutter web app by implementing a runtime check for Web Crypto API (crypto.subtle) support, ensuring stability in non-secure or legacy browser environments.
