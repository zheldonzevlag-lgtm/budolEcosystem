System Admin Manual v483

Configuration Requirements

System admins should ensure that the web environment is served over HTTPS for production to enable full encryption features in flutter_secure_storage.

If the app is served over http:// (non-localhost), the system will automatically degrade to non-encrypted storage to maintain functionality, but this should be avoided for production data.

Repayment Verification

Admins can verify BudolPay repayment attempts in the gateway logs (port 8080) or directly in the payment-service logs. Successful repayments will update the order status from unpaid to paid via the /api/payment/checkout endpoint.
