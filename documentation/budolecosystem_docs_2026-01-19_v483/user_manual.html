Paying for Unpaid Orders

If you placed an order using BudolPay but did not complete the payment immediately, you can now pay for it later:

  1. Go to your Order History.
  2. Find the order with status "Order Placed" and payment status "Unpaid".
  3. Click the green "Pay Now" button.
  4. Follow the prompts to complete your BudolPay transaction.

Web Browser Support

The BudolPay mobile app (Web version) now supports more browsers and environments. If you are using a non-secure connection (e.g., internal company network without HTTPS), the app will still allow you to log in and use basic features, although biometrics may be disabled for security reasons.
