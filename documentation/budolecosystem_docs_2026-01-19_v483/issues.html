Known Issues v483

I-483-01: Biometric Login Unavailable on Non-Secure Web

On web environments served via http://, biometric authentication (Face ID/Fingerprint) is not available as browsers restrict these APIs to secure contexts (HTTPS).

Status: Known limitation of web standards.

I-483-02: Local Storage Persistence

In private browsing modes or when "Clear Cookies and Site Data on Close" is enabled, secure storage data (like PIN or login tokens) will be lost when the browser is closed.

Status: Expected behavior.
