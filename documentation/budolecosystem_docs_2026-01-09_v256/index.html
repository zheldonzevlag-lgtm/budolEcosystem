budolEcosystem Docs v256

Date: 2026-01-09 | Version: v256

Overview

This update fixes the issue where transaction history was not updating in real-time in the mobile app. The TransactionHistoryScreen now listens to the RealtimeService for incoming transaction events.

Key Changes

  • TransactionHistoryScreen: Integrated RealtimeService.transactionStream listener to automatically prepend new transactions to the list.
  • Stream Management: Added StreamSubscription handling in initState and dispose to prevent memory leaks.
  • Mobile Build: Incremented version to 1.2.21+21.
