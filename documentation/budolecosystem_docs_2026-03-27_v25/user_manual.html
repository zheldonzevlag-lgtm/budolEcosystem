User Manual – v25

Quick (Phone-Only) Registration

This feature lets you create a BudolShap account using only your Philippine mobile number. No password or email address is required initially.

  1. Click Sign Up on the homepage.
  2. Select the Quick (Phone Only) tab.
  3. Solve the Shield Challenge math question (e.g., What is 3 + 5?). This confirms you are human.
  4. Enter your Philippine mobile number in the format 9XXXXXXXXX (the +63 prefix is added automatically).
  5. Click Create Account.
  6. You're in! A temporary profile is created. You can complete your full profile (name, email, KYC) in Account Settings later.

Standard Registration

  1. Click Sign Up on the homepage.
  2. Select the Standard tab.
  3. Solve the Shield Challenge.
  4. Enter your email address, phone number, and a strong password (minimum 8 characters, must include uppercase, lowercase, number, and special character as per BSP requirements).
  5. Click Create Account.
  6. Check your email for a verification link (sent within 5 minutes).

Frequently Asked Questions

Q: My phone number says "already registered" but I never signed up.
A: This was a known bug (v25 Bug #001) that has been fixed in this release. If you still see this error, please clear your browser cache and try again.
Q: What is the Shield Challenge?
A: A simple math question that proves you are a real person, not a bot. It replaces traditional image CAPTCHAs for a faster experience.
Q: Can I add a password to my Quick Registration account later?
A: Yes. Go to Account Settings → Security to set a permanent password at any time.
