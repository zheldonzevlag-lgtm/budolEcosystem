Shield Challenge & Registration Security

Approved

Secure automated registration gatekeeper and ecosystem validation consistency.

🛡️ Project Goal

This plan integrates a math-based CAPTCHA (Shield Challenge) as a gatekeeper for the account creation process in **budolID** (SSO), **budolShap** (Storefront), and **budolPay Admin**. It also stabilizes production email services and ensures "Quick Registration" (Phone Only) flows are not blocked by standard password complexity requirements.

Compliance Note: PCI DSS Requirement 8.3 requires multi-factor or secure gatekeeping for critical administrative and authenticated access. This math challenge serves as a "Pre-Auth Shield".

Proposed Changes

  • Implemented isQuickReg bypass for password complexity regex (allowing auto-generated 6-digit credentials).
