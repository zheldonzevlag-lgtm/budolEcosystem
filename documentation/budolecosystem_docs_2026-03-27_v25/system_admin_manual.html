2. Service Startup Order

  1. Start budolID on port 8000: 
    cd budolID-0.1.0 && npm start
  2. Start budolPay on port 8001: 
    cd budolpay-0.1.0 && npm start
  3. Start budolShap on port 3001: 
    cd budolshap-0.1.0 && npm start

Or use the root npm start (runs concurrently).

4. Clearing Stale Processes (Windows)

$ports = @(3001,8000,8001,9000)
foreach ($p in $ports) {
    $proc = (Get-NetTCPConnection -LocalPort $p -EA SilentlyContinue).OwningProcess
    if ($proc) { Stop-Process -Id $proc -Force -EA SilentlyContinue }
}

5. Vercel Deployment

cd budolshap-0.1.0
powershell -ExecutionPolicy Bypass -File .\scripts\test_scripts\deploy-vercel.ps1

Ensure NEXT_PUBLIC_SSO_URL is set in the Vercel Environment Variables dashboard to point to the production budolID endpoint.
