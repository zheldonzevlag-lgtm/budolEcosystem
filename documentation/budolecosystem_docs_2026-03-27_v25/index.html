Quick Registration Stabilization & Shield Challenge

VERSION v25 RELEASE DATE: 2026-03-27 DEPLOYED: budolshap.vercel.app

Summary

v25 resolves a critical false-positive "ALREADY TAKEN" error during Quick (Phone-Only) Registration, caused by a Node.js 18+ IPv6 resolution bug. It also introduces the Shield Challenge (Math-based CAPTCHA) on all registration forms for bot protection.

✅ Fixed

  • Node.js 18+ IPv6 localhost → IPv4 127.0.0.1 fix
  • Quick Registration password bypass for phone-only accounts
  • Production-safe NEXT_PUBLIC_SSO_URL fallback support

🛡️ Added

  • Shield Challenge (MathCaptcha) on all registration flows
  • registrationType flag propagation (budolShap → budolID)
  • Service-specific debug identifiers in error responses
