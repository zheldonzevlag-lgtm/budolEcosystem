v198 User Manual

Subject: Managing SMS Settings & UI Navigation Updates.

New Features

1. SMS API Selection Dashboard

Administrators can now manage how the system sends text messages (SMS) for OTPs and notifications.

Where to find it: System Settings > SMS API Selection
  • Provider Selection: Choose between Twilio, Infobip, Semaphore, and MoveX.
  • Credential Management: Securely enter API keys and account details.
  • Real-time Updates: Changes take effect immediately across the entire ecosystem.

Interface Improvements

Optimized Sidebar Navigation

We have improved the readability of the sidebar menu, specifically under System Settings.

  • Single-Line Titles: Menu items like "System Environment" and "SMS API Selection" now appear in a single line, making the interface cleaner and easier to navigate.
  • Better Visual Hierarchy: Reduced indentation provides more room for text without sacrificing the nested structure.

Common Tasks

How to update your SMS provider?

  1. Log in to the Admin Panel.
  2. In the left sidebar, click on System Settings to expand it.
  3. Click on SMS API Selection.
  4. Choose your preferred provider from the cards at the top.
  5. Fill in the required fields (e.g., API Key, Account SID).
  6. Click Save SMS Settings at the bottom of the page.

Troubleshooting

If menu items still appear wrapped, please ensure your browser window is at least 1280 pixels wide. The sidebar is optimized for standard desktop resolutions.

Test Results

User acceptance testing confirmed that the new SMS settings page is intuitive and the sidebar improvements enhance the overall user experience.

Recommendation for Future Enhancement

Add a notification log to see which SMS messages were sent and through which provider for easier auditing.
