System Admin Manual: Category Icon Management

Overview

Admin users now have direct control over category icons. This manual explains how to use the new Icon Picker and manage icon overrides.

Changing a Category Icon

  1. Navigate to the Admin Panel > Categories.
  2. Find the category you wish to edit and click the Edit (Pencil) icon.
  3. In the modal, locate the Category Icon section.
  4. Use the Search Box to find a specific icon (e.g., "Phone", "Gift", "Coffee").
  5. Select the desired icon from the grid. The preview will update immediately.
  6. Click Save Changes to apply the update globally.
[UI Reference: The searchable icon grid appears above the image upload section in the category modal]
Note: If no icon is manually selected, the system will use "Smart Detection" to automatically pick an icon based on the category name or slug.

Best Practices

  • Consistency: Use similar icon styles for categories within the same level (e.g., Level-1 categories should all have distinct, solid icons).
  • Clarity: Pick icons that clearly represent the products within the category to aid user navigation.
  • Resetting: To return to an automatic icon, click the "Reset" link in the Icon Picker section.

Troubleshooting

Icon is not updating: Clear your browser cache or perform a hard refresh (Ctrl+F5). The icons are rendered client-side and may be cached.

Icon appears as a generic box: This happens if the component name in the database does not match any Lucide component. Use the picker to re-select a valid icon.
