Roadmap & Future Enhancements

Consolidated recommendations for the next phases of visual category modernization.

Phase 3: Category Brand Colors

Allow administrators to assign specific hex colors to each category. This color would then be used as the background/border for icons and category chips globally, creating "mini-brands" for each section (e.g., Electronics is blue, Fashion is pink).

Phase 4: L3 Specific Iconography

Expand the icon picker specifically for Level-3 categories. Currently, most specific categories use smaller dots or plain text. High-traffic L3 categories could benefit from their own unique icons in the mobile view.

Phase 5: Animated Icon States

Integrate `lottie-react` or similar for specific featured categories, allowing icons to "come to life" (e.g., a phone ringing, a package opening) when a user stays on the homepage section.

Phase 6: Image-to-Icon AI Utility

Implement an AI tool in the Admin panel that analyzes a category's uploaded image and suggests the most relevant Lucide icon (e.g., detecting a shoe in the image and suggesting the 'User' or 'Package' icon).
