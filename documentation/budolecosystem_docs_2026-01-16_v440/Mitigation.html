Mitigation Strategies v440

M-440-01: Regex Robustness

The regex pattern was updated to include \₱ and ₱ explicitly to handle Philippine Peso symbols and common variations in branding suffixes.

M-440-02: Fallback Logic

If the regex fails to find a match, the system falls back to displaying the original text as-is, ensuring no information is lost due to formatting errors.

M-440-03: Performance Optimization

In Flutter, RichText with TextSpan is used for efficient rendering. In React, React.Fragment and memoized components prevent unnecessary re-renders of branded text.
