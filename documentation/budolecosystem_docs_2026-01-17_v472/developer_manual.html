Branding Utility

The core branding logic resides in ui_utils.dart. The formatBudolPayText method uses a regular expression to identify "budol" followed by a suffix. It now explicitly handles the ₱ay suffix and normalizes any Pay input to use the Philippine Peso sign.

// Pattern updated to include the Peso sign
final pattern = RegExp(r'(budol)(_?)([a-zA-Z\₱]+)', caseSensitive: false);

// Normalization logic
if (suffixLower == "pay" || suffixLower == "₱ay") {
  suffixPart = "₱ay";
}
