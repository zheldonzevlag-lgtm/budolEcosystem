BudolEcosystem Documentation v472

Release Date: 2026-01-17

Status: Compliance Verified

Purpose

This version implements a comprehensive branding update to replace the "P" in "budolPay" with the Philippine Peso sign (₱), resulting in the new branded name: budol₱ay. This change is applied across all mobile application screens, services, and internal components to ensure total brand consistency.

Implementation Phases

  • Phase 0: Identified all occurrences of "budolPay" and the formatting logic in UIUtils.
  • Phase 1: Updated UIUtils.formatBudolPayText to normalize "Pay" to "₱ay".
  • Phase 2: Updated hardcoded strings and user-facing labels in main.dart, home_screen.dart, login_screen.dart, qr_scanner_screen.dart, biometric_service.dart, and discovery_service.dart.
  • Phase 3: Verified consistent rendering across the entire application using automated verification scripts.
