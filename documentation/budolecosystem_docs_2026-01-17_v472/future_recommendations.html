Future Recommendations v472

SVG Logos:

Consider replacing text-based branding in the Splash and Login screens with high-quality SVG logos that incorporate the ₱ symbol for better visual fidelity across different screen densities.

Multi-Language Support:

As the branding now uses local symbols, consider implementing full i18n support to allow for Tagalog and other regional languages in future releases.

Dynamic Branding:

Implement a theme-based branding service that can fetch the latest brand name and symbols from the backend, allowing for ecosystem-wide branding updates without app-store releases.
