System Admin Manual

Maintenance and deployment instructions for v402.

🔧 Maintenance Operations

Dashboard Monitoring

System administrators should monitor the Admin Dashboard (port 3000) for UI stability. The v402 update ensures that searches do not cause the application to crash, even if the database contains incomplete transaction records.

Deployment

Standard npm start in the apps/admin directory is required to apply the changes. No database migrations are necessary for this update.
