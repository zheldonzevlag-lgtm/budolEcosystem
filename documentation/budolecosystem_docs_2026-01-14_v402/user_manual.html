User Manual (Admin)

Guide for using the stabilized transaction search feature.

🔍 Searching Transactions

The Global Transactions page allows you to search through all ecosystem financial flows in real-time. With the v402 update, the search is now more robust.

Reference ID Search

Enter the unique transaction reference ID to filter the list instantly.

Email Search

Search by sender or receiver email address to track specific user activity.

💡

The search results update instantly as you type. If a transaction lacks a reference ID, it will simply be excluded from reference-based searches rather than causing an error.
