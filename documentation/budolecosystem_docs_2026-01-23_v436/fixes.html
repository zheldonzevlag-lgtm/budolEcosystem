Fixes & Updates - v436

[FIX] Store Orders Search ReferenceError

Issue: Runtime ReferenceError: searchTerm is not defined in app/store/orders/page.jsx at line 973.

Root Cause: The UI implemented a search input but forgot to define the corresponding state variable in the React component.

Resolution: Defined searchTerm and setSearchTerm using useState('') in the StoreOrders component.

Impact: High. The entire Store Orders page was crashing upon render for any store owner.
