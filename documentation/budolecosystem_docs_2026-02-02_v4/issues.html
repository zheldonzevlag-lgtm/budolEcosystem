Issue Log & Resolutions

I-PROF-001: EPERM Error on Prisma Generate

Description: npx prisma generate failed with permission denied because the query engine binary was locked by active Node.js processes.

Resolution: Terminated conflicting processes using taskkill /f /im node.exe.
I-PROF-002: TypeScript "as any" Syntax in CJS

Description: Test scripts (CommonJS) failed when using TypeScript-style type assertions (as any).

Resolution: Removed all type assertions from test scripts to maintain JS compatibility.
I-PROF-003: Stale Modal Data

Description: Edit modal showed data from the previously edited user when reopened for a different user.

Resolution: Updated onClose handler in employees/page.tsx to reset selectedEmployee state.
