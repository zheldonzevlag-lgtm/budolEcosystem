User Manual - Employee Profile Updates

1. Navigating to Employees

Log in to the Admin Dashboard and click on the Employees tab in the sidebar.

2. Editing a Profile

Locate the employee you wish to update. Click the Edit icon (pencil) in the Actions column.

3. Updating Information

In the popup modal, you can now reliably update:

  • First Name & Last Name
  • Phone Number (displayed in the new Contact column)
  • Department (displayed in the new Dept column)
  • System Role

4. Verifying Changes

After clicking Save Changes, the table will automatically refresh. You should see the updated phone number and department immediately in the "Contact & Dept" column.
