Operational Guide: Phase 1 Implementation

1. Environment Variable Management

Admins must ensure that the JWT_SECRET is synchronized across all Vercel projects. If the secret is rotated, it must be updated in all services simultaneously to prevent SSO failures.

2. Deployment Commands

To deploy the core services using the new configurations:

# Deploy budolID
cd budolID-0.1.0
vercel deploy --prod

# Deploy budolAccounting
cd budolAccounting-0.1.0
vercel deploy --prod

# Deploy budolpay Gateway
cd budolpay-0.1.0/services/api-gateway
vercel deploy --prod

Risk & Mitigation

  • Risk: Database connection failure due to incorrect DATABASE_URL in .env.vercel.
  • Mitigation: Always verify connectivity using prisma db pull locally after switching to the vercel environment.
