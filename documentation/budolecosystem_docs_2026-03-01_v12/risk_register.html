Risk Register - Compliance Focus (v5.5.0)

Risk ID Description Category Impact Likelihood Mitigation Strategy
R-COMP-01 Insider Threat: Unauthorized privilege escalation by Admin Security Critical Low Enforced Maker-Checker for all role modifications.
R-COMP-02 Bypass of MFA for sensitive profile edits Compliance High Medium Hardcoded MFA gating at the API level (Phase 3).
R-COMP-03 Data Integrity: Incorrect phone/email format causing sync failure Operational Medium High Implemented normalization logic (E.164) and frontend validation.
R-COMP-04 Maker acting as Checker for their own request Regulatory Critical Low Server-side validation blocking same-user approval.
