Knowledge Base

Welcome to the centralized documentation hub for Budol Ecosystem v3.3.24. This version introduces critical updates to the address system, validation logic, and database schema.

What's New in v3.3.24 New

Release Date: February 24, 2026

  • Optional Barangay: Full stack support for optional barangay fields in addresses.
  • Zod Validation: Migrated from toast errors to Zod schema validation for robust error handling.
  • Prisma Client v4: Custom output path @prisma/client-custom-v4 to resolve Windows file locking.
  • Address Pre-filling: Enhanced user experience with auto-filling of known user details.
  • Optional Zip Code: Zip code is now optional in address forms and database schema.
  • Store Address Fix: Resolved critical validation error for store addresses.
