User Manual: Address Management Updates

Welcome to the updated Address Management system (v3.3.24). We have made it easier for you to manage your addresses.

What's New?

1. Optional Barangay

You are no longer required to enter a "Barangay" when adding or editing an address. This field is now optional. If your address does not have a specific barangay, you can leave it blank.

2. Automatic Name Filling

When you add a new address, your First Name and Last Name will be automatically filled in based on your profile information. You can still edit them if needed.

3. Improved Error Messages

We have improved the error messages you see when filling out forms. Required fields will be clearly marked, and if you miss something, specific guidance will appear right below the field.

How to Add an Address

  1. Go to your Profile or Checkout page.
  2. Click on "Add Address".
  3. Fill in the details. Note that "Barangay" is marked as Optional.
  4. Click "Save Address".

Troubleshooting

Issue: "Missing required fields" error.
Solution: Check if you have filled in all fields marked with a red asterisk (*). Barangay is NOT required.
