System Admin Manual Update v158

Vercel Hobby Plan Constraints

  • Execution Timeout: 10 seconds per request.
  • Bandwidth: 100GB per month.
  • Concurrent Requests: Limited (not specified, but subject to throttling).

Domain Configuration

All core service domains have been successfully created and configured in Vercel. Admins should verify that each domain is correctly linked to its respective microservice repository and production branch.

S3 Bucket Configuration

For the image repository, ensure the S3 bucket is configured with:

  • Block Public Access: Enabled (for KYC privacy).
  • CORS: Allow GET and PUT from budolpay.vercel.app.
  • IAM Policy: Grant s3:PutObject and s3:GetObject to the service user.

Monitoring

Use the Vercel Analytics and Logs tabs to monitor service health. Set up alerts for 5xx errors which may indicate database connection timeouts.
