System Admin Manual: Infrastructure & Database Synchronization

Version: v148 | Date: 2026-01-04

Database Schema Updates

The budolid database was reset to align with the latest Prisma schema. Ensure the following command is run after any schema changes:

npx prisma db push --force-reset

Warning: This will wipe all data in the SSO database. Use only in development.

Environment Configuration

Services are configured to run on a local network IP (192.168.1.24). If the host IP changes, update the following environment variables in the startup scripts or .env files:

  • LOCAL_IP
  • AUTH_SERVICE_URL
  • WALLET_SERVICE_URL
  • GATEWAY_URL

Operational Runbook

Manual Admin Synchronization

  1. Run node prisma/reset-admin.js in budolID-0.1.0.
  2. Note the generated User ID.
  3. Update budolpay-0.1.0/scripts/reset-admin.js with the same ID.
  4. Run node scripts/reset-admin.js in budolpay-0.1.0.
