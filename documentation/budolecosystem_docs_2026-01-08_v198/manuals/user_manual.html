User Manual Updates - v198

Improved Login Security

Fixed: Invalid Authentication Request

We've fixed an issue where the app would ask for a PIN even if you hadn't set one yet. This usually happened after a security update or when logging in from a trusted device for the first time.

What to Expect

If you haven't set up a 6-digit PIN, the app will now automatically direct you to verify your identity via a 6-digit OTP sent to your phone. This ensures you can always access your account safely.

Steps to Resolve

  1. Enter your phone number on the login screen.
  2. Wait for the 6-digit OTP code to arrive via SMS.
  3. Enter the code in the app.
  4. Once logged in, you may be prompted to set up a new secure PIN for future logins.

Note: Your PIN is a 6-digit number that you should keep private. Never share your PIN or OTP with anyone, including budolPay staff.
