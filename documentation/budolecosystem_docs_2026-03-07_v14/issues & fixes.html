Issues & Fixes Log (v14.0.0)

Issue #1: RDS Instance Creation Limit. Attempting to create 5 db.t4g.micro instances resulted in a QuotaExceeded error.
Fix: Switched to logical database partitioning within a single RDS instance. Updated Task Definition environment variables to point to independent databases on the same host.
Issue #2: pg_restore Unsupported Version. Error: "unsupported version (1.16) in file header" when using pg15 client.
Fix: Upgraded Management Host client tools to PostgreSQL 17 via DNF. Successfully bypassed header version checks.
Issue #3: RDS Connection Restriction. Initial restore attempt failed due to Security Group blocking the Management Host.
Fix: Dynamically updated RDS Security Group to allow inbound 5432 from the ECS/Management SG.
