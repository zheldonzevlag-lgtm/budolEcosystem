Developer Manual (v14.0.0)

Structure Overview

The Singapore Regional Rebuild uses a Consolidated Database Architecture. Developers must use service-specific DATABASE_URL paths sharing the same RDS host.

Restoration Workflow

Restoration is handled via a Management Bridge. The script restore_rds.sh automates the PG17 restore process.

Reference File: /scripts/restore_rds.sh 

# Manual Restoration Trigger
bash /home/ec2-user/restore_rds.sh

Database Verification (Lambda)

To audit row counts without EC2 access, use the VPC-Bridge Lambda.

Reference File: /scripts/lambda_db_verify.js 

// Sample Connection Config

PCI DSS Compliance - Data at Rest

All RDS instances have Storage Encryption enabled at build time using the default AWS KMS key.
