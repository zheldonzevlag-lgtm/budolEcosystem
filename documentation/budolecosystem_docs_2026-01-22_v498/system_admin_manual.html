System Admin Manual: Timezone Management

This manual covers the system-level implications of the v498 timezone standardization update.

Compliance Monitoring

Admin users can now monitor compliance telemetry with higher accuracy. All system health checks and audit logs are now synchronized to UTC while being displayed in Asia/Manila time.

System Health Checks

All microservices now return a timestamp field in their /health endpoints using ISO UTC format. This ensures that monitoring tools can correctly order events across different servers.

Audit Logs (BSP Circular 808)

Audit logs generated via the Admin Dashboard or Backend Services now strictly follow the UTC standard for storage, ensuring integrity during forensic audits.
