User Manual: Consistent Time Display

Welcome to the updated Budol Ecosystem. This update improves how time is displayed across the platform.

What's New?

  • Accurate Timestamps: All order times, return requests, and transaction histories are now consistently displayed in Philippine Standard Time (PST).
  • Export Improvements: CSV exports for orders and returns now use a standardized date format that matches your local time, making record-keeping easier.
  • Real-time Notifications: Status updates and chat messages now show the exact time they occurred in your local timezone.

What do I need to do?

No action is required on your part. The system automatically handles the timezone conversion to ensure you always see the correct time for your region.
