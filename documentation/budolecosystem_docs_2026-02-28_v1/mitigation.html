Mitigation Plan

M-001: Scroll Direction Isolation

Risk: R-001 (Scroll Jitter)

Action: Implemented touch-action: pan-x (implicit via overflow-x) and ensured flex container has no vertical overflow on mobile.

M-002: Responsive Class Control

Risk: R-002 (Desktop Logic Failure)

Action: Used Tailwind sm:hidden combined with index-based rendering range. Verified that startIndex still correctly updates the 4-item window on desktop.

M-003: Fixed Dimension Enforcement

Risk: R-003 (Item Squashing)

Action: Applied flex-shrink-0 to all scrollable items to preserve their aspect ratio and readability on various mobile screen widths.
