User Manual: New Mobile Navigation

Swiping Product Images

You can now easily swipe left or right on the product thumbnails located below the main image/video on mobile devices. This allows you to quickly browse all available product media without having to tap small navigation arrows.

Browsing Categories

Category icons on the homepage now support horizontal scrolling. Simply swipe across the category row to see more shopping options. This saves you from scrolling down too far to find what you need.
