User Manual - Version 112

Feature Updates

1. App Branding

The app icon has been updated to the official dark background design with the Peso symbol. This ensures consistent branding across the budolEcosystem.

2. Login Accessibility

The login issue has been resolved. Users can now sign in using their budolID credentials even when using the release version of the app on a local network.

Support

If you still encounter a "404" or "Connection Timeout" error, please verify that your mobile device is connected to the same Wi-Fi network as the budolID server.
