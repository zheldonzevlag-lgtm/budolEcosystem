System Admin Manual - Version 112

Environment Configuration

The mobile app is configured to connect to the following local services:

  • SSO Service: http://192.168.1.24:8000
  • API Gateway: http://192.168.1.24:8080

Ensure that the host machine (192.168.1.24) is reachable from the Android device/emulator and that ports 8000 and 8080 are open in the firewall.

Build Instructions

To generate the updated APK with the correct branding and fixes:

cd budolPayMobile
flutter clean
flutter pub get
flutter pub run flutter_launcher_icons:main
flutter build apk --release

Branding Assets

The primary branding asset is assets/app_icon.png. This file is used to generate all Android-specific icon resources in android/app/src/main/res/.
