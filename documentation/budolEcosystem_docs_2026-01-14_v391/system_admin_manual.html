System Admin Manual - v391

Release Management

The Android application version 1.2.88 is now the baseline for the current deployment cycle. Admins should ensure that any existing installations are upgraded to this version to benefit from the refined Face Login logic (v390) and stability fixes.

APK Deployment

The production APK is located in the /android-installer/ directory. Admins should distribute budolPay 1.2.88.apk to users or upload it to the internal distribution portal.
