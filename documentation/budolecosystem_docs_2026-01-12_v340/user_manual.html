User Manual - v340

Admin Dashboard Overview

The Admin Dashboard (BudolPay Admin) provides centralized management for the ecosystem's financial and user operations.

Security Audit & Compliance

Access the Security tab to run full system audits. This version introduces IP tracking for all security-sensitive actions, fulfilling PCI DSS requirements.

Settlement Management

New Accounting features allow administrators to track merchant settlements, net amounts, and fee deductions in real-time.

Consumer Marketplace (BudolShap)

The marketplace allows users to browse products, manage carts, and execute payments via BudolPay.

  • Mobile Access: Use the Android installer to access the marketplace from mobile devices via the 192.168.1.17 network.
  • Real-time Updates: Notifications for order status changes and payment confirmations are now processed via the optimized build.

Getting Help

For technical issues, contact the System Administrator or refer to the Developer Manual for error code definitions.
