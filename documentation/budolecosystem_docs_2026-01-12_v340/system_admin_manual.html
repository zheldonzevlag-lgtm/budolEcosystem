System Admin Manual - v340

Environment Configuration

Ensure all .env files are standardized to the local LAN IP for cross-device accessibility.

Standard IP: 192.168.1.17
Database: PostgreSQL (port 5432)
Admin Port: 3000
Consumer Port: 3001

Build Procedures

1. Admin Dashboard Build

Execute the following command to generate the production build for the Admin application:

npm run build --prefix budolpay-0.1.0/apps/admin

2. Consumer Marketplace Build

Execute the following command for the Consumer application:

npm run build --prefix budolshap-0.1.0

Post-Build Verification

Critical: After any Prisma schema change, you MUST run npx prisma db push on the production database before restarting services.
  • Check .next/standalone directory for output.
  • Verify that 192.168.1.17 is reachable from mobile devices in the same network.
  • Ensure the splash screen in the Android app (v0.1.0) reflects the correct version.

Maintenance

Logs for the build process are stored in the terminal output. Any DYNAMIC_SERVER_USAGE errors during build are expected for routes using force-dynamic.
