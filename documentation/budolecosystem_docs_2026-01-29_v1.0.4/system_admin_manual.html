System Admin Manual v1.0.4

Environment Configuration

This release requires strict synchronization of environment variables across the microservices ecosystem.

1. JWT Secret Sync

Mandatory: The JWT_SECRET in budolID-0.1.0/.env must match the JWT_SECRET in budolshap-0.1.0/.env.

2. Database Schema Setup

Ensure the PostgreSQL database contains the budolid schema. Run the following if necessary:

CREATE SCHEMA IF NOT EXISTS budolid;

Service Management

To start the ecosystem with the new configurations:

  1. Navigate to the root directory: cd budolEcosystem
  2. Run: npm start
  3. Select Option 1 (LOCAL) to apply environment changes.

Troubleshooting

  • 401 Unauthorized: Verify JWT_SECRET matches in both budolID and budolshap.
  • Database Connection Error: Check DATABASE_URL in budolID has the correct search_path=budolid.
