Issues & Fixes v1.0.4

Issue: 401 Unauthorized in SSO Flow

Symptoms: User logs in via budolID, redirected to budolShap, but /api/auth/me returns 401.

Root Cause: budolID was using a different JWT_SECRET than budolShap, causing token verification to fail in the callback.

Fix: Unified JWT Secret

Change: Updated budolID index.js to use the same fallback JWT_SECRET as budolShap and ensured environment variables are synced.

Issue: Prisma Schema Collision

Symptoms: budolID failed to find EcosystemApp table or complained about missing columns in User table.

Root Cause: Both apps were pointing to the public schema in the same database, leading to table naming conflicts.

Fix: Schema Namespacing

Change: Configured budolID to use schema=budolid and search_path=budolid in its DATABASE_URL.
