Release Overview v1.0.4

Date: 2026-01-29 | Release Manager: Budol AI Orchestrator

Purpose & Rationale

The v1.0.4 release focuses on resolving critical SSO (Single Sign-On) authentication issues between budolID and budolShap. Following the implementation of LAN access in v1.0.3, a regression was identified where users encountered a 401 Unauthorized error when attempting to login via budolID due to mismatched JWT secrets and database schema isolation issues.

Implementation Task List

Investigate SSO 401 Unauthorized error in budolShap flow
Sync JWT secrets between budolID and budolShap
Fix Prisma schema isolation for budolID (search_path)
Re-seed test user (tony@budol.id) with correct credentials
Automated verification of SSO flow via repro script
Create v1.0.4 documentation suite
Final Phase 10: Git Branch & Push
