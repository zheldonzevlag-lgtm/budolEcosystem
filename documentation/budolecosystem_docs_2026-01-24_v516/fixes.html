Issues & Fixes - v516

Issue: "Unknown Merchant" or Hardcoded Name Display

Users reported that payments always showed "budolShap Merchant" even when purchasing from different stores. This made transaction verification difficult in the mobile app.

Fix: Implemented dynamic store name propagation from the order metadata to the QR payload.

Issue: Transaction ID Truncation

In the Flutter mobile app, long transaction IDs were being cut off on smaller screens.

Fix: Wrapped transaction details in Expanded widgets with softWrap: true and reduced font size to 11px for better fit.
