User Manual: Official Mobile Release

budolPay Mobile v1.2.36

We are excited to announce the release of version 1.2.36 of the budolPay Mobile app. This version is specifically optimized for LAN-based ecosystems.

Auto-Discovery

The app now automatically finds the server on your network. You no longer need to type in the IP address manually. Simply open the app, and it will connect to your local budolEcosystem instance.

Version Display

You can now see the app version clearly displayed on the splash screen and the login screen. This helps you ensure you are running the latest version optimized for your network.

How to Install

The installer can be found in the android-installer folder in the root of the project:

/android-installer/budolPay_1.2.36.apk

Copy this file to your Android device and open it to install the update.
