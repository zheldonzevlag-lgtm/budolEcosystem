budolEcosystem Docs v255

Date: 2026-01-09 | Version: v255

Overview

This update implements dynamic real-time provider switching in the BudolPay mobile app. The system now automatically transitions between Socket.io and Pusher based on remote system settings without requiring an app restart.

Key Changes

  • RealtimeService: Implemented ApiService listener to detect configuration changes.
  • Provider Lifecycle: Added robust disconnect() methods to both Socket and Pusher services to ensure clean resource cleanup during switching.
  • Mobile Build: Incremented version to 1.2.20+20 with sanitized versioning display.
