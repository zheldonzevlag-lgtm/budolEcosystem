User Manual v87: Accessing BudolPay from Anywhere

Local Network Access

You can now access the BudolPay web app from any device connected to the same Wi-Fi. Simply enter the following address in your browser:

http://192.168.1.24:8080

Remote Access (Testing)

If you are testing from a different location, your administrator will provide you with a temporary Ngrok URL (e.g., https://abcd-1234.ngrok-free.app). Use this URL to access the application securely over the internet.
