Developer Manual v87: Multi-Network Access

0.0.0.0 Binding

To allow other devices on your LAN to access the Flutter web app, use the --web-hostname 0.0.0.0 flag when running the app.

flutter run -d chrome --web-hostname 0.0.0.0 --web-port 8080

Dynamic Host Resolution

The ApiService now uses Uri.base.host to automatically determine the correct backend IP when running in a browser environment.
