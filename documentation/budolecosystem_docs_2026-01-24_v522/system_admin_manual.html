System Admin Manual

Guidelines for administrators regarding the order management system.

Order Identification

When viewing the orders list, the "Customer" column now correctly identifies the user who placed the order. Below the customer name, you will find the "Shipping To" section which contains the recipient's name and address. This distinction is vital for verifying orders and communicating with the correct account holder.

Troubleshooting Data Mismatches

If an order appears with "Unknown Customer", it may indicate a data integrity issue where the user record associated with the order has been deleted or is missing. In such cases, refer to the shipping address name for backup identification.
