⚙️ v20 System Admin Manual

🚀 All production services were updated to Revision 26 on March 22, 2026.

🛡️ Automated DB Schema Sync (VPC Internal)

If you encounter P2021 (Table not found) in logs, run this one-time VPC Migrator task from the AWS CLI. This connects to your Private RDS through Fargate:


aws ecs run-task --cluster budol-ecosystem-cluster --task-definition budol-shap-task:25 \
--overrides '{"containerOverrides": [{"name": "budol-shap", "command": ["npx", "prisma", "db", "push"]}]}'

🔄 ECS Service Resync (Master Sync)

To refresh the 13-service internal wiring, run the deployment script targeting the latest master-sync.json:

python register_and_deploy.py budol-shap-task-27-master-sync.json

1. Key Rotation Management

This release implements 64-character high-entropy secure keys for SSO and BudolPay. These keys must be synchronized across 100% of ecosystem services to function.

  • BUDOLPAY_API_KEY: Used for app-to-gateway authorization.
  • BUDOLID_API_KEY: Used for secure SSO session verification.

2. ECS Service Update Flow

To roll out future changes, use the deploy_task.py master script to ensure all services are updated consistently.

python deploy_task.py [TASK_JSON_FILE]

3. Internal Gateway Routing

For production, always favor internal.budol.internal addresses over public DuckDNS addresses to avoid loopback latency and DNS expiration issues.

Gateway internal path: http://budolpay-payment.budol.internal:8004/
