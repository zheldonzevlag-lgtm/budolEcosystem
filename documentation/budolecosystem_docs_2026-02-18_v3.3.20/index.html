Version 3.3.20 - Profile Edit & UI Enhancements

Purpose & Rationale

This release focuses on improving the User Profile Experience by enhancing data privacy controls and validation. Key updates include the ability to toggle visibility of First and Last Name fields, stricter validation for required fields, and ensuring seamless email persistence across the ecosystem.

Technical Summary

  • Profile Visibility Controls: Added "Eye" icon toggles for First Name and Last Name fields in both Web and Mobile apps.
  • Enhanced Validation: Enforced required fields (First Name, Last Name, Email) with real-time feedback.
  • UI/UX Improvements: Split single "Name" field into "First Name" and "Last Name" in the Web App for better granularity and consistency with Mobile.
  • Forensic Audit Trail UI: (Carried over from v3.3.19) Fixed text overlap and added column headers in Admin Security page.
