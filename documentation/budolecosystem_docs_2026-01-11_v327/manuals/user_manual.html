User Manual - v327

Using the Debug Log for Troubleshooting

If you are still experiencing issues with biometric login in version 1.2.51, we have added a troubleshooting tool directly inside the app.

Step 1: Check Version

Ensure you see 1.2.51 at the bottom of your screen. If not, please update to the latest APK.

Step 2: Capture the Error

  1. Tap the fingerprint icon on the login screen.
  2. Scroll down to the very bottom of the login page.
  3. You will see a section titled "Debug Logs".
  4. This section will list exactly what the app is doing when you tap the icon.

Step 3: Report the Issue

Please take a screenshot of the Debug Logs section and send it to your system administrator. This will show us if the phone is rejecting the request, if biometrics are not enrolled, or if there is a critical error.
