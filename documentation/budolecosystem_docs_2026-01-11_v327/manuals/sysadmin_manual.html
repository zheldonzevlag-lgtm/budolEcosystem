Troubleshooting Biometric Taps

If a user reports that "nothing happens" when tapping the biometric icon, ask them to:

  1. Install v1.2.51.
  2. Tap the biometric icon.
  3. Scroll to the bottom of the login screen to view the Debug Logs section.
  4. Take a screenshot of these logs for diagnosis.

These logs will show if the app is even detecting the tap, if it's failing at the enrollment check, or if the OS prompt is being suppressed by the device manufacturer's security policy.
