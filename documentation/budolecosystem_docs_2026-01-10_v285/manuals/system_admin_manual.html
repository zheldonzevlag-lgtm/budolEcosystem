System Admin Manual - v285

1. Overview

This version focuses on client-side stability and resolving issues encountered during LAN testing across different browsers and computers.

2. Maintenance & Operations

2.1. Handling Browser Extension Interference

In LAN environments, users may have various browser extensions installed (e.g., download managers, password managers). These extensions often inject attributes into HTML elements, which can cause Next.js hydration mismatches.

Mitigation: suppressHydrationWarning has been applied to critical UI elements (buttons, inputs) to ignore these injected attributes during the hydration process.

3. Dependencies

No new infrastructure dependencies were introduced in this version. The fixes are entirely within the application code of budolShap.
