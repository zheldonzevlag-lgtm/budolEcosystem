Developer Manual - v439

Global Branding Standardization

This version enforces the use of reusable branding components to ensure consistency across the budol-* ecosystem. Developers MUST use these components instead of raw text or manually styled spans.

Branding Components

  • React (Web): Use the <BudolPayText text="budolPay" /> component from @/components/payment/BudolPayText.
  • Flutter (Mobile): Use the UIUtils.formatBudolPayText(text) utility from ../utils/ui_utils.dart.

Color Palette

  • Slate 400 (#94A3B8): Used for the "budol" prefix.
  • Emerald 500 (#10B981): Used for "Pay", "Shap", or "Care" suffixes.

Code Implementation - React

Example from return/page.jsx:


import BudolPayText from '@/components/payment/BudolPayText';

// Correct usage:
<p className="text-2xl font-bold">
    <BudolPayText text="budolShap" />
    <span className="text-slate-800">.</span>
</p>

Code Implementation - Flutter

Example from marketing_ads_screen.dart:


import '../utils/ui_utils.dart';

// Correct usage:
UIUtils.formatBudolPayText(
  slide['title'],
  baseStyle: const TextStyle(
    fontSize: 32,
    fontWeight: FontWeight.bold,
    color: Colors.white,
  ),
)
