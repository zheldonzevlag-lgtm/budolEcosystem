👥 User Manual - v19.0.0 Update

What's New?

In this update, we have significantly improved the Admin Settings module. You can now save your Email and SMS configurations without worrying about underlying system version mismatches.

How to update SMTP Settings

  1. Log in to the Admin Dashboard.
  2. Navigate to Settings > Email Settings.
  3. Select GOOGLE or your preferred provider.
  4. Enter your credentials and click Update.

Note: If the system cannot send the email immediately, it will safely log the OTP to the background logs for recovery.
