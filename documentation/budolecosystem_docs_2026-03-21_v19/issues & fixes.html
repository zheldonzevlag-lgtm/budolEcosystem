🩹 Issues & Fixes - v19.0.0

Issue 1: Internal Server Error on Admin Settings Save

Cause: Prisma Client (v18) was unaware of the newly added emailProvider columns in the RDS database. This caused an Unknown Argument error during updateSystemSettings.

Fix: Implemented a Raw SQL Fallback layer in lib/settings.js to handle specific sensitive configuration fields bypassing Prisma's validation schema.

Issue 2: OTP Emails falling back to Console

Cause: Missing Transporter configuration in the database meant the lib/email.js function refused to use available ECS environment variables.

Fix: Enhanced the createTransporter logic to respect process.env overrides for SMTP_USER, SMTP_PASS, etc. (available in Revision 11+).
