System Admin Manual - v341

Environment Configuration

The Flutter Web application is built to run on the standard 192.168.1.17 network. No server-side environment variables are required for the static build, as the api_service.dart is hardcoded to the LAN IP.

Build Procedures

1. Flutter Web Build

Execute the following command to generate the production artifacts:

flutter build web --release --prefix budolPayMobile

Deployment & Serving

Current Status: RUNNING
Serving Directory: budolPayMobile/build/web
URL: http://192.168.1.17:8080
Command: npx http-server budolPayMobile/build/web -p 8080 -a 192.168.1.17
Critical: Ensure that the firewall on the host machine allows traffic on port 8080 for LAN access.

Post-Build Verification

  • Verify index.html existence in build/web.
  • Check main.dart.js size (typically 1.5MB - 3MB).
  • Confirm manifest.json points to the correct icons for PWA support.
