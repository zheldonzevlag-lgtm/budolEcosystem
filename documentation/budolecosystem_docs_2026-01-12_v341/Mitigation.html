Mitigation Plan - v341

M-341-01: CanvasKit Fallback Mechanism

Risk: WebAssembly module fails to load.

Action: The build is configured to automatically fall back to the HTML renderer if CanvasKit initialization fails, ensuring basic UI functionality remains accessible.

M-341-02: Microservice CORS Standardization

Risk: Web app cannot communicate with backend services.

Action: All microservices (Auth, Wallet, etc.) have been updated to allow requests from the LAN IP 192.168.1.17 via standard CORS middleware.

M-341-03: PWA Manifest Validation

Risk: Icons or splash screens do not display correctly on mobile browsers.

Action: The manifest.json and index.html meta tags have been verified to point to valid artifacts in the icons/ and splash/ directories.
