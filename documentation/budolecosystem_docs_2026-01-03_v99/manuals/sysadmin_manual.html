Reference ID Auditing

Format Change

The standard format for transaction references has changed from INTENT-<timestamp> to JON-<timestamp>.

Auditing Procedures

When searching for transactions in the database or logs, ensure you use the correct prefix:

SELECT * FROM "Transaction" WHERE "referenceId" LIKE 'JON-%';

Legacy Support

Existing transactions with the INTENT- prefix will remain valid and fully accessible. The system is designed to handle multiple prefix formats simultaneously as it performs exact string lookups during webhook processing.
