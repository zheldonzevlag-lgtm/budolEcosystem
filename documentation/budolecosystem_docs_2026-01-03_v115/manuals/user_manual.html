User Manual - Version 115

Icon Verification

The official app icon should now consistently show the dark background with the pink Peso symbol on all modern Android devices.

Troubleshooting

If you still see the red circular icon after installing this update:

  1. Uninstall the current app.
  2. Install the latest budolPay 1.0.apk.
  3. If it still persists, try restarting your phone to refresh the system icon cache.
