System Admin Manual - Infrastructure & Ports

Ecosystem Port Configuration

The budolPay Mobile app has been configured to connect to the existing ecosystem services without modifying their original ports. This ensures compatibility with other applications.

Service Port Endpoint Type Role in Mobile App
budolID 8000 Auth API User Login / SSO
budolPay Gateway 8004 Transaction API Payments & Wallet
budolPay Mobile 8081 Web Preview Frontend Service

How to Run the Application

Depending on your environment, use one of the following methods to run the budolPay Mobile app:

1. Development Mode (Hot Reload)

For active development with real-time updates:

cd budolPayMobile
flutter run -d chrome

2. Web Preview (Production Build)

To run the app as it appears in the ecosystem preview:

  1. Build the project: 
    cd budolPayMobile
flutter build web
  2. Serve the build:

    Use any static file server (e.g., http-server) to serve the build/web directory:

    http-server build/web -p 8081

3. Accessing the Preview

Once the server is running, the app is accessible at:

URL: http://localhost:8081

4. Android Installation (Physical Device)

To install and run the app on your connected Android phone:

  1. Prerequisites:
    • Enable Developer Options and USB Debugging on your phone.
    • Connect your phone via USB and ensure it is detected (run flutter devices).
  2. Run directly: 
    cd budolPayMobile
flutter run
  3. Build APK for manual install: 
    cd budolPayMobile
flutter build apk --release

    The installer will be located at: build/app/outputs/flutter-apk/app-release.apk

Deployment & Build Process

To update the mobile web preview, follow these steps:

  1. Navigate to budolPayMobile directory.
  2. Run flutter build web to generate the build/web folder.
  3. The preview server is automatically configured to serve this directory on port 8081.
Note: Do not use --wasm flag during build as the QR scanner package is currently incompatible with Wasm compilation.

Operational Runbook: Connection Issues

If the mobile app cannot login or process payments:

  • Check budolID: Ensure the service on port 8000 is running (npm start in ecosystem root).
  • Check Gateway: Ensure port 8004 is reachable.
  • CORS Policy: Verify that the Gateway and ID services allow requests from http://localhost:8081.
