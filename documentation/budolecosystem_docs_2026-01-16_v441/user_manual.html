Branding Refinement

In this version, we have refined the appearance of our service names to be clearer and more consistent.

What's Changed?

  • Better Visibility: The "budol" prefix is now a slightly darker shade of gray, making it easier to read on all screens.
  • Consistent Size: Service names like budolShap and budolLoan now perfectly match the size of other services in the app, even if you have increased your phone's font size settings.

Example: budolPay
