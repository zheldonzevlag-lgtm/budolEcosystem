🚀 Future Recommendations v494

1. Implement useSWRInfinite for Orders

Moving from manual pagination to useSWRInfinite would simplify realtime updates across multiple pages and reduce the complexity of useEffect hooks in the Orders UI.

2. WebSocket-Based Bulk Updates

For bulk actions (e.g., marking 50 orders as shipped), implement a single "bulk-updated" socket event instead of 50 individual "order-updated" events to further reduce client-side processing.

3. Offline Queue Sync

Enhance the mobile app to queue status updates locally when offline and sync them via the same realtime patching logic once connectivity is restored.
