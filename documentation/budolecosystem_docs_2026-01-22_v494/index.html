Budol Ecosystem v494

Release Date: 2026-01-22

This release implements Realtime Order Tab Sync and Enhanced Realtime Notifications. It ensures that when an order's status changes, it automatically moves to the correct status tab in both buyer and seller interfaces without manual refresh. Additionally, notifications now feature a professional "Image 2" style (white background, clean typography) and use user-friendly status labels.
