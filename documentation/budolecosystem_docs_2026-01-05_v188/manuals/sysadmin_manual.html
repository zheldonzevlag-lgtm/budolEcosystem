System Admin Manual - v188

Maintenance Update

This update addresses a UI synchronization issue where document rotation states were not correctly reflected after closing and reopening the verification modal.

Deployment Notes

  • No database migrations are required for this update.
  • Update only affects the Admin Panel front-end logic.
  • Ensure the Admin API (/api/users) is reachable and healthy to support silent state refreshes.

Verification

  1. Open the User Management page.
  2. Open a user's verification documents.
  3. Rotate an image and wait for the "saving" spinner to disappear.
  4. Close the modal and reopen it.
  5. The image should maintain its new orientation.
