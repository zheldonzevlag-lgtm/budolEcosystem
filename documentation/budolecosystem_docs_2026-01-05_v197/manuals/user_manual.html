v197 User Manual

Subject: Improved navigation for BudolPay Admin.

What's New?

We've updated the sidebar navigation to make it cleaner and easier to find system settings.

1. New Icons

Every menu item now has a descriptive icon to help you identify sections at a glance (e.g., a Wallet for Accounting, a Shield for User Verification).

2. Grouped System Settings

To reduce clutter, "System Environment" and "Map & Location" are now neatly tucked under "System Settings".

  • Click System Settings to expand or collapse the menu.
  • The menu will automatically stay open if you are currently working in any of the settings pages.

3. Improved Indentation

Sub-items are now clearly indented, making the hierarchy easy to follow.

Feedback

If you have suggestions for further improving the navigation, please reach out to the development team.
