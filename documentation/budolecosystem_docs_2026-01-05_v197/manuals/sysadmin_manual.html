v197 System Admin Manual

Subject: Administrative impacts of the sidebar reorganization.

Overview

The v197 update reorganizes the BudolPay Admin Panel navigation. No infrastructure changes or database migrations are required for this update.

Navigation Changes

Administrative users will now find "System Environment" and "Map & Location" grouped under a new "System Settings" collapsible menu in the sidebar.

Configuration Paths

  • System Environment: /settings/environment - Managed environment variables and service parameters. (Note: /settings will automatically redirect here).
  • Map & Location: /settings/location - Configuration for Google Maps, Geoapify, and Radar providers.

Deployment Instructions

  1. Pull the latest changes from the main branch.
  2. Rebuild the admin application: 
    npm run build -w admin
  3. Restart the admin service or container.

Troubleshooting

If the sidebar appears broken or the submenu does not expand:

  • Clear the browser cache to ensure the latest Sidebar.tsx bundle is loaded.
  • Verify that the lucide-react package is correctly installed (v0.284.0+ recommended).
