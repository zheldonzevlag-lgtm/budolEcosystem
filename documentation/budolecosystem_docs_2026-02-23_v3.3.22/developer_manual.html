Developer Manual: Geocoding System

Version 3.3.22 introduces a centralized Geocoding API to handle address resolution securely.

New API Endpoint: /api/system/geocode

This endpoint abstracts the underlying map provider (Nominatim or Geoapify) and handles necessary headers and API keys.

Usage

// Reverse Geocoding (Coordinates to Address)
GET /api/system/geocode?lat=14.5995&lng=120.9842
// Returns: Single Object

// Forward Geocoding (Address to Coordinates)
GET /api/system/geocode?q=Makati%2C+Philippines
// Returns: Array of Objects

Response Format (Single Object)

{
  "display": "Formatted Address String",
  "main": "Primary Name (e.g., Street, Building)",
  "secondary": "Secondary Name (e.g., City, Region)",
  "raw": { ... }, // Raw provider response
  "coordinates": [lat, lng]
}
