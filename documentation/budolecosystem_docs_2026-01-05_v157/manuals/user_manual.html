New Feature: P2P Transfer via Phone Number

You can now send money to friends and family using just their phone number! No need to remember long email addresses.

How to use:

  1. Open the budolPay Mobile App.
  2. Tap on Send Money or Cash In.
  3. In the "Recipient Email or Phone" field, enter the 11-digit phone number (e.g., 09171234567).
  4. Enter the amount and tap Send.
  5. The system will automatically find the user and complete the transfer.

Workflow Changes

The "Send Money" screen has been updated to be more intuitive. It now accepts both identifiers in a single input field, reducing friction for users.
