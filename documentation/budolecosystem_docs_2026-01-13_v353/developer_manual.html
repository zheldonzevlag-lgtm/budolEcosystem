Technical Explanation

The persistent black screen on Web was caused by a combination of factors:

  1. Residual lucide_icons dependency in pubspec.yaml causing potential font load failures on Web.
  2. Missing Material context in the custom builder in main.dart for Web rendering.
  3. Potential race conditions in MarketingAdsScreen timer during page transitions.
