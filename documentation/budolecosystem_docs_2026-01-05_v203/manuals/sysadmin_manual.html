System Admin Manual - v203

Configuring Notification Providers

Administrators can now configure active notification providers through the Admin Panel under Settings > Notifications.

1. Supported Providers

Channel Provider Required Settings
Email BREVO brevoApiKey, smtpUser
Email GMASS gmassApiKey, smtpUser
SMS ZERIX zerixApiKey
SMS ITEXTMO itextmoApiKey, itextmoClientCode

2. Database Schema Changes

The User table has been updated. Ensure the database is migrated using:

npx prisma db push

3. Environment Variables

Ensure the following environment variables are set if using default SMTP:

  • SMTP_HOST
  • SMTP_PORT
  • SMTP_USER
  • SMTP_PASS

4. Monitoring

Logs for notification delivery can be found in the service console outputs prefixed with [Notification].
