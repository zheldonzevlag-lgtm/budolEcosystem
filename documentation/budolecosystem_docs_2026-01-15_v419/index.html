Executive Summary

Version 419 resolves a critical dependency conflict in the BudolPay monorepo involving the Prisma ORM. A version mismatch between prisma and @prisma/client was causing build failures and runtime warnings. This release standardizes all Prisma-related dependencies to version 6.19.1 across all 9 packages in the monorepo.

Key Fixes

  • Dependency Standardization: Updated 9 package.json files to use exact version 6.19.1 for both prisma and @prisma/client.
  • Build Pipeline Recovery: Resolved MODULE_NOT_FOUND errors by cleaning up workspace-specific node_modules and reinstalling from the root.
  • Prisma Generation: Successfully regenerated the Prisma client without version mismatch warnings.

Implementation Task List
