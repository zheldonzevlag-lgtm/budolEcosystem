UI/UX IMPROVEMENT

Unified SSO Error Handling

Version 2.2.1 • Updated: April 10, 2026 • Status: Deployed

📌 Purpose

Replaced the generic 401 Unauthorized plain text response with a professional inline error alert on the BudolID login page. This improvement includes a "shake" micro-animation to provide immediate haptic-visual feedback to the user on authentication failure.

🚀 Technical Solution

  • Dynamic Injection: The GET /login route now detects an error query parameter.
  • State Preservation: Redirection back to the form preserves the apiKey and redirect_uri to prevent session dropping.
  • Micro-Animations: CSS @keyframes shake implemented for premium UX feel.
