Phase 8: Rationale

The decision to force navigation to the Login screen from Marketing Ads was made to prevent unauthorized access in cases where a previous session's token might still be valid but the user expects a fresh login experience after seeing onboarding ads. This enhances security compliance with BSP guidelines on session management.
