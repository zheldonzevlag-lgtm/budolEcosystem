Developer Manual

Technical Explanation

The Budol Ecosystem is a multi-project environment containing several microservices. This release addresses the configuration and dependency issues that prevented a smooth local startup.

Key Changes

  • Monorepo Dependency Linkage: Ran npm install in all sub-projects (budolID, budolpay, budolshap, budolAccounting) to ensure local packages like @budolpay/database are correctly linked.
  • Workspace Name Correction: Corrected the admin workspace name in budolpay-0.1.0/apps/admin/package.json from admin to @budolpay/admin to match the runner script.
  • Prisma Schema Fix: Fixed the Prisma schema in api-gateway by adding the missing url parameter to the datasource block.
  • Database Schema Isolation: Updated .env.connection.local to use the budolpay schema locally, ensuring consistency with the production environment and avoiding collisions.
  • Environment Path Fix: Corrected the .env path in payment-gateway-service to correctly point to the monorepo root's .env file.
