User Manual

Local Application Access

After the ecosystem starts, you can access the applications at the following local URLs:

  • BudolShap (Main App): http://localhost:3001
  • BudolPay (Admin Dashboard): http://localhost:3000
  • BudolID (SSO): http://localhost:8000
  • BudolAccounting: http://localhost:8005

Using BudolPay

BudolPay is our financial service integrated with BudolShap. You can use it for payments, transfers, and wallet management. Ensure you have an account created through BudolID (SSO) to access all features.

Troubleshooting

If you encounter issues while accessing the applications, ensure that all services are running and your local network configuration is correct. Contact your system administrator for further assistance.
