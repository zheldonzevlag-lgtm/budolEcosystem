Mitigation Plan - v449

M-449-01: Rendering Fallback

Risk: Flutter engine fails to paint the first frame within the expected buffer.

Action: Implemented a CircularProgressIndicator in the MarketingAdsScreen build method that shows if slides are not yet initialized, preventing a total black screen.

M-449-02: JS Communication Timeout

Risk: The JavaScript call to remove the splash screen hangs or fails.

Action: The splash removal is now triggered by multiple screens (Splash, Login, Home) as a fail-safe. If one fails, the next screen mount will attempt the removal again.

M-449-03: State Persistence Integrity

Risk: SharedPreferences corruption leads to navigation loops.

Action: Added validation logic in ApiService.init() to reset suspicious or malformed host/state configurations.
