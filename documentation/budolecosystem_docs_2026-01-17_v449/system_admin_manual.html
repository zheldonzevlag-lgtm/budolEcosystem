System Admin Manual - v449

Web Deployment

This version includes critical fixes for Flutter Web. When deploying the web build (flutter build web), ensure that the index.html still contains the #splash overlay, as the app now relies on specific JS signals to remove it.

Monitoring

Admins should monitor the browser console for "WEB_RENDER_OK" logs to verify that the rendering guard is functioning correctly on user devices.

Compliance: This release maintains compliance with NPC and BSP regulations regarding application stability and availability.
