Infrastructure Requirements

  • API Gateway: Must expose port 8080 to the network where mobile devices reside.
  • SSL/TLS: Mobile OS requirements necessitate valid SSL certificates for all API endpoints.

Environment Variables

  • BUDOLPAY_GATEWAY_URL: URL of the API Gateway.
  • BUDOLID_SSO_URL: URL for authentication services.
  • FLUTTER_TELEMETRY_DISABLED: Set to true in CI/CD environments.

Deployment Procedures

  1. Standard Flutter build process: flutter build apk or flutter build ios.
  2. Run flutter config --no-analytics on build agents.
  3. Verify connectivity from a mobile device to the Gateway URL.
