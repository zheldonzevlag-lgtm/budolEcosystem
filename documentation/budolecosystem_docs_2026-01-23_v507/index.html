v507

Purpose & Rationale

The v507 release focuses on Payment Summary UI Enhancements in the mobile application. This update improves transparency for users by displaying detailed transaction identifiers and store-specific information during the payment confirmation process.

Key Objectives:

  • UI Enhancement: Restructured the Transaction Details section to include Transaction ID, Store Name, Order Number, and Reference ID.
  • Transparency: Provided clear mapping of payment data from QR codes to the summary screen.
  • Standardization: Maintained consistent styling using existing UI utility components.
