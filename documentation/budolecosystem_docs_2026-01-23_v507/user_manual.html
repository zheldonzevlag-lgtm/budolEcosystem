User Manual - v507

Enhanced Payment Summary

When you scan a QR code to pay, the Payment Summary screen now shows more detailed information to help you verify your transaction.

  • Merchant: The brand or organization you are paying.
  • Transaction ID: A unique identifier for this specific payment attempt.
  • Store Name: The specific branch or store location.
  • Order Number: Your unique order reference from the merchant.
  • Reference ID: The payment intent identifier for tracking.

Installation (v506)

The latest version of the mobile app is available as an APK in the android-installer directory.

  • File Name: budolPay 1.3.24.apk
  • Installation: Transfer the APK to your Android device and open it to install. Ensure "Install from Unknown Sources" is enabled.

API Host Configuration (v505)

The mobile application now remembers your API Gateway host address. You only need to set it once, and it will be saved for future sessions.

  • How to Set: Open the Host Configuration dialog from the settings or initial setup screen.
  • Manual Entry: Enter the IP address and port (e.g., 192.168.1.5:8080) and click Save.
  • Network Scan: Click Scan Network to automatically discover the gateway on your local network.

For Store Owners: Dashboard Search (Inherited v504)

You can now search for orders directly from your store dashboard. Use the search bar to filter by:

  • Order ID: Enter the unique ID of the order.
  • Customer Details: Search by customer name or email.
  • Product Names: Find orders containing specific items.
