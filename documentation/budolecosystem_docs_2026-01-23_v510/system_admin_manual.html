System Admin Manual v510

System Impact

This update ensures the budolShap order processing system remains stable by aligning code logic with the database schema. No changes to server infrastructure or environment variables are required.

Database Integrity

Admins should note that the system currently uses a binary inStock status for products. Numerical stock levels are not maintained in the database. When a product is marked as inStock: false in the admin dashboard, it will be immediately unavailable for new orders.

Monitoring

Monitor the POST /api/orders endpoint for 500 Internal Server Error responses. The previous PrismaClientValidationError has been resolved, but any new schema mismatches will appear in the server logs.
