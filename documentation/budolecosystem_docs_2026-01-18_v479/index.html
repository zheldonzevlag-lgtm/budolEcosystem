Release Summary

This release (v479) focuses on Forensic Audit Logging & BSP Circular 808 Compliance. It standardizes the createAuditLog helper across critical services (Auth, Wallet, Transaction, Settlement), ensuring that every financial and security action is logged with IP, User Agent, and Manila Time (UTC+8). Furthermore, it integrates these logs with real-time admin notifications via the API Gateway.
