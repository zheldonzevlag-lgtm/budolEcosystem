Service Port Configuration

The following ports must be correctly configured in .env files for proper ecosystem communication:

Service Name Default Port Env Variable (in budolShap)
budolID (SSO) 8000 NEXT_PUBLIC_BUDOLID_URL
budolPay (Auth) 8001 -
budolPay (Wallet) 8002 -
budolAccounting 8005 BUDOLACCOUNTING_URL

Troubleshooting Order Visibility

If a user reports that they cannot see their orders after payment:

  1. Verify the user's token contains userId or id.
  2. Check the application logs for [Token] Successfully verified token for user: ...
  3. If a 403 Forbidden error appears in /api/orders, ensure lib/auth.js is using the v427 normalization logic.
