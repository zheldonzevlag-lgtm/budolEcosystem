Troubleshooting Connection Issues

🔍 New: Test Connection Tool

If you are experiencing a "Timeout" error during login, you can now diagnose the problem directly in the app:

  1. On the Login screen, tap the Network Settings icon (top right).
  2. Tap the Test Connection button.
  3. Wait for the results. If either "Auth Service" or "Gateway API" shows FAIL, your phone cannot talk to the server.

💡 Quick Fixes

  • Check Wi-Fi: Ensure your phone is on the exact same Wi-Fi network as the computer running the server.
  • Turn off VPN: Disable any active VPNs or Ad-blockers (like AdGuard) as they can interfere with local network connections.
  • Restart Wi-Fi: Sometimes toggling your phone's Wi-Fi off and on can resolve routing issues.
