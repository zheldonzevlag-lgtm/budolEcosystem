Future Recommendations (v18)

Implement CI/CD via AWS CodePipeline

Instead of manual PowerShell scripts, integrate GitHub/AWS CodePipeline to automatically build and deploy images on every git push to the master branch.

Shared Prisma Base Image

To reduce build times, create a base Docker image containing pre-installed Node.js, Prisma, and common dependencies used across all services.

ECS Auto-Scaling

Configure auto-scaling policies based on CPU/Memory utilization to handle traffic spikes during "Budol" sales events.

VPC Endpoint for ECR

Utilize VPC Endpoints for ECR and S3 to reduce NAT Gateway costs and improve security within the private subnets.
