User Manual - v51

Order Placement restored

This update fixes a technical issue that was preventing customers from completing their orders in budolShap.

What's Fixed?

  • Checkout Success: The "Place Order" button in the Order Summary now correctly initiates the payment process without errors.
  • Payment Redirect: Customers will now be correctly redirected to the BudolPay or PayMongo checkout pages after clicking "Place Order".

Modified Workflows

Placing an Order

  1. Add items to your cart.
  2. Proceed to the Checkout page.
  3. Review your order in the Order Summary.
  4. Click Place Order.
  5. You will be redirected to the secure payment gateway to complete your transaction.

If you previously encountered an error message starting with "Unexpected token", this has been resolved and you can now proceed with your purchase.
