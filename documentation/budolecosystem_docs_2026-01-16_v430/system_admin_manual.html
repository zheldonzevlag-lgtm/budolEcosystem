System Admin Manual - Version v430

Environment Configuration

Ensure that the following environment variables are correctly set in .env.vercel or the local environment:

  • JWT_SECRET: Must be consistent across all services in the ecosystem (budolID, budolShap, budolPay).
  • NEXT_PUBLIC_SSO_URL: URL of the budolID service.

Authentication Flow

The system now supports Authorization: Bearer [token] headers for admin verification. This provides a fallback if cookies are not being correctly transmitted by the client browser.
