System Admin Manual - Version 1.2.65

Deployment Instructions

1. Web Hosting

The web build is located in /budolPayMobile/build/web. Upload these files to your web server's root directory. Ensure that the server is configured to serve index.html for all unknown routes (Single Page Application configuration).

2. Android Installation

The latest APK is available at:

/android-installer/budolPay 1.2.65.apk

3. Version Verification

After deployment, verify the version by checking the top-left corner overlay (in development mode) or the splash screen information.

Build Command

To rebuild the web and android apps:

flutter build apk --release; flutter build web --release --base-href /
