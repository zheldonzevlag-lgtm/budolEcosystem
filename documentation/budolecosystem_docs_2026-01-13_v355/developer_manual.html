Developer Manual - Version 1.2.65

Overview

This version addresses the critical black screen issue observed on Flutter Web after the splash screen. The primary cause was identified as rendering failures in the complex RichText logic and potential missing default text styles in the MaterialApp builder.

Technical Changes

1. DefaultTextStyle Injection

Added a DefaultTextStyle wrapper inside the Material widget in the MaterialApp builder. This ensures that any Text or RichText widgets have a valid fallback style, preventing rendering crashes on some browser environments.

File: main.dart

2. Simplified MarketingAdsScreen Rendering

Replaced RichText with Text.rich for better compatibility and simplified the regex-based branding logic. Added ValueKey to slides to ensure proper widget recycling.

File: marketing_ads_screen.dart

3. MultiProvider Refactoring

Updated MultiProvider to use the standard constructor instead of .value for the primary services to ensure proper lifecycle management.

File: main.dart
