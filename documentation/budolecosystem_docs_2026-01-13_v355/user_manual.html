User Manual - Version 1.2.65

Welcome to the latest version of budolPay. This version improves the reliability of the marketing ads shown during the app's initial launch.

1. New Service Spotlight: budolAkawntng

The marketing sequence now includes our new budolAkawntng service. This service acts as the central hub for monitoring and auditing all your financial transactions across the Budol Ecosystem.

2. Improved Web Experience

If you are using the web app, you will now see the application contained within a realistic mobile device frame. This provides a more consistent experience between the mobile app and the web portal.

3. Navigation

The marketing ads will automatically cycle every 5 seconds. You can skip the sequence at any time by tapping the "Get Started" button or swiping to the end.
