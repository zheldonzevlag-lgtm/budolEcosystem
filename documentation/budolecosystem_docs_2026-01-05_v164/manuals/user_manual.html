Better App Stability & Reliability

← Back to v164 Index

New Error Recovery Screen

We've fixed the issue where the app would sometimes show a completely white/blank screen. Now, if something goes wrong, you'll see a helpful error screen with a "Reset" button.

How to use the Reset Button:

If you see an error message, simply tap "Reset App & Logout". This will clear any stuck settings and take you back to the login screen, where you can start fresh.

Faster Loading

The app now loads faster. If it takes more than 5 seconds to load your previous session, it will automatically move you to the login screen so you aren't left waiting on a loading screen indefinitely.

Transaction History Fix

Your recent transactions (money sent and received) should now appear correctly on the home screen. We've fixed a bug that caused the app to crash when trying to display certain types of payments.

What to do if the app is still "White":

  1. Ensure you have updated to version 1.1.5+1 (check App Info).
  2. Check your internet connection.
  3. If you are using a local server, tap the Gear Icon on the login screen to verify the Server IP address.
