🔧 Sync Workflow (The Warp Sync)

To keep production in sync with local dev, we use the budol_v21_master_sync.py tool. This script ensures that any changes to your local environment variables or ports are immediately reflected in the AWS ECS Task Definitions.

# Execution command:
python budol_v21_master_sync.py

# Logic:
1. Register local .json tasks to AWS ECR/ECS
2. Extract working Networking (SG/Subnets) from budol-id-service
3. Create/Update services in the 'budol-ecosystem-cluster'
