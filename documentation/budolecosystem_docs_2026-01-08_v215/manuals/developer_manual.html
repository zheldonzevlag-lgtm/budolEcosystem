Developer Manual Updates v215

Password Hashing Logic

The application uses bcryptjs for password hashing. When updating passwords manually, developers should use the hashPassword utility or follow this pattern:

const salt = await bcrypt.genSalt(10);
const hashedPassword = await bcrypt.hash(newPassword, salt);

Database Impact Analysis

This change directly modifies the password field in the User table of the budolshap database. It does not affect session tokens or other user metadata, but will require the user to re-authenticate if their session expires.
