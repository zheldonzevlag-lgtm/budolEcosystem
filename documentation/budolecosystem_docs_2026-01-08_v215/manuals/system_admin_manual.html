Manual Password Reset Procedure

To manually reset a user's password when the UI/email flow is unavailable:

  1. Navigate to budolshap-0.1.0/scripts/.
  2. Run the update script: node scripts/update-user-password.mjs.
  3. Verify with: node scripts/verify-user-password.mjs.

Note: Ensure the .env.local file contains the correct DATABASE_URL for the environment you are targeting.
