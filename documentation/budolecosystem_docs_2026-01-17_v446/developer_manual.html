Flutter Web Rendering Fixes

This version addresses the "black screen" issue on Flutter Web where the Marketing Ads screen failed to render after the splash screen was removed.

1. Navigation Timing Adjustment

Modified splash_screen.dart to increase the delay before removing the web splash overlay. This ensures the target screen has enough time to mount and prepare its first frame.

// splash_screen.dart
if (kIsWeb) {
  debugPrint('SplashScreen: Removing web splash in 500ms...');
  await Future.delayed(const Duration(milliseconds: 500));
  callJsMethod('removeSplashFromWeb');
  await Future.delayed(const Duration(milliseconds: 100));
}
