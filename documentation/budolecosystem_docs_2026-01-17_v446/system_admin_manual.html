Web Deployment Considerations

Important: The Flutter web app now includes a 500ms delay for splash removal. This is normal behavior and intended to prevent rendering issues on slower client hardware.

1. Viewport Settings

The web app is containerized to a fixed size of 393x780. Administrators should ensure that deployment environments (like frames or mobile browser simulators) respect these dimensions for the best user experience.

2. Debugging

In kDebugMode, the app will display a green label ADS_READY at the top left of the marketing screen, along with the detected layout constraints. This can be used to verify environment scaling.
