Fixes Log - v446

Fixed: Flutter Web Black Screen after Splash

Issue: Users reported a black screen on the Flutter web app after the splash screen finished, preventing marketing ads from showing.

Root Cause: A race condition between the JavaScript-based splash removal and the Flutter engine's first frame rendering of the MarketingAdsScreen.

Resolution:

  • Increased web splash removal delay to 500ms.
  • Added RepaintBoundary to the MarketingAdsScreen.
  • Used LayoutBuilder to ensure stable constraints on web.
  • Added debug logging to track layout constraints in real-time.
