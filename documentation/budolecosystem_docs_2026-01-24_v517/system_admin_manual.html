Deployment Guide

To deploy the budol₱ay Mobile v1.3.32 APK:

  1. Ensure the Android device has "Install from Unknown Sources" enabled.
  2. Transfer the APK from \budolEcosystem\android-installer\budolPay 1.3.32.apk to the device.
  3. Run the installer and follow the on-screen prompts.
  4. Verify the version number (v1.3.32) on the splash screen.

Environment Configuration

The app connects to the Budol Ecosystem API. Ensure the API_HOST environment variable is correctly set during build time or configured via the in-app debug settings if applicable.
