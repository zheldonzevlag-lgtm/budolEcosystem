Developer Manual - v517

Technical specifications and code changes for the budol₱ay Mobile v1.3.32 release.

1. Navigation Stack Reset

Implemented a complete navigation stack reset when returning to the home screen from a successful payment. This prevents users from navigating back into the payment flow using the hardware back button.

// Implementation in qr_scanner_screen.dart Navigator.of(context).pushNamedAndRemoveUntil( Routes.home, (route) => false, );

Reference: qr_scanner_screen.dart

2. Compact UI & Dynamic Metadata

Refactored the transaction details section to use a 12px font size and added real-time date/time formatting using the intl package.

// Date/Time formatting final now = DateTime.now(); final paymentDate = DateFormat('MMMM dd, yyyy').format(now); final paymentTime = DateFormat('hh:mm a').format(now); // Compact row implementation _buildDetailRow('Order Number', orderId), _buildDetailRow('Payment Date', paymentDate), _buildDetailRow('Payment Time', paymentTime),

Reference: qr_scanner_screen.dart

3. Version Synchronization

Synchronized version numbers across pubspec.yaml and ApiService to ensure consistency in the splash screen and about sections.
