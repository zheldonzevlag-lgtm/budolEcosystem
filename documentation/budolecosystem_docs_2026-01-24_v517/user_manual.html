User Manual - v517

Payment Summary Screen

The payment summary screen now displays more detailed information for your transactions:

  • Order Number: A unique identifier for your purchase.
  • Payment Date: The date the transaction was processed.
  • Payment Time: The exact time of the transaction.

The text is now more compact (12px) to allow more information to be visible without scrolling.

Navigation

When you tap "BACK TO HOME" after a successful payment, the app will return you to the main home screen. You can then start a new transaction or view your balance.
