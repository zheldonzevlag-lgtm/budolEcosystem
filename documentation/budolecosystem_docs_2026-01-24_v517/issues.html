Known Issues - v517

UI/UX

  • The 12px font size might be too small for users with visual impairments. Accessibility options for font scaling are not yet fully implemented.
  • Payment date/time uses device local time, which may differ from the server time in cross-timezone scenarios.

Functional

  • Occasional delay in redirecting to Home screen if network latency is extremely high during session refresh.
