Network Configuration & LAN Access

In v346, we've improved the way the budolPay Mobile Web app handles LAN connectivity. As a System Admin, ensure the following configurations are in place for optimal performance.

1. Gateway and Auth Ports

The app expects the following services to be accessible on the server's IP:

  • API Gateway: Port 8080
  • Auth Service: Port 8001
  • Mobile Web App: Port 9000

2. LAN Accessibility

Important: The Flutter web server is now configured to listen on 0.0.0.0 (all interfaces) to allow access from other devices on the same network.

To start the ecosystem with full LAN support, use the central runner:

npm start

3. Troubleshooting Loading Issues

If the app is unreachable, verify:

  • The server's firewall allows traffic on ports 8001, 8080, and 9000.
  • The LOCAL_IP in the .env files is correctly set to the server's current LAN IP (e.g., 192.168.1.17).
  • All microservices are running (check the console output of the Central Runner).
