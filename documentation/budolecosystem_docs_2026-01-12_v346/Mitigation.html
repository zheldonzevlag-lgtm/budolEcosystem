Mitigation Strategies

R-346-01: Background network calls fail silently.

Mitigation: Implement notifyListeners() and localized error handling in the background initialization. Ensure screens that depend on this data have appropriate loading states or error indicators.

R-346-02: Host detection logic fails in complex environments.

Mitigation: Provide a manual host override in the Settings screen (implemented as _customHost) allowing users or admins to manually set the API host if auto-detection fails.

R-346-03: Bypassing marketing screen engagement.

Mitigation: Move important announcements or promotions to a non-intrusive banner or "What's New" section within the Home screen, rather than forcing a splash-to-marketing flow every time.
