Enhancements for v347 and Beyond

  • Progressive Loading: Implement skeleton screens in the Home and Wallet screens to provide better visual feedback while background data is being fetched.
  • Service Worker for Web: Explore using service workers to cache static assets and improve initial load time for the web app.
  • Centralized Logging: Implement a centralized error logging service (e.g., Sentry) to better track and diagnose loading issues in production/LAN environments.
  • Automated Health Checks: Add a health-check page in the Admin panel that verifies connectivity between the Mobile Web App and all microservices.
