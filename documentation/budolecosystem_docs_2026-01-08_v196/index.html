Release Summary: v196 (Verification Service Port Conflict Fix)

This version documents the resolution of a critical EADDRINUSE error in the Verification Service (KYC). The issue was caused by a redundant app.listen call in the service's entry point, which attempted to bind to port 8006 twice during startup.

Key Improvements

  • Service Stability: Eliminated redundant listener logic that prevented the service from starting.
  • Code Quality: Cleaned up the service initialization sequence to match ecosystem standards.
  • Environment Awareness: Refined the conditional startup logic for local vs. production environments.
