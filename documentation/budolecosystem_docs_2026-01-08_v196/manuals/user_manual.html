User Manual Updates - v196

What's New?

Resilient KYC Verification

The system's background verification service (KYC) has been stabilized. Users will no longer experience intermittent "Service Unavailable" errors when attempting to upload documents or check their verification status.

Impact on User Workflow

There are no changes to the user interface or workflow. This update is a behind-the-scenes stability fix that ensures the following features work reliably:

  • Document Upload: Faster and more reliable processing of ID documents.
  • Status Checking: Real-time updates on your verification tier (e.g., Semi-Verified, Fully Verified).
  • KYC Processing: Smoother transition between registration and full account access.

Recommendations

Users who previously encountered issues while uploading their IDs are encouraged to try again. The service is now fully operational and monitored for stability.
