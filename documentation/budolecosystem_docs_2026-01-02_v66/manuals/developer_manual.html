1. Layout Height Adjustment

Following user feedback, the container height for the web preview has been optimized for better visibility on a wider range of desktop monitors.

1.1 Container Specification

The height property in the BudolPayApp builder within main.dart was updated from 852px to 780px.

// Updated in main.dart
width: 393, // Standard Pro width
height: 780, // Optimized height for desktop preview

1.2 Rationale

A height of 780px ensures that the entire mobile frame (including bezels and margins) is visible on standard 1080p displays without requiring excessive scrolling, while still maintaining a clear mobile aspect ratio.
