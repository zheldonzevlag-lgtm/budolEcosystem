Developer Manual - v453

Compilation Fix: kDebugMode Import

The previous release (v452) introduced a debug reset button in the SplashScreen but missed the necessary import for kDebugMode. This has been resolved by updating the flutter/foundation.dart import.

splash_screen.dart 
import 'package:flutter/foundation.dart' show kIsWeb, kDebugMode;

UI Integration: Smooth Page Indicator

The smooth_page_indicator package is now fully integrated into the MarketingAdsScreen. This replaces the custom-built pagination logic with a more robust and visually appealing solution.

marketing_ads_screen.dart 
SmoothPageIndicator(
  controller: _pageController,
  count: _slides.length,
  effect: const ExpandingDotsEffect(
    dotHeight: 8,
    dotWidth: 8,
    expansionFactor: 3,
    spacing: 8,
    dotColor: Colors.white24,
    activeDotColor: Color(0xFFF43F5E),
  ),
),

Files Impacted

  • splash_screen.dart: Fixed missing import for kDebugMode.
  • marketing_ads_screen.dart: Verified smooth_page_indicator implementation.
