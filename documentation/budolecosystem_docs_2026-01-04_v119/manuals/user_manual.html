User Manual: Understanding Documentation Versioning

User Experience Version 119 | 2026-01-04

Purpose

To inform documentation users about the updated versioning system and how to navigate the documentation hub effectively.

Explanation

Documentation versions now follow a sequential build number (e.g., v117, v118, v119). This change makes it easier to track the progress of the BudolEcosystem and identify the most recent updates.

Issues & Fixes

Issue: Confusion between different versioning schemes.
Fix: Simplified the interface to show only the latest relevant build numbers in the navigation bar.

Technical Details

Users can navigate between versions using the "Previous Version" and "Next Version" links at the bottom of each page.

Risk & Mitigation

Risk: Users might get lost in deep version histories.
Mitigation: Added a "Main Index" link to all pages for quick navigation back to the root.

Recommendation for Future Enhancement

Add a search bar to the documentation hub to allow users to find specific implementation details across all versions.
