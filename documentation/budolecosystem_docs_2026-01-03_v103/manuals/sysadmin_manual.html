SysAdmin Manual: Infrastructure & Configuration

Service Updates

The following services require updates to support the new polling mechanism:

Service Port Required Change
payment-gateway-service 8004 Expose /payments/status/:referenceId endpoint.
api-gateway 8080 Ensure /payments route is correctly proxied to port 8004.

Environment Variables

No new environment variables are required for this update. Ensure BASE_URL is correctly set in payment-gateway-service to generate accurate redirect links.

Operational Runbook

Monitoring Polling Traffic

Admins should monitor the logs of payment-gateway-service for high frequencies of status checks. If traffic exceeds 100 requests per second per node, consider increasing the polling interval from 3s to 5s in index.js.

Rollback Procedure

  1. Revert services/payment-gateway-service/index.js to the version tagged v102.
  2. Restart the service: pm2 restart payment-gateway-service.
