Issue & Fix Log

Issue: BudolID seeding failed due to missing tables in budolshap_1db.

Cause: The service was attempting to use the 'public' schema which was occupied by BudolShap tables.

Fix: Updated BudolID .env to use schema=budolid and ran prisma db push.

Issue: SSO redirects failing on mobile devices in LAN.

Cause: Redirect URIs were hardcoded to localhost or old IP (192.168.1.24).

Fix: Implemented dynamic IP detection and updated seed script to use process.env.LOCAL_IP.
