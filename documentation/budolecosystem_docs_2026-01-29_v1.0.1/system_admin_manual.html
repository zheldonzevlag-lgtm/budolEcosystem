System Administration Guide v1.0.1

Startup & Management

To start the entire ecosystem in local mode with LAN support:

npm start

Select option 3 to skip environment override if already configured, or 1 for fresh local setup.

LAN Access Verification

  1. Identify host IP using ipconfig.
  2. Ensure budolID-0.1.0/.env has LOCAL_IP set correctly.
  3. Run npm run prisma:seed --prefix budolID-0.1.0 to sync redirect URIs.
Important: Port 8000 (SSO), 3001 (App), and 8080 (Gateway) must be allowed through the host firewall for LAN access.
