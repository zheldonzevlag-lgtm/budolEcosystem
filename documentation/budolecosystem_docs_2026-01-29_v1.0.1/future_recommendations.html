Future Enhancements & Recommendations

1. mDNS / Bonjour Integration

Implement budol.local hostname resolution using the bonjour-service to eliminate IP-based dependency entirely for local development.

2. Centralized Configuration Server

Transition from .env files to a central configuration service (like Spring Cloud Config or a custom BudolConfig) to push updates to all microservices in real-time.

3. SSL/TLS for LAN

Generate and distribute local CA certificates to enable HTTPS on 192.168.x.x addresses, improving security for cross-device testing.
