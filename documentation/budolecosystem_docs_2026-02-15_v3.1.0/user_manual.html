User Manual: Login Toggle & Email Auth

Step-by-step guide for using the restored login toggle and email authentication features in budolPay Mobile.

New Feature

1. Accessing the Login Method Toggle

On the login screen, you will now see a clear toggle at the top of the input area allowing you to switch between Mobile Number and Email Address login methods.

[UI Placeholder: Login Method Toggle with 'Mobile' and 'Email' options]
  • Tap Mobile to login using your registered phone number.
  • Tap Email to login using your email address and password.

2. Logging in with Mobile Number

When 'Mobile' is selected, enter your phone number in the input field. The system will automatically check if the number is registered and proceed to the next step (OTP verification).

[UI Placeholder: Mobile Input Field with Phone Icon]

3. Logging in with Email Address

When 'Email' is selected, two fields will appear: Email Address and Password.

[UI Placeholder: Email Address and Password Input Fields]
  • Enter your registered email address.
  • Enter your password. Use the eye icon 👁️ to toggle password visibility.
  • Tap the arrow button to sign in.

Security Tips

  • Never share your password or OTP with anyone, including Budol Ecosystem staff.
  • Ensure you are using the official budolPay mobile app.
  • Always logout if you are using a shared device.
