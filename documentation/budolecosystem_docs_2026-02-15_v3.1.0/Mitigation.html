Mitigation Strategies

  • R-310-1 (Credential Exposure): Use obscureText: true for password fields and ensure all communication is over HTTPS in production.
  • R-310-2 (Session Mismatch): Unified session storage using FlutterSecureStorage across both login methods.
  • R-310-3 (Input Collision): Separate controllers (_phoneController, _emailController) and explicit state clearing on toggle switch.
