User Manual - v54

Welcome to Budol Ecosystem v54

This update brings improved reliability to your payment experience and better synchronization for Administrator profiles.

New & Improved Features

NEW Robust Payment Redirection

Payments now correctly redirect back to BudolShap regardless of whether you are accessing the store via localhost or a local network IP address. This ensures a seamless "Return to Merchant" experience.

FIX Permanent Admin Profile Updates

Administrators can now update their display names and profile details without fear of them reverting on the next login. All changes are now synchronized across the entire ecosystem instantly.

IMPROVEMENT High-Contrast Login

The BudolID login button has been updated to a bold black color for better visibility and accessibility during the sign-in process.

Common Questions

Why did my payment fail when using my phone?

In previous versions, the system was locked to localhost. With v54, we have enabled dynamic IP support. If you still encounter issues, ensure your phone is on the same Wi-Fi network as the server.
