Developer Manual - v54

Purpose

To enable cross-network (LAN/IP-based) testing and ensure absolute financial precision across all microservices by standardizing URL resolution and amount parsing.

Key Technical Updates

1. Dynamic URL Resolution for Payment Gateways

To support testing on LAN IPs (e.g., 192.168.x.x), the payment-gateway-service now dynamically resolves the localShapUrl based on the incoming request host if an environment variable is not provided.

Fix: Replaced hardcoded localhost:3001 with request-host detection in payment-gateway-service/index.js. 
// Dynamic resolution in payment-gateway-service
const requestHost = req.headers.host;
let localShapUrl = process.env.BUDOLSHAP_URL;

if (!localShapUrl && requestHost) {
    const hostIp = requestHost.split(':')[0];
    localShapUrl = `http://${hostIp}:3001`;
}

2. Standardized Amount Precision

Following the v53 standardization, all financial microservices (Wallet, Transaction, Gateway) now strictly use Number() parsing and integer arithmetic for calculations to avoid floating-point drift.

// Transaction Service Standard
const amountCents = Math.round(Number(amount) * 100);
const feeCents = 1500; // PHP 15.00
const totalDeduct = (amountCents + feeCents) / 100;

3. SSO Bi-Directional Synchronization

Admin profile changes are now synchronized across the ecosystem. When an Admin updates their name in budolPay, the change is propagated back to budolID to prevent reversion on the next login.

Impact Analysis

  • Network: Developers can now test the full checkout flow from mobile devices or other computers on the same LAN using IP addresses.
  • Finance: Elimination of "off-by-one-centavo" errors in large transaction batches.
  • SSO: Consistent identity management across BudolShap and BudolPay Admin dashboards.

Recommendations for Future Enhancement

  • Service Discovery: Implement a lightweight service discovery mechanism to replace IP-based host detection.
  • Audit Logs: Add detailed ledger entries for every balance modification to facilitate easier financial reconciliation.
