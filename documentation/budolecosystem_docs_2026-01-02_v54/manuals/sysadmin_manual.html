SysAdmin Manual - v54

Infrastructure Overview

The Budol Ecosystem now supports dynamic host resolution for development and staging environments. This version focuses on port 8004 consolidation for payment services.

Environment Configuration

Ensure the following variables are set in your .env files to support LAN-based redirection:

Service Variable Recommended Value Description
budolPay Gateway BUDOLSHAP_URL http://[LAN-IP]:3001 Explicit override for the BudolShap frontend.
budolShap BUDOLPAY_GATEWAY_URL http://[LAN-IP]:8004 Ensures the frontend talks to the correct gateway port.

Critical Note: Port 8004 Migration

The payment intent service has been migrated from the generic API Gateway (8080) to the dedicated Payment Gateway Service (8004). All firewall rules and proxy configurations must allow traffic on this port.

Deployment Steps

  1. Update budolpay-0.1.0/services/payment-gateway-service/index.js with the dynamic URL logic.
  2. Restart all microservices using the ecosystem manager.
  3. Verify the /health endpoint on port 8004 is reachable.
