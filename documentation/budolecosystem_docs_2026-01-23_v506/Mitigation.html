Mitigation Strategy - v505

M-505-03: Host Format Validation

Risk: Incorrect host format entered by user (e.g., protocols, trailing slashes) causing API failures.

Mitigation: Implemented _normalizeHost function in api_service.dart to strip protocols and trailing slashes before saving to persistent storage.

M-504-03: COD Liquidity (Inherited)

Risk: Delays in crediting seller wallets for COD orders.

Mitigation: Automated credit trigger in updateOrderStatus upon DELIVERED status confirmation, backed by transaction logs for manual audit.
