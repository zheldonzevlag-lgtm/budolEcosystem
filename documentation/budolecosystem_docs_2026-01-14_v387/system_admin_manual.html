System Admin Manual - Budol Ecosystem v387

Infrastructure Update

This version requires no backend infrastructure changes but necessitates an update to the mobile app version 1.2.83 for all field testers.

Biometric Model Integrity

The system now enforces strict SHA256 validation. Admins should verify that the facenet.tflite model served via OTA or bundled in the APK matches the official hash: 54660297ebad23b7106a8ccd05f2f2f616b3d39bf7d7e82e148cf8ed7e27ae1a.

Troubleshooting Biometrics

If users report "Biometric Failure", check:

  • Device support for biometrics (Android 6.0+).
  • Correct model file presence in assets.
  • App permissions for CAMERA and USE_BIOMETRIC.
