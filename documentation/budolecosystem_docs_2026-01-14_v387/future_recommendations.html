Future Recommendations - Budol Ecosystem v387

Technical Debt & Enhancements

  • Git LFS Implementation: Moving large TFLite models to Git LFS to keep the main repository lightweight.
  • Biometric Encryption: Encrypting the stored face embeddings using device-specific keys from the Secure Enclave / Keystore.
  • Liveness Detection: Implementing 3D depth checks or motion-based liveness (blink, head turn) during the face login process.
  • Automated Testing: Adding unit tests for the SHA256 validation logic in FaceEmbeddingService.

Regulatory Alignment

As the biometric system matures, it is recommended to conduct a formal Privacy Impact Assessment (PIA) in accordance with NPC requirements to document the handling of sensitive biometric data.
