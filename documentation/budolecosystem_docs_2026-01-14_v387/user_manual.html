User Manual - Budol Ecosystem v387

Enhanced Login Experience

Version 1.2.83 introduces a cleaner biometric login prompt. When prompted for "Biometric login for budolPay", you can now use your fingerprint or face recognition as configured in your phone's settings.

Face Recognition Login

The custom face recognition (Selfie Login) has been improved for reliability. Ensure you are in a well-lit area and looking directly at the camera when using this feature.

Security Note

If biometric authentication fails multiple times, you will be prompted to enter your 6-digit PIN as a secure fallback.
