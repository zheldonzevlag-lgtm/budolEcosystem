SysAdmin Manual - v111

Release Artifacts

The production-ready installer for the Android application is located at:

D:\IT Projects\budolEcosystem\budolPayMobile\build\app\outputs\flutter-apk\app-release.apk

Installation Requirements

  • Android Version: Minimum API Level 23 (Android 6.0).
  • Permissions: Requires Camera (for QR scanning) and Internet access.

Deployment Environment

Ensure that the backend services (API Gateway, Auth Service, and Wallet Service) are properly configured and accessible from the mobile network. The app is currently configured with the following production URLs:

  • API Gateway: https://budolpay-api-gateway.vercel.app
  • Auth Service: https://budolid-auth.vercel.app
