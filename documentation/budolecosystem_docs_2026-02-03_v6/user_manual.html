User Manual: Monitoring Forensic Audit Logs

Accessing Audit Logs

Go to Settings > Audit Logs in the Admin Dashboard. This page provides a high-level view of all security events within the ecosystem.

Live Realtime Updates

The Audit Log page now features a Live status indicator. New events will appear at the top of the list automatically without requiring a page refresh.

Provider Status Badge

Next to the page title, you will see a status badge indicating the current realtime technology:

  • PUSHER: Low-latency live updates via Pusher.
  • SOCKET_IO: Live updates via internal socket server.
  • POLLING: Silent updates occurring every 10 seconds.

Audit Details

Each entry includes:

  • Timestamp: Precise time of the event (Manila Time).
  • User: Profile picture, name, and email of the actor.
  • Device Info: OS and device type (Mobile/Desktop).
  • Location: City, Country, and IP Address.
  • Map Link: Direct link to Google Maps if GPS data is available.

Searching and Filtering

Use the search bar at the top right to filter logs by user name, email, device, or IP address. Realtime updates will continue to stream in the background, but will only be displayed if they match your current search criteria.
