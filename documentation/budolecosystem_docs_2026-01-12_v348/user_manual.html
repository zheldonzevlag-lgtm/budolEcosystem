Login Process

  1. Open the budolPay app.
  2. In the "Mobile Number" field, enter your 11-digit number starting with 09.
  3. The app will automatically add hyphens for readability (e.g., 0917-XXX-XXXX).
  4. If the number is invalid, you will see an error message below the field.
  5. The "Continue" button will turn pink and become clickable only once a valid 11-digit number is entered.
