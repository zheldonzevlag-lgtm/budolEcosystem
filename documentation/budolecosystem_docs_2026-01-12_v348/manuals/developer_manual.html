1. Phone Input Formatting

A custom TextInputFormatter was implemented to automatically inject hyphens into the mobile number as the user types.

File: login_screen.dart

class PhoneInputFormatter extends TextInputFormatter {
  @override
  TextEditingValue formatEditUpdate(
    TextEditingValue oldValue,
    TextEditingValue newValue,
  ) {
    final text = newValue.text.replaceAll('-', '');
    if (text.length > 11) return oldValue;

    var newText = '';
    for (var i = 0; i < text.length; i++) {
      newText += text[i];
      if ((i == 3 || i == 6) && i != text.length - 1) {
        newText += '-';
      }
    }
    // ... selection logic
  }
}

2. Real-time Validation Logic

The _validatePhone method checks the input against Philippine mobile number standards (11 digits, starting with '09').

void _validatePhone(String value) {
  final digits = value.replaceAll('-', '');
  setState(() {
    if (digits.isEmpty) {
      _isPhoneValid = false;
      _phoneError = null;
    } else if (!digits.startsWith('09')) {
      _isPhoneValid = false;
      _phoneError = 'Must start with 09';
    } else if (digits.length < 11) {
      _isPhoneValid = false;
      _phoneError = 'Enter 11 digits';
    } else {
      _isPhoneValid = true;
      _phoneError = null;
    }
  });
}

5. Compliance Verification

  • NPC (Data Privacy): Mobile numbers are validated client-side to ensure only valid formats are transmitted. No sensitive data is logged.
  • Cybersecurity Law: Input sanitization via TextInputFormatter prevents injection of non-numeric characters at the entry point.
