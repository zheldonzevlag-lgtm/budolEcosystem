Version 348

Enhanced Phone Number Input with Real-time Validation and Hyphen Formatting.

Release Overview

This version focuses on improving the user experience during the login phase by implementing a structured phone number input field. It ensures data integrity by enforcing a 11-digit format starting with '09' and provides visual feedback to the user.
