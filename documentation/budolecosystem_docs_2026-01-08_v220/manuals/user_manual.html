User Manual: Enhanced Features & Fixes

1. Persistent Profile Updates

Admin users can now update their profile details (First Name, Last Name, and Account Type) in the BudolShap Admin Dashboard. These changes will now stick and will not be reverted when you log in via SSO.

2. Face Verification Flow

The Face Verification screen in the BudolPay mobile app is now fully functional. Users can upload their selfie for identity verification. Once uploaded, the account status will automatically transition to "FULLY_VERIFIED," unlocking higher transaction limits.

3. Sending Money (P2P)

If you encountered a "Verification required" error when sending money, ensure your account is upgraded. Fully verified users can now send money to other BudolPay users without restriction. If you still see an error, please contact support to verify your KYC tier.

Note: All error messages are now displayed in a clear, readable format instead of raw code or technical pages.
