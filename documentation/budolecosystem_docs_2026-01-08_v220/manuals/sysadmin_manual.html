SysAdmin Manual: Infrastructure & Data Maintenance

1. Port & Service Mapping

Ensure the following services are running and accessible on the local network:

Service Port Endpoint
API Gateway8080/api/*
Auth Service8001/api/auth/*
Transaction Service8003/api/transactions/*
Verification Service8006/api/verification/*
BudolID5000/users/*

2. Manual KYC Tier Upgrades

If a user is restricted by "BASIC" tier limits, use the upgrade-user-kyc.mjs script to elevate them:

node upgrade-user-kyc.mjs

Database Requirements: The user must have a valid record in the BudolPay PostgreSQL database linked to their mobile number.

3. Troubleshooting Middleware Issues

If req.body is reported as undefined in logs, check the initialization order in index.js. The express.json() parser MUST be initialized before routes are attached to the app or router.

4. Log Monitoring

Monitor logs for [Transaction] Transfer Error: Verification required. This indicates a user tier mismatch. Ensure users intending to use P2P transfers are FULLY_VERIFIED in the User table.
