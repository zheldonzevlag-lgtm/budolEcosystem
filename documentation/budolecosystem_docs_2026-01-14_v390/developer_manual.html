Developer Manual - v390

Face Login Enablement Logic Refinement

In version 390, we decoupled the face login availability from the device-wide biometric toggle. This was done to support users who have registered a custom face template via TFLite but might have disabled general biometrics in the app settings.

Key Changes in LoginScreen

The _checkBiometrics method now explicitly checks for a stored face template:


// [login_screen.dart](file:///d:/IT%20Projects/budolEcosystem/budolPayMobile/lib/screens/login_screen.dart#L55-L65)
Future<void> _checkBiometrics() async {
    final biometricService = context.read<BiometricService>();
    final available = await biometricService.isAvailable();
    final enabled = await biometricService.isEnabled();
    final faceTemplate = await biometricService.getStoredFaceTemplate();
    if (mounted) {
      setState(() {
        _isBiometricAvailable = available;
        _isBiometricEnabled = enabled;
        _hasFaceTemplate = faceTemplate != null;
      });
    }
}

The _handleFaceRecognitionLogin method now allows entry if _hasFaceTemplate is true, even if _isBiometricEnabled is false:


// [login_screen.dart](file:///d:/IT%20Projects/budolEcosystem/budolPayMobile/lib/screens/login_screen.dart#L104-L111)
Future<void> _handleFaceRecognitionLogin() async {
    final biometricService = context.read<BiometricService>();
    final faceService = context.read<FaceEmbeddingService>();
    
    // If face is registered, we allow it even if general biometric toggle is off
    if (!_isBiometricEnabled && !_hasFaceTemplate) {
      _showError('Face login is not enabled. Please login with your PIN once to enable it.');
      return;
    }
    ...
}

Version Synchronization

The app version has been incremented to 1.2.87 across the ecosystem to reflect these logic changes.

  • pubspec.yaml: version: 1.2.87+87
  • api_service.dart: _appVersion = '1.2.87'

A new test script test_face_login_v390.dart was created to validate this behavior using manual mocks for BiometricService and ApiService.
