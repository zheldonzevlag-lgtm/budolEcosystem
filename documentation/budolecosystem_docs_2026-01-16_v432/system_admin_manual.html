System Admin Manual - v432

Administrative Overview

Release v432 is a visual branding update. No database schema changes or backend service configuration updates are required for this release.

Configuration Changes

None. The branding is handled entirely within the mobile application frontend code.

Deployment Instructions

1. Ensure the latest version of budolPayMobile is built and deployed to the respective app stores or web servers.

2. Verify that the UIUtils class is correctly integrated into all transaction-related screens.

Monitoring

Monitor application logs for any rendering exceptions related to RichText or TextSpan. No significant impact on performance is expected.
