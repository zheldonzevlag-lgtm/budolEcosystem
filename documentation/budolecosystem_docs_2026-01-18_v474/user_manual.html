User Manual: Biometric Login

Version 474 makes it easier to use your device's security features to log in to budol₱ay.

What's New?

The login screen now intelligently shows only the security icons that your phone supports.

  • 🖐️ Fingerprint: Only shown if your phone has a fingerprint scanner.
  • 👤 Face Recognition: Only shown if your phone supports face unlock or if you've set up a custom Face ID in the app.

How to use Biometric Login:

  1. Enter your phone number as usual.
  2. On the PIN screen, look for the security icon next to the PIN box.
  3. Tap the icon to trigger your phone's security check.
  4. Once verified, you will be logged in instantly!
Tip: If the icons look gray or "faded", it means you haven't enabled Biometric Login in the Settings menu yet. Log in with your PIN once, go to Settings, and turn on "Biometric Login".
