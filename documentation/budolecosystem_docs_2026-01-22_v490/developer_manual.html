Developer Manual - Version 490

Order ID Transparency Implementation

Technical Rationale

Truncated order IDs (e.g., slice(0, 8)) caused ambiguity in support tickets and audit logs. By displaying the full UUID-based order ID, we ensure 100% precision in identification across all platforms (Buyer, Seller, Admin, and BudolPay).

Removed truncation in the customer list and order details modal.
