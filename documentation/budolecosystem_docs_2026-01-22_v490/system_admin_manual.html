System Admin Manual - v490

This version focuses on UI transparency and audit trail integrity.

Monitoring & Maintenance

  • Audit Logs: All order-related logs will now contain full order IDs. Ensure log aggregators are configured to handle full UUID strings.
  • Database: No schema changes were made. The id field remains a string/UUID.
  • Notifications: Real-time notifications for overdue shipping now broadcast the full order ID.

Support Troubleshooting

When receiving support requests, admins can now directly copy-paste the full order ID from any UI screen (Buyer, Seller, or Admin) to perform database queries or manual interventions.
