Release Overview - v490 (2026-01-22)

Release Date: January 22, 2026 | Version: 490

Primary Objective

Ensuring complete order identification transparency across all user interfaces by removing order ID truncation.

Key Changes

  • Removed slice() operations from order ID displays in Buyer Orders, Seller Dashboard, and Admin Portal.
  • Updated BudolShapShippingModal to show full order IDs.
  • Fixed notification payload in OverdueShippingSweep API to include full order numbers.
  • Standardized BudolPayText branding while maintaining data integrity.

Compliance Status

Verified: NPC (Data Privacy), Cybersecurity Law. Full ID display improves auditability and dispute resolution clarity.
