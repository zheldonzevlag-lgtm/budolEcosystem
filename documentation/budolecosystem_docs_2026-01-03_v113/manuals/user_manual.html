User Manual - Version 113

Installation

To install the app on your Android device:

  1. Transfer the budolPay 1.0.apk file to your phone.
  2. Open the file and allow installation from "Unknown Sources" if prompted.
  3. The app will appear as budolPay 1.0 with the dark Peso icon.

Key Features

  • Seamless SSO Login: Connect once to your budolID.
  • QR Payments: Instantly pay at budolPay merchants by scanning a code.
  • Live Balance: Monitor your wallet funds in real-time.

Enjoy using the official budolPay 1.0!
