System Admin Manual v407

Service Management

Admin dashboard and core services must be monitored for uptime. Use PM2 for process management in production environments.

  • Auth Service: Port 8001
  • Admin Dashboard: Port 3000
  • API Gateway: Port 8080

Forensic Audit Monitoring

System administrators should regularly review the Security section of the Admin Dashboard. The dashboard now updates in real-time, allowing for immediate forensic analysis of security events.

Critical events to monitor:

  • SECURITY_MOBILE_IDENTIFY_FAILED: Pattern may indicate reconnaissance or phishing.
  • SECURITY_MOBILE_LOGIN_PIN_FAILED: May indicate brute-force attempts.
  • SECURITY_LOGOUT: Confirms session termination and aids in tracking active sessions for compliance (BSP Circular 808).

Compliance Operations

Ensure that database backups are performed daily and audit logs are archived according to the 7-year retention policy mandated by local regulations.
