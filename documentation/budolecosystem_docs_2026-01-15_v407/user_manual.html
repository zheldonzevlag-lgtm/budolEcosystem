Mobile App Login

Users can log in using their registered mobile number. For security, the system will prompt for a 6-digit PIN on trusted devices.

  1. Enter your mobile number.
  2. If the device is trusted, enter your 6-digit PIN.
  3. If the device is new, you will receive an OTP for verification.

Biometric Authentication

Once logged in, users can enable Face Recognition or Fingerprint login for faster access. This can be toggled in the Security Settings of the app.

Security Transparency

Every login and logout attempt is recorded in our forensic audit trail to ensure the safety of your account. If you notice any unusual activity, please contact support immediately.

Admin Dashboard Access

Administrators can view real-time transaction data and security logs through the Admin portal. Ensure you use a secure connection and never share your credentials.
