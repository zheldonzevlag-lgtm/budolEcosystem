Release Overview

Purpose: This release addresses critical forensic audit log gaps in the mobile login flow, implements real-time provider-agnostic audit trail updates, and ensures session termination (logout) events are recorded for compliance.

Rationale: Compliance with BSP Circular 808, NPC, and PCI DSS requires real-time forensic visibility and immutable records of all authentication lifecycle events, including secure logouts.
