System Admin Manual - v426

Database Maintenance

Admin intervention was required to correct a user record in the User table. Ensure that DATABASE_URL in .env is correctly formatted for PostgreSQL.

Important: Always backup the database before performing manual updates on user records.

Environment Configuration

The system now supports automatic provider detection. Admins should no longer need to manually toggle provider flags in db-config.js.
