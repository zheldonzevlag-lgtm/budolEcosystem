User Manual: budolPay v1.3.73

Welcome to the latest version of budolPay Mobile. This update ensures seamless synchronization with the Vercel production environment.

1 Verify Version

Upon opening the app, the splash screen must display budolPay v1.3.73. If you see an older version, please reinstall the APK from the android-installer directory.

2 Login with Phone

Enter your 11-digit phone number (e.g., 09171234567). The app will automatically handle the +63 normalization for the Vercel backend.

3 OTP Verification (New Device)

If you are logging in from a new device, a 6-digit verification code will be sent to your registered Email and/or SMS. Enter this code to proceed.

4 Security PIN

Enter your 6-digit security PIN to unlock your wallet. If you have not set up a PIN, you will be prompted to create one immediately.

Important Security Note

Never share your OTP or PIN with anyone. BudolPay staff will never ask for these codes.
