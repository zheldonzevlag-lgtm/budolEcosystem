System Admin Manual: Mobile Production Build

1. Vercel Production Build

To generate the production APK pointing to Vercel, use the specialized PowerShell script:

cd budolPayMobile .\scripts\build_android_vercel.ps1

This script enforces flutter analyze and flutter test before building the APK with --dart-define=IS_VERCEL_BACKEND=true.

2. Environment Variables (Vercel)

The following variables must be configured in the Vercel dashboard for the admin app:

  • JWT_SECRET: Must match the ecosystem secret for shared auth.
  • DATABASE_URL: Postgres connection string (sin1 node).
  • SSO_URL: Point to the BudolID provider on Render.

3. Android Manifest Security

Production releases use android:networkSecurityConfig to block cleartext HTTP. Only HTTPS traffic to budolpay.vercel.app is permitted. Local IP ranges are exempted only in debug builds.

4. APK Distribution

The final APK is stored at:

d:\IT Projects\clone\budolEcosystem\android-installer\budolPay 1.3.73.apk
