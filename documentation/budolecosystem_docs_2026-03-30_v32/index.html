BudolPay v32 - Mobile Vercel Sync

Version: 32.0.0 Date: 2026-03-30 IN PROGRESS / STABILIZED

Purpose & Rationale

This version bridges the gap between the local Express-based microservices and the Vercel-based Next.js production environment. It resolves the "Account not found" 404 error on Vercel by porting the critical mobile authentication endpoints to the Next.js App Router.
