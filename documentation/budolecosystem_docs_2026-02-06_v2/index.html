Release: 2026-02-06 v2 | BudolShap UI Optimization

Release Summary

This release addresses visibility and responsiveness issues in the registration modal of BudolShap. It implements responsive constraints (max-height, scrollability) to ensure the full registration form is accessible across all device sizes, including camera capture and selfie preview components. This ensures compliance with NPC (National Privacy Commission) guidelines for clear UI interactions during identity verification.
