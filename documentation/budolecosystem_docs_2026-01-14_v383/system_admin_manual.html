System Admin Manual - v383

Release Management

This release introduces the 1.2.80 Android build. Admins should ensure the APK is distributed to the correct channels (Internal Testers / Google Play Console).

Deployment Path: All mobile installers must be stored in \budolEcosystem\android-installer with the naming convention budolPay N.N.N.apk.

Installation Steps

  1. Download the APK from the installer directory.
  2. Enable "Install from Unknown Sources" on the target Android device.
  3. Run the installer and verify the version 1.2.80 on the splash screen.

Troubleshooting with Debug Window

In v383, the Face Service Debug Window has been made full-screen. To access and use it:

  1. Navigate to the Settings screen in the app.
  2. Toggle the Show Debug Logs switch.
  3. The full-screen log window will appear, displaying real-time events from the TFLite interpreter and camera services.
  4. Error Detection: Logs highlighted in RED indicate critical failures (e.g., TFLite initialization errors).
