Developer Manual - v65

1. Branding Identity Update

The app logo and text branding have been updated to align with the new visual identity.

1.1 Philippine Peso Logo

The primary logo in login_screen.dart was changed from a generic wallet icon to a stylized Philippine Peso (₱) sign.

// Implementation in login_screen.dart
Container(
  padding: const EdgeInsets.all(20),
  decoration: BoxDecoration(
    color: const Color(0xFFF43F5E).withOpacity(0.1),
    shape: BoxShape.circle,
  ),
  child: const Text(
    '₱',
    style: TextStyle(
      fontSize: 80,
      fontWeight: FontWeight.bold,
      color: Color(0xFFF43F5E),
    ),
  ),
)

1.2 Brand Text Styling

The "budolPay" text now uses a split color scheme for better contrast and brand recognition:

  • budol: Slate 400 (#94A3B8)
  • Pay: Rose Red (#F43F5E)
// Implementation using RichText
RichText(
  text: TextSpan(
    style: const TextStyle(fontSize: 32, fontWeight: FontWeight.bold),
    children: [
      TextSpan(text: 'budol', style: TextStyle(color: Color(0xFF94A3B8))),
      TextSpan(text: 'Pay', style: TextStyle(color: Color(0xFFF43F5E))),
    ],
  ),
)

2. Web Preview Container Adjustments

The web preview container dimensions were adjusted in main.dart to better reflect the proportions of the iPhone 17 Pro.

  • Width: 393px (Standardized Pro Width)
  • Height: 852px (Standardized Pro Height)

These dimensions ensure the app doesn't appear "too long" on desktop displays while maintaining a mobile aspect ratio.
