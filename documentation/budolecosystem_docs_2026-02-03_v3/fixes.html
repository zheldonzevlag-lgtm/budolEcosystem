Issues and Fixes

Issue: Account Creation TimeoutException

Symptoms: Users see a red snackbar with TimeoutException when attempting to register.

Cause: Hardcoded 10-second timeout in ApiService.dart was insufficient for the full registration lifecycle (database + notification sending).

Fix: Increased Timeout to 30s

Updated all auth methods in ApiService.dart to use .timeout(const Duration(seconds: 30)).

Issue: Pusher Configuration Missing

Symptoms: Gateway logs showed [Pusher] Configuration missing.

Cause: Gateway was looking for PUSHER_APP_ID instead of the REALTIME_PUSHER_APP_ID stored in the database.

Fix: Updated Gateway Key Lookup

Modified api-gateway/index.js to prioritize REALTIME_PUSHER_* keys.
