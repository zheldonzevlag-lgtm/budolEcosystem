v1.3.7

Release Overview: KYC Standards Alignment

Version 1.3.7 marks a significant milestone in aligning the BudolShap ecosystem with industry-leading e-commerce standards (Shopee model) and Philippine regulatory requirements (NPC, BSP, BIR).

Key Improvements

  • Tiered Seller Architecture: Differentiated data models for Individual and Business sellers.
  • Real-time OCR Integration: Automated data extraction using google_mlkit_text_recognition for KYC documents.
  • KYC Data Contracts: Defined structured JSON schemas for multi-document verification.
  • Regulatory Alignment: Built-in infrastructure for TIN and Business Registration tracking.
  • OTP Flow Continuity: Fully integrated the v1.3.6 registration-to-verification UX optimization.
  • Proactive KYC Visibility: Integrated account verification status and submission links into the primary Navbar/Profile menu for both desktop and mobile views.
