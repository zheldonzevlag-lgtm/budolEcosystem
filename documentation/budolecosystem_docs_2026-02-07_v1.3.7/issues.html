Known Issues: v1.3.7

1. KYC: Document Upload UI

The current schema supports Individual/Business tiers, but the frontend UI for uploading multiple business documents (BIR 2303, DTI/SEC) is still in development for Phase 2.

2. Auth: OTP Rate Limiting

Aggressive OTP requests from the same IP/Phone number currently rely on basic backend throttling. More granular rate limiting based on device fingerprinting is recommended for v1.3.8.

3. UI: Mobile Sidebar Sync

The new "Comparative Analysis" link in the documentation sidebar may require manual cache clearing in some mobile browsers to appear correctly.
