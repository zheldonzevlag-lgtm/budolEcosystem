HTTPS & SSL Configuration

All ecosystem traffic is now routed through Port 443 with SSL termination at the Application Load Balancer.

Secrets Management

ECS Tasks retrieve sensitive data directly from AWS Secrets Manager. Note that Full ARNs are now used in task definitions to ensure reliable resolution across regions.

Service Startup Fixes

To address startup crashes in standalone environments:

  1. Redis Bypass: Since internal Redis is not yet provisioned, REDIS_URL is set to redis://127.0.0.1 to prevent connection timeouts during initialization.
  2. Hostname Resolution: HOSTNAME is set to 0.0.0.0 to allow Next.js and Node.js services to bind correctly without resolving AWS internal Task IDs.
  3. Telemetry: NEXT_TELEMETRY_DISABLED=1 is set for performance and to avoid external DNS calls.
