Executive Summary

This release fixes a critical UX blocker where users were unable to set up Face Recognition login even after a successful PIN-based authentication. The root cause was identified as the PIN not being persisted to secure storage unless the user explicitly enabled general biometrics during the login phase. This version introduces an on-demand PIN verification dialog within the Settings screen to bridge this gap securely.
