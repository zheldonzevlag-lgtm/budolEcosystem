Developer Manual

Technical Explanation

The Reference ID (Payment Intent ID) was added to the success and failure screens on the payment return page. This was achieved by extracting the payment_intent_id from the URL search parameters and displaying it using the BudolPayText component for consistent styling.

Code Changes

File: app/payment/return/page.jsx

Modified the ReturnContent component to display intentId. Also added break-all and text-right classes to handle long IDs gracefully.

// Success State
<div className="flex justify-between mb-2">
    <span className="text-slate-600">Order ID:</span>
    <span className="font-medium text-slate-800 text-right break-all ml-4"><BudolPayText text={order.id} /></span>
</div>
<div className="flex justify-between mb-2">
    <span className="text-slate-600">Reference ID:</span>
    <span className="font-medium text-slate-800 text-right break-all ml-4"><BudolPayText text={intentId} /></span>
</div>
