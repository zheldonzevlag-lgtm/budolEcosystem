System Admin Manual - v389

1. Deployment

Ensure the Android APK v1.2.85 is distributed via the internal installer path: \budolEcosystem\android-installer\budolPay_1.2.85.apk.

2. Troubleshooting

If users report "No face profile found" even after registration:

  • Check if the user has cleared app data (which deletes local secure storage).
  • Verify the device has a working front camera.
  • Check the "Face Registration Debug" logs in the app's Settings screen.
