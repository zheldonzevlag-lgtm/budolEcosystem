Mitigation Plan - v389

Mitigation for Custom Face Login Security

Strategy: Multi-Layered Validation

  • Liveness Check: ML Kit Face Detection ensures the user is a real person (checking for eye blinking and expressions).
  • High Similarity Threshold: FaceNet comparison uses a 0.8 threshold, which is statistically significant for 512-dimensional embeddings.
  • Local-Only Storage: Templates never leave the device, mitigating server-side breach risks.
