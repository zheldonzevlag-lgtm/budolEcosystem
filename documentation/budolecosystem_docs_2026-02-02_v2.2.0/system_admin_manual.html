System Admin Manual: Configuring Ecosystem Parameters

Introduction

The Budol Ecosystem administrative dashboard provides a unified interface for managing environment-wide settings. With v2.2.0, these settings are partitioned into functional groups to improve administrative efficiency.

1 Accessing Settings

Navigate to the "System Settings" item in the sidebar. You will now see sub-links for specific configuration domains.

2 Configuring Realtime Services

Under "Realtime Settings", you can toggle between Pusher and Socket.io. Ensure that the PUSHER_APP_ID and Cluster are correctly set for production environments to avoid transaction status delay.

3 Managing Notifications

The "Notification Settings" panel allows you to configure Resend for emails and Twilio for SMS. These services are critical for OTP delivery and account security alerts.

4 Security Configuration

Use the "Security Config" section to manage session lifetimes and account lockout policies. Modifications here are logged immediately to the audit trail for compliance purposes.
