User Manual - Admin Settings

Managing System Settings

The Admin Dashboard now features a grouped settings interface for easier navigation and management of system-wide configurations.

1. Accessing Settings

Log in to the BudolPay Admin Dashboard and click on System Settings in the left sidebar. You will see new submenus:

  • Environment Variables: Core system paths and IPs.
  • Payment Gateways: API keys for Maya, GCash, etc.
  • Realtime Settings: Pusher/WebSocket configuration.
  • Notification Settings: Email (Resend) and SMS (Twilio) keys.
  • Security Config: Session timeouts and login attempt limits.

2. Updating a Setting

Scroll to the desired section or click the submenu link. Update the value in the input field. For security reasons, sensitive keys (secrets) are masked with dots. Click Update to save changes.

3. Realtime Method Selection

In the Realtime Settings section, you can choose the communication method (Pusher, Socket.io, or Polling) from a dropdown menu. This takes effect immediately for all connected clients.

Safety Precautions

  • Validation: The system validates input formats (e.g., numeric values for timeouts).
  • Audit Logs: All changes are recorded with the timestamp and the administrator's ID.
  • Rollback: If a setting causes instability, refer to the Audit Logs to revert to previous values.
