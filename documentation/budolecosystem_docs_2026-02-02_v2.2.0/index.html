Release v2.2.0: Unified Settings Navigation

Release Date: 2026-02-02 | Version: 2.2.0

Overview

This release introduces a hierarchical navigation system for administrative settings, specifically addressing the missing "Realtime" and "Notification" configuration panels. The settings engine now supports logical grouping, allowing for a cleaner and more scalable administrative interface.

Core Changes

  • Grouped Settings Engine: Refactored the settings backend to group parameters by functional domain (System, Payment, Realtime, Notification, Security).
  • Dynamic Sidebar Submenus: Added specialized sub-links for Realtime and Notification settings with automatic active-state tracking.
  • Anchor-Based Navigation: Implemented smooth-scroll anchor links for quick access to specific settings blocks.
  • New Configuration Domains:
    • Realtime Settings: Manage Pusher, Socket.io, and polling intervals.
    • Notification Settings: Configure Resend (Email) and Twilio (SMS) integration.
    • Security Config: Manage session timeouts, login attempts, and JWT secrets.

Compliance Verification

NPC Compliance: Navigation sub-links are purely UI-based and respect underlying API permission checks. Sensitive keys remain masked (isSecret: true).

BSP/PCI DSS: Enhanced visibility of security settings (session timeouts, lockouts) allows admins to quickly verify compliance parameters.
