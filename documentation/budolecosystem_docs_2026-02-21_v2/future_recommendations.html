Future Recommendations & Enhancements

Version: 2.0 | Date: 2026-02-21

1. Automated Database Backups

Priority: High

Currently, backups are manual or script-triggered. Implement automated daily backups to cloud storage (e.g., AWS S3, Google Cloud Storage) to ensure disaster recovery.

2. Rate Limiting for Image Uploads

Priority: Medium

While basic rate limiting exists for login, image uploads via Cloudinary or base64 fallback should be rate-limited per user to prevent abuse and storage costs.

3. Image Optimization Pipeline

Priority: Low

Implement a server-side image optimization pipeline (e.g., using Sharp) to further compress images before storage if client-side compression is bypassed or insufficient.

4. Monitoring Dashboard

Priority: Medium

Create an admin dashboard to visualize system health, including database size, active users, and error rates from Sentry.

5. End-to-End Testing Suite

Priority: High

Expand the current Jest unit tests to include end-to-end tests using Cypress or Playwright to simulate user flows like checkout and product management.
