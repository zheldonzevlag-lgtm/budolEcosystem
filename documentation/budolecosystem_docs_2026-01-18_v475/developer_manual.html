Settings Biometric Implementation

The biometric and face recognition settings are managed within the SettingsScreen component. This implementation ensures that users can selectively enable or disable authentication methods based on hardware availability.

Core Logic: Biometric Toggle

Toggling the biometric switch updates the app's internal state and persists the preference to secure storage.

[settings_screen.dart:L82-99]
Future<void> _toggleBiometrics(bool value) async {
  final biometricService = context.read<BiometricService>();
  if (value) {
    await biometricService.setEnabled(true);
    // ... show snackbar
  } else {
    await biometricService.setEnabled(false);
    await biometricService.clearStoredPin();
  }
}

Service Layer

The BiometricService acts as the abstraction layer for both OS-level biometrics (LocalAuth) and custom face templates (SecureStorage).
