Custom Painter Implementation

The QRScannerOverlayPainter handles the heavy lifting of drawing the dark overlay, clear cutout, and neon corner accents.

Code Reference: qr_scanner_screen.dart


// Drawing the corners
canvas.drawPath(
  Path()
    ..moveTo(left, top + cornerSize)
    ..lineTo(left, top + radius)
    ..arcToPoint(Offset(left + radius, top), radius: const Radius.circular(radius))
    ..lineTo(left + cornerSize, top),
  cornerPaint,
);
