v39: QR Scanner Enhancement

Compliance: PCI-DSS / BSP Circular 808

Release Date: 2026-04-02

Purpose

This release upgrades the QR Code Scanning experience in BudolPay with a high-fidelity "techno-style" UI. The goal is to improve scanning accuracy by providing clear guided frames and adding essential utility features like torch control and gallery scanning.

Key Enhancements

  • Techno Overlay: Custom-painted square frame with neon corner accents.
  • Laser Animation: Real-time animated vertical scanning line for visual feedback.
  • Flashlight Support: Integrated torch toggle for low-light environments.
  • Gallery Scanning: Support for analyzing QR codes from saved images.
  • Haptic Feedback: Medium-impact vibration upon successful barcode detection.
