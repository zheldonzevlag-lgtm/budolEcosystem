User Manual - SSO Login (v1.0.3)

Logging in with budolID

To access any application within the Budol Ecosystem, follow these steps:

  1. Navigate to the application URL (e.g., BudolShap).
  2. You will be redirected to the budolID login page.
  3. Enter your registered email and password.
  4. Upon successful authentication, you will be redirected back to the original application.
