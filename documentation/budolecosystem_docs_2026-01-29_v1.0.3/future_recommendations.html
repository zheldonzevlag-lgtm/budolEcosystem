Future Enhancements & Recommendations (v1.0.3)

Technical Improvements

  • Auto-IP Discovery: Implement a script to automatically detect the host's LAN IP and update .env and database seed automatically on startup.
  • Refresh Token Flow: Add OAuth2 standard refresh tokens to extend sessions without requiring full re-login.
  • Service Health Dashboard: Integrate a real-time monitoring dashboard in start-ecosystem.js to show the status of all 12 services.
  • Dockerization: Move services to Docker containers with internal networking to eliminate host port/IP conflicts.

Security Enhancements

  • MFA Integration: Add TOTP-based Multi-Factor Authentication to the BudolID SSO flow.
  • Rate Limiting: Implement middleware to prevent brute-force attacks on the SSO login form.
  • Audit Logs: Create a dedicated service for centralized audit logging across the entire ecosystem.
