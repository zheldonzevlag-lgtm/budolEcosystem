System Admin Manual - v540

Service Management

The Budol Ecosystem is managed via a central runner script. To start all services:

npm start

Select option 3 (SKIP) to use existing environment variables, or 1 (LOCAL) for fresh local setup.

Database Maintenance

If orders are reported missing, run the database verification script:

node scripts/test_scripts/budolShap/v2/verify-orders-db.mjs

Troubleshooting Webhooks

Monitor the logs of the budolShap application for webhook delivery failures. The endpoint for BudolPay webhooks is:

/api/webhooks/budolpay

Configuration

Ensure LOCAL_IP is correctly set in the root .env file to allow cross-service communication in a LAN environment.
