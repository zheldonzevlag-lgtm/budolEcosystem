Slate Branding Implementation

The requirement was to change the "budol" prefix color to slate while keeping the service suffix (Loan, Shap, Express) in its thematic color. This was achieved by updating the RichText widget in marketing_ads_screen.dart.

Technical Details

The RichText widget now uses an explicit TextStyle for the "budol" TextSpan, overriding the default white color with Slate 400 (#94A3B8).

Code Changes

In marketing_ads_screen.dart, the _buildSlide method was modified:


// lib/screens/marketing_ads_screen.dart

RichText(
  text: TextSpan(
    style: const TextStyle(
      color: Colors.white,
      fontSize: 32,
      fontWeight: FontWeight.bold,
      letterSpacing: -0.5,
    ),
    children: [
      TextSpan(
        text: slide['title'].toString().startsWith('budol') ? 'budol' : slide['title'].toString(),
        style: const TextStyle(color: Color(0xFF94A3B8)), // Slate 400
      ),
      if (slide['title'].toString().startsWith('budol'))
        TextSpan(
          text: slide['title'].toString().substring(5),
          style: TextStyle(color: slide['color']),
        ),
    ],
  ),
),

Impact Analysis

  • Visual Hierarchy: The use of slate for "budol" creates a more sophisticated look and emphasizes the specific service name (e.g., "Loan") which remains in the theme color.
  • Consistency: This change is applied across all slides in the marketing carousel.
