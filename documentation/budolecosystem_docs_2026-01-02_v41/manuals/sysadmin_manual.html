System Admin Manual: v41 UI Deployment

Version: v41.0.0 | Date: 2026-01-02

Deployment Details

This update modifies the frontend template served by the budolID service.

Verification Steps

  1. Access http://[SERVER_IP]:8000/login.
  2. Open Developer Tools (F12) and toggle "Device Toolbar" to simulate mobile viewports (e.g., iPhone SE, Pixel 5).
  3. Confirm the card width is constrained properly and the shield icon appears within a light blue circle.
Service Status: The ecosystem runner automatically detects changes if using nodemon (dev mode). If in production mode, restart the service manually.
