System Admin Manual: Notification Providers

Version: 2026-01-01_v38

Configuration Interface

Admins can now manage Email and SMS settings via Admin Dashboard > Settings > Notifications.

Email Providers

1. Google Mail / SMTP Default

  • SMTP Host: smtp.gmail.com
  • SMTP Port: 587
  • Auth: Use App Passwords for Gmail accounts.

2. Brevo (Sendinblue)

  • API Key: Found in Brevo SMTP & API settings.
  • Sender Email: Must be a validated sender in Brevo.

SMS Providers

1. Console Log Development

Logs all SMS content to the server terminal. Useful for testing without cost.

2. Zerix / iTextMo / Viber

Require valid API Keys and Client Codes from the respective providers.

Migration & Fallbacks

If a provider is selected but credentials are missing, the system will attempt to fallback to environment variables (SMTP_USER, SMTP_PASS) or log to the console as a last resort.

Verification

After updating settings, trigger a "Resend Verification" from any user profile to verify the new provider configuration is active and working correctly.
