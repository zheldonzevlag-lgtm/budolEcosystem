User Manual: Managing Notification Providers (Admin)

Version: 2026-01-01_v38

Overview

As an administrator, you can now control how the system sends emails and SMS messages. This allows you to switch between providers if one service goes down or if you find a more cost-effective option.

1. Accessing Settings

Go to the Admin Dashboard, click on Settings in the sidebar, and select Notifications.

2. Switching Email Providers

Choose between "Google Mail / SMTP" or "Brevo". Enter the required credentials like SMTP server details or API keys. Click Save Changes to apply.

3. Switching SMS Providers

Select your preferred SMS provider (Zerix, iTextMo, or Viber). Ensure you have a valid account and API key from the provider. For development or testing, use "Console Log" to see messages in the server output.

4. Saving and Verifying

Always click the Save Changes button at the top right after making adjustments. The system will immediately begin using the new provider for all outgoing OTPs and notifications.
