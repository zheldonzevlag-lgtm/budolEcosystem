Release Summary

Version 501 focuses on resolving the 1-minute latency issue in order status synchronization between Lalamove webhooks and the Budol Ecosystem UI. By optimizing real-time event triggering and implementing an intelligent caching strategy for system settings, we have achieved near-instantaneous UI updates for both buyers and sellers.
