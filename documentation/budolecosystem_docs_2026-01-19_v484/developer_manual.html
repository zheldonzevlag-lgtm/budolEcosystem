Developer Manual v484

Technical Explanation

Version v484 addresses the breaking changes in flutter_secure_storage: 9.2.4 and enhances the stability of web deployments by implementing a manual fallback to SharedPreferences when Web Crypto APIs are unavailable. Additionally, it migrates all deprecated Flutter 3.x members (withOpacity, dart:js) to modern standards (withValues, dart:js_interop) and improves async context safety.

3. Asia/Manila Timezone Compliance

Standardized date and time formatting using a centralized DateUtils helper in formatters.dart. This ensures all transaction timestamps are displayed in UTC+8, regardless of the user's device settings.

// formatters.dart
class DateUtils {
  static DateTime toManila(DateTime dateTime) {
    return dateTime.toUtc().add(const Duration(hours: 8));
  }

  static String formatDateTime(DateTime dateTime) {
    return DateFormat('MMM dd, yyyy hh:mm a').format(toManila(dateTime));
  }

  static String formatTimeOnly(DateTime dateTime) {
    return DateFormat('hh:mm a').format(toManila(dateTime));
  }
}

Applied to transaction_details_screen.dart, home_screen.dart, and transaction_history_screen.dart.

4. Flutter 3.x UI Modernization

Migrated all instances of withOpacity to the new withValues(alpha: ...) API to resolve deprecation warnings and future-proof the UI components.

2. Robust Secure Storage Fallback & JS Interop

Improved isWebCryptoSupported() in js_helper_web.dart by migrating from deprecated dart:js to modern dart:js_interop. It now checks for isSecureContext using type-safe JS interop.

// js_helper_web.dart (Modern dart:js_interop implementation)
bool isWebCryptoSupported() {
  try {
    final JSAny? isSecureContext = _window.getProperty('isSecureContext'.toJS);
    final bool isSecure = isSecureContext != null && (isSecureContext as JSBoolean).toDart;
    
    final JSAny? crypto = _window.getProperty('crypto'.toJS);
    bool hasSubtle = false;
    if (crypto != null && crypto is JSObject) {
      final JSAny? subtle = crypto.getProperty('subtle'.toJS);
      hasSubtle = subtle != null;
    }
    return isSecure && hasSubtle;
  } catch (e) {
    return false;
  }
}

Implemented _readSecure, _writeSecure, and _deleteSecure helpers in api_service.dart and biometric_service.dart.

// api_service.dart
Future<String?> _readSecure(String key) async {
  try {
    if (kIsWeb && !isWebCryptoSupported()) {
      final prefs = await SharedPreferences.getInstance();
      return prefs.getString('fallback_$key');
    }
    return await _secureStorage.read(key: key);
  } catch (e) {
    return null;
  }
}
