System Overview v484

Release Date: 2026-01-19

Purpose & Rationale

This release (v484) addresses critical Flutter Web Secure Storage issues, timezone accuracy, and prepares the official Android build v1.3.21.

  • Secure Storage Fallback: Implemented a robust fallback mechanism that automatically switches to SharedPreferences when running in non-secure web contexts.
  • Manila Time Compliance: Standardized all transaction time displays to Asia/Manila (UTC+8) across the Home, History, and Details screens.
  • Android Release v1.3.21: Successfully built and deployed the official Android installer with the latest stability and security fixes.
  • Code Hygiene: Resolved all Flutter 3.x deprecations (withOpacity → withValues) and context safety warnings.
