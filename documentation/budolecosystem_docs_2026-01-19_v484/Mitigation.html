Mitigation Strategy v484

M-484-01: Manual Secure Storage Fallback

To mitigate the loss of encrypt parameter in flutter_secure_storage: 9.2.4, we implemented a manual fallback to SharedPreferences. This ensures that even in non-secure web contexts (where Web Crypto is unavailable), the app can still persist session data without crashing.

M-484-02: Secure Context Validation

Improved the isWebCryptoSupported() check to explicitly validate isSecureContext. This provides a more reliable detection of environment capabilities than simply checking for the presence of crypto.subtle.
