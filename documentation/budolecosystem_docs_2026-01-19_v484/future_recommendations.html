Future Roadmap v484

Short-term Enhancements

  • Standardized API Response Timezone: Ensure the backend API also returns ISO strings with explicit +08:00 offset to further simplify client-side parsing.
  • Secure Context Enforcement: Implement a mandatory redirect to HTTPS for production web instances to ensure encryption is always active.

Long-term Strategy

  • Offline Repayment: Explore local queueing of payment intent for BudolPay when the device is offline, with automatic sync when connectivity returns.
  • Universal Biometric Bridge: Develop a custom JS bridge that provides a unified biometric API across different browser engines, reducing dependency on standard crypto.subtle for non-critical identity checks.
