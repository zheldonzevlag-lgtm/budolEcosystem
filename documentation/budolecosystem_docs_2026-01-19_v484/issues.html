Known Issues v484

I-484-01: SharedPreferences Fallback Persistence

The fallback storage uses SharedPreferences which is not encrypted. On non-secure web origins, session data is stored in plain text in the browser's localStorage. This is a trade-off for usability in non-production environments.

Status: Risk accepted for development/staging.

I-484-02: Deprecated dart:js Usage

The js_helper_web.dart still uses the deprecated dart:js library. While functional, it should eventually be migrated to dart:js_interop to future-proof the application.

Status: Planned for v485+.
