Release Notes - v1.4.0

v1.4.0

Release Date: 2026-02-09

Title: Admin-Only OCR Transparency & Security Hardening

Purpose

This release addresses privacy concerns regarding the visibility of AI-extracted data during the KYC process. It ensures that sensitive OCR results are only visible to authorized KYC approvers (admins), while providing users with transparent feedback about the confidential processing of their data.

Key Changes

  • Frontend Hardening: Hidden the "AI Extraction Results" section in the KYCWizard for regular users.
  • API Security: Implemented server-side data masking in the KYC status route to prevent unauthorized access to raw OCR data.
  • Admin Visibility: Updated admin utilities to ensure KYC approvers retain full visibility of extracted data for verification purposes.
  • Compliance: Enhanced alignment with NPC (National Privacy Commission) and BSP standards for data minimization and role-based access control.

Compliance Status

PCI DSS BSP NPC BIR Cybersecurity Law
