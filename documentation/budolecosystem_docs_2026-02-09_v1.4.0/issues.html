Known Issues - v1.4.0

  • Mock Testing Complexity: Testing the full async file upload flow in KYCWizard.test.jsx requires complex mocks for Tesseract.js and Canvas API in JSDOM, leading to simplified test coverage for the visibility logic.
  • Data Masking Consistency: While the API masks `fullName` and `idNumber`, other custom fields added in future KYC tiers may require manual updates to the masking logic in app/api/user/kyc/route.js.
