Executive Summary

Version 420 addresses a UI redundancy issue on the BudolShap login page where two "Login with budolID" buttons were visible. This was caused by the button being present in both the AuthForm component and the LoginPage wrapper. The redundant button was removed from the page wrapper to maintain component consistency and improve the user experience.

Key Fixes

  • UI Cleanup: Removed the duplicate SSO button from the LoginPage in BudolShap.
  • Logic Synchronization: Updated the SSO button in AuthForm to include the redirect_uri parameter, ensuring the SSO flow returns to the application correctly.
  • Component Integrity: Ensured that AuthForm remains the single source of truth for authentication UI elements.
