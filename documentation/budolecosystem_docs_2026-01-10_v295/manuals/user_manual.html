User Manual - v295

Accessing the System via LAN

With the improvements in v295, you can now access the budolEcosystem more reliably from other devices on the same network (e.g., your mobile phone or another laptop).

Steps to Access

  1. Ensure the host machine has started the services (npm start).
  2. On your secondary device, open a web browser.
  3. Enter the LAN URL provided by the administrator (e.g., http://192.168.1.24:3001).
  4. The login and redirection process should now work seamlessly without redirecting you back to localhost.

Known Issues & Solutions

  • Login Loop: If you are redirected to 127.0.0.1 when logging in from a remote device, please ask the administrator to run the network:sync script.
  • Service Unavailable: Ensure both the device and the host are on the same Wi-Fi network.
