Budol Ecosystem Release v352

Post-Logout Session Leakage Prevention & Navigation Integrity

Release Overview

This release (v352) addresses a critical UI bug where the application would attempt to fetch the user's wallet balance immediately after a successful logout, resulting in an "unauthorized" or "session expired" error snackbar appearing on the login screen. This was caused by race conditions during navigation and active polling timers in the HomeScreen and WalletScreen.

Key Improvements

  • Authentication Guarding: Implemented explicit isAuthenticated checks in all data-fetching methods (_fetchBalance, _fetchTransactions) across core screens.
  • Navigation Integrity: Prevented background data refreshes when the user is transitioning away from authenticated states.
  • UI Stability: Eliminated the jarring "Session expired" snackbar that previously appeared after intentional logouts.
  • Version Sync: Incremented app version to 1.2.58.
