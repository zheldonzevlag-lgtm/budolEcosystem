Dependencies & Technology Choice - Phase 8

Core Libraries

Library Purpose Why Chosen?
react-hook-form Form State Management Performance (minimizes re-renders) and easy integration with validation libraries like Zod.
zod Schema Validation Type-safety and ease of defining complex validation rules (e.g., matching passwords, checking array lengths).
@dnd-kit/core Drag & Drop Interactions Modern, accessible, and lightweight compared to older libraries like react-beautiful-dnd. Supports touch devices well.
framer-motion Animations Declarative animations for the wizard steps make the UI feel smoother and more polished.
react-hot-toast Notifications Simple, lightweight, and customizable toast notifications for user feedback.
next/dynamic Dynamic Imports Used for CKEditor to prevent Server-Side Rendering (SSR) issues, as it relies on `window` object.
