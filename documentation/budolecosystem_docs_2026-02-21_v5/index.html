Documentation Index - v5.0

Date: 2026-02-21

Version: 5.0

Focus: Product Creation Wizard, Image Handling, Testing Framework

Summary of Changes

  • Completed implementation of AddProductWizard with step-based navigation.
  • Refactored DragDropImageUpload for stable sorting using unique IDs.
  • Fixed accessibility issues in form fields (labels associated with inputs).
  • Resolved Jest configuration conflicts and added unit tests for the wizard.
  • Integrated store association (storeId) into product creation flow.
  • Fix: Enabled persistence for Shipping Dimensions, Condition, and Pre-Order fields.
  • Schema: Added preOrder boolean field to Product model.
  • UI: Implemented scrollable product image gallery (limit 4) with floating navigation buttons.
  • UX: Improved product draft handling - added "Restore Draft" button instead of auto-restore.
  • AI: Added "Magic Write" button for auto-generating product descriptions.
