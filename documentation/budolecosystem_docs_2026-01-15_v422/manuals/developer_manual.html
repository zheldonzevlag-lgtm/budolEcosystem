Technical Rationale

The absence of the Admin menu was traced to an inconsistent state in the User model where accountType and isAdmin flags were not aligned with administrative requirements. Additionally, the product listing components (LatestProducts.jsx, BestSelling.jsx) utilize SWR to fetch data from /api/products, which filters by inStock: true. Empty results were due to a lack of active product records.

    where: { email: email },
    data: {
        isAdmin: true,
        accountType: 'ADMIN',
        role: 'ADMIN'
