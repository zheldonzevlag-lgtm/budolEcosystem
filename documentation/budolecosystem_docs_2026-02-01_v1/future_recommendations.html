Future Recommendations & Enhancements

1. Authentication & Security

Biometric Passkeys: Transition from standard biometrics to FIDO2/WebAuthn Passkeys for passwordless authentication across Web and Mobile.
Hardware Security Modules (HSM): Integrate with cloud HSM for signing financial transactions to meet the highest level of BSP security standards.

2. User Experience

Social Login Consolidation: Allow users to link Google/Apple IDs to their existing dual-identifier (Email/Phone) accounts.
Offline Transaction Support: Implement limited offline QR payment capabilities using pre-signed transaction vouchers for areas with poor connectivity.

3. System Scalability

Redis Caching for Uniqueness: Implement a distributed Redis cache for email/phone uniqueness checks to reduce direct Prisma/DB hits during high-traffic registration periods.
