Release v1.3.58

Version: v547 (Release)

App Version: budolPay 1.3.58

Date: 2026-02-01

Status: Verified Production Ready

Executive Summary

This release implements the Ecosystem-wide Dual-Identifier Login system and Real-time Uniqueness validation. Users can now sign in using either their registered phone number or email address across all Budol Ecosystem applications (Web & Mobile). Additionally, real-time uniqueness checks ensure that no two accounts share the same identifier, aligning with BSP Circular 808 and PCI DSS standards for identity management.

Key Features

  • Dual-Identifier Login: Support for Email or Phone Number input on Web and Mobile login screens.
  • Real-time Uniqueness: 800ms debounced validation for Email and Phone during registration and login.
  • Backend Synchronization: Unified /login/mobile/identify endpoint with cross-identifier search logic.
  • BSP Compliance: Server-side phone number normalization (639XX / 09XX) for consistent identity mapping.

Impact Analysis

  • User Experience: Reduced friction during login by allowing multiple identifiers.
  • Security: Enhanced audit logging for identification attempts (Forensic Audit aligned).
  • System Stability: Debounced checks prevent API flooding while providing immediate feedback.
