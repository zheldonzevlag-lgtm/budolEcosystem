System Admin Manual: v1.3.58

1. Monitoring Authentication Services

Administrators should monitor the health of the auth-service and budol-id-sso services. These are responsible for the dual-identifier login logic.

# Check service status
pm2 status auth-service
pm2 status budol-id-sso

2. Log Verification

Authentication logs can be found in the centralized logging system. Look for IDENTIFIER_CHECK and LOGIN_ATTEMPT tags to debug issues with email/phone login.

3. Manual User Intervention

If a user reports that their phone number is "already taken" but they do not own the account, administrators must verify the identity via manual KYC and update the database record if necessary.

# Find user by phone
node scripts/find_user.js --phone 09123456789
