Future Recommendations (v40)

1. Webhook Implementation

Currently, the KYC verification is PENDING by default. Implement a webhook handler to receive auto-verification results from providers like Jumio or Onfido.

2. Blob Storage Migration

Document photos are currently stored as bytea in Postgres. For scale, migrate to Vercel Blob or AWS S3 and store signed URLs in the remoteUrl field.

3. Real-time Status via WebSockets

Update the mobile app via Pusher or Socket.io when verification status changes to improve user retention.
