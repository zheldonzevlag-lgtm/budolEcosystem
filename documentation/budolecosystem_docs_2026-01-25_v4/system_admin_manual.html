System Admin Manual

Deployment Instructions

Web Application

The web application is deployed via Vercel. Ensure that the .env.vercel file is up to date with the latest API endpoints.

Mobile Application

The Android APK is generated and stored in the /android-installer directory. To deploy:

  1. Retrieve budolPay 1.3.34.apk from the installer directory.
  2. Distribute via internal MDM or upload to the Google Play Console for internal testing.
  3. Ensure the backend services are running and accessible from the mobile device network.

Monitoring

Monitor the payment logs for any "N/A" Order IDs, which might indicate a data mapping issue in the frontend components.

Maintenance

Version control is managed via Git. Each release should be tagged appropriately.
