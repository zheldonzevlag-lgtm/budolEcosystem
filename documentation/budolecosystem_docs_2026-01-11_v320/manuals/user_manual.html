Version 320

User Manual

User guide for budolPay Mobile version 1.2.47.

Purpose

This manual explains how to use budolPay Mobile 1.2.47, focusing on biometric login and how to safely enable and disable it.

What's New in 1.2.47

Functional biometric login

You can authenticate using fingerprint or face recognition when enabled in Settings.

Version update

The app is updated to 1.2.47 with build and version synchronization improvements.

Explanation & Rationale

Biometric login improves convenience while keeping security controls intact. Your device performs the biometric verification; budolPay does not store or transmit biometric data.

How to Enable Biometric Login

  1. Log in using your phone number and PIN.
  2. Open Settings.
  3. Enable the Biometric Login toggle.
  4. Follow the in-app confirmation prompts.

How to Use Biometric Login

  1. Open the app.
  2. On the Login screen, tap the biometric icon.
  3. Authenticate when your device shows the biometric prompt.
  4. If successful, your PIN is auto-filled and login proceeds.

How to Disable Biometric Login

  1. Open Settings.
  2. Disable the Biometric Login toggle.
  3. Your stored PIN is removed from secure storage and biometric login stops working.

Issues & Fixes

  • Issue: Biometric icon on the Login screen did not initiate authentication.
    Fix: Biometric authentication now triggers the OS prompt and can auto-fill the PIN.
  • Issue: Users could not control biometric login usage.
    Fix: Settings now include an enable/disable toggle.

Risk & Mitigation

  • Risk: Shared device access.
    Mitigation: Only enrolled biometrics can authenticate; disable biometric login when needed.
  • Risk: Lost device.
    Mitigation: Use device-level lock protections and follow incident reporting procedures.

Dependencies

  • Biometrics must be enrolled in your device settings.
  • Some device policies may disable biometrics.

Technical Details

The app uses OS secure storage to protect your stored PIN and relies on OS biometric prompts for verification.

Test Results

  • Release build 1.2.47 is available as android-installer\budolPay_1.2.47.apk.
  • Biometric prompt appears on supported devices when enabled.
  • Disabling biometric login prevents PIN auto-fill.

Recommendation for Future Enhancement

Add an optional biometric re-authentication timeout after the app is backgrounded.
