User Manual - Version 411

What's New

Version 1.2.95 (v411) includes several usability and security improvements:

  • Automatic Phone Formatting: Phone numbers now automatically format as 09XX-XXX-XXXX.
  • Currency Precision: All transaction amounts are now restricted to 2 decimal places to prevent errors.
  • Phone-Only Transfers: For improved security, Send Money and Cash In now exclusively use phone numbers.

How to Check Your Version

  1. Open the BudolPay app.
  2. Look at the bottom of the splash screen during startup.
  3. You should see budolEcosystem 1.2.95.
