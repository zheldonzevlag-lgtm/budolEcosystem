User Manual: Verifying the SSO Login Fix

1. Introduction

If you were previously unable to log in to the budolPay mobile app due to database errors, this update resolves the issue by correcting the local database credentials.

2. Verification Steps

Step 1: Restart the Ecosystem
Open your terminal and run npm start.
Step 2: Select LOCAL Environment
Type 1 and press Enter. This will update all .env files with the correct r00t password.
Step 3: Try Logging In
Open the budolPay mobile app and attempt to log in. The "Prisma Initialization Error" or "Authentication Failed" error should no longer appear.

3. Issues Fixed

  • Fixed "Authentication failed" error in budolID service.
  • Fixed "Protocol error" when services were missing their database URLs.
