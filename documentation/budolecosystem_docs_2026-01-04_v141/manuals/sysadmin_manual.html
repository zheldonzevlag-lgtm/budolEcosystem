SysAdmin Manual: Database Credential Management

1. Overview

This manual covers the configuration of local database credentials across the ecosystem.

2. Local Credentials

Standard Local Password: The ecosystem is now standardized to use r00t as the default password for the postgres user in local development environments.

3. Updating Credentials

If your local Postgres password is different from r00t, you must update the following files before running npm start:

  • [service-folder]/.env.connection.local

After updating these files, run npm start and choose Option 1 (LOCAL) to propagate the changes to the active .env files.

4. Troubleshooting

  • Authentication Failed: Double check that your local Postgres server is running on port 5432 and accepts the postgres:r00t combination.
  • Prisma Protocol Error: Ensure you are using the latest version of the Central Runner (v141) which isolates environment variables.
