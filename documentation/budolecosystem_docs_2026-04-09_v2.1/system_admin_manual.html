System Administrator Manual: v1.1.0

Monitoring the Ecosystem

Use the scripts/test_scripts/verify-ecosystem-health.cjs tool to check service status. This tool now correctly monitors the BudolID SSO service via the /api/health endpoint.

Environment Configuration

Ensure LOCAL_IP is set in .env to allow mDNS discovery. Run npm run db:local to refresh local database connection strings.
