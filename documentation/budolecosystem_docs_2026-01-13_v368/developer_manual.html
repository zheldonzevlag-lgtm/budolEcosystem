Face Cropping Integration

This version enhances the face embedding process by cropping the detected face from the full image frame before inference.

1. Service Update

The generateEmbedding method in face_embedding_service.dart now accepts an optional Rect? faceArea parameter.

// Cropping Logic
if (faceArea != null) {
  faceImage = img.copyCrop(
    originalImage, 
    x: faceArea.left.toInt(), 
    y: faceArea.top.toInt(), 
    width: faceArea.width.toInt(), 
    height: faceArea.height.toInt()
  );
}

2. UI Integration

Both settings_screen.dart and login_screen.dart have been updated to pass the face.boundingBox to the service.

// Example from SettingsScreen
final template = await faceService.generateEmbedding(
  file, 
  faceArea: face.boundingBox
);

3. Accuracy Rationale

FaceNet (the underlying model) is trained on 160x160 tightly cropped face images. Providing the full frame (which might be 1080p or higher) forced the model to look at background noise, significantly degrading embedding quality and matching similarity.
