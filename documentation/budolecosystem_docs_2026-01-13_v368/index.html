Executive Summary

This release fixes a reliability issue where face embeddings were failing or inaccurate because the model was processing the entire camera frame instead of just the face. By integrating the bounding box from ML Kit with the image package's cropping functionality, we now isolate the face before generating the embedding. This significantly increases the success rate of biometric registration and login.
