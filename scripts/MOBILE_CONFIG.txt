================================================================================
BUILD COMMANDS FOR PRODUCTION
================================================================================

Android APK (Production):
--------------------------
flutter build apk --release \
  --dart-define=GATEWAY_URL=https://api.budolpay.budol.duckdns.org

iOS (Production):
-----------------
flutter build ios --release \
  --dart-define=GATEWAY_URL=https://api.budolpay.budol.duckdns.org

Local Development:
------------------
flutter run --dart-define=API_HOST=localhost

Connect Physical Device to Local Server:
----------------------------------------
flutter run --dart-define=API_HOST=192.168.1.10

================================================================================
ENVIRONMENT VARIABLES
================================================================================

The ApiService uses these Dart defines:
- API_HOST     : Server hostname (default: localhost)
- GATEWAY_URL  : Full gateway URL (overrides API_HOST)
- AUTH_URL     : Auth service URL override
