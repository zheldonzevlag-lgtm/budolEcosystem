### REQUIRED VERCEL DASHBOARD CONFIGURATION FIXES:
For the new pushes to succeed on Vercel, please adjust the project setups in your Vercel Dashboard manually.

**For `budolid-sso` project in Vercel:**
1. Navigate to: **Settings -> Build and Deployment**
2. **Framework Preset:** Change this from `Next.js` to **`Other`** (This project is an Express app, not a Next app).
3. **Root Directory:** Edit this and set it to `budolID-0.1.0` (it is currently empty/`./`).
4. **Install Command:** Set to `npm install`
5. **Build Command:** Set to `npm run build`

**For `budolpay-0.1.0` project in Vercel:**
1. Navigate to: **Settings -> Build and Deployment**
2. Ensure **Framework Preset** is **`Next.js`**
3. Ensure **Root Directory** is `budolpay-0.1.0/apps/admin` (If you want to build the Admin app explicitly) or if building from monorepo root, set root to `budolpay-0.1.0` and configure the Build Command to `npm run build --workspaces` (which is standard).

After securing these settings, you may click "Redeploy" in the deployment tabs to automatically fetch the fixed `main` branch.
